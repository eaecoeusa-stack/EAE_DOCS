Controller Compatibilities

Overview

This topic describes the compatibilities of the following controllers with EcoStruxure Automation Expert:

  • ATV dPAC

  • M251 dPAC

  • M262 dPAC

  • M580 dPAC

  • Soft dPAC

  • HA Soft dPAC

The exact firmware version of these controllers are specified in the Product Information of the EcoStruxure Automation Expert - Release Notes.

NOTE: The compatibility version of these controllers with EcoStruxure Automation Expert is for the version 25.0. For other version, please refer to the Compatibilities and Dependencies - User Guide of the corresponding EcoStruxure Automation Expert version.

ATV dPAC

Use EcoStruxure Automation Expert version 25.0 only when configuring an ATV Distributed PAC with firmware version 25.0.

M251 dPAC

Use EcoStruxure Automation Expert version 25.0 only when configuring a Modicon M251 Distributed PAC with firmware version 25.0.

M262 dPAC

Use EcoStruxure Automation Expert version 25.0 only when configuring a Modicon M262 Distributed PAC with firmware version 25.0.

M580 dPAC

Use EcoStruxure Automation Expert version 25.0 only when configuring a BMED581020(C) and a BMECRD0100(C) with firmware version 25.0.

Use EcoStruxure Automation Expert version 25.0 only when configuring a Soft dPAC with firmware version 25.0.

HA Soft dPAC

Use EcoStruxure Automation Expert version 25.0 only when configuring an High Availability Soft dPAC with firmware version 25.0.
