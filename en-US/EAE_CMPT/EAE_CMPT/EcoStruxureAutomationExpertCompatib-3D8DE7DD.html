EcoStruxure Automation Expert Compatibility with OS

Overview

This topic describes which OS versions are compatible with EcoStruxure Automation Expert components:

  • Buildtime

  • HMI

  • Archive

  • Soft dPAC

EcoStruxure Automation Expert - Buildtime

EcoStruxure Automation Expert - Buildtime is compatible with the following versions of Microsoft Windows 10 Professional - x64:

  • Version 1809

  • Version 1903

  • Version 1909

  • Version 2004

  • Version 20H2

  • Version 21H1

  • Version 21H2

  • Version 22H2

NOTE: For Microsoft Windows earlier than version 1903, the installation of the latest available Windows updates for .NET may be required, otherwise the EtherNet/IP scanner may not work correctly. Verify the version number of the file C:\Windows\Microsoft.NET\Framework\v4.0.30319\WPF \PresentationFramework.dll. If the version is earlier than 4.8.4390.0, install the latest available .NET using Windows Update. The installation of multiple .NET updates and multiple reboots may be required.
EcoStruxure Automation Expert - Buildtime is also compatible with the following version of Microsoft Windows 10 IoT Enterprise LTSC 2021 - x64:

  • Version 21H2

EcoStruxure Automation Expert - Buildtime is also compatible with the following versions of Microsoft Windows 11 Professional - x64:

  • Version 21H2

  • Version 22H2

  • Version 23H2

EcoStruxure Automation Expert - Buildtime is also compatible with the following versions of Microsoft Windows 11 Professional - x64:

  • Version 21H2

  • Version 22H2

  • Version 23H2

EcoStruxure Automation Expert - Buildtime is also compatible with the following version of Microsoft Windows Server 2019 - x64:

  • Version 1809

EcoStruxure Automation Expert - Buildtime is also compatible with the following version of Microsoft Windows Server 2022 - x64:

  • Version 21H2
