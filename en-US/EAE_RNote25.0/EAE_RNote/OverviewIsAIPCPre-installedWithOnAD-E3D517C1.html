Known Limitations

Compatibility Version

The Essential Edge Controller prepackaged offer is currently only available in version 24.1.

IP address issue on unarchiving

When unarchiving an attached solution, the following error may appear: "eth1 is not defined (2001)". In this case, the IP address is not displayed in the Deploy & Diagnostic section. However, the user can still log in to the device and deploy the application. Assign any valid IP address to eth1 to resolve the issue. 

Full screen mode

On some monitors, the EcoStruxure Automation Expert HMI does not display in full screen on the Essential Edge device, even when full-screen mode is enabled in the HMI configuration settings

Resolution issue

On some monitors, the user can face a resolution issue. This prevents the HMI from rendering correctly and view fully the deployed application To address this issue, unplug and plug again the HDMI cable on the Essential Edge. This solves the issue.
