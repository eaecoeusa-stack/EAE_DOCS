EAE Buildtime and Runtime Installation (Windows)

Installing EcoStruxure Automation Expert involves the installation of two separate software packages:

1. EcoStruxure Automation Expert - Buildtime: the buildtime software package provides the engineering environment and the options:

  • AVEVA Industrial Graphics Editor

  • Asset Link for Bulk Engineering

  • Asset Link for AVEVA OMI

NOTE: EcoStruxure Automation Expert - Buildtime is tracking usage information of this software as part of the Schneider Electric software improvement program. If you don't want to participate to this program, it can be disabled for your computer in the Settings tab of the Schneider Electric Software Update that is also installed on your computer.

2. EcoStruxure Automation Expert - Runtime: the runtime software includes the following options:

  • HMI - User Guide

  • Soft dPAC - User Guide

  • Archive - User Guide

Local Windows Defender firewall rules are created. If you are using another firewall software, you must configure the rules described in chapter Computer Firewall. Installing EcoStruxure Automation Expert also installs the following components:

  • Schneider Electric Software Update

  • EcoStruxure Automation Expert EtherNetIP Master DTM library

  • FDTEnv32

  • MS SQL Server 2022 LocalDB (V16.0.1000.6)

  • Microsoft Windows Desktop Runtime 8.0.16 (x64) (V8.0.16.34817)

  • MS ASP.NET Core - 8.0.16 (x64) (V8.0.16.34817)

  • MS Edge WebView 2 Runtime (x64) (V136.0.3240.76)

  • Microsoft Visual C++ 2015-2022 Redistributable (x64) (V14.44.35112.1)

  • Microsoft Visual C++ 2015-2022 Redistributable (x86) (V14.44.35112.1)

NOTE: Any modification to select or unselect the options Asset Link for Bulk Engineering, Asset Link for AVEVA OMI or AVEVA Industrial Graphics Editor requires to launch the EcoStruxure Automation Expert - Buildtime installer again, and then to select Modify.
NOTE: In case of simultaneous installation of AVEVA System Platform 2020 R2 SP1, AVEVA options and EcoStruxure Automation Expert - Buildtime on the same computer, you must also install the Hot Fix - IMS1755821 on top of ASP 2020 R2 SP1.
NOTE: Install EcoStruxure Automation Expert - Buildtime to the default installation path. If you choose a custom installation path, ensure that the total path length does not exceed 85 characters. If the selected path exceeds this limit, the installation will fail with error 'Error 1304. Error writing to file<FileName>. Verify that you have access to that directory.'

Installing EAE Buildtime (Windows)

If you have already installed EcoStruxure Automation Expert Platform, make sure there is no remaining active reservations. For more details, see Multi-User Environment - User Guide.

Launch the EcoStruxure Automation Expert - Buildtime installer and follow the instructions.

NOTE: Launching the installer may take some time.

  1. Double-click the buildtime installation file, EcoStruxure Automation Expert - Buildtime – 2x.x.exe, to begin the installation wizard.

  2. When the Windows User Access Control requests permission to install the buildtime software on your PC, click Yes to proceed with installation.

  3. The installer is automatically extracted.

  4. Select the language for the installation and click OK. The installer begins then automatically.

  5. The Welcome to the Installation Wizard screen opens. Click Next to proceed to the next step.

  6. The End User Licensing Agreement (EULA) screen is displayed. Accept the terms, then click Next to proceed to the next step.

  7. In the Destination Folder screen, open the release notes and read them carefully. Afterwards, accept the default installation folder (Program Files), or navigate to and specify a different location, then click Next.

  8. In the Custom Setup screen, you can select to install one or more of the following features:

    • AVEVA Industrial Graphics Editor

    • Asset Link for Bulk Engineering

    • Asset Link for AVEVA OMI

    • Documentation

    • Select the language with the subfeatures for that you want the documentation to be installed.

    • The English documentation is always installed and cannot be unselected.

      After making your selection, click Next to continue.

      NOTE: If you want to change the installation by adding or removing any of these features on your PC, you need to re-launch the EcoStruxure Automation Expert - Buildtime installer executable, select the Modify option, and then continue to select the listed features.

  9. In the Ready to Install the Program screen, click Install to begin the installation.

  10. The EcoStruxure Automation Expert EtherNetIP Master DTM screen displays, showing the progress of installing EcoStruxure Automation Expert EtherNetIP Master DTM library.

  11. The Installing Schneider Electric Software Update screen displays, showing the progress of installing Schneider Electric Software Update.

  12. The Installing EcoStruxure Automation Expert - Buildtime screen displays, showing the progress of installing EcoStruxure Automation Expert - Buildtime.

  13. In the Installation Wizard Completed screen, you can select:

    • Open Release Notes. By default, the release notes are displayed after installation.

    • Reset the existing tool settings of current version of EcoStruxure Automation Expert. This overwrites any pre-existing settings stored on the host PC with EcoStruxure Automation Expert default settings.

    NOTE: This option is available only when you had installed already another EcoStruxure Automation Expert version on your PC.

  14. Click Finish to complete installation.

Upon the completion of the installation, shortcuts to open EcoStruxure Automation Expert are added to the Start menu and to the desktop

Installing EAE Runtime (Windows)

Launch the EcoStruxure Automation Expert - Runtime installer and follow the instructions.

  1. Double-click the runtime installation file, EcoStruxure Automation Expert - Runtime – 2x.x.exe, to begin the installation wizard.

  2. When the Windows User Access Control requests permission to install the runtime software on your PC, click Yes to proceed with installation.

  3. The installer is automatically extracted.

  4. Select the language for the installation, then click OK. The installer begins automatically.

  5. The Welcome to the Installation Wizard screen opens. Click Next to proceed to the next step.

  6. The End User License Agreement (EULA) is displayed. Accept the terms, then click Next to proceed to the next step.

  7. In the Destination Folder screen, open the Release Notes and read it carefully. Afterwards, accept the default installation folder (Program Files), or navigate to and specify a different location. then click Next.

  8. In the Custom Setup screen, the available EcoStruxure Automation Expert - Runtime applications are displayed.

    NOTE: To continue, select at least one of the listed applications. If you want to change the installation by adding or removing any of these applications on your PC, you need to re-launch the EcoStruxure Automation Expert - Runtime installer executable, select the Modify option, and then continue to select the listed applications.

    After you make your selections, click Next.

  9. Select your application(s) :

    1. If you select the Archive application the following prerequisites must be satisfied before installing and using the EcoStruxure Automation Expert - Archive (Windows):

      • PostgreSQL V13 or later for Windows x86-64 must be installed.

        NOTE: NOTE: For cybersecurity reasons, it is recommended to install the latest PostgreSQL version supported by EcoStruxure Automation Expert - Archive.

      • pgAdmin 4 and pgAgent must be installed.

      • Administrator rights are required on the PC where PostgreSQL, EcoStruxure Automation Expert - Archive, pgAdmin 4 and pgAgent are installed.

        Please refer to the Archive User Guide in order to complete the installation/configuration.

      • The online help center : EcoStruxure Automation Expert - Archive (Windows) Installation Instructions.

      • · The EcoStruxure Automation Expert installation Package (PDF format): Archive Subfolder/ Chapter : EcoStruxure Automation Expert - Archive (Windows) Installation Instructions.

    2. If you select Softdpac application, please refer to the Softdpac User guide in order to complete the installation and configuration. This user guide is accessible from :

      • The online help center : Install Soft dPAC on Windows™

      • The EcoStruxure Automation Expert Expert installation Package (PDF format): Softdpac Subfolder/ Chapter : EcoStruxure Automation Expert - Archive (Windows) Installation Instructions.

        NOTE: Softdpac HA is only available on Ubuntu/Debian.
