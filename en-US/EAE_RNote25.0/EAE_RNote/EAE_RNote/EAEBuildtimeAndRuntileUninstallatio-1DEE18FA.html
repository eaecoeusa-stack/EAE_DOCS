EAE Buildtime and Runtime Uninstallation (Windows)

Use the standard Microsoft Windows procedure to remove the EcoStruxure Automation Expert software from your computer.

Before uninstalling EcoStruxure Automation Expert - Buildtime, ensure that the EcoStruxure Automation Expert application is closed. If the application remains open during the uninstallation process, the uninstallation will proceed, but the application will stay open. Consequently, an exception dialog will appear when any operation is attempted within the application.

The following components are not uninstalled during the uninstallation of EcoStruxure Automation Expert software and require a manual uninstallation using the standard Microsoft Windows procedure:

  • Schneider Electric Software Update

  • EcoStruxure Automation Expert EtherNetIP Master DTM library

    NOTE: Uninstallation is only possible by using Windows standard uninstallation procedure.

    • FDTEnv32

    • MS SQL Server 2022 LocalDB (V16.0.1000.6)

    • Microsoft Windows Desktop Runtime 8.0.16 (x64) (V8.0.16.34817)

    • MS ASP.NET Core - 8.0.16 (x64) (V8.0.16.34817)

    • MS Edge WebView 2 Runtime (x64) (V136.0.3240.76)

    • Microsoft Visual C++ 2015-2022 Redistributable (x64) (V14.44.35112.1)

    • Microsoft Visual C++ 2015-2022 Redistributable (x86) (V14.44.35112.1)

Uninstalling EAE Buildtime (Windows)

You can uninstall the EcoStruxure Automation Expert - Buildtime software applications one at a time in the Microsoft Windows™ Apps and Feature window.

To uninstall a software application:

  1. On your PC, open Start > Settings > Apps and Features.

  2. Click the EcoStruxure Automation Expert software application you want to uninstall. The dialog to uninstall the Buildtime application opens.

  3. Click Uninstall.

  4. An uninstall wizard opens. Follow the steps presented in the wizard to uninstall the selected application.
