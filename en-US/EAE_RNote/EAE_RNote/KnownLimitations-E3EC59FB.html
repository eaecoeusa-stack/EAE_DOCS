RSTP Port Usage Clarified

The Soft dPAC and Soft dPAC HA user guides have been updated to clearly specify that the BMECRD0100 service port (ETH1) must not be included in an RSTP loop. Only ports ETH2 and ETH3 are supported for use in RSTP configurations.

Docker Compatibility for Soft dPAC and Soft dPAC HA

Scope: Soft dPAC Docker Linux & Soft dPAC HA

Soft dPAC supports only Docker versions prior to 28. Installation will fail if an incompatible version is detected. Version 27.x is required, with 27.5 recommended.

Boot Project Deletion Availability

The option to delete a boot project in the Deploy and Diagnostic view is available only when a boot project has already been configured.
