State Manager

State Manager CAT which is part of SE.App2StateManagement has been refactored. Customer migration is not expected to be affected as only internal codes have been changed to fix some identified bugs and limitations. Also, AVEVA System Platform template will be now available for State Manager as well. 

MMM Specific Symbols and Graphical Objects

New symbols have been added to Motors, Valves, Digital Input and Digital Outputs with MMM segment in focus. However, the symbols are generic and part of SE.App2CommonProcess library and hence available to use in any project of any segment.

Conveying Library Redesign

Multiple CATs from the Conveying library has been redesigned and will be available as part of SE.App2BaseConveying library.

Single Line Power Monitoring Library Redesign

Single Line Power Monitoring library CATs have been redesigned and our now available as part of SE.App2SingleLinePowerMonitoring. These CATs are compatible with SE.AppSequence, SE.App2Base, SE.App2CommonProcess and all other "SE.App2xxx" libraries.

MMM Library

Bag Filter, Belt Conveyors & Motor StandBy blocks are now available as part of SE.App2MMM library for MMM segment needs.
