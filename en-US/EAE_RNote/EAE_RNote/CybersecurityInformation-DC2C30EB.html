Cybersecurity Information

For information on cybersecurity for EcoStruxure Automation Expert controllers, refer to the EcoStruxure Automation Expert - Cybersecurity User Guide.

Compliance

Our system conforms to the California SB-327 Information Privacy: Connected Devices bill by requiring the user to generate a new means of authentication before granting initial access. This includes the Modicon M580 Distributed PAC, Altivar dPAC Module VW3A3530D, Modicon M251 Distributed PAC, and Modicon M262 Distributed PAC. Industry vulnerabilities are regularly identified and mitigated in order to minimize cybersecurity risks.

Improved Security Robustness

This release of EcoStruxure Automation Expert includes several cybersecurity fixes addressing vulnerabilities and improving security robustness.
