Product Information

Overview

The EcoStruxure Automation Expert offer includes:

  • EcoStruxure Automation Expert - Buildtime, a software that allows you to design, test and maintain your control and HMI applications.

  • EcoStruxure Automation Expert - HMI, which allows you to run the HMI application, hosted on an HMIST6 or device running the Microsoft Windows operating system, or a device running the Linux Ubuntu operating system.

  • EcoStruxure Automation Expert - Archive, which provides the storage space for alarms and data loggings, hosted on a device running a Microsoft Windows operating system, or a device running a Linux Ubuntu operating system.

  • A range of dPAC controllers:

    • EcoStruxure Automation Expert - Soft dPAC, a dPAC controller, hosted on a Linux or Microsoft Windows operating system.

    • EcoStruxure Automation Expert - High Availability Soft dPAC, a redundant version of the standard EcoStruxure Automation Expert - Soft dPAC, used in a high availability system configuration.

    • Altivar dPAC Module VW3A3530D, a dPAC controller, hosted on an Altivar variable speed drive.

    • Modicon M251 Distributed PAC, a standalone dPAC controller.

    • Modicon M580 Distributed PAC, a standalone dPAC controller.

    • Modicon M262 Distributed PAC, a standalone dPAC controller.

NOTE: The firmware version installed on the connected dPAC controllers, must be the same as the installed EcoStruxure Automation Expert version.
