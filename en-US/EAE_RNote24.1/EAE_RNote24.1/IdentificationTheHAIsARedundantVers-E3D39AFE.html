EcoStruxure Automation Expert - High Availability Soft dPAC

Identification

The EcoStruxure Automation Expert - High Availability Soft dPAC (HA) is a redundant version of the standard EcoStruxure Automation Expert - Soft dPAC. It allows continuous operation during a single fault condition at the controller or network level.

The EcoStruxure Automation Expert - HA Soft dPAC can control I/Os using the BMECRD0100 device and other X80 I/O modules.

Refer to BMECRD0100 X80 Remote I/O for a description of the supported X80 modules.

Refer to the High Availability chapter, available in the Appendix of the EcoStruxure Automation Expert User Manual for a description of the supported MIBs.

Known Limitations

Limited Support of System Features

The following features are not currently supported by: 

  • OPC-UA Client 

  • Modbus Server 

  • EtherNet/IP 

  • EcoStruxure Automation Expert - Archive

Single Instance

EcoStruxure Automation Expert - HA Soft dPAC can support only one instance per iPC. No additional software can be installed on the iPC hardware platform. 

Single Resource

A High Availability control application supports only a single resource. Multiple resources in EcoStruxure Automation Expert cannot be deployed to an EcoStruxure Automation Expert - HA Soft dPAC.
