Safety Related Recommendations

Online Change Features

EcoStruxure Automation Expert intends to offer the capability to make application updates and to deploy them into devices, while they are running, without requiring the overall system execution to stop.

These online change features must be used only for design and commissioning operations. These features must not be used in operational conditions, except for EcoStruxure Automation Expert - Soft dPAC and Modicon M262 Distributed PAC devices.

For any other dPAC device, the following effects may occur while performing online change in operational condition:

  • Glitches on I/Os going to fallback

  • Hardware and fieldbus scanning cycle timeout

Therefore careful examination, understanding and design of the application is required to prevent or solve such issues. Refer to the EcoStruxure Automation Expert Applications Design Guidelines Reference Manual, available in the online help.

 WARNING
Online Change
Use online change features for application design, tests, and commissioning purposes only.
Failure to follow these instructions can result in death, serious injury, or equipment damage.

Multi-User Management

EcoStruxure Automation Expert allows several users to connect to and interact with the same devices, in order to work collaboratively with the same solution. This multiuser capability requires caution to use, especially to ensure that users actions are coherent towards a common and shared goal for the solution (otherwise unexpected results may occur).

 WARNING
CONCURRENT ACCESS TO DEVICES
When allowing several users to perform simultaneous or sequential modifications to the application under execution in the devices, ensure that these modifications do not have adverse effects on the safety of the local and system level application execution.
Failure to follow these instructions can result in death, serious injury, or equipment damage.

EcoStruxure Automation Expert - Soft dPAC

EcoStruxure Automation Expert - Soft dPAC can be installed on Linux and Microsoft Windows operating systems. Within standard Linux and Microsoft Windows operating systems, the accuracy of the time management is not high enough to match real-time requirements, therefore only non-time-critical processes must be executed (such as analytics and orchestration for example).

 WARNING
UNEXPECTED APPLICATION BEHAVIOR
Do not use Soft dPAC on Linux without real-time support or Microsoft Windows operating systems to process application tasks that require real-time level performances.
Failure to follow these instructions can result in death, serious injury, or equipment damage.

Manual Editing of Solution Files

Editing solution files directly from Windows Explorer can lead to inconsistencies that EcoStruxure Automation Expert - Buildtime is not able to fix.

 WARNING
UNEXPECTED APPLICATION BEHAVIOR
Do not edit manually Solution files directly from their disk location.
Always use EcoStruxure Automation Expert - Buildtime to edit the Solution.
Failure to follow these instructions can result in death, serious injury, or equipment damage.
