Before installing AVEVA System Platform 2023 on a device running Microsoft Windows 11, #122730 a specific Microsoft PowerShell script must be installed. 

To download and install the script, go to support.microsoft.com, search for KB5022083 Change in how WPF-based applications render XPS documents, and follow the steps to complete the procedure.

  • .NET Desktop Runtime (x64) 6.0.12 or later

Online Change Limitations

Online Change Limitations During Library Development

To perform a successful Online Change of already referenced libraries, follow these recommendations when developing your libraries: 

  • Online Change not compatible with the following block types: 

  • Visible blocks in a library — all block types, CAT, Basic, Composite, SubApp, Adapter and Function 

  • Rename or remove function block types 

  • Rename or remove ports from the interface of function block types 

  • Modifying symbolic link names 

  • Modifying the data type of a port, only if the value can be converted to the new type 

  • Rename or remove UDTs 

  • Rename or remove members of a structure 

  • Reduce the range of a subrange 

  • Reduce an array size 

  • Rename or remove enumeration items 

Online Change is compatible with: 

  • Adding new ports 

  • Adding new symbolic links 

  • Adding new blocks 

  • Algorithm changes and/or ECC changes for basic blocks 

In case blocks or ports of an interface need to be renamed or removed, the current block should be kept unchanged and market as obsolete, and a new block with new interface and name should be added. 

Test Online Change of the modified blocks of the library before releasing the library.
