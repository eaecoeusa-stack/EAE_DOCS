Modicon Edge IO

EcoStruxure Automation Expert will allow user to integrate Modicon Edge IO offer as distributed IO node over Ethernet IP in control system architecture. This is Schneider Electric new cost-effective offer with advance functionalities.

User can instantiate Edge IO avatar from Physical View and configure network settings.

User can configure and parameterize Edge IO head and modules in offline mode.

User can read and write Discrete & Analog data over Ethernet IP. Modules supported are as followed:

  • NTSNEC1200

  • NTSDDI1602K

  • NTSDDO0802XK

  • NTSAMI0400

  • NTSAMO0400

User can control and monitor Discrete & Analog module inEcoStruxure Automation Expert - HMI.

Platform supported:

  • Non-HA SoftdPAC over Ethernet IP.

  • Non-HA & HA SoftdPAC over Modbus TCP.

Limitations:

  • Online capabilities like diagnostic, configuration, parameterization…etc. are not supported. However, user can do offline configuration in EcoStruxure Automation Expertand export configuration file. This file can be imported and deployed to Edge IO modules using web editor.

  • Modbus TCP interface to configure Edge IO is not seamless, however generic Modbus TCP device hardware CAT can be used to exchange data with Edge IO.
