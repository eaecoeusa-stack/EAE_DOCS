New Features

Reliable cross communication

Modicon M251 Distributed PAC now supports Reliable Cross Communication (HASRT).

For information on how to optimize cross-commmunication between devices, refer to Cross-Communication Links in the EcoStruxure Automation Expert User Manual. 

EtherNet/IP and Ethernet Interfaces

EtherNet/IP scanner (implicit messages) can only start on the Ethernet 2 interface. 

If the BMTM3 Bus is also configured, the EtherNet/IP scanner will start 10 seconds after TM3 bus initialization. 

Explicit messages can be sent/received through both Ethernet interfaces. 

The EIP_EXPLICIT function block can be used 30 seconds after DPAC_FULLINIT 

ModbusSerial/RTU

The Modicon M251 Distributed PAC supports only one serial Modbus RTU client bus.

Performances

The recommended EtherNet/IP configuration is 8 devices with an RPI equal to 20ms and an EtherNet/IP bus cycle time equal to 40ms. 

In addition, do not exceed the following limits for EtherNet/IP implicit exchanges: 

  • Maximum number of devices: 16 

  • Maximum number of connections: 16 

  • Maximum number of input packets per second: 1000 

  • Maximum number of output packets per second: 1000 

The Modicon M251 Distributed PAC may enter ErrorHalt state in case the following features are used simultaneously: 

  • Communication services 

  • Function blocks algorithms 

  • Maximum number of input packets per second: 1000 

  • I/O activity 

Perform the following steps to allow the Modicon M251 Distributed PAC to enter in RUNNING state: 

  1. Reduce CPU load by: 

  2. Increasing TM3 bus cycle time 

  3. Increasing EtherNet/IP scanner RPI & EtherNet/IP bus cycle time 

  4. Reducing application events rate 

  5. Reducing Modbus TCP and RTU activities 

  6. Reducing communication activities with other dPACs (consider changing the function block mapping)

  7. Deploy the application to the Modicon M251 Distributed PAC 

For more information on applications design, refer to the EcoStruxure Automation Expert Applications Design Guidelines Reference Manual, available in the online help.
