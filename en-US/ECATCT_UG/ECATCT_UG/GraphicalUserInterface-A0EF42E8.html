Graphical User Interface

Here is an overview of the graphical user interface of EtherCAT Configuration Tool:

  1. Menus; Shows current status or mode of the EtherCAT Configuration Tool and allows your to change it.

  2. Toolbar.

  3. Project Explorer pad; Shows different views of the current network configuration and allows you to change it by adding or removing Main-devices, Sub-devices and modules.

  4. Device Editor pad; Shows information about the selected device, like process variables or PDO mappings. It also allows you to change this information.

  5. Short Info pad; Shows the name, description and vendor about selected device.

  6. Messages pad; Shows the notifications that appeared, e.g. when the EtherCAT Main-Device has changed its operation state or when a Sub-device has been removed from (or added to) the EtherCAT network.
