Menus

EtherCAT Configuration Tool has couple of menus:

  • File menu

  • View menu

  • Network menu

  • Settings menu

  • Help menu

File Menu

Here are the commands available in the File menu:

  • New; Start a new configuration.

  • Open; Open an existing configuration.

  • Save; Save the current configuration.

  • Print; Print the current configuration.

  • Add Maindevice-Unit; Add a new Main-device unit to the configuration.

  • ESI-Manager; Opens the ESI Manager dialog box to add, delete or export ESI and SCI* files.

  • EMI-Manager; Opens the EMI Manager dialog box to add, delete or modify EMI files.

  • Recent Projects; Open a recent project.

  • Exit; Closes EtherCAT Configuration Tool.

(*) SCI (Sub-device Configuration Information): Helps configure how the Sub-device behave within the EtherCAT Configuration Tool.

View Menu

Here are the commands available in the View menu:

  • Messages Window; Shows or hides the Messages pad.

  • Short Info Window; Shows or hides the Short Info pad.

  • Expert Mode; Activate or deactivate expert mode.

  • Motion Mode; Activate or deactivate motion mode.

  • Refresh; Update the current view.

Network Menu

Here are the commands available in the Network menu:

  • Scan EtherCAT Network; Scans the connected network for Sub-devices.

  • Edit Topology; Opens the Edit Topology dialog box.

  • Export ENI File; Creates an ENI file.

  • Export ENI Variants; Creates an ENI variants.

  • Import ENI File; Imports and ENI file.

  • Process Variables; Export the process variable.

  • EEPROM File; Export the EEPROM file.

  • Network Mismatch Analyzer (active only in diagnosis mode); Opens the Network Mismatch Analyzer dialog box.

  • Line Crossed Analyzer (active only after scan); Opens the Line Crossed Analyzer dialog box.

  • Inspection Report (active only in diagnosis mode); Opens the Inspection Report dialog box.

  • Clear Error Counters (active only in diagnosis mode); Clears the error counters of all connected Sub-devices.

  • Acknowledge All Warnings (active only in diagnosis mode); Clear the yellow warning icon of all Sub-devices.

  • Self Test Scan; Executes a self test routine for EtherCAT Main-Device and the network.

  • Rescue Scan; Executes the rescue scan. Can help if for example frames get lost with a switch.

  • Take Snapshot (active only for local or remote system); Takes a snapshot from the current diagnosis state and saves it into a capture file.

  • Automatic Snapshot Mode (active only for local or remote system); Activates the automatic snapshot mode to take snapshots based on the configured rules.

  • Snapshot (active only for offline diagnosis system); Changes the active snapshot.

  • EoE Endpoint Configuration; Activates EoE Endpoint support in diagnosis mode.

Settings Menu

Here are the commands available in the Settings menu:

  • Language; Changes the current language.

  • Theme; Changes the current theme.

  • Message Level; Change the current message level.

  • Capture File; Opens the Capture File dialog box.

  • Project Template; Opens the Project Template dialog box.

Help Menu

Here are the commands available in the Help menu:

  • Show User Guide; Shows this user guide.

  • Show Log File; Shows the log file.

  • Check for Update; Enables or disables automatic update check (once per month). Also a manual update check is performed on activation.

  • About; Shows the about dialog box.
