Offline Configuration

In this mode, you can configure the EtherCAT network in the EtherCAT Configuration Tool by manually adding Sub-devices to the network.

When clicking on Offline Configuration, it opens the Select MainDevice Unit dialog box. There choose your wanted Main-device unit.

Once done, it automatically opens the Append EtherCAT SubDevice dialog box. There choose the Sub-devices that you want to add in your EtherCAT network.

NOTE: If you can not find the wanted Sub-device or if you want to use your own ESI file, you can edit this list by using the ESI Manager.

Next, you can finalize the configuration of your EtherCAT network by setting the Main-device and setting the Sub-devices.
