About the Document

Document Scope

The EtherCAT Configuration Tool - User Guide is here to give you all the information needed about the EtherCAT Configuration Tool software for you to be able to configure your EtherCAT network on EcoStruxure Automation Expert as you want it.

In this user guide, you will then find:

  • An introduction about EtherCAT Configuration Tool, available in here.

  • How to configure your EtherCAT network, available in here.

  • The description of all the graphical user interfaces of EtherCAT Configuration Tool, available in here.

NOTE: The EtherCAT Configuration Tool is used when configuring your EtherCAT network on EcoStruxure Automation Expert. To have more information about EtherCAT on EcoStruxure Automation Expert, refer to the EtherCAT chapter in the EcoStruxure Automation Expert - User Manual.

Validity Note

This document is valid for EcoStruxure Automation Expert version 25.0 or any subsequent supporting version(s).
NOTE: Always refer to the online documentation to access the latest version of the document.
