Variables Tab

The PDO Mapping tab is available in the Device Editor pad when selecting in the Project Editor a Sub-device.

In this tab, you can see and tune, if allowed, the variables of the Sub-device.

  • Variables; Lists the variables coming from the ESI file or that will be generated from the configurator.

  • Edit Variable

    • Move Up; Move upper in the list the selected variable.

    • Move Down; Move lower in the list the selected variable.

    • New Alias; Opens the Add Alias dialog box.

    • Edit Alias; Opens the Edit Alias dialog box.

    • Delete Alias; Deletes the alias of the selected variable.

  • New; Opens the Add Variable dialog box.

  • Edit; Opens the Edit Variable dialog box.

  • Delete; Deletes the selected variable.

  • Add to watch list; Adds to global watch list the selected variable.
