Device Editor Pad

The Device Editor pad allows you to read and write information of the selected Main-device or Sub-device.

The tab you see in the Device Editor pad depends if:

  • The Expert Settings are enabled or disabled.

    NOTE: To know more about Expert Settings, refer to the View menu topic.

  • The element selected in the Project Explorer pad is a Main-device or a Sub-device.

Then find here all the tabs available for a Main-device:

  • MainDevice tab.

  • Process Data Image tab.

  • Trace Data tab.

  • Advanced Options tab, available with Expert Settings enabled.

  • Distributed Clocks tab, available with Expert Settings enabled.

Finally, find here all the tabs available for a Sub-device:

  • General tab.

  • Modules tab.

  • PDO Mapping tab.

  • Variables tab.

  • Advanced Options tab, available with Expert Settings enabled.

  • Distributed Clock tab, available with Expert Settings enabled.

  • Init Commands tab, available with Expert Settings enabled.

  • CoE Object-Dictionary tab, available with Expert Settings enabled.
