Edit PDO Dialog Box

The Edit PDO dialog box is opened when you click on Add or Edit buttons in the PDO view of the PDO Mapping tab of the Device Editor pad.

This dialog box allows you to add or edit a PDO.

  • General

    • Name; Name of the PDO.

    • Index; Index of the PDO.

      NOTE: The index value can be entered in hexadecimal or decimal.

  • Optional

    • Exclude; Select the PDOs which cannot be activated if this PDO is activated.

  • Flags

    • Mandatory; PDO cannot be deleted.

    • Fixed Content; Content of PDO cannot be changed.

    • Virtual PDO; PDO has no entries.

  • Direction

    • TxPdo; Input PDO.

    • RxPdo; Output PDO.

  • Entries; List of configured PDO entries.

  • Add; Opens the Add PDO Entry dialog box.

  • Delete; Deletes the selected PDO entry.

  • Edit; Opens the Edit PDO Entry dialog box.

  • Up; Moves upper in the list the selected PDO entry.

  • Down; Moves lower in the list the selected PDO entry.
