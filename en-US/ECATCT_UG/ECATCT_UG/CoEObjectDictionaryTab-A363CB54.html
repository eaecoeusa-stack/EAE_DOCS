CoE Object Dictionary Tab

The Init Commands tab is available in the Device Editor pad when selecting in the Project Editor a Sub-device and with the Expert Settings enabled.

NOTE: To know more about Expert Settings, refer to the View menu topic.

In this tab, you can see and edit the offline CoE object dictionary.

  • Value; List of the CoE object dictionary entries coming from the ESI file or is generate from the configurator. The Flags column inform if the entry is an PDO entry and if it can be edited. It follows the format 'AA BB C D (EE FF GG)':

    • AA; Mapping as RX PDO or not

    • BB; Mapping as TX PDO or not

    • C; Backup flag

    • D; Settings flag

    • EE; Access rights for PreOp (RO, WO, RW)

    • FF: Access rights for SafeOp (RO, WO, RW)

    • GG; Access rights for Op (RO, WO, RW)

  • Filter Setting Flag; If activated, shows setting data.

  • Edit Value; Edits value in CoE object dictionary, only if writable.

  • Write; Writes value to CoE object dictionary, only if writable.

  • Update; Changes the selected entry.

  • Reset; Resets the selected entry to the ESI default value.
