PDO Mapping Tab

The PDO Mapping tab is available in the Device Editor pad when selecting in the Project Editor a Sub-device.

This tab has two different views selectable on the top right corner of the tab:

  • PDO view.

  • FMMU/SM view.

PDO View

In this view, you can see the current PDO mapping. For some Sub-device types, you can activate or deactivate some PDO configurations.

  • List of inputs or outputs checkbox; Signals if PDO is used for the current configuration or not.

  • Buttons (available when Expert Settings are enabled)

    NOTE: To know more about Expert Settings, refer to the View menu topic.

    • Download PDO Configuration; Downloads the PDO Configuration.

    • Add; Opens, if allowed by the ESI, the Edit PDO dialog box.

    • Delete; Deletes the selected list if allowed by the ESI.

    • Edit; Opens, if allowed by the ESI, the Edit PDO dialog box.

    • Up; Moves the selected PDO upper in the selected list.

    • Down; Moves the selected PDO lower in the selected list.

    • Load PDO Information; If EtherCAT Configuration Tool is connected to the control system, you can lead the PDO information directly from the Sub-device.

When right clicking on the inputs and outputs lists, here are the commands available:

  • Expand All

  • Collapse All

  • Select All

  • Unselect All

FMMU/SMU View

In this view, you can see some information about FMMU and SyncManager.

  • FMMU; List of available FMMUs coming from the ESI file.

  • SM (SyncManager); List of available SyncManagers coming from the ESI file.
