Init Commands Tab

The Init Commands tab is available in the Device Editor pad when selecting in the Project Editor a Sub-device and with the Expert Settings enabled.

NOTE: To know more about Expert Settings, refer to the View menu topic.

In this tab, you can view and update, if allowed, the current configured init commands.

  • Init Commands; List of init commands coming from the ESI file or is generate from the configurator. The Access

    column informs if the init commands can be edited (RW = Read / Write) or not (RO = Read-Only).

  • Edit Value; Edits the selected init command.

  • New; Opens the Add CoE Init Command dialog box.

  • Copy; Copies the selected init command.

  • Edit; Opens the Edit CoE Init Command dialog box.

  • Delete; Deletes the selected init command.

  • Move Up; Moves upper in the list the selected command init.

  • Move Down; Moves lower in the list the selected command init.
