Modules Tab

The Modules tab is available in the Device Editor pad when selecting in the Project Editor a Sub-device.

In this tab, you can assign modules into the specific slots. You can also change the setting for downloading the slot configuration to the Sub-device.

  • <<; Connects the selected module (from the right list) to the selected slot (from the left list). If the slot is already connected, the module is inserted and the subsequent modules is moved (if this is supported from the Sub-device).

  • X; Disconnect the wanted module from the selected slot (from the left list).

  • Download Slot Configuration; If checked, download the slot configuration.

  • Load Modules; Loads modules from a connected Sub-device.

NOTE: The modules can also be connected and disconnected by using the Append and Remove commands in the Project Explorer pad.
