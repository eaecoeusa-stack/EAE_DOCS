Online Configuration

You can use this mode if Sub-devices are connected to the engineering workstation by scanning the EtherCAT network configuration.

When clicking on Offline Configuration, it opens the Select MainDevice Unit dialog box. There choose your wanted Main-device unit.

Next, the Device Editor pad opens at the MainDevice tab. There you can choose the network adapter which is connected to the control system:

NOTE: Please ensure that the selected network adapter is connected to the EtherCAT Sub-devices.

The EtherCAT Configuration Tool then scans the control system and adds the network configuration to the Project Explorer pad.

Next, you can finalize the configuration of your EtherCAT network by setting the Sub-devices that have been added.
