Edit PDO Entry Dialog Box

The Edit PDO Entry dialog box is opened when you click the Edit button in the Edit PDO dialog box.

This dialog box is used to edit a PDO entry without a CoE Object-Dictionary.

  • General

    • Name; Name of the PDO entry.

    • Comment; Comment of the PDO entry.

    • SwappingSwapping mode of the PDO entry.

  • Settings

    • Index; Index of the PDO entry.

      NOTE: The index value can be entered in hexadecimal or decimal.

    • SubIndex; Subindex of the PDO entry.

      NOTE: The subindex value can be entered in hexadecimal or decimal.

    • Datatype; List of available datatypes.

    • Bit Length; Length of the PDO entry in bits.

    • Show Base Datatypes; Shows only base datatypes.

  • CoE Object-Dictionary; Key value pairs of internal settings data or data. According to the flags this data can sometimes be read-only or can be written too when writable.
