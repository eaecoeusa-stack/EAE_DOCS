Abstract

The ATVDisplay is a hardware CAT available under the VSD_EmbeddedFunctions in the IoATV library.

The ATVDisplay hardware CAT is connected to other IEC61499 function blocks, within the application via the Symbolic Links.

The ATV dPAC Display Configurator allows to configure the HMI screens and variables.

NOTE:

In the 24.1 release, there is a limitation in the development of the ATV dPAC display feature. This limitation affects the number of values supported by an event on the IThis component of a CAT. Specifically, if you try to display a value that is the 33rd linked to the same event, it will not be updated on the Graphic Display Terminal. Therefore, we are limited to 32 values per event on the IThis component.

This limitation only affects the ATV dPAC Display feature and does not impact any other features. If you do not use this feature, you can disregard this limitation.

In the 24.0 release, this limitation was even stricter, allowing only 16 values per event.
