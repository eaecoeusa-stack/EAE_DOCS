atsEtiDecoder

Overview

The atsEtiDecoder is a composite function block inside SE.IoATV Library.

The atsEtiDecoder composite function block is connected to other IEC 61499 function blocks, within the application.

Composite function blocks are collections of basic function blocks or other composites with lower level functionality to achieve a higher level functionality.

atsEtiDecoder is used to decode the Internal State Register (Logical Address 3206), to bit level (True or False) at function block level to be used in the application easily.

atsEtiDecoder is compatible with altivar soft starter ats430, ats480 and ats490.
