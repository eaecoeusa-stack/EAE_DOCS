Integration of ALARM_BIT function block into AltivarVs

Abstract

The ALARM_BIT function block has already been preconfigured in the AltivarVs application CAT.

The ALARM_BIT function block is used to process warning messages at an Alarm Grid on HMI, which offers a platform to show warnings on the HMI in form of a list.

Warnings of AltivarVs application CAT report drive errors (not drive warnings), and the warning Code is visible for AltivarVs.

Description

The element Alarm Grid shows warnings, sent by the AltivarVs application CAT. This means that this CAT is instantiated in the project from ready-to-use libraries and that the functionality to send warnings is already implemented.

You need to create an alarm grid control panel for the HMI and the CAT warning messages used are displayed automatically at runtime.

When the drive detects an error, you will see Drive Fault in an alarm grid on the HMI.
