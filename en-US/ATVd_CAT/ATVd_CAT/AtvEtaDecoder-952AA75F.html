atsEtaDecoder

Overview

The atsEtaDecoder is a composite function block inside SE.IoATV Library.

The atsEtaDecoder composite function block is connected to other IEC 61499 function blocks, within the application.

Composite function blocks are collections of basic function blocks or other composites with lower level functionality to achieve a higher level functionality.

atsEtaDecoder is used to decode the ETA CIA402 State Register (Logical Address 3201), to bit level (True or False) at function block level to be used in the application easily.

atsEtaDecoder is compatible with altivar soft starter ats430, ats480 and ats490.
