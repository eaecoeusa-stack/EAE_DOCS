Integration of ATVCyclicScanner to EcoStruxure Automation Expert Solution

  1. Configure your physical device:
    1. Click on the System icon to open the System tab.
    2. Click on the Physical Devices icon to open the Physical Devices editor.
    3. Drag and drop your altivar device to the Physical Devices editor.
    4. Click on your altivar device to display its properties, then select ATV dPAC for the Communication Card Reference.
    5. Click on the ATV dPAC module to display its properties, then type the IPV4 Address of your ATV dPAC
  2. Configure your logical device:
    1. Click on the Logical Devices icon to open the Logical Devices editor.
    2. Click Add Device icon to add a new logical device.
    3. Select SE.DPAC:ATV_dPAC as a Type, then click OK.
    4. For the Physical Device column, select your ATV dPAC module (Already created in the Physical Devices editor).
  3. Configure your hardware configuration
    1. Click on the Hardware Configuration icon to open the Hardware Configuration editor.
    2. Right-click in Hardware Configuration editor, then select Add.
    3. Select the bus element type SE.IoATV.ATVdPACxVW3A3530D, then click Add.
    4. Right click on the added bus ATVdPACxVW3A3530D, select the configuration module SE.IoATV.ATVSpeedControl, then click Add.
    5. Right click on the added bus ATVdPACxVW3A3530D, select the configuration module SE.IoATV.ATVCyclicScanner, then click Add.
      The following figure displays the result of this sub-step:
  4. Configure the properties editor
    1. Right-click on the configuration module ATVCyclicScanner, then select Properties Editor.
      The following figure illustrates the Properties Editor window of ATVCyclicScanner for the Input Scanner:The following figure illustrates the Properties Editor window of ATVCyclicScanner for the Output Scanner:
    2. Click the add button to add the Input Scanner parameters:
      After clicking the add button, the following safety message is displayed:Accept the safety message to add the Input Scanner parameters.
    3. Click the icon to select the Input Scanner parameter.
      NOTE: You can select the scanner parameters in two ways:

      • Use the search bar and type the parameter code, label, or address.

      • Scroll down until you locate the required parameter.

    4. Select the required input scanner parameter, then click ADD.
    5. Repeat the sub-steps b, c, and d until you add all the required input scanner parameters.

      NOTE: You can also add the required parameter by simply typing its Modbus Address.
    6. Select the Output Scanner tab.
    7. Repeat the sub-steps b, c, and d until you add all the required output scanner parameters.
