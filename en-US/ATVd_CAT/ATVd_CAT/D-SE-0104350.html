IoEmbeddedATV6xx AnalogOutputs

Abstract

The IoEmbeddedATV6xx AnalogOutputs faceplate allows you to monitor the analog output channels. You can observe the values of the parameters, force the values, set the range and select the units.

 WARNING
LOSS OF CONTROL
  • Verify that forcing the value does not result in unsafe conditions.
  • Verify that the fact to close the tab while a forced value is active does not result in unsafe conditions.
Failure to follow these instructions can result in death, serious injury, or equipment damage.

Description

Usage

This faceplate will open when the corresponding active zone of IoEmbeddedATV6xx HMI is clicked.

The following figure explains the elements of IoEmbeddedATV6xx AnalogOutputs faceplate:

Hardware Channels

The detailed description of IoEmbeddedATV6xx AnalogOutputs faceplate elements is as follows:

Element Description Read-Write/Read-Only Comments

AQ1-AQ2

Output Channels

Read-Write

AQ1 and AQ2 are the output channels.

You can either allow the control logic to send outputs, or you can force the output channels.

Present Values

Read-Only

Click on the PV button to monitor the present values in the Analog Output channels.

Range Values selection

Read-Write

Range button enables you to select the Range of Values for Analog output channels.

Units

Read-Write

Click on the Unit button to define the units of measurement.

Force

Read-Write

Click the F button to force the required analog channel/s.

SYM

Symlink

Read-Only

Symlink associated to the channel.

HW

Hardware

Read-Only

Indicates the real-time data from the Hardware.

SIM/PV

Simulated/Program Value

Read-Write

The SIM/PV and HW values are the same, if the current value of the channel is not forced.

If the current value of the channel is forced, the SIM/PV value is equal to the forced value

MODE

Mode

Read-Only

Type configured for the channel

STATUS

Channel Status

Read-Only

Indicates the status of the channel.
