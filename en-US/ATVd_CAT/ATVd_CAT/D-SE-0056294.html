About This Menu

Following parameters can be accessed on the Graphic Display Terminal by pressing the OK key on the [DI7 Frequency measured] PFC7 parameter.

[DI7 Pulse Input Assign] PI7A

It displays all the functions associated with the pulse input in order to verify, for example, for compatibility problems.

If no functions have been assigned, [No] NO is displayed.
