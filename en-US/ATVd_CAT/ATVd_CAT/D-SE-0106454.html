RelayModuleVW3A3204 - Properties

Abstract

This HW CAT allows the IEC61499 application to control and monitor the Relay module.

Description

Usage

This element can be added to the Hardware Configuration Device in order to develop and implement applications for RelayModuleVW3A3204, based on the IEC61499 Standard.

The Symbolic Links are directly connected to the IEC61499 application.

Hardware Configuration

  1. In EcoStruxure Automation Expert, open the System tab → Logical Devices → Add Device that should have SE.DPAC.ATV_dPAC as a type → Click OK.

  2. Right click on the device to open the HW Configurator.

  3. Right click in the HW Configuration window to add the ATVdPACxVW3A3530D → Right click in the added ATVdPACxVW3A3530D to add a RelayModuleVW3A3204.

Hardware Properties Editor

Right click on the RelayModuleVW3A3204 to open the Hardware Properties Editor which is a window that allows to edit and configure the Hardware parameter settings.
