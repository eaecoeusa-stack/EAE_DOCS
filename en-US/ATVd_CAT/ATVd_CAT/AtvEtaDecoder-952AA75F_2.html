atvEtaDecoder

Overview

The atvEtaDecoder is a composite function block inside SE.IoATV Library.

The atvEtaDecoder composite function block is connected to other IEC 61499 function blocks, within the application.

Composite function blocks are collections of Basic function blocks or other composites with lower level functionality to achieve a higher level functionality.

atvEtaDecoder is used to decode the ETA CIA402 State Register (Logical Address 3201), to bit level (True or False) at function block level to be used in the application easily.

atvEtaDecoder is compatible with ATV6••, ATV9••, ATV340 and ATV320.
