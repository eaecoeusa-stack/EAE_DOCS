AltivarVs - Application CAT

Abstract

The AltivarVs is an application CAT inside SE.IoATV Library.

The AltivarVs acts as interface between an application and the ATVSpeedControl hardware CAT.

The AltivarVs improves the performance and simplifies the application of VSD_Control hardware CAT by offering enhanced IO scanner functionality, increased parameters, and more feedback information. Additionally, AltivarVs has the capability to operate with a larger number of logics.

The AltivarVs application CAT is connected to other IEC 61499 function blocks, within the application.

Data interface

Inputs

The following table provides the details of Data Interface Input parameters for AltivarVs:

Variable

Type

Description

CommandReady

BOOL

Drive Ready State Command

  • = 0: The state of the drive is NST

  • =1: The state of the drive is RDY

NOTE:

  • The state of the drive changes only if the REQ_RDY event is executed.

  • When the motor is running, Command Ready does not impact its operation. It serves as an optional input to transition from NST to RDY if necessary.

CommandRun

BOOL

Run or Stop the Motor

  • = 0: Stops the motor.

  • = 1: Runs the Motor.

CommandDirection

BOOL

Forward or reverse direction of rotation

  • = 0 : Starts the motor run in forward direction.

  • = 1: Starts the motor run in reverse direction.

NOTE: To allow the reverse direction on ATV Slaves:

On the display terminal: select: select [Complete settings] > [Command and Reference] > [Reverse Disable] > [NO]

CommandQuickStop

BOOL

Quick Stop Command

  • = 0: no reset command is present.

  • = 1: the motor decelerates very quickly.

ResetError

BOOL

  • = 0: No error reset is present.

  • = 1: Reset the error in the drive if it is resettable.

SetpointSpeedPct

REAL

Speed setpoint value in percentage (%) when AtvuseRPMSpeed = 0.

ATVSetPointSpeedRpm

REAL

Speed setpoint value in revolutions per minute.(RPM) when AtvuseRPMSpeed = 1.

AtvUseUserValue

BOOL

Allows to select the unit of the speed setpoint.

  • = 0: the unit is %

  • = 1: the unit is RPM

AtvRampAccelerationSec

REAL

Ramp acceleration value in seconds (s) based on parameter [Ramp increment]INR.

AtvRampDecelerationSec

REAL

Ramp deceleration value in seconds (s) based on parameter [Ramp increment]INR.

ExternalFault

BOOL

External fault is triggered.

  • 0 = No fault is triggered.

  • 1 = Fault is triggered.

AtvStatusETA

UINT

ETA status of the drive, it shows the status of the drive comprising several parameters.

Bit

Parameter

0

= 1 : Ready to switch on, awaiting power section line supply

1

=1 : Switched on, ready

2

= 1 : Operation enabled, running

3

= 1 : Operating State Fault (error detected)

4

Voltage enabled, power part connected to supply mains

  • = 0 : Power part not connected to supply mains

  • = 1 : Power part connected to supply mains

NOTE: When the drive is powered by the power section only, this bit is always at 1.

5

= 0 : Quick stop active

6

= 1 : Switch on disabled, power section line supply locked

7

  • = 0 : No warning

  • = 1 : Warning active

8

Drivecom reserved (= 0)

9

Remote: command or reference via fieldbus

  • = 0 : Command or reference given via the graphic display terminal or remote display terminal

  • = 1 : Command or reference via fieldbus

10

Target reference reached (Steady state)

  • = 0 : The reference is not reached

  • = 1: The reference has been reached

NOTE: When the drive is in speed mode, this is the reference frequency.

11

Internal limit active, reference outside limits (Reference exceeded < LSP or > HSP)

  • = 0 : The reference is within the limits

  • = 1 : The reference is not within the limits

NOTE: When the drive is in speed mode, the limits are defined by the [Low speed] (LSP) and [High speed] (HSP) parameters.

12

Reserved (= 0)

13

Reserved (= 0)

14

STOP via stop button

  • 0 : STOP button not active

  • = 1 : Stop triggered by the STOP button on the graphic display terminal or the remote display terminal

15

Direction of rotation

  • = 0 : Forward rotation at output

  • = 1 : Reverse rotation at output

NOTE: The combination of Bits 0, 1, 2, 4, 5 and 6 defines the state in the DSP 402 state chart (refer to fieldbus manuals).
