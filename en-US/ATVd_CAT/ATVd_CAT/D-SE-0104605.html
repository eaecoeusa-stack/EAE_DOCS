ATVSpeedControl - sDriveInterface

Abstract

ATVSpeedControl - sDriveInterface is the HMI representation for ATVSpeedControl Hardware CAT.

ATVSpeedControl - sDriveInterface allows you to monitor the data interface between the application and the ATVSpeedControl Command, to facilitate the interface between the IEC 61499 application, and the Altivar drive.

In particular, it displays the computation parameters of the Normalized Motor Speed (expressed in % or rpm).

If additional information is required on the data interface values during program execution, the Add Watch function can be used.

You can use the ATVSpeedControl - sDriveInterface, only when a Hardware CAT of a Drive has been instantiated, and the HMI representation of the selected drive dragged and dropped onto the HMI canvas.

Description

Add the ATVSpeedControl HW CAT under ATVdPACxVW3A3035D.

The following figure illustrates the HMI symbol for ATVSpeedControl CAT: Variable Speed Drive Interface.

Usage

The following procedure describes the steps to use ATVSpeedControl CAT HMI in your solution:

Step

Action

1

After Adding the ATVSpeedControl Hardware CAT under ATVdPACxVW3A3035D and building the solution, the Hardware CAT Instances will be automatically added:

2

Create a new Canvas then drag and drop the ATVSpeedControl CAT instance inside the Canvas. A new window will pop up stating that two options are available: sDrive and sDriveInterface.

3

Choose sDriveInterface and check the Status of the Device and the configuration parameter in HMI.
