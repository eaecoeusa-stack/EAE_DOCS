[Stop configuration] STT– Menu

Access

[Complete settings] [Generic functions] [Stop configuration]

About This Menu

This function allows to manage the type of stop mode. If a stop is requested, the drive performs a stop according to [Type of stop] STT.

NOTE: This function cannot be used with some other functions.

[Type of stop] STT

Normal stop mode.

This parameter allows to configure the stop of the drive on removal of the run command.

The setting of this parameter is not taken into account when the active command channel is set to a communication channel and [Control Mode] CHCF is not set to [I/O profile] IO. In this case, the stop mode is defined by the parameter [SwitchOnDisable Stp] DOTD .

Setting

Code / Value

Description

[On Ramp]

RMP

The stop follows a ramp determined according to [Deceleration] DEC (or [Deceleration 2] DE2 depending on which one is active at the corresponding stop request).

It can be requested:

  • At removal of the run command or at stop request (such as pressing the STOP/RESET button), according to STT.

  • Drive switching from the operating state Operation enabled to the operating state Switched on depending on DOTD.

Factory setting

[Fast stop]

FST

The stop follows a ramp determined by [Deceleration] DEC (or [Deceleration 2] DE2) divided by [Ramp Divider] DCF coefficient.

It can be requested:

  • At removal of the run command or at stop request (such as pressing the STOP/RESET button), according to STT.

  • Using a digital input or a bit depending on FST.

[Freewheel Stop]

NST

Power stage is disabled.

It can be requested:

  • At removal of the run command or at stop request (such as pressing the STOP/RESET button), according to STT.

  • Using a digital input or a bit depending on NST.

  • Using a configurable threshold FFT.

  • Drive switching from the operating state Operation enabled to the operating state Switched on depending on DOTD.

[DC injection]

DCI

DC voltage is injected in the winding of the motor. See below the profile of this DC injection.

It can be requested:

  • At removal of the run command or at stop request (such as pressing the STOP/RESET button), according to STT.

  • Using a digital input or a bit depending on DCI.

[Freewheel Stop] NST

The motor stops in freewheel when the assigned input or the bit changes to 0. If the input returns to state 1 and the run command is still active, the motor will only restart if [2/3-Wire Control] TCC is set to [2-Wire Control] 2C and if [2-wire type] TCT is set to [Level] LEL or [Level With Fwd Priority] PFO. If not, a new run command must be applied.

[Freewheel stop Thd] FFT

Freewheel stop threshold.

Speed threshold below which the motor switches to freewheel stop.

This parameter allows switching from a ramp stop or a fast stop to a freewheel stop when the speed drops below a low speed threshold.

This parameter can be accessed if:

  • [Type of stop] STT is set to [Fast stop] FST or [On Ramp] RMP, and

  • [Auto DC Injection] ADC is configured, and.

  • [Brake assignment] BLC is not configured.

Setting

Description

0.2...599.0 Hz

Setting range

Factory setting: [Freewheel stop Thd] FFT default value is 0.4% of FRS(S), with a minimum of 0.2 Hz.

[Fast Stop Assign] FST

The motor stops in fast stop when the assigned input or the bit changes to 1. (bit in [I/O profile] IO at 0). Fast stop function is activated as long as [Fast Stop Assign] FST is set to 1, whatever the channel which controls the drive.

If the input returns to state 1 and the run command is still active, the motor will only restart if [2/3-Wire Control] TCC is set to [2-Wire Control] 2C and if [2-wire type] TCT is set to [Level] LEL or [Level With Fwd Priority] PFO.

If not, a new run command must be applied.

NOTE: This function cannot be used with some other functions.

[Ramp Divider] DCF

Fast Stop deceleration ramp reduction coefficient.

When the fast stop function is activated the current deceleration time [Deceleration] DEC and [Deceleration 2] DE2 is divided by [Ramp Divider] DCF, so the selected output ramp deceleration is equal to input ramp deceleration divided by [Ramp Divider] DCF.

Value 0 corresponds to a minimum ramp time. Make sure that this value is compatible with the inertia being driven.

Setting

Description

0...10

Setting range

Factory setting: 4

[DC Injection Assign] DCI

DC injection assignment.

This function is used in application which need :

  • To stop the motor if it is running at low speed.

  • To apply and remove braking torque at stop.

 WARNING
UNINTENDED MOVEMENT
  • Do not use DC injection to generate holding torque when the motor is at a standstill.
  • Use a holding brake to keep the motor in the standstill position.
Failure to follow these instructions can result in death, serious injury, or equipment damage.

This parameter can be accessed if:

  • [Brake assignment] BLC is set to [No] NO, or

  • [Motor control type] CTT is not set to [Sync. mot.] SYN or [Sync.CL] FSY or [Reluctance Motor] SRVC or [SYN_U VC] SYNU.
    NOTE: If conditions are not respected, the error [Invalid Configuration] CFI triggered.

DC injection braking is initiated when the assigned input or bit changes to state 1. Before the DC injection occurs at any speed, a demagnetizing time must be respected. To perform the demagnetization, the voltage is decreased linearly during a time equivalent to the rotor time constant of the motor.

If DCI current is injected, [DC Inj Level 1] IDC is set to 1.

When the DC injection is activated, the measured current is adjusted to [DC Inj Level 1] IDC thanks to a filter initialized at the present current value.

If [DC Injection Assign] DCI function is still activated after [DC Inj Time 1] TDI, the DC current is automatically reduced to [DC Inj Level 2] IDC2.

When [DC Injection Assign] DCI function is deactivated, no more current is injected.

If the input returns to state 0 and the run command is still active, the motor will only restart if [2/3-Wire Control] TCC is set to [2-Wire Control] 2C and if [2-wire type] TCT is set to [Level] LEL or [Level With Fwd Priority] PFO. If not, a new run command must be applied.

When the stop requested mode is set to DC injection stop, first [DC Inj Level 1] IDC is injected during [DC Inj Time 1] TDI time, then [DC Inj Level 2] IDC2 is injected during [DC Inj Time 2] TDC time.

NOTE: This function cannot be used with some other functions.

DC injection level 1.

NOTICE
OVERHEATING
Verify that the connected motor is properly rated for the DC injection current to be applied in terms of amount and time.
Failure to follow these instructions can result in equipment damage.

This parameter can be accessed if:

  • [Type of stop] STT is set to [DC Injection Assign] DCI, or
