ATVSpeedControlLight - Application

Abstract

The ATVSpeedControlLight function block is a lighter built of the ATVSpeedControl function block and used to command the Altivar variable speed drive. It has fewer block parameters and hence takes less memory and has a faster response.

The ATVSpeedControlLight function block is connected to other IEC 61499 function blocks, within the application.

Description

Data Interface

Type

Description

CommandSetpointSpeedRpm

AtvUseRpmSpeed

BOOL

  • RPM when AtvuseRPMSpeed = 1

ResetError

BOOL

1 = Clear drive Error

ExternalFault

BOOL

External fault.

  • 0 = Remove external fault.

  • 1 = apply external fault.

The following table provides details regarding the Data Interface Outputs for ATVSpeedControlLight CAT:

Name

Type

Description

FeedbackRunForward

BOOL

Motor forward rotation feedback

  • = 0: Stopped

  • = 1: Run Forward

FeedbackRunReverse

BOOL

Motor Reverse rotation feedback

  • = 0: Stopped

  • = 1: Run Reverse

FeedbackSpeedPct

REAL

Speed Feedback in percentage (%)

AtvFeedbackSpeedRpm

REAL

Current motor speed in rpm (rotation per minute).

NOTE: The AtvFeedbackSpeedRpm value is less than or equal to the maximum speed value, where the maximum speed value is calculated as follows:

  • FRSS: Synchronous motor nominal frequency.

  • PPNS: Synchronous pole pairs number.

Note that the maximum speed value corresponds to 100% of FeedbackSpeedPct.

Error

BOOL

1 = A detected Error in the control block or in the device

Ready

Bool

1 = The device is enabled and free of detected errors.

LocalOverride

Bool

1 = The device is being controlled from external source.
