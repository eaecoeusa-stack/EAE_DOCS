ATVCyclicScanner - Properties

Abstract

The ATVCyclicScanner is a hardware CAT available under the VSD_EmbeddedFunctions in the IoATV library.

The ATVCyclicScanner hardware CAT is connected to other IEC61499 function blocks, within the application via the Symbolic Links.

Description

The ATVCyclicScanner hardware CAT is used to read and write the device parameters cyclically.

The ATVCyclicScanner parameters define the reading and writing functionality for the hardware CAT, and they can be used in the function block inside the application.
