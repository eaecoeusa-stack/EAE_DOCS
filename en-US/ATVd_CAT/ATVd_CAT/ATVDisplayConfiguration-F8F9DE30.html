ATVDisplay Configuration

  1. In ATV dPAC Displays tab, Right-click on your physical device (ATV9•• in this example), then select Create New Application.
    A window appears asking you to confirm the safety message:
    ATV dPAC Graphic Display Terminal HMI fallback logic

    You are about to configure the ATV dPAC Graphic Display Terminal HMI.

     WARNING
    LOSS OF CONTROL
    As the application variables can be set from the Altivar Graphic Display Terminal HMI:
    • Perform a Failure Mode and Effects Analysis (FMEA) of your application, and apply preventive and detective controls before implementation.
    • Provide a fallback state for undesired control events or sequences.
    • Provide a separate or redundant control paths whenever required.
    • Supply appropriate parameters, particularly for limits.
    • Review the implication of communication link interruptions and take actions to mitigate (for example, but not limited to, connect the ConnectionStatus input Symbolic Link from ATVDisplay Hardware CAT (SE.IoATV Library) to an applicative fallback logic, in order to detect and react to a Graphic Display Terminal disconnection).
    • Provide independent paths for control functions (for example, emergency stop, over-limit conditions, and fault conditions) according to the safety analysis and applicable codes, and regulations.
    • Apply local accident prevention and safety regulations and guidelines.
    • Test each implementation of this library for proper operation before placing it into service.
    Failure to follow these instructions can result in death, serious injury, or equipment damage.
  2. Type the name of your application, then click Ok.
    Result: A new application is created.
  3. Create a new Display (Monitor).
    1. Right-click on your Boiler application, then select Add New Display.
    2. Double-click on your display, type the name of your display, then press Enter.
    3. Select your display, then select the type of your menu (Standard in this example).
    4. Click Save
  4. Add new sub-display (Pressure)
    1. Right-click on your display, then select Add New Display.
    2. Double-click on your display, type the name of your display, then press Enter.
    3. Select your display, then select the type of your menu (Display - Single Bargraph in this example).
    4. Click Save
    Result:
  5. Drag and drop the CAT instances for variable selection
    1. Drag and drop the application CAT instance of your variable (Pressure).
    2. Select the variable that you cant to configure, then configure its parameters.
    3. Click Save .
  6. Add new sub-display (Temperature)
    1. Right-click on your display, then select Add New Display.
    2. Double-click on your display, type the name of your display, then press Enter.
    3. Select your display, then select the type of your menu (Display - Single Bargraph in this example).
    4. Click Save
    Result:
  7. Drag and drop the CAT instances for variable selection
    1. Drag and drop the CAT instance of your variable (Temperature).
    2. Select the variable that you cant to configure, then configure its parameters.
    3. Click Save .
  8. Move the display to the desired location in the tree view
    1. Select the Pressure display.
    2. drag and drop it bellow the Temperature display.
  9. Navigate through the customized tree view on the graphic display terminal.
    1. Open the deploy and Diagnostic Editor.
    2. Compile, login, deploy and run the device actions.
    3. On the graphic display terminal select, [Main menu] > [Controller Application] > [Application HMI] to navigate through the customized tree view on the graphic display terminal of your application.

    If the Controller Application menu does not appear on the Graphic Display Terminal after you configure security operation, you need to restart the runtime. 

    To do this in  EcoStruxure Automation Expert, follow these steps

    1. Open the Deploy and Diagnostics tab.

    2. Select Device actions.

    3. Select Restart runtime.

    NOTE:

    • In EcoStruxure Automation Expert 24.1, the Controller Application menu is found on the main menu 9. Controller Application  of the Graphic Display Terminal for all ATV drives.

    • In EcoStruxure Automation Expert 24.0:

      • For ATV6•• and ATV9••, the menu is on the main menu 9. Controller Application.

      • For ATV340, the menu is inside the 6. Communication menu.
