IoEmbeddedATV340Ex30to75kW Relay & Digital Outputs

Abstract

IoEmbeddedATV340Ex30to75kW Relay & Digital Outputs faceplate sends the output command and control signals.

You may additionally force a Digital output signal or signals, by over-riding the control logic.

 WARNING
LOSS OF CONTROL
  • Verify that forcing the value does not result in unsafe conditions.
  • Verify that the fact to close the tab while a forced value is active does not result in unsafe conditions.
Failure to follow these instructions can result in death, serious injury, or equipment damage.

Description

Usage

This faceplate opens when the corresponding active zone is clicked in the IoEmbeddedATV340E HMI Symbol

The following table provides details regarding elements of the Relay & Digital Outputs faceplate:

Element

Description

Read-Write/Read-Only

Comments

R1...R3,DQ

Relay outputs R1, R2, R3 and DQ

Read-Write

These are the Relay output command signals, to the respective relays.

You may, additionally force the output command signals, over-riding the control-logic.

Force button

Read-Write

Click on F button, to force the Digital Inputs.

Force Indication Status

Read-Only

This button indicates the status if a Digital Input has been forced.

Blue color indicates that a Digital Input is forced

Note: Grey color indicates that no input is forced

Note: If an input has been forced in any other faceplate, and no input is forced in current faceplate, even then the indicator displays blue color.
