ATVFrequencyControl - sDrive

Abstract

ATVFrequencyControl - sDrive is the HMI representation of the ATVFrequencyControl module. You can access several faceplates by clicking different active zones of ATVFrequencyControl - sDrive, to perform the following actions:

  • Monitor and take manual control of the VSD .

  • Visualize the key Motor parameter values.

  • Behavior: Configure the VSD parameters.

Defining the HMI Security Group

Follow these steps to Define the HMI Security Group of your ATVFrequencyControl hardware CAT:

Step

Action

1

Configure your physical device:

  1. Click on the System icon to open the System tab.

  2. Click on the Physical Devices icon to open the Physical Devices editor.

  3. Drag and drop your altivar device to the Physical Devices editor.

  4. Click on your altivar device to display its properties, then select ATV dPAC for the Communication Card Reference.

  5. Click on the ATV dPAC module to display its properties, then type the IPV4 Address of your ATV dPAC

2

Configure your logical device:

  1. Click on the Logical Devices icon to open the Logical Devices editor.

  2. Click Add Device icon to add a new logical device.

  3. Select SE.DPAC:ATV_dPAC as a Type, then click OK.

  4. For the Physical Device column, select your ATV dPAC module (Already created in the Physical Devices editor).

3

Create users:

  1. Select View > User Management.

  2. Type the name of your new user.

  3. Type the password.

  4. Select the required checklists.

  5. Specify the security level for the user, ranging from 0 (minimum) to 15 (maximum).

  6. Repeat steps b to e for each user you need to create.

Result:

4

Configure the security:

  1. Click on the Physical Devices icon to open the Physical Devices editor.

  2. Click start monitoring icon .

  3. Right-click on the ATV dPAC, select Reset security, Click Yes to confirm the Reset security safety message, then type the old credentials of your device.

  4. Right-click on the ATV dPAC, select Configure security, select Trust to trust the certificate information, then type the old credentials of your device.

Result:

5

Configure your hardware configuration:

  1. Click on the Hardware Configuration icon to open the Hardware Configuration editor.

  2. Right-click in Hardware Configuration editor, then select Add.

  3. Select the bus element type SE.IoATV.ATVdPACxVW3A3530D, then click Add.

  4. Right click on the added bus ATVdPACxVW3A3530D, select the configuration module SE.IoATV.ATVFrequencyControl, then click Add.

6

Drag and drop the CAT instances to the canvas:

  1. Click Build.

  2. Right-click Canvases, then select New resolution.

  3. Select the resolution you want, then click Finish.

  4. Right-click on the created resolution, select new item, type the name of your item, then click Finish.

  5. Drag and drop the Cat instance of your ATVFrequencyControl to the canvas, then select sDrive.

7

Configure the security group level of your hardware CAT:

  1. Select the dropped CAT instance to display its properties.

  2. Set the security group of the different parameters of the CAT instance.

NOTE: If the user's level is equal to or higher than the security group level of the following parameters, then he is authorized to perform the associated actions..

Parameter

Associated Actions

Force

  • To select the mode (Auto/Force).

  • To login to the Webserver.

Mode

To start the Test Without Device (On/Off).

Parameter

  • To apply Autotuning.

  • To erase Autotuning.

Reset

To clear the detected error if it is resettable and the error is no longer present, by clicking the CLEAR button.

Setpoint

  • To monitor the actual Hardware values with Reference Values: Frequency, Speed, Torque, Current and Power

  • To set the speed value in rotations per minute (RPM) or percentage (%).

  • To select the unit of the speed (rotations per minute (RPM) or percentage (%)).

  • To send the command to the motor (Run/Stop/Quick stop).

  • To select the direction (Forward/ Reverse).

  • To select the status of Auto-Stop Inhibition (On/Off).

    • On: Motor continues running.

    • Off: motor stops running.

    NOTE: Auto-Stop Inhibition function disables the automatic stop of the connected drive, when the faceplate is no longer accessible (example: changing the canvas).

  • Trend: use this zone, to observe and monitor the graphical representation of the drive parameter values.

  • Commands: to select the drive operating modes.

  • Configuration: to configure the ATV drive settings.
