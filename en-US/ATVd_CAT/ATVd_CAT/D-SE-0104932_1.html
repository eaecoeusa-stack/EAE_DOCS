IoEmbeddedATV340x075to22kW HMI Symbol

Abstract

IoEmbeddedATV340x075to22kW is the HMI representation of the IO Embedded ATV340 module. You can access various faceplates by clicking on the corresponding active zones of the IO Embedded ATV340 HMI.

Each faceplate allows you to observe values, monitor parameters, set Range of values, select Units and force the Analog/Digital Inputs.

Defining the HMI Security Group

Follow these steps to Define the HMI Security Group of your IoEmbeddedATV340Ex30to75kW hardware CAT:

Step

Action

1

Configure your physical device:

  1. Click on the System icon to open the System tab.

  2. Click on the Physical Devices icon to open the Physical Devices editor.

  3. Drag and drop your altivar device to the Physical Devices editor.

  4. Click on your altivar device to display its properties, then select ATV dPAC for the Communication Card Reference.

  5. Click on the ATV dPAC module to display its properties, then type the IPV4 Address of your ATV dPAC

2

Configure your logical device:

  1. Click on the Logical Devices icon to open the Logical Devices editor.

  2. Click Add Device icon to add a new logical device.

  3. Select SE.DPAC:ATV_dPAC as a Type, then click OK.

  4. For the Physical Device column, select your ATV dPAC module (Already created in the Physical Devices editor).

3

Create users:

  1. Select View > User Management.

  2. Type the name of your new user.

  3. Type the password.

  4. Select the required checklists.

  5. Specify the security level for the user, ranging from 0 (minimum) to 15 (maximum).

  6. Repeat steps b to e for each user you need to create.

Result:

4

Configure the security:

  1. Click on the Physical Devices icon to open the Physical Devices editor.

  2. Click start monitoring icon .

  3. Right-click on the ATV dPAC, select Reset security, Click Yes to confirm the Reset security safety message, then type the old credentials of your device.

  4. Right-click on the ATV dPAC, select Configure security, select Trust to trust the certificate information, then type the old credentials of your device.

Result:

5

Configure your hardware configuration:

  1. Click on the Hardware Configuration icon to open the Hardware Configuration editor.

  2. Right-click in Hardware Configuration editor, then select Add.

  3. Select the bus element type SE.IoATV.ATVdPACxVW3A3530D, then click Add.

  4. Right click on the added bus ATVdPACxVW3A3530D, select the configuration module SE.IoATV.IoEmbeddedATV340x075to22kW, then click Add.

6

Drag and drop the CAT instances to the canvas:

  1. Click Build.

  2. Right-click Canvases, then select New resolution.

  3. Select the resolution you want, then click Finish.

  4. Right-click on the created resolution, select new item, type the name of your item, then click Finish.

  5. Drag and drop the Cat instance of your SE.IoATV.IoEmbeddedATV340x075to22kW to the canvas.

7

Configure the security group level of your hardware CAT:

  1. Select the dropped CAT instance to display its properties.

  2. Set the security group of the CAT instance, by default it is 3 for the force.

NOTE: If the user's level is equal to or higher than the security group level, then he has the authority to, observe the values, monitor the parameters, set the range of the values, select the units and force the Analog/Digital Inputs.

Settings

The IO Embedded ATV340 HMI symbol has various active zones. You can click on the various active zones, to access the corresponding faceplate.

You can access faceplates to monitor, observer values, force values, select Range values and select Units, related to the following active zones:

  • Analog Inputs

  • Analog Outputs

  • Digital Inputs

  • Digital Outputs

  • Relay Outputs
