Memory consumption of AltivarVs vs MotorVS

Abstract

The usage of MotorVS application CAT from SE.AppCommonProcess library could limit the number of motor servers. If the limitation occurs, it is recommended to use AltivarVs from SE.IoATV library instead.

Description

The table below displays a comparison of the memory consumption between MotorVS application CAT from SE.AppCommonProcess library versus AltivarVs from SE.IoATV library.

Summary

You can deploy up to 4 MBTCP servers and 1 ATVSpeedControl using MotorVS application CAT from SE.AppCommonProcess library. Therefore, deploy is not possible with more than 5 MBTCP servers.
