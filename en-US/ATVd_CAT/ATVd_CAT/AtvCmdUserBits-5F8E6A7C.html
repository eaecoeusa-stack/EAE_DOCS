atvCmdUserBits

Overview

The atvCmdUserBits is a composite function block available in IoATV library. It connects to other IEC61499 function blocks, within the application via Symbolic Links.

atvCmdUserBits can be used to write on command bits 11 to 15 using ATVSpeedControl hardware CAT or ATVFrequencyControl hardware CAT.

These command bits can be assigned to specific drive functions using SoMove or the ATV Graphic Display Terminal. Refer to the Altivar Programming Manual for additional details.
