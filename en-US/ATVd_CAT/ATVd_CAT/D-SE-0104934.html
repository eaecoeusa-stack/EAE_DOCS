IoEmbeddedATV340x075to22kW AnalogInputs

Abstract

The IoEmbeddedATV340x075to22kW AnalogInputs faceplate allows you to monitor the analog input values

You can observe the values of the parameters, force the values, set the range of values and select the units.

 WARNING
LOSS OF CONTROL
  • Verify that forcing the value does not result in unsafe conditions.
  • Verify that the fact to close the tab while a forced value is active does not result in unsafe conditions.
Failure to follow these instructions can result in death, serious injury, or equipment damage.

Description

Usage

This Analog Inputs faceplate will open when the corresponding active zone of IoEmbeddedATV340 HMI is clicked.

The detailed description of IoEmbeddedATV340x075to22kW AnalogInputs faceplate elements is as follows:

Element

Description

Read-Write/ Read-Only

Comments

AI1-AI2

Analog Input channels AI1-AI2

Read-Only

You can monitor the values from the required Analog Channel/s.

Present Values

Read-Write

Click the PV button to display the present values in the required analog channel/s.

Force button

Read-Write

Click the F button to force the required analog channel/s.

Range selection

Read-Write

Click the Range button to select the range of the values.

Unit

Read-Write

Click the Unit button to select the required Units.

Force Indicator

Read-Only

This element indicates if an Analog Channel has been forced.

Blue color indicates that one or more channels have been forced

Grey color indicates that no channel has been forced

NOTE: If an input has been forced in any other faceplate, and no input is forced in the current faceplate, the Force indicator shows it with the blue color.

SYM

Symlink

Read-Only

Symlink associated to the channel.

HW

Hardware

Read-Only

Indicates the real-time data from the Hardware.

SIM/PV

Simulated/Program Value

Read-Write

The SIM/PV and HW values are the same, if the current value of the channel is not forced.

If the current value of the channel is forced, the SIM/PV value is equal to the forced value.

MODE

Mode

Read-Only

Displays the range selected.

STATUS

Channel Status

Read-Only

Indicates the status of the channel.
