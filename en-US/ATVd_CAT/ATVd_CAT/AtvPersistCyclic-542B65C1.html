atvPersistCyclic

Overview

The atvPersistCyclic is a composite function block within the SE.IoATV Library, It connects to other IEC 61499 function blocks in the application.

Composite function blocks are collections of basic function blocks or other composites with lower level functionality to achieve a higher-level functionality.

The atvPersistCyclic ensures that critical data is not lost during power interruptions by saving the necessary values automatically.

The atvPersistCyclic is compatible with ATV9••, ATV6•• and ATV340.

The figure below displays The atvPersistCyclic composite function block.

NOTE: If you plan to operate at low frequencies with only a few variables, using atvPersistEvent will put less load on the CPU. However, if you aim for high-frequency operations with many variables, atvPersistCyclic will be the better choice for managing CPU load.
