sAtv340Ex30to75kW

Abstract

sAtv340Ex30to75kW is the graphical representation of Altivar Machine ATV340 variable speed drives for safety and heavy duty applications from 30 to 75kW.

Description

  1. In EcoStruxure Automation Expert , open the System tab → Logical Devices → Add Device that should have SE.DPAC.ATV_dPAC as a type → Click OK.

  2. Right click on the device to open the HW Configurator.

  3. Right click in the HW Configuration window to add the ATVdPACxVW3A3530D and build the solution.

    NOTE: Check that the ATV Hardware CAT instances are properly created in the solution.

  4. Create a new Canvas → Drag and drop ATVdPACxVW3A3530D Hardware CAT Instance to the EcoStruxure Automation Expert Canvas Editor for creating the HMI Symbols of Altivar drive and ATVdPAC Module → Choose sAtv340Ex30to75kW→ Click OK.

  5. The sAtv340Ex30to75kW generic symbol will appear in the solution.
