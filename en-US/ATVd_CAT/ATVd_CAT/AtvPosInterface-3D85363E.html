atvPosInterface

Overview

The atvPosInterfacer is a composite function block within the SE.IoATV Library, It connects to other IEC 61499 function blocks in the application.

Composite function blocks are collections of basic function blocks or other composites with lower level functionality to achieve a higher-level functionality.

The atvPosInterface is used to allow choosing the behavior of the drive when the positioner has finished its movement: either to stay in RUN mode or switch back to RDY mode (to allow braking in the case of vertical applications for example).

The atvPosInterface is also used to properly initialize the encoder when starting the application.

The atvPosInterface is compatible with ATV9•• and ATV340.
