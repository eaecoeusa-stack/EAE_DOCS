ATVPosInterface2 - Application CAT

Abstract

The ATVPosInterface2 is an application CAT inside SE.IoATV Library Application CAT section.

The ATVPosInterface2 easily interface with ATVPosControlLight, PositionerVSDBaseExt function block and AtvPosControl Hardware CAT for motor position control and brake control. refer to ... for more information

The ATVPosInterface2 CAT is connected to other IEC 61499 function blocks, within the application.
