IoEmbeddedATV9xx - Properties

Abstract

The Hardware CAT allows the IEC 61499 application to control and monitor the embedded I/Os of the drive.

Description

Usage

This element can be added to the Hardware Configuration Device in order to develop and implement applications for IoEmbeddedATV9xx module, based on the IEC 61499 Standard.

The Symbolic Links are directly connected to the IEC 61499 application.

Hardware Configuration

  1. In EcoStruxure Automation Expert, open the System tab → Logical Devices → Add Device that should have SE.DPAC.ATV_dPAC as a type → Click OK.

  2. Right click on the device to open the HW Configurator.

  3. Right click in the HW Configuration window to add the ATVdPACxVW3A3530D → Right click in the added ATVdPACxVW3A3530D to add a IoEmbeddedATV9xx.
