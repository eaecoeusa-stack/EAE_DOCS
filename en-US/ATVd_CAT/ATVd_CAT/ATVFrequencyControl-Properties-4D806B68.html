ATVFrequencyControl - Properties

Abstract

The ATVFrequencyControl Hardware CAT allows the IEC 61499 application to control and monitor an Altivar Variable Speed Drive with an ATV dPAC module.

The ATVFrequencyControl Hardware CAT supports frequency command and it can be used to control the drive at low frequency.

Description

Data Interface

The following table provides details regarding the Data Interface Inputs for ATVFrequencyControl:

Name

Type

Description

CommandReady

BOOL

Drive Ready State Command

  • 0 = NST

  • 1 = RDY

CommandRunForward

BOOL

1 = the motor starts running in the direction selected

CommandRunReverse

BOOL

1 = the motor starts running in the reverse direction selected

CommandQuickStop

BOOL

1 = the motor decelerates very quickly

CommandSetpointSpeedPct

REAL

Speed setpoint in Percentage when AtvUseRPMSpeed = False

ResetError

BOOL

1 = Clear Error

ATVCommandSetpointSpeedHz

WORD

Speed setpoint in Hz when AtvUseFreqSpeed = True

AtvRampAccelerationSec

WORD

Drive Acceleration Time

Note : Range of Acceleration time is based on setting of Drive Parameter INR

AtvRampDecelerationSec

WORD

Drive Deceleration Time

Note : Range of this Parameter based on setting of Drive Parameter INR

AtvUseFreqSpeed

BOOL

Frequency as setpoint

ExternalFault

BOOL

External fault.

  • 0 = Remove external fault.

  • 1 = apply external fault.

Name

Type

Description

FeedbackSpeedPct

Real

Speed feedback in percentage (%).

FeedbackRunForward

BOOL

1 = The Drive runs in forward direction.

FeedbackRunReverse

BOOL

Error

BOOL

LocalOverride

BOOL

Ready

BOOL

1 = The device is enabled and free of detected errors.

AtvStatusETA

WORD

This output shows the status of ATV device.

AtvStatusETI

WORD

This output shows the status of internal status

AtvFeedbackSpeedRpm

WORD

Current motor speed in rpm (rotation per minute).

NOTE: The AtvFeedbackSpeedRpm value is less than or equal to the maximum speed value, where the maximum speed value is calculated as follows:

  • FRSS: Synchronous motor nominal frequency.

  • PPNS: Synchronous pole pairs number.

Note that the maximum speed value corresponds to 100% of FeedbackSpeedPct.

AtvStatusHMIS

WORD

This output shows the status of HMI.

AtvFrequencyHz

WORD

This parameter displays the estimated motor frequency in Hz.

AtvTorquePct

WORD

Current motor Torque in percentage (%)

AtvPowerPct

WORD

This output shows actual drive electrical power in kW. (Resolution is 0.00).

AtvCurrentA

WORD

Present motor current in 0.1A increment

AtvErrorCode

WORD

Existing Error Code

AtvWarning

BOOL

1 = An warning has been activated for the device.

AtvWarningCode

WORD
