ATVPosControlLight - Properties

Abstract

The ATVPosControlLight Hardware CAT allows the IEC 61499 application to control and monitor an Altivar Variable Speed Drive with an ATV dPAC module.

The ATVPosControlLight Hardware CAT can be used for position control when the drive profile is set to I/O. It features a separate Position control faceplate, offering:

  • Fast Start and Stop with no ramp, ensuring efficient and dynamic control.

  • Precise control.

Hardware Configuration

  1. In EcoStruxure Automation Expert, open the System tab → Logical Devices → Add Device that should have SE.DPAC.ATV_dPAC as a type → Click OK.

  2. Right click on the device to open the HW Configurator.

  3. Right click in the Hardware Configuration window, then add ATVdPACxVW3A3530D → Right click on the added ATVdPACxVW3A3530D, then add ATVPosControlLight.

Data interface description

Data Inputs:

The following table provides details regarding the Data Interface Inputs for ATVPosControlLight:

Name

Type

Description

CommandRun

BOOL

1 = Enable the run command operation in the direction selected.

ResetError

BOOL

1 = Reset the error in the drive if the error is resettable.

CommandSetpointSpeedRpm

REAL

Speed setpoint value in RPM

The following table provides details regarding the Data Interface Outputs for ATVPosControlLight:

Name

Type

Description

Error

BOOL

Displays the Error status

  • 1 = A detected Error is active in the control block or in the device.

REAL

Speed Feedback in Rpm

Ready

BOOL

1 = The drive is in ready state.

LocalOverride

BOOL

Hardware Properties Editor

The Hardware Properties Editor window allows you to edit and configure the Hardware parameter settings, which are transferred to the device upon the application start-up.

NOTE:

  • These parameters will override the existing parameter settings stored in the Altivar device.

  • If these parameters are modified during run-time by a source external to EcoStruxure Automation Expert (e.g. the Altivar device’s Webserver, SoMove, Modbus parameter write), then the IEC 61499 application might be impacted.

Right click on ATVPosControlLight, then select Properties Editor:

Result:

When you select the Properties Editor, a new window appears indicating to select the dedicated Encoder Type.

Once you select the relevant encoder, the Property Editor of ATVPosControlLight is opened with relevant encoder parameters.
