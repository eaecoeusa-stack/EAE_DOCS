ATV_DD_OUT

Abstract

The ATV_DD_OUT is a service function block inside SE.IoATV Library Service module.

The ATV_DD_OUT function block is connected to other IEC 61499 function blocks, within the application.

The main function of the service function block is to write parameter to the ATV Drive.

The ATV_DD_OUT function block has capability to read up to 8 drive specific parameters.
