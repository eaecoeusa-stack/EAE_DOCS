ATV_DD_IN

Abstract

The ATV_DD_IN is a service function block inside SE.IoATV Library.

The ATV_DD_IN function block is connected to other IEC 61499 function blocks, within the application.

The main function of the service function block is to read parameters from the ATV Drives, using their modbus address.

The ATV_DD_IN function block has capability to read up to 8 drive specific parameters.
