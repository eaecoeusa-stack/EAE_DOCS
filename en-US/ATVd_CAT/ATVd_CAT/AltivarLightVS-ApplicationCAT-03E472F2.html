AltivarVsLight - Application CAT

Abstract

The AltivarVsLight is an application CAT inside SE.IoATV Library Application CAT section.

The AltivarVsLight is an easy-to-use, simple AltivarVsd application CAT.

The AltivarVsLight can connect to both light and normal hardware CAT. Hence, It can connect with ATVSpeedControlLight and ATVSpeedControl hardware CAT.

The AltivarVsLight CAT is connected to other IEC 61499 function blocks, within the application.

Data interface

Inputs

The following table provides the details of Data Interface Input parameters for AltivarVsLight:

Variable

Type

Description

CommandReady

BOOL

Drive Ready State Command

  • = 0: The state of the drive is NST

  • =1: The state of the drive is RDY

NOTE:

  • The state of the drive changes only if the REQ_RDY event is executed.

  • When the motor is running, Command Ready does not impact its operation. It serves as an optional input to transition from NST to RDY if necessary.

CommandRun

BOOL

Run or Stop the Motor

  • = 0: Stops the motor.

  • = 1: Runs the Motor.

CommandDirection

BOOL

Forward or reverse direction of rotation

  • = 0 : Starts the motor run in forward direction.

  • = 1: Starts the motor run in reverse direction.

NOTE: To allow the reverse direction on ATV Slaves:

On the display terminal: select [Complete settings] > [Command and Reference] > [Reverse Disable] > [NO]

CommandQuickStop

BOOL

Quick Stop Command

  • = 0: no reset command is present.

  • = 1: the motor decelerates very quickly.

ResetError

BOOL

  • = 0: No error reset is present.

  • = 1: Reset the error in the drive if it is resettable.

AtvSetpointSpeedPct

REAL

Speed setpoint value in percentage (%) when AtvuseRPMSpeed = 0.

AtvUservalueSetPoint

REAL

Speed setpoint value in revolutions per minute.(RPM) when AtvuseRPMSpeed = 1.

AtvUseUserValue

BOOL

Allows to select the unit of the speed setpoint.

  • = 0: the unit is %

  • = 1: the unit is RPM

ExternalFault

BOOL

External fault.

  • 0 = Remove external fault.

  • 1 = apply external fault.

Outputs

The following table provides the details of Data Interface Output parameters for AltivarVsLight:

Variable

Type

Description

FeedbackRunForward

BOOL

Motor forward rotation feedback.

  • = 0: Stopped or running in the reverse direction.

  • = 1: Run Forward.

FeedbackRunReverse

BOOL

Motor Reverse rotation feedback.

  • = 0: Stopped or running in the forward direction.

  • = 1: Run Reverse.

Ready

BOOL

  • 0 = The drive is not in ready state.

  • 1 = The drive is in ready state.

Error

BOOL

The error status.

  • 0 = No Error.

  • 1 = A detected Error is active in the control block or in the device.

FeedbackSpeedPct

REAL

Speed Feedback in percentage (%).

AtvFeedbackValue

REAL

Speed feedback in revolutions per minute (RPM).

LocalOverride

BOOL

Local override signal.

  • =0: The device is controlled by EcoStruxure Automation Expert application mode.

  • =1: The device is controlled by an external source (EcoStruxure Automation Expert force mode, or the Graphic Display Terminal, or SoMove control panel).
