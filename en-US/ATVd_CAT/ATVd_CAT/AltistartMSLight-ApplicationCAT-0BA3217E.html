AltivarSoftStarter - Application CAT

Abstract

The AltivarSoftStarter is an application CAT inside SE.IoATV Library Application CAT acts as interface between an application and ATS480Mbtcp, or ATS480Eip hardware CAT.

The AltivarSoftStarter application CAT

The AltivarSoftStarter application CAT is connected to other IEC 61499 function blocks, within the application.

The application CAT AltivarSoftStarter can be used with ATS480Mbtcp and ATS480Eip with 2 symlink inputs and 12 symlink outputs.
