(1) To update the firmware for this specific case only:

  • Use EcoStruxure Automation Device Maintenance 20.2.351.2.

  • Install Automation Device Maintenance - ATV Plugin 20.2.351.2 (available in EcoStruxure Automation Expert 21.1 installation package) to enable the firmware update, refer to the Readme file instructions for further information.

(2) To update the firmware use the latest version of EcoStruxure Automation Device Maintenance 3.3 (available in EcoStruxure Automation Expert 25.0 installation package).

(3) Downgrading is possible but not recommended, as new features will not be supported.

Firmware Upgrade Path: Step-by-Step:

Upgrading from a version older than 23.0:

  • If your current version is 20.1:

    1. First, update to 21.1

    2. Then, update to 21.2

  • From 21.2, you can upgrade directly to 23.0

  • Once you're on 23.0, you're compatible with all newer firmware versions

Upgrading from 23.0 or later:

  • You can directly install the target firmware version using the standard firmware package, no intermediate steps needed.

NOTE: After updating to firmware version 21.1 or later, you may notice a Security File Corrupt [SPFC] error, restart the drive to clear the error.

Pre-configuration

There are four main steps to update the firmware of the device:

  1. Pre-configuration.

  2. Update the ATV drive firmware.

  3. Update the ATV Distributed PAC firmware.

  4. Update the labels and languages of the Graphic Display Terminal (VW3A1111).

Use the following instructions to setup the firmware with EcoStruxure Automation Device Maintenance.

Step Action

1

Download and install EcoStruxure Automation Device Maintenance (Available in EcoStruxure Automation Expert installation package)

NOTE:

  • To upgrade/downgrade any firmware versions between 21.2 and 23.0, it is recommended to use the latest version of EcoStruxure Automation Device Maintenance 3.2 (available in EcoStruxure Automation Expert 23.0 installation package).

  • To upgrade/downgrade any firmware versions between 20.2 and 21.2, it is recommended to use the version of EcoStruxure Automation Device Maintenance which is compatible with the current firmware of the drive or the ATV Distributed PAC, as well as the ATV Plugin (available in EcoStruxure Automation Expert installation package, to enable the firmware update).

2

Configure the discovery mode

NOTE: You can select the device discovery mode as Automatic or Manual discovery. In case of:

  • Automatic discovery, the tool will periodically send information over the network in the background and receive information from the responding devices.

  • Manual discovery, you can manually discover the devices connected in the network when required.

  1. Click Settings menu.

  2. Click Discovery.

  3. Select the Discovery mode (Automatic or Manual)

  4. Select the scanners that will be taking part in discovery (DPWS-IPv6 and Modbus TCP-IPv4). Use this setting to help prevent scanning of any devices that you want to avoid.

  5. Click OK.

NOTE: For more information about how to configure the scanners (DPWS and Modbus TCP), refer to EcoStruxure Automation Device Maintenance online help.

3

Copy-paste the ATV Firmware packages (*.fwp files and *.sedp files), to the default EcoStruxure Automation Device Maintenance Data Packages folder: by default it is C:\Users\Public\Documents\Schneider Electric\Data Packages

NOTE:

  • Always update the firmware of the Altivar drive and the ATV Distributed PAC to match the version of EcoStruxure Automation Expert, as mentioned in ATV Distributed PAC system compatibility table.

  • The firmware packages can be found in EcoStruxure Automation Expert installation package (Example: EcoStruxure Automation Expert installation package > Firmware > Prod > SEDP_ATVD_*.zip)
    The copied and pasted packages will then be displayed in EcoStruxure Automation Device Maintenance in DATA PACKAGE tab.

4

Before updating the firmware:

  • Confirm that EcoStruxure Automation Expert version supports the firmware version you are installing.

  • Close EcoStruxure Automation Expert discovery view in the Topological View.

  • Stop EcoStruxure Automation Expert monitoring in the Topological View is recommended.

5

Before updating the firmware, confirm that the ATV Distributed PAC runtime state is CLEAN, READY, or STOPPED as follows:

ATV Distributed PAC State Upgrade Status
CLEAN ALLOWED
READY ALLOWED
STOPPED ALLOWED
RUNNING ALLOWED
NOTE: EcoStruxure Automation Device Maintenance stops the ATV Distributed PAC if its in the run state before starting the firmware update. It’s not mandatory to stop using EcoStruxure Automation Expert.
