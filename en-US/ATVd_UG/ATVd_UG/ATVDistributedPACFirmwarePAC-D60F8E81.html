NOTE:

  • You need to update the firmware of ATV Distributed PAC after updating the firmware of the drive with the appropriate firmware version.

  • The following procedure is used with EcoStruxure Automation Device Maintenance 3.2, thus, it can be used to update any firmware version between 21.2 and 23.0

Step Action

Connect the ATV Distributed PAC physically

Connect the Ethernet cable between the ATV Distributed PAC port and the computer where EcoStruxure Automation Device Maintenance is running, then turn on the drive if it is not already on.

Discover the ATV Distributed PAC automatically or add it manually

  • Discover the drive automatically

    Step

    Action

    1

    Click the Add icon to add a new device.

    2

    Select Scan Network.

    3

    Updating the settings

    In the Configuration step:

    1. Select DPWS, for Discovery Protocols.

    2. Select all the needed network adapters, for Used Network Adapters.

    3. Click Scan.

    4

    In the Discovery Results step:

    1. Select your device.

    2. Click Next.

    NOTE: Some firewall software may block the discovery. If you are unable to discover the device, disable firewall or consult your system administrator.

    5

    In the Summary step, click Confirm.

    6

    Result: The discovered devices are listed in the DEVICES tab.

  • Add the ATV Distributed PAC manually using Modbus TCP scanner (mbap-IPv4)

    Step

    Action

    1

    Click Add to add a new device.

    Result:

    Result: The Add Device window opens.

    Select Catalog.

    2

    Select the Commercial Reference of your drive.

    NOTE: To add the drive manually using Modbus TCP scanner (mbap-IPv4), select ATV***, not ATV*** (modernized).

    3

    Select MODBUS (TCP) for Connection.

    4

    Type the IP Address of your drive (port 502).

    5

    Click Add Device.

  1. Click Device certificate icon.

Login to the ATV Distributed PAC

Login the ATV Distributed PAC over Modbus TCP scanner (mbap-IPv4) discovery

Step

Action

1

Click Set credentials icon

2

Type the credentials

For Device User Name, type ADMIN

For Device password, type the Ethernet option module credentials.

NOTE: :

  • If the password is not modified via SoMove or Webserver, the default Ethernet option module credentials can be used, and are available from the Graphic Display Terminal (VW3A1111) in: [Main Menu] > [Communication] COM > [Comm parameters] CMP > [Opt Eth Config] ETO > [User authentication] SECO > [Default Pwd Eth Opt] WDPO

  • If the password is modified but you don’t know the new credentials, the drive Ethernet option module credentials can be reset from the Graphic Display Terminal (VW3A1111) in: [Main Menu] > [Communication] COM > [Comm parameters] CMP > [Opt Eth Config] ETO > [User authentication] SECO > [Reset Eth Opt Pwd] RWPO > [Yes] Yes

3

Click Save and Connect.

Login the ATV Distributed PAC over DPWS scanner (https-IPv6) discovery

Step

Action

1

Click Set credentials icon

2

Type the credentials

  • Case 1: The security is not Configured (New user credentials).

    • For Device User Name, type installer.

    • For Device password, type Inst@ller1.

  • Case 2: The security is configured: use EcoStruxure Automation Expert User Management credentials.

    NOTE: If the security is configured but you don’t know EcoStruxure Automation Expert User Management credentials, the User Management credentials can be reset, refer to the chapter Local reset of the ATV Distributed PAC security for more information.
NOTE: For firmware version 21.1 (3.1IE04_B03) or earlier, use EcoStruxure Automation Device Maintenance webserver credentials, available in the Graphic Display Terminal (VW3A1111) if the security is enabled, or Anonymous login if the security is disabled

3

Click Save and Connect.

Start the firmware update

NOTE: Whether you selected the firmware package from DEVICE / LOADING tab or EXTENSIONS tab, follow these instructions:

  1. Click Deploy icon from the DEVICES tab.

  2. Select Firmware.

  3. In the Select a Category step, select Next.

    The following figure shows the steps 1, 2 and 3:

  4. In the Prerequisite step, click Next.

  5. In the Review the Selection step, Click Next if you want to perform the firmware update from EcoStruxure Automation Device Maintenance tool.

  6. In the Confirmation step, read the safety messages then click Confirm.

When the firmware is finalized

  1. Press OK on the Graphic Display Terminal (VW3A1111).

  2. Restart the drive.

Result:

When the firmware update is complete, the current firmware version is updated and the update info shows the icon indicating that the firmware update was successful.
