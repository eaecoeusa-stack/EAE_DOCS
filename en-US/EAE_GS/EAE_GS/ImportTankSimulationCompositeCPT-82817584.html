Importing the Tank Simulation Composite

In this topic, as mentioned in the topic Finalize the System Editor, a composite named TankSim will be added to the solution.

This composite is used to mimic the standard reactions of the Filling sequence’s instruments. Thanks to this composite, artificial feedbacks will be given to the sequence, as if the instruments were really reacting to the commands sent.

NOTE:

Hence, there is no need to force values and trigger events of the instruments as done in the test of the part The Filling Sequence CAT.

As described in the topic Import Content, import the composite named TankSim. Its file is located in the Windows folder C:\Program Files\Schneider Electric\EcoStruxure Automation Expert - Buildtime 25.0\GettingStartedProject\Composites and its name is GettingStarted_TankSim.Composite.

The composite TankSim should then appear in the Composite node of the Solution Explorer pad.

Now, add this composite to the TR1 layer of the Whole_Milk_Tank application and name it TR1Sim.
