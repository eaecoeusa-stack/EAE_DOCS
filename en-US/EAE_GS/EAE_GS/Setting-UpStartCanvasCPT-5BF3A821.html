Setting Up the Start Canvas

The ‘Start Canvas’ is the default section that appears on the top of each canvas, once online.

To access this default section, right-click on the resolution node containing the two edited canvases and select Open Start Canvas.

 
In this section, a name button will be added. It will display the name of the instruments by clicking on it. The button is named ButtonTagname and is available in the library SE.AppBase.
 

To add ButtonTagname:

  1. Drag and drop it into the section.

  2. Save and close the canvases tabs.

The canvas development is completed.
