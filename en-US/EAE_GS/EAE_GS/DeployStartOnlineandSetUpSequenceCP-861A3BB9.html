Deploying, Starting Online and Setting up Sequence

Like in the previous part, before to be able to run the sequence online few steps need to be achieved.

Deploying the Project

In the Deploy and Diagnostic editor, compile, deploy and start running the sequence by achieving the following actions:

  1. Compile

  2. Runtime Simulation > Start

  3. Login

  4. Deploy > Clean

  5. Deploy > Deploy

  6. Device Actions > Run

After having done these actions, the project and all its elements are deployed and running online. The project is now ready to be started online.

Starting Online the Canvas

Open the canvas online by selecting on the resolution Start Local HMI.

You then need the credentials set during the Creating a User Account topic.

NOTE:

If you don’t know what are the credentials, you can reset them on the User Management tool as shown in the topic Running the Application.

After having done these actions, the canvas is started online.

Setting-Up the Sequence

With the canvas opened online, few actions need to be done to get the sequence ready.

  • Check the alarms. Two alarms are currently active: TR1LSL01 and TR1LT01.low.

    It is due to the current tank level. As it is at 0%, it is below the minimal value triggering the digital input and below the low alarm level of the analog input set in the Offline Parametrization.

    Acknowledge them.

  • Check the Filling sequence overview.

    The command buttons of the sequence overview are currently disabled because the hand is not given to the operator. To be able to command the sequence, the ownership of the sequence has to be given to the operator.

    Open the face plate of the sequence by clicking on Start Filling sHead and change the sequence owner from Program to Operator.

After having done these actions, the active alarms are known and the command buttons of the Filling sequence are enabled. The sequence is ready to be started.
