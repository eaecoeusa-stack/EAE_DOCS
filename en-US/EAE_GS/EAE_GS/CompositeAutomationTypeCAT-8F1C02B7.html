Composite Automation Type (CAT)

EcoStruxure Automation Expert - Buildtime implements application logic (control intelligence) and includes the visualization of the application on the HMI/SCADA level. These two perspectives are unified within the CAT.

From control point of view A CAT object can be seen as a composite function block containing itself basic, service, or composite function blocks.
From HMI point of view A CAT contains various symbols that can be used in visualization images (canvases) as well as detailed images (faceplates). Those symbols make easy the design of a detailed view and enable parameterizing of the control logic.
A CAT contains additionally one service function block used to gather control part and HMI.
