Creamery Application

The tutorial application that is presented here after is based on a typical creamery application.

The full factory process is presented but only the reception tank number 1 (TR1) and its sequence, is detailed in this Getting Started document.

NOTE:

Even if the Getting Started document doesn’t cover the whole application, it will introduce and detail useful features and libraries to get started with in EcoStruxure Automation Expert.

The application tends to be practical by simulating a realistic creamery and by introducing:

  • The IEC61499 standard

  • EcoStruxure Automation Expert features

  • Libraries such as SE.AppCommonProcess and SE.AppSequence

  • Canvas creation
