Opening the Second Valve

As previously described in the topic Purpose of the Connections, the digital input TR1LSH01:

  • First, receives manual orders from the operator.

  • Then, this command is transmitted to the valve TR1V02.

NOTE:

This action is the same as Opening the First Valve: TR1LSH01 receives commands from the operator through the HMI, transmits them to a Transmission composite.

As a second Transmission composite will be used to converts the Boolean command of TR1LSH01 into an adapter readable by TR1V02.

There are two methods to add a second Transmission composite into a layer using the one already existing:

  • Method 1

    Select the Transmission function block and copy/paste it (CTRL+ C then CTRL + V).

  • Method 2

    Select the Transmission function block, drag it, hold down CTRL and drop it somewhere else.

NOTE:

These methods will reuse the name of the existing object and increment it.

To achieve this second connection, proceed the following steps using one of the two previous methods:

  1. Add a Transmission composite and name it TransVal2.

  2. Link TR1LSH01.CNF to TransVal2.REQ and TR1LSH01.Pv to TransVal2.Value.

  3. Link TransVal2.ISc to TR1V02.ISc.
