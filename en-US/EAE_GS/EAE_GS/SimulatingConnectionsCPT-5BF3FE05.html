Simulating Connections

Before simulating the connection, the ownership of all instruments has to be set back to Program.

NOTE:

The connections won’t be able to write commands to the instruments if their ownership is owned by the operator.

However, as described in the topic Purpose of the Connections, some instruments are turned On manually, by the operator. These instruments are:

  • TR1LSL01

  • TR1LSH01

  • TR1M01

  • TR1LT01

When it is possible, the ownership should remain to the operator. For the digital input, the value has to be overridden in the setting faceplate.

 

As developed in the topics Opening the First Valve and Opening the Second Valve, when the Override Values of TR1LSL01 and TR1LSH01 are set to On, the open command is sent to the valves TR1V01 and TR1V02.

In the event that the simulation is already activated, the state will directly turn to Open. Otherwise, the status will be set to Opening and will be waiting for the simulation to be activated.

 
NOTE:

The digital inputs are activating an alarm when turning On because their original purpose is to alert when turning On, not to turn On and Off instruments. Again, this example doesn’t claim to be realistic. The ‘Whole Milk Factory’ development will be.

As developed in the topic Opening the Third Valve, TR1V01 and TR1V02 need to be opened, and TR1M01 On for the simulation to open the third valve.

About the alarms ‘High’ and ‘High High’

The alarms ‘High’ and ‘High High’ are currently active.

NOTE:

They are active when the process value is higher than them.

In our case:

  • the process value has been set to 50

  • the ‘High’ and ‘High High’ are set to 0.

There are two methods to manually change these ‘High’ and ‘High High’ values:

  • Method 1

    Grab the arrows of these alarms and raise them up.

     

  • Method 2

    Set the setpoint of the ‘High’ and ‘High High’ alarms in their faceplate.

    NOTE:

    As developed in the topic Turning ‘ON’ the Digital Output, the last digital output is commanded On when TR1V03 is Open and when TR1LT01’s process value is higher than or equal to the constant 82.5.

Then try the following configurations:

  1. Open TR1V03 and set the process value to 82.

  2. Set the process value to 82,5.

  3. Set the process value to 85.

  4. Turn Off TR1LSL01.

You can now play with the instruments and try by yourself various other configurations or even try to develop other connections between function blocks.
