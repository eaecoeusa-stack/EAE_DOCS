Creating the Alarm Canvas

In this canvas, all the alarms generated by the instruments will be handled.

  1. Create a canvas and name it Alarms.

     

  2. Select the Alarm Grid tool (inside the editing menu of this new canvas).

  3. Add an alarm grid panel in the canvas:

    • First, zoom out with the mouse wheel so all the canvas be visible, then click on the left corner and keep pressing.

    • Move the mouse until the alarm grid rectangle covers most of the canvas, then release pressing.

     
The alarm grid panel is added into this canvas.
