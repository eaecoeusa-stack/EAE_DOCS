Adding a Function Block

There are four different methods to add a function block.

  • Method 1: Searching in the Solution Explorer pad tree view

    From the tree view of the Solution Explorer pad:

    1. Expand all the nodes and sub-nodes of a library, if the function block is external, or of the project, if it is internal.

    2. Expand the nodes until finding the function block needed and select it.

    3. Then, drag-and-drop it into the layer.

     

    If the function block is a CAT:

    1. The Define new name dialog box opens.

    2. Type the instance name and keep empty the Asset Tag name of the function block.

     
    NOTE:

    The instance name is the designation of the function block. It has to be unique. However, the CAT name, here MotorCyc, is shared with all the function blocks from the same CAT.

    For more simplicity, the term ‘instance name’ will be referred as ‘name’ in the rest of the document.

    The Asset Tag name is the application name given optionally to CATs.

  • Method 2: Searching in the Solution Explorer pad search bar

    From the Solution Explorer pad:

    1. Type the CAT name of the function block in the search bar.

      While typing, it will show all the function blocks whose name starts alike.

    2. Once found, select the function block and drag-and-drop it into the layer.

       

    3. If the function block is a CAT, the Define new name dialog box opens. Fill it as defined in the first method.

    The search engine supports wildcards. For instance, try to add the character * at the beginning of your search, like shown in the following image.

  • Method 3: Searching in the right-click list

    1. Right-click anywhere in the layer to display the right-click list.

       

    2. Select between the various categories available, the one of the function blocks until arriving to it.

    3. Then, select it to add it to the layer.

    NOTE:

    Using this function block adding method, the Define new name dialog box doesn’t appear. So, the name must be changed afterwards.

  • Method 4: Searching with shortcut

    1. Select the layer.

    2. Press simultaneously CTRL + W.

       

    3. Then, type the CAT name of the function block needed.

      A result list with all the function blocks matching the search appears.

    4. Then, either double-click on the right one or press Enter after selecting it.

      It will display the breadcrumb of the function block.

    5. Press Enter to validate.

      NOTE:

      This way, the Define new name dialog box doesn’t appear. So, the name must be changed afterwards.

NOTE:

With both methods 2 and 4, you should already know the name of the function block to find it. Whereas with both methods 1 and 3, it is the path to find the function block that must be known.

In addition, with both methods 1 and 2, you can specify directly the instance name and the Asset Tag name (even if not used in the Getting Started) while creating a CAT. Whereas with both methods 3 and 4, you need to manually define the instance name and the Asset Tag name after the creation. Anyhow, both instance name and Asset Tag name can be renamed afterwards.

Those four methods end up with the same result. So, choose the method which you are the most comfortable with.
