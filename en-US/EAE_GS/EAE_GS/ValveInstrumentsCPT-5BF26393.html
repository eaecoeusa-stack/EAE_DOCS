Valve

The three valves used in the TR1 units (TR1V01, TR1V02 and TR1V03) are standard.

These three valves:

  • Are controlled by a digital signal.

  • Have only two positions available: fully open or fully closed.

Suitable function block: Valve (SE.AppCommonProcess library)

Now, using one of the four methods to add a function block:

  1. Add three function blocks Valve into the layer TR1.

  2. Name them TR1V01, TR1V02 and TR1V03.

NOTE:

To get further details about the function block Valve select the function block and press F1.
