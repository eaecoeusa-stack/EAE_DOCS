Referencing Libraries

Now, to be able to correctly develop the solution, specific libraries need to be referenced by the project.

To reference a library, click on the library and select Project Library References.

 
The Project Library Reference dialog box appears.
 

The Project Library References window is divided into 2 areas:

1 | List of libraries already referenced to the solution.

2 | List of all libraries installed in EcoStruxure Automation Expert.

To reference a new library in the project, proceed the following steps:

  1. Find the wanted library in the list of installed libraries.

  2. Tick the checkbox of the library to select its latest library version.

    This action adds the library in the list of referenced libraries.

  3. Add this way the following libraries:

    • SE.App2Base

    • SE.App2CommonProcess

    • SE.AppSequence

    NOTE:

    These three libraries are added with the latest library version available on your Engineering workstation. As consequences, for the rest of the document, do not pay attention to the library version indicated in the graphics.

  4. Once the three libraries have been added, click on Apply to validate the referencing. It will reference the three libraries into the project and the solution (see image below).
