Starting Online HMI

First step of the testing is to Start online the HMI so it will be possible to interact with the instruments and see their connections working.

  1. Select the Start Local HMI option from the resolution containing all the canvases.

     
    The HMI is building and the Login dialog box appears.
     

  2. Type the username and the related password defined in either topic: Creating a User Account or Running the Application.

    Then validate.

    The HMI is now open online and ready to be used.
    A settings of EcoStruxure Automation Expert allows to remove the login authentication so that opens right away the HMI online.

    This parameter is only available when running in simulation. The authentication is mandatory when not in simulation anymore.

  3. Untick the option named Security Active of the Simulation Runtimes page to remove the authentication for simulation HMI.

Once opened, the online HMI looks like this:

1 | All the simulated instruments.

2 | The buttons related to the two canvases. Click on them to switch from a canvas to another.

3 | Tagname button. Click on it to make the names of the instruments appear or disappear.

4 | The flashing red rectangle around instruments indicates that an alarm is active. The three valves have an active alarm.

NOTE:

The alarms are flashing because they haven’t been acknowledged yet. Go to the ‘Alarms canvas’ to access to the ‘Alarm control panel’.

NOTE:

The control panel of this canvas gathers all the alarms that haven’t been acknowledged yet or that are still active.

On the control panel, all the alarms that are new or still unacknowledged are noticeable with the following characteristics:

  • Flashing red cell in the SC column.

  • Exclamation marks !!! in the Ack column.

NOTE:

Currently, the alarms of the three valves are unacknowledged. So they all have these characteristics.

To acknowledge an alarm:

  1. Click on the three exclamation marks (!!!) of the alarm line.

    Once acknowledged, there are two options for the alarm:

    • Option 1: The alarm is not active anymore.

      So, the line of the alarm and the red rectangle around the instrument both disappear.

    • Option 2: The alarm is still active.

      So, the three exclamation marks (!!!) get replaced by a X and the SC cell stops flashing and the red rectangle stops flashing too.

      NOTE:

      The alarm line will remain that way as long as the alarm is active. As soon as it is not active anymore, then both the alarm line and the red rectangle disappear.

  2. Acknowledge the three alarms appearing in the alarm control panel.

    NOTE:

    The alarm lines don’t disappear. It means that the three alarms are still active.
