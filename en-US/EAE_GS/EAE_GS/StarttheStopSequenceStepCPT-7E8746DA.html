Starting the Stop Sequence Step

As seen with the Filling sequence, the first action to do is to add the SeqHead function block that will lead and control the sequence. Name it StopHead.

NOTE:

As defined in the topic Stop Filling Sequence, the sequence starts when the Filling sequence is getting the command to stop.

The status of a sequence is given by the output variables of its SeqHead function block. Hence, to know if the Filling sequence is commanded to stop, then the StartHead.SeqStopped has to be checked. When this variable turns TRUE, then the Stop sequence is permitted to start.

Link StartHead.SeqStopped to StopHead.PermStart.

To update the PermStart variable and permit the start, the REQ_PERM event has to be triggered. To do so, a E_PERMIT function block will be needed.

NOTE:

This function block lets pass the received events only if its input variable is TRUE.

To inform about the change of status, the event StartHead.STATE_IND will be checked. This event is triggered every time the state of the sequence changes. The E_PERMIT function block will then be used to filter the triggers of the STATE_IND event. The E_PERMIT will let pass the triggers only when StartHead.SeqStopped is TRUE, which means when the sequence transitions to stop.

  1. Add ans E_PERMIT function block and name it StartStopSeq.

  2. Link StartHead.STATE_IND to StartStopSeq.EI.

  3. Link StartHead.SeqStopped to StartStopSeq.PERMIT.

  4. Link StartStopSeq.EO to StopHead.REQ_PERM.

As programmed, the permissive command of the Stop sequence will turn TRUE when Filling sequence is commanded to stop.

A last step is needed: the start order of the sequence. Something has to trigger the START input event of the SeqHead to make it begin. In opposition to the Filling sequence that is started by the operator, this Stop sequence has to be started automatically. It has to start by itself when the Filling sequence is commanded to stop.

To do so, link StartHead.CMD_RESET to StopHead.START.

The SeqHead function block is now completely set. Cross reference the link between StartHead and the two function blocks StartStopSeq and StopHead.
