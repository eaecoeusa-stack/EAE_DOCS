Introduce how to engineer an application by developing from scratch a full application which contains: function blocks, CATs and canvases.

Most of the time, to realize an application on EcoStruxure Automation Expert, the engineering process is divided in 3 parts:

Engineering with ready-made objects All objects needed for the project development (motors, valves, sensors and so forth) are included into an existing library. In that case, it can be easily and quickly implemented in the solution.
Engineering of objects If a specific function block is missing, custom objects can be built for the solution. They can also be reused for future ones. This functionality provides maximum flexibility.
Load and start the runtimes Once the engineering process is completed, it is needed to load the project into the control unit and start the runtime for visualization (HMI).
