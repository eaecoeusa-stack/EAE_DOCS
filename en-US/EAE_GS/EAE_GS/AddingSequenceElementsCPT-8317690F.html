Adding Sequence Elements

The last step of this canvas modification is to add several elements related to the sequence and its steps.

The first element to add is the quick access of the sequence. This element is giving an overview of the sequence, its current status and it also gives access to the command of the sequence (start, stop, held, reset).

NOTE:

The overview of the sequence is in CAT Instances > Application > TankFillingSequence > StartHead. From the Select generic symbol dialog box appearing, select sQuickAccess.

Add it and position the element above the reset button and change its AliasName from Name to Filling.

The overview of the sequence is completely set up.

The other elements to add to the canvas are descriptions for each step of the Filling and Stop Filling sequences.

It will inform the user about the active step and for how long.

Add to the canvas the instance of each step, each SeqHead and each SeqTerminate.

NOTE:

For the sequence step instances, select sStepwithTransition in the Select generic symbol dialog box. For the SeqHead, select sHead.

Position one element below the other, with StartHead at the top and SeqTerminate at the bottom.

Do the same for the Stop Filling Sequence too.

The name and description for each of them (beside the SeqTerminate) have to be changed. Here are the path to find the parameters depending on their type:

SeqHead SeqStep
Name Symbol > Display Name Symbol > Step Name
Description

Doesn’t have description

Symbol > Step Comment

Modify the parameters seen above for each element based on the following table:

Element Name Description
StartHead Start Filling No Applicable
Step1 Set Owner Program Local
Step2 Agitator ON None
Step3 Valve 1 Open None
Step4 Valve 2 Open None
Step5 Valves 1 & 2 Close None
Step6 Agitator OFF None
Step7 Set Owner Program Remote
StopHead Stop Filling No Applicable
StopStep1 Valves 1 & 2 Close None
StopStep2 Agitator OFF None

Add two lines, one for each sequence, to link the SeqHead instance to the first SeqStep of its sequence.

The canvas process is now fully completed. It includes the instance of the instruments and of the sequences. The status and aspect of each instance will be constantly updated to allow the user to have a full and clear overview of the tank, its instruments and its filling process.
