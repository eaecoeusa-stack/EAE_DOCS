Creating the Main Canvas

To create a canvas:

  1. Go to the Solution Explorer pad in the Canvases node.

     

    Under this node, there is by default only the Without resolution node

    NOTE:

    All the canvases created under this resolution node will have this resolution.

    For the Getting Started, let’s add one resolution under which the canvases will be created:

    1. Right click on Canvases and select New resolution.

       

    2. In the Start Canvas dialog box opening, select the size, here 1280x980. Once done, click on Finish to create the resolution.

       

  2. Add it a canvas under the newly created resolution.

     
    The File Wizard dialog box appears.
     

  3. In this dialog box, change the name from Canvas1 to WholeMilkTank and press Finish to validate the creation.

    NOTE:

    Keep empty the Title and Tooltip. These items can be edited later in the Properties pad of the canvas.

     
  4. The main canvas has been created.
