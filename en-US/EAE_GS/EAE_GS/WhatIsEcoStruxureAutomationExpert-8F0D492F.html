What Is EcoStruxure Automation Expert?

EcoStruxure Automation Expert is an ecosystem for automation, conforming to IEC61499 standard. It offers a new automation paradigm based on object-oriented model and event-based mechanisms.This complete ecosystem mainly comprises:

  • EcoStruxure Automation Expert – Buildtime, the engineering tool to design, commission and maintain the application and related hardware and software components.

  • Distributed Programmable Automation Controllers (dPACs) to host the running application.

  • HMI to allow interaction with operator while application is running.

  • Archive engine for historization, trend and alarm management.

  • Interface to AVEVA System Platform (ASP) for supervising the application.

Asset centric EcoStruxure Automation Expert is asset centric. Indeed, real devices as well as whole applications (e.g., air conditioning systems) can be represented by software objects: the Composite Automation Types (CATs). CATs contain at the same time, control intelligence and visualization. They can be reused with low effort in any project.
Hardware independence EcoStruxure Automation Expert is hardware independent. As the control intelligence is designed at the application level, it is possible to run the same application using one or more dPACs. If new considerations appear along with the design, operating or maintenance phases, it is possible to change with low effort the dPACs structure of the application.
Event driven EcoStruxure Automation Expert is relying on event mechanism. Indeed, it is possible to execute part of the application on-demand, as new events in the system are occurring.
NOTE:

Highly solicitated parts of the application can be isolated and deserve a dedicated dPAC.

Distributed controlling EcoStruxure Automation Expert removes the consideration to get a fixed system structure. Communications that occur between dPAC are transparently managed.
NOTE:

The application can be deployed across several dPACs with low effort.
