EcoStruxure Automation Expert - Buildtime

Once the solution has been created, the EcoStruxure Automation Expert - Buildtime appears.

This is where all menus, editors, pads, tools and more from EcoStruxure Automation Expert are accessible.

When first open, EcoStruxure Automation Expert - Buildtime looks like the image below:

The workspace of EcoStruxure Automation Expert consists of one window containing the following elements:

1 | Menu bar

2 | Ribbon bar

3 | Various tool windows (Pads), docked on the side areas of the window, such as Solution Explorer, Log View and Symbolic Links.

4 | Editor in the middle of the window. Here Deploy and Diagnostic.

  • All pads can be hidden/shown and can be moved around, repositioned.

  • Additional windows are available and can be displayed by using one of the shortcuts of the View menu of the main tool bar.

  • From there, it is possible to select a specific layout (window and toolbox arrangements) or save the current layout.
