SeqTerminate Function Block

The SeqTerminate function block is positioned after the last SeqStep in the ISeqChain adapter series.

When the last SeqStep is completed, it then activates the SeqTerminate.

This one will inform the SeqHead that the sequence reached the terminate step so that the SeqHead can get the state completed.

 
NOTE:

To know more about this function block, refer to the SeqTerminate topic into SE.AppSequence library. To access it, search SeqTerminate in the Solution Explorer pad and press F1 to the result.
