Compiling, Deploying and Running the Application

At this stage, the application is fully developed, mapped and initialized on the resource.

Now, the device has to be compiled, deployed and put to run mode.

To achieve all these actions:

  1. Go to the Deploy and Diagnostic editor.

    NOTE:

    All the logical devices previously set in the topic Setting the Logical Devices. are visible. It is from this menu that they will be compiled, deployed and all the simulation actions.

  2. As the device is simulated and does not use physical hardware, switch the Active Network Profile setting to Local Test.

  3. With this editor, you can deploy and diagnose all the logical devices defined in the Logical Devices editor, as detailed in the topic Setting the Logical Devices.

    To simulate the Getting Started project, perform the following actions on the EcoRT_0 logical device with one of the methods detailed afterwards:

    1. Compile: To compile the application.

    2. Runtime Simulation > Start: To start the simulation, as this project is in Local Test.

    3. Login: To connect the project to the simulation.

    4. Deploy > Clean: To clean the device from any previous program.

    5. Deploy > Deploy: To deploy the application on the device.

    6. Device Actions > Run: Start running the simulation.

    NOTE:

    In the Getting Started, only a few actions are used. To find a list describing all the actions available with this editor, refer to the User Manual.

  • Method 1: Manually on one logical device

    Click on the Action drop-down list from the logical device line and select the wanted actions.

    The actions are done only to this specific logical device.

  • Method 2: Manually on several logical devices

    Tick one by one all the logical devices required or all of them with the general Logical Device Name checkbox. Then click on the Action button and select the wanted actions in the drop-down list.

    The actions are done to all the logical devices selected.

  • Method 3: Automatically on one logical device

    Click on the Deploy to Run Assistant button of EcoRT_0 logical device to automatically perform all the wanted actions.

    Login the device if not connected, then in the Deploy to Run Assistant dialog box appearing click Deploy.

     
    NOTE:

    By default the selected actions in the Deploy to Run Assistant dialog box are the same actions as the ones listed above.

NOTE:

If during the deployment an error about persistent data appears, do the following action: Persistent Data > Delete Persistent Data then deploy again.

This happens if the new parameter set is in conflict with the parameter that is currently loaded on the device.

The device is now online and running.
