Creating Solution

To develop an application on EcoStruxure Automation Expert, the first step is to create a solution that handles the software and hardware application of a project.

To create a solution:

  1. Select New in the File menu of EcoStruxure Automation Expert.

    The New Solution dialog box appears.
     

    This dialog box gives the choice to create a solution following one of these templates:

    Full solution template (Windows HMI) All the libraries available in this Engineering station will be referenced in the solution. Then, the user has access to the functionalities of all its libraries, even some that may not be used. This large amount of library may confuse the user who looks for a specific element. The HMI in this template is compatible with Windows OS.
    Material Handling (Multi-OS HMI) A few libraries will be referenced in the solution for the user to be able to handle material. The HMI in this template is compatible with multiple OSs.
    Starter Kit template (Windows HMI) Only the basic libraries will be referenced in the solution. The user will have to reference by hand let on its needed libraries. The HMI in this template is compatible with Windows OS.
    Starter Kit template (Multi-OS HMI) Only the basic libraries will be referenced in the solution. The user will have to reference by hand let on its needed libraries. The HMI in this template is compatible with multiple OSs.
    NOTE:

    Choose the Starter Kit (Windows HMI) template for this Getting Started application.

  2. Then, add the name GettingStarted and click on Create button.

     
    A Properties dialog box appears to fine tune the metadata of the solution.
     
    NOTE:

    All these metadata are optional to create the solution.

    Then either fill them or let them empty.

    Finally press Finish to create the solution.
