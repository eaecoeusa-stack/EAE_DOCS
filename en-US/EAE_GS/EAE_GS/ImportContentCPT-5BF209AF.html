Importing Content

In addition to importing the libraries, it is possible to import content.

While referrencing a library adds a group of content, potentially from different types (CAT, composite, graphics…) the import adds one specific content.

For the needs of the Getting Started, the imported content is:

GettingStarted_Transmission.Composite.

This content is a composite function block. As described in the topic Three Kinds of Function Blocks, it will realize certain actions once its events are triggered.It is working like a CAT, component explained in Composite Automation Type (CAT), except that the composites don’t have HMI representation..
NOTE:

This composite is used to transmit information between the tank TR1’s instruments. It is available in the following Windows folder C:\Program Files\Schneider Electric\EcoStruxure Automation Expert - Buildtime 25.0\GettingStartedProject\Composites. The composite will be used in the section Setting-Up the Instruments.

Add the imported content into the correct category of the project.

In this case, since it is a composite:

  1. Select the sub-node Composite then, Import.

     
    A Windows page opens for you to look for the composite.

  2. Search in the Windows folder containing the composite, then open it.

    The Import summary window opens. This window indicates what content will be added.

  3. Validate it by clicking on OK.

     
    The content is now added to the project.
