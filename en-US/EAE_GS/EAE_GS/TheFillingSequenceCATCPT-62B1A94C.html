The Filling Sequence CAT

Purpose of this section

Create the Filling sequence in a CAT that will be used to automatically fill the TR1 tank of the creamery.

Main Topics

  • Defining the Filling sequence.

  • Defining the SE.AppSequence library and its function blocks used in the sequence.

  • Creating a CAT.

  • Developing the Filling sequence in the CAT.

  • Running and testing the Filling sequence CAT.

To start up, you can either:

  • Proceed with the same EcoStruxure Automation Expert solution as used in the previous part.

  • Start this topic with the official archived solution.

NOTE:

If you choose to start with the official archived solution, you will be able to find the file in the following Window folder C:\Program Files\Schneider Electric\EcoStruxure Automation Expert - Buildtime 25.0\GettingStartedProject\Archive files. The file is named GettingStarted_BeginningFillingSequenceCAT.sln.

During the opening of the archived solution, you may get a message telling that specific version of libraries are missing. It is because a version referenced into the solution is not available on your device.

If the message appears, pick up the mentioned libraries and skip it. Then follow the next action for each library:

  1. Right click on the project in the Solution Explorer pad.

  2. Select Change Library Reference Version.

  3. In the Select installed Library for version change dialog box, select a library from the Reference Version change from: section

  4. Select in the Reference Version change to: section the latest version of this library then click OK.

  5. Compile the solution and notice that it is successfully built.

Doing so replace all the older libraries found by the newer libraries available in your solution.
