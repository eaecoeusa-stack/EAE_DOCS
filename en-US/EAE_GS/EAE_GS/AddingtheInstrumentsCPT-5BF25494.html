Adding the Instruments

This sub-section describes how to add all the function blocks of the TR1 instruments.

These function blocks are standard type and come from the SE.AppCommonProcess library.

They will be added into the layer TR1 of the Whole_Milk_Tank application.

NOTE:

To get exhaustive information about the instruments available in this library, select SE.AppCommonProcess in the Solution Explorer pad and pressing F1.
