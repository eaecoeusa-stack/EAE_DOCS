Set Up the Application

To set up the application, the step-by-step development need to be correctly organized.

So, to set up the application:

  1. Open the Applications and layers editor.

    The current application is named APP1.

    To bring more meaning, it will be renamed Whole_Milk_Tank.

  2. Rename the application, first by selecting the name and then by clicking a second time on it or by right clicking on the name and selecting rename.

    The name becomes editable.

     
    NOTE:

    Do not double-click to rename the application. Otherwise, instead of renaming, it will open the application.

    The application is now renamed.

  3. Expand the application to get access to its layers.

    As the Whole milk factory is divided into units, each of the unit will have its own layer, with all its instruments and sequences inside.

    NOTE:

    Since the Getting Started only focuses on the TR1 units, then only one layer will be added.

  4. By default there is one layer: Default. Click on the layer to select it and click a second time to make it editable or right click on it and select Rename (as for editing the application name).

    Then, name it TR1.

     
Double click on it to open the TR1 layer of the Whole_Milk_Tank application.

In the top part of the image is the breadcrumb indicating which layer of which application is currently shown.

It is then possible to easily access the other applications and the different layers either by:

  • Clicking into the breadcrumb and then typing the full path.

  • Clicking on one of the rightwards arrow. By doing so, it will show all the other elements available at this level. For instance if you click on the rightwards arrow on the right of Whole_Milk_Tank it will show all the layer below Whole_Milk_Tank available, here only TR1. The same with the one on the right to Applications, it will show all the applications available.
