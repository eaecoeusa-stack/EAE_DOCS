EcoStruxure Automation Expert - Archive Commissioning

Overview

To commission the EcoStruxure Automation Expert - Archive, the following configuration and deployment steps need to be completed:

  • Create a new database instance in PostgreSQL.

  • Deploy the archive link.

  • Deploy the HMI device configuration.

  • Deploy the dPAC device configuration to EcoStruxure Automation Expert - Soft dPAC.

  • Compile and deploy the application to EcoStruxure Automation Expert - Soft dPAC.

  • Optionally, configure the disk space management setting to backup the data before delete in pgAdmin 4, using the pgAgent jobs.

To learn more about how to create and use an archive database, watch: .

  • How to Create an Archive Database in EcoStruxure Automation Expert, and

  • How to Use an Archive Database in EcoStruxure Automation Expert.

After creating the archive database, deploying the archive link and dPAC device configuration, you can test the variable and alarm archiving on your local computer.
