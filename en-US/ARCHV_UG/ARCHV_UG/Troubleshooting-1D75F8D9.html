Troubleshooting

Error while reinstalling Archive (Ubuntu/Debian)

If you encounter any of the following issues while installing EcoStruxure Automation Expert - Archive (Ubuntu and Debian):

  1. "Configuring eae-archive" with following message appears that cannot be closed: "Password requirements are not met, please re-enter the new password."

  2. The Terminal window shows following error: dpkg: error processing package eae-archive (--configure): installed eae-archive package post-installation script subprocess returned error exit status 30

Execute the following steps in Terminal window to resolve the issue:

  1. Uninstall EAE-Archive package: sudo apt-get --purge remove eae-archive

  2. Clear Debconf storage: echo PURGE | sudo debconf-communicate eae-archive

  3. To fix "dpkg was interrupted, ... ... " message: sudo dpkg --configure -a

  4. Reinstall the required EcoStruxure Automation Expert - Archive (Ubuntu and Debian) package: sudo apt install ./<EAE Debian Package Name>
