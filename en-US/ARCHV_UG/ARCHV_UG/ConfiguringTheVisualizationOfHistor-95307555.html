Configuring the Visualization of Historized Alarms

  1. In your solution logic, configure an ALARM_BIT function block to generate alarm events. Refer to Graphic Editors for EcoStruxure Automation Expert HMI in the EcoStruxure Automation Expert User Guide for information how to configure a function block for alarm events.
  2. In the Solution Explorer pad, select Solution > Canvases, then double-click the canvas that you want to use to view archived alarms.
    Result: The canvas opens for editing.
  3. In the Designer editor, add an Alarm Grid control ( ) to the canvas, then save the canvas.
  4. Select the Alarm Grid control in the canvas to display its properties.
  5. In the Properties pad, under the Misc property area, set the AlarmType property to Archive.
  6. Compile and deploy the HMI.
    NOTE: For testing, right-click the resolution and click Start Local HMI.
    Result: The configured EcoStruxure Automation Expert - HMI service is updated and the updated canvas can be displayed. For more information on the HMI, refer to EcoStruxure Automation Expert - HMI in the EcoStruxure Automation Expert User Manual.
  7. In the running canvas, right click the Alarm Grid control.
    Result: A menu is displayed where different configuration options can be selected.
  8. Click Filter Alarms, then in the Alarm Filter dialog, select the path to the alarm archive to add it to the Alarm Grid control.
    Result: Archived alarms are displayed in table format.
