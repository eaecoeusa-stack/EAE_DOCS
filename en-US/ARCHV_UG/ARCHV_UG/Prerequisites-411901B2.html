Prerequisites

The following prerequisites must be satisfied before installing and using the EcoStruxure Automation Expert - Archive (Windows):

  • PostgreSQL V13 or later for Windows x86-64 must be installed.

    NOTE: For cybersecurity reasons, it is recommended to install the latest PostgreSQL version supported by EcoStruxure Automation Expert - Archive.

  • If you still use older version of PostgreSQL for your Archive database and you want to keep your recorded data, read the instructions on how to upgrade it under the section EcoStruxure Automation Expert - Archive Database Migration.

    NOTE: PostgreSQL, pgAdmin 4 and pgAgent are not installed with the installation of EcoStruxure Automation Expert and must be installed manually. Instructions on how to install these components are provided under the sections PostgreSQL Installation, pgAdmin 4 Configuration, pgAgent Installation, pgAgent Configuration.

    Administrator rights are required on the PC where PostgreSQL, EcoStruxure Automation Expert - Archive, pgAdmin 4 and pgAgent are installed.

 CAUTION
Failure to follow these instructions can result in injury or equipment damage.

Turning Off Hibernation Mode

To support continuous archiving, turn off hibernation (or sleep) mode for the PC hosting EcoStruxure Automation Expert - Archive (Windows). To do this, configure the Microsoft Windows Power Options for the PC hosting EcoStruxure Automation Expert - Archive as follows:

  • Hard disk > Turn off hard disk after > On battery: Never

  • Hard disk > Turn off hard disk after > Plugged in : Never

  • Sleep > Sleep after > On battery : Never

  • Sleep > Sleep after > Plugged in : Never

  • Sleep > Allow hybrid sleep > On battery : Off

  • Sleep > Allow hybrid sleep > Plugged in : Off

  • Sleep > Hibernate after > On battery : Never

  • Sleep > Hibernate after > Plugged in : Never

NOTE: If the host PC enters hibernation mode, or if its hard disk turns off, you must restart the host PC to restart archiving.
 WARNING
INTERRUPTION OF SERVICES
Always deactivate power options that could interrupt services execution on Microsoft Windows PC when running EcoStruxure Automation Expert - Archive (Microsoft Windows) or EcoStruxure Automation Expert - Soft dPAC (Microsoft Windows).
Failure to follow these instructions can result in death, serious injury, or equipment damage.
