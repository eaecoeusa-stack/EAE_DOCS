Reset Forgotten Password for Super User (postgres)

PostgreSQL uses the pg_hba.conf configuration file stored in the database data directory to control the client authentication.


To reset the password for the postgres user, you need to follow these steps:


Windows:

  1. Backup the pg_hba.conf file by copying it to a different location.
    Path: C:\Program Files\PostgreSQL\XX\data\pg_hba.conf

  2. Edit the pg_hba.conf file and change all local connections from scram-sha-256 to trust. By doing this, you can log in to the PostgreSQL database server without using a password.
  3. Restart the PostgreSQL server.
    PostgreSQL can be restarted from Services. Alternatively, execute the following command from the Command Prompt as the Administrator.
    pg_ctl -D "C:\Program Files\PostgreSQL\16\data" restart
  4. Connect to PostgreSQL database server using any tool such as psql or pgAdmin, PostgreSQL will not require a password to log in.
    psql -U postgres
  5. Execute the following command to set a new password for the postgres user.
    postgres=# ALTER USER postgres WITH PASSWORD 'new_password'
  6. Restore the pg_hba.conf file, restart the PostgreSQL database server, and connect to the PostgreSQL database server with the new password.

Linux:

  1. Backup the pg_hba.conf file by copying it to a different location.
    Path : /etc/postgresql/14/main/pg_hba.conf

  2. Edit the pg_hba.conf file and add this entry at the end of the file.
    # Allow local connections (Unix socket)
    local all all trust
  3. Restart the PostgreSQL server.
    $ sudo systemctl restart postgresql
  4. Connect to PostgreSQL database server using any tool such as psql or pgAdmin, PostgreSQL will not require a password to log in.
    $ psql -U postgres
  5. Execute the following command to set a new password for the postgres user.
    postgres=# ALTER USER postgres WITH PASSWORD 'new_password'
  6. Restore the pg_hba.conf file, restart the PostgreSQL database server, and connect to the PostgreSQL database server with the new password
