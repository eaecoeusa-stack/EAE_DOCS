Defining Variables for Historization

  1. In the Solution Explorer pad, select Solution > Canvases, then double-click the canvas that you want to use to monitor archived variable values.
    Result: The canvas opens for editing.
  2. In the Designer editor, add a History Trend control () to the canvas, then save the canvas.
  3. Compile and deploy the HMI.
    NOTE: For testing, right-click the resolution and click Start Local HMI.
    Result: The configured EcoStruxure Automation Expert - HMI service is updated and the updated canvas can be displayed. For more information on the HMI, refer to EcoStruxure Automation Expert - HMI in the EcoStruxure Automation Expert User Manual.
  4. In the running canvas toolbar, click the Archive List On/Off icon of the History Trend control ().
    Result: The Add Paths to Trend dialog opens.
  5. In the Add Paths to Trend dialog, select the archived variable values to monitor and click Add Paths.
    Result: The selected variable values are added to the History Trend control.
  6. Click the left and right control arrows to monitor the value change of the archived values over time.
