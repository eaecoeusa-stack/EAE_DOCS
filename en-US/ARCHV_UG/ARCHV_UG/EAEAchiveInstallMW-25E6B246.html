EcoStruxure Automation Expert - Archive (Windows) Installation Instructions

Overview

PostgreSQL is not installed with EcoStruxure Automation Expert - Archive (Windows). A supported PostgreSQL version for Windows x86-64 must be installed and configured before the installation of EcoStruxure Automation Expert - Archive.

The tools pgAdmin 4 and pgAgent must also be installed. These components are required for the use of EcoStruxure Automation Expert - Archive and its databases.

The EcoStruxure Automation Expert - Archive (Windows) installation wizard lists all the supported PostgreSQL versions installed on the PC. When you select a PostgreSQL version and a database exists for the selected version, the data directory and the port number of this PostgreSQL version are displayed. If there is no existing searchive database found for the selected PostgreSQL version, a new Archive database named searchive, and users named searchive and seservice are created. The searchive database is the template for the Archive database, created in EcoStruxure Automation Expert for your solution.

For any modification to the EcoStruxure Automation Expert - Archive feature installation (for example, select for installation or Remove), you need to launch the EcoStruxure Automation Expert - Runtime installer again.

The installation of EcoStruxure Automation Expert - Archive (Windows) requires to complete the following installation procedures:

  1. Install PostgreSQL and its components.

  2. Install EcoStruxure Automation Expert - Archive.

  3. Configure pgAdmin 4.

  4. Install pgAgent.

  5. Configure pgAgent.

  6. Configure pgAgent jobs.
