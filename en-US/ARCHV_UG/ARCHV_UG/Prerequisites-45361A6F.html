Prerequisites

The following prerequisites must be satisfied before installing the EcoStruxure Automation Expert - Archive (Ubuntu and Debian):

  • The PC where EcoStruxure Automation Expert - Archive (Ubuntu and Debian) and its components are installed must be connected to the internet.

PostgreSQL V14.0 is installed with the installation of EcoStruxure Automation Expert - Archive (Ubuntu and Debian). pgAdmin 4 and pgAgent must be installed and configured manually as part of the installation.

Sudo Privileges

In Ubuntu and Debian, the sudo command is used to execute commands with elevated privileges, typically as the root user or another specified user with administrative rights. This command is necessary to install new software on your PC.

To determine if your user account has sudo privileges, follow these steps:

  1. Open a terminal or command prompt on your PC running Ubuntu, or Debian.

  2. Type the following command: sudo -v.

  3. Press Enter.

    Result: If you have sudo privileges, you are prompted to enter your password.

  4. Type your password and press Enter.

Result:

  • If your password is accepted, and you have sudo privileges, no output or error message is displayed. This indicates that your user account has sudo privileges.

  • If you receive an error message or a message indicating that you are not allowed to use sudo, then your user account does not have sudo privileges. In this case, contact your system administrator to request the required privileges.
