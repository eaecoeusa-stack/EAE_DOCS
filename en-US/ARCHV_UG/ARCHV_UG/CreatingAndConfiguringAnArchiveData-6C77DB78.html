Configuring an Archive_Database Object

Create and configure an Archive_Database object. This Archive_Database object is used later to create the database for EcoStruxure Automation Expert - Archive in PostgreSQL.

  1. In System > Logical Devices, click Add Device.
  2. In the New Device dialog, enter a Name (for example, ArchiveDB1) and select the SE.DPAC:Archive_Database type, then click OK.
    Result: A new Archive_Database object is created.
  3. Set the Network Profile to (Default), then for the new Archive_Database object, select the runtime device Runtime_X under Physical Device.
  4. Click the new Archive_Database object to display its configurable properties in the Properties pad.
  5. Configure the following Archive_Database object properties:

    Property

    Description

    Misc
    Database type

    PostgreSQL (fixed).

    Login Retry-Time

    The time between login retries in milliseconds.

    Connection Timeout

    The connection timeout in milliseconds.

    Archivelink Device

    Name of the Archive_Link object that you created, used to establish a connection to the archive database.

    PostgreSQL
    User ID

    Use one of the default profile IDs that are pre-configured in PostgreSQL:

    • searchive

    • seservice

    Password

    Use the password of searchive or seservice user set during the installation of EcoStruxure Automation Expert - Archive.

    Database Name

    By default, the database name is the name of the EcoStruxure Automation Expert - Buildtime solution (for example, MyArchiveSolution1).
