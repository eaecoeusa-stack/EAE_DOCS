Defining Groups of Variables for Historization

Before performing these steps, click on Build in the Home menu to create the new CAT instances.
  1. In the Solution Explorer pad, select the project node, then right-click and select Configurations > History Trend.
    Result: The History Trend Configuration Manager opens.
  2. In the History Trend Configuration Manager, right-click the Configurations node and select New.
    Result: The History Trend Configuration dialog opens.
  3. In the History Trend Configuration dialog, in the Series field, click the ellipsis (...) to open the TrendSeries Collection Editor and click Add.
    Result: The Select Archive Path dialog opens.
  4. In the Select Archive Path dialog, select the path or paths to the variable values that you want to monitor and click OK, then return to the History Trend Configuration Manager, where the selected variable configuration is displayed, and click Close.
  5. In the Solution Explorer pad, select Solution > Canvases, then double-click the canvas that you want to use to monitor archived variable values.
    Result: The canvas opens for editing.
  6. In the Designer editor, add a History Trend control () to the canvas, then save the canvas.
  7. Compile and deploy the HMI.
    NOTE: For testing, right-click the resolution and click Start Local HMI.
    Result: The configured EcoStruxure Automation Expert - HMI service is updated and the updated canvas can be displayed. For more information on the HMI, refer to the HMI - User Guide.
  8. In the running canvas toolbar, click the Configuration icon of the History Trend control ().
    Result: The History Trend Configuration Manager opens.
  9. In the History Trend Configuration Manager, select a configuration and click OK to apply the selected configuration to the History Trend control.
    Result: The variables assigned to the selected configuration are added to the History Trend control.
  10. Click the left and right control arrows to view the value change of the archived values over time.
