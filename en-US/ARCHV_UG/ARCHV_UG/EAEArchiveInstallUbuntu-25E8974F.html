EcoStruxure Automation Expert - Archive (Ubuntu and Debian) Installation Instructions

Overview

The EcoStruxure Automation Expert - Archive installer is provided in the Archive subfolder of the EcoStruxure Automation Expert installation package.

When you install EcoStruxure Automation Expert - Archive (Ubuntu and Debian), PostgreSQL is installed as part of the installation.

pgAdmin 4 and pgAgent must also be installed. These components are required for the use of EcoStruxure Automation Expert - Archive and its databases.

The installation of EcoStruxure Automation Expert - Archive (Ubuntu and Debian) requires to complete in order the following installation procedures:

  1. Verify the Archive installation file (.deb).

  2. Install EcoStruxure Automation Expert - Archive and PostgreSQL.

  3. Install and configure pgAdmin 4.

  4. Install pgAgent.

  5. Configure pgAgent jobs.
