PostgreSQL Installation

EcoStruxure Automation Expert - Archive (Windows) supports PostgreSQL Windows x86-64 versions from V13.x to V17.x.

The PostgreSQL installer sets up a PostgreSQL Server which runs as a Windows service named postgresql-x64–XX–PostgreSQL Server XX (XX is the installed PostgreSQL major version). A database named postgres and a database superuser (postgres) are also created.

Multiple PostgreSQL major versions and their compatible minor versions can be installed on the same PC. Information on data directory, port configuration, available databases, users, roles and tablespaces of each major version of the installed PostgreSQL servers is available in pgAdmin 4.

  1. Download the latest supported major PostgreSQL version for Windows from https://www.postgresql.org/download/windows/.
  2. Run the downloaded PostgreSQL installer and follow the setup wizard.
  3. Select all components of PostgreSQL, including PostgreSQL Server, pgAdmin 4, Stack Builder and Command Line Tools, and click Next.
    NOTE: The PostgreSQL installer creates the databases in a default data directory. You can modify this directory during the installation.
  4. In the Password setup dialog, enter a password for the database superuser (postgres) and click Next.
    NOTE:

    • Use a password that follows the standard security requirements: 8–64 characters long, containing at least one uppercase, one lowercase and one of the following special characters — !$*%,&('/=@)^;#[<>.?+_-:|]. Other characters might cause issues during PostgreSQL installation. The EcoStruxure Automation Expert - Archive installation is showing an informational message when these password requirements are not followed. The superuser password can be changed then at any time within the pgAdmin 4 after the PostgreSQL Server installation.

    • Remember this password as it is required for later use, for example, during EcoStruxure Automation Expert - Archive installation, pgAdmin 4 configuration, or to access your Archive databases.

  5. In the Port setup dialog, enter a port number for the PostgreSQL Server of this PostgreSQL version, and continue the installation.
    NOTE: The default port number is 5432. If multiple PostgreSQL versions are installed on the PC, a unique port number is proposed by the PostgreSQL installation wizard, or an available unique port number must be set manually for each version. This port number is stored in the postgresql.conf file, located in the data directory of each installed PostgreSQL version.
    Result: PostgreSQL Server, pgAdmin 4, Stack Builder, and Command Line Tools are installed.
  6. In the Completing the PostgreSQL Setup Wizard dialog, clear the Stack Builder check box to prevent from launching its application after closing the setup wizard, then click Finish.
    Result: The Application Stack Builder shortcut is created in the PostgreSQL XX folder of the Windows Start menu. The PostgreSQL installation is completed.
    NOTE: In PostgreSQL XX, XX is the installed PostgreSQL major version.
  7. Verify that the PostgreSQL service is installed and running correctly before proceeding with the EcoStruxure Automation Expert - Archive installation.
