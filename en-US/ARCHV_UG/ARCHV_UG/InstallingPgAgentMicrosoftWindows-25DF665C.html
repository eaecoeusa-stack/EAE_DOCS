pgAgent Installation

An internet connection is required on the PC to download and install pgAgent.
  1. Open the Application Stack Builder from the Windows Start menu, select the previously installed PostgreSQL version and click Next.
    NOTE: The Stack Builder installer requires internet connection to download the pgAgent installation package. If there is no internet connection available on the PC, you can install the Stack Builder application from the PostgreSQL installer on a PC with internet connection, download the pgAgent package, then install it on the target PC.
  2. In the list of available applications, click the Add-on, tools and utilities category and select the pgAgent check box, then click Next.
  3. If required, modify the download directory of the pgAgent installer, or keep the default directory, then click Next.
  4. Select the Skip installation check box and click Next, then click Finish.
    Result: The pgAgent installer is available in the folder specified during the execution of the Application Stack Builder.
  5. Open the folder, containing the pgAgent installer (for example, C:\Users\Admin) and run the edb_pgagent_pgXX.exe file as administrator, then click Next to launch the installation.
    NOTE: XX in the filename represents the version number of the pgAgent installer for the related major PostgreSQL version.
  6. Click Next and follow the setup wizard.
  7. In the PostgreSQL Installation Details dialog, enter the password of the database superuser (postgres), set during the PostgreSQL installation, then click Next.
  8. In the pgAgent service account dialog, enter a password for the Windows (system) user and click Next, then follow the setup wizard.
    NOTE: The password must contain at least 15 characters, of which at least one special character, one number, one uppercase and one lowercase letter.
    NOTE: Modify the default Windows (system) username (postgres) to avoid confusion with database superuser.
  9. Click Next, then click Finish to complete the pgAgent installation.
