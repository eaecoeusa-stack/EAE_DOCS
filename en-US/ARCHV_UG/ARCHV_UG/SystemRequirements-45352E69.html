Microsoft Windows Configuration

Microsoft Windows operating system functions such as (non-exhaustive list) automatic update, power options or system notifications may interrupt operation of EcoStruxure Automation Expert - Archive. To avoid such interruptions, Microsoft Windows operating system may need additional configuration. Contact your local administrator before installing EcoStruxure Automation Expert - Archive .


Automatic Root Certificates Group Policy

The EcoStruxure Automation Expert - Archive installation might be unsuccessful and an Invalid digital signature error message might appear.

EcoStruxure Automation Expert - Archive might not function properly, and error messages might appear. To configure the automatic root certificates update:

1. From the Windows Start menu, open the Local Group Policy Editor.

2. Click the Local Computer Policy setting, then double-click Computer Configuration.

3. Double-click Administrative Templates > System > Internet Communication Management > Internet Communication settings > Turn off Automatic Root Certificates Update.

4. Click either Not Configured or Disabled, if the setting is enabled.

5. Click OK.



Required User Rights

Computer administration rights are required for the installation of EcoStruxure Automation Expert - Archive and the Archive Manager.
