Prerequisites

The following tasks must be completed in order to configure and use EcoStruxure Automation Expert - Archive successfully:

  • During application design:

    • Define variables to be monitored — single or as a preconfigured group.

    • Define alarms whose history to be recorded.

    • Configure the project to support archiving.

    • Configure the HMI to display trends and/or alarm history.

    • Test the configuration to make sure it works properly.

  • During commissioning:

    • Install EcoStruxure Automation Expert - Archive either on Windows, or, Ubuntu and Debian operating system, following the system requirements.

    • Configure the archiving strategy for disk space management.

    • Create the Archive database.

    • Deploy each dPAC and HMI device of the archiving configuration.

  • Optionally, during operation:

    • Monitor historized input variable trends.

    • View historized alarms.

  • For maintenance purposes:

    • Backup the stored data before delete.

    • Restore the backed up data.

    • Upgrade EcoStruxure Automation Expert - Archive.

    • Migrate the Archive database.

    • Uninstall EcoStruxure Automation Expert - Archive.

The following additional tools are required for the installation and use of EcoStruxure Automation Expert - Archive:

  • PostgreSQL.

  • pgAgent — a tool used to create scheduled jobs within pgAdmin 4, for example, regular database backups and cleanup.

  • pgAdmin 4 — a tool that displays for each major version of the installed PostgreSQL servers:

    • its data directory and port configuration

    • the available databases

    • users and roles

    • tablespaces

pgAdmin 4 and pgAgent are tools, used with PostgreSQL for managing and maintaining the EcoStruxure Automation Expert - Archive database.
NOTE: These tools are not included in the EcoStruxure Automation Expert - Archive installation and must be installed separately, following the instructions provided in this user guide.

The word Archive used in this user guide, refers to EcoStruxure Automation Expert - Archive.
