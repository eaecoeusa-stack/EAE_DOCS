Deploying the Archive Link (Windows)

When the Archive link is deployed, the data from the dPAC devices is collected and sent to the database. Then HMI clients can access the archived data in the PostgreSQL database.

To deploy the Archive link:

  1. Open the Deploy and Diagnostic editor.

  2. Click the Action menu of the Archive_Link object, and select Deploy > Deploy Archive Link Configuration.

.

To confirm that the Archive link deployed properly, and to manage and monitor the Archive link in the future, you can use Archive Manager 2x.x. This is a software tool, installed automatically when you install EcoStruxure Automation Expert - Runtime. To open it, click Start > EcoStruxure Automation Expert 2x.x > Archive Manager 2x.x.

The Archive Manager 2x.x interface consists of a Service Status tab and an Archive Links tab.

Use the Service Status tab to display the number of archive links configured and running, the service port number and the date the service started running. Because only one link is configured for the Archive_Link object, this tab confirms that the link is configured and running.

The Archive Links tab presents a list of the active Archive links. Select an Archive link and click the right mouse button to display a menu containing the following commands you can use to manage the selected Archive link:

Command

Description

Log In

Opens the Enter Credentials dialog where you can enter your management service username and password. When a valid username and password combination is entered, the remaining commands are enabled.

NOTE: The valid username and password combination are configured in the Management Service property of the Archive_Link object. By default, managementservice is used for both the username and password.

Start

Manually starts an Archive link. This command is impacted neither by enable nor disable status.

Stop

Manually stops an Archive link. This command is impacted neither by enable nor disable status.

Enable

Activates the automatic start and stop of an archive link depending on Archive service command. As enabled, it starts the archive link automatically when the archive service starts (for example, when Windows boots).

Disable

De-activates the automatic start and stop of an Archive link depending on Archive service command. As disabled, it does not impact the Archive link if the archive service starts or stops.

Show Service Log Directory

Displays the directory containing logs of the archiving service.

Show Archive Link Log Directory

Displays the directory containing logs of the selected Archive link.

Assign Licenses

Disabled and not used.

Delete

Removes the Archive link configuration files from the system where it is deployed.

Log Out

Ends the Archive link management session.
