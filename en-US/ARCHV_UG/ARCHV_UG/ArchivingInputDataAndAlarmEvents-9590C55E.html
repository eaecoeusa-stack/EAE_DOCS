Designing the Application for Archiving

Overview

To design your application for archiving, after the installation of all the required components, you need to perform the following configuration steps:

  • Add archive runtimes to your configuration.

  • Configure an Archive_Database object.

  • Configure an Archive_Link object.

  • Configure the dPAC device to enable both the archiving process and the Archive link.

  • Optionally, enable process variable archiving for the dPAC object.

  • Optionally, enable alarm archiving on the dPAC object.
