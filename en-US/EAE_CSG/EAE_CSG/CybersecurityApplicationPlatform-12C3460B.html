Cybersecurity Application Platform

Schneider Electric's Cybersecurity Application Platform (CAP) is a solution that unifies different independent cybersecurity components and services into one single platform. It allows the user to organize, control, and monitor the cybersecurity of an entire system.

CAP is recommended for medium to large companies that want to secure the entirety of the EcoStruxure Automation Expert system and beyond.

The set of CAP services are the minimum set that are recommended:

  • CAP core.

  • OT dashboard and SIEM Graylog.

  • Identity provider/AD integration.

  • Patch management.

  • Acronis CyberProtect: Backup and restore, and Endpoint protection.

  • Anomaly detection, IDS Nozomi.

  • Claroty Secure Remote Access.

NOTE: The CAP platform provides a Domain Controller image hosting an Active Directory Service with pre-configured group policies. Alternatively the CAP platform can be integrated with an existing Active directory.

The solution is scalable allowing the user to choose within the available cybersecurity components:

  • The core and essential services: User management, logging centralization, system management, secured remote access...

  • The advanced functionalities: End-point protection, backup and restore, intrusion detection, patch management...

CAP should be hosted in a dedicated server inside the DMZ network of the site.

This Cybersecurity - User guide then focuses on CAP to provide solution on essential and advanced cybersecurity components.

NOTE: For more information about available CAP features, refer to the Appendix topic Additional Documentation.
