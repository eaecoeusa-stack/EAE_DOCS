Interactive Logon Banner

Logon banners are used to:

  • Warn when users access the systems for illegal, malicious or in unauthorized manner.

  • Define what is considered acceptable use of the system for the authorized users.

Users must agree to the terms and conditions to successfully log in. Thus, it enables the organization to prosecute users that fail to comply these conditions.

For EcoStruxure Automation Expert, it is recommended to set up an interactive logon message for all engineering workstations. There are two ways to configure interactive logon messages:

  • Locally on every engineering workstation if Active Directory is not used on the plant network. This must be done on every engineering workstation and is not recommended for large plants.

  • On the Active Directory server by setting up group policies or user policies. This must be done on the Active Directory server hosted on the plant.

NOTE: To know how to:

  • Install and set up an Active Directory server, if not present yet on your system, refer to the CAP documentation.

  • Configure Active Directory and set up group policies, refer to Windows documentation.

  • Set custom logon banner, refer to Interactive Logon Message text - Windows 10 | Microsoft Learn

A logon banner can also be applied to any Edge I/O device. Refer to the following topics in the Modicon Edge I/O Configurator and Web Interface - User Guide: Security Banner Subpage and Security Settings Configuration with Cybersecurity Admin Expert Software.
