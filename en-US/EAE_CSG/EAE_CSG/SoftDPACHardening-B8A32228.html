Soft dPAC Hardening

End-users are responsible for applying security patches on their machines when using Windows and Linux operating system distributions. This especially applies for the EcoStruxure Automation Expert - Soft dPAC (Linux) due to the flexibility to use any Linux distribution that supports docker.

Linux Platform Hardening

  • The Standard Ubuntu Linux installation and Soft dPAC installation procedure install a system that is secure by default. Refer to CIS Ubuntu Linux 24.04 LTS Benchmark v1.0.0 for more information on how to securely configure the Linux host OS.

  • Perform operating system installation (OS) from an offline image. Make any updates to the OS either by a complete offline re-installation or an APT proxy - or APT via web proxy. Any installation or update done online (connected to Internet) must be subsequently reviewed/scanned/inspected to mitigate against the device being used as an attack vector on the system. Refer to the topic IEC 62443-3-3 Conformance for the recommended Soft dPAC and platform versions and their compatibility, or contact Schneider Electric support for guidance.

ASRock Industrial iEP-5010G iPC Hardening

Install the Linux image of the ASRock iEP-5000G iPC according to the Linux Platform Hardening guidelines.

Refer to Security Guidelines For ASRock Industrial iEP-5010G Series Version 1.0, August 2024, for further guidance.

Physical Security

  • Secure the ASRock iEP-5000G iPC in a room and cabinet that are physically locked to help prevent access to USB, Micro SD, and HDMI ports.
