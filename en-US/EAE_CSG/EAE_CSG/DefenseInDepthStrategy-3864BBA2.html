Defense in Depth Strategy

The Schneider Electric defense in depth strategy integrates a set of related process and systems to help provide higher levels of security.

The basic layers of Schneider Electric’s Defense in Depth approach are:

  • Risk Assessment: A systematic security analysis of the Control System environment and related systems.

  • Security Plan: Built on the results of the risk assessment.

  • Training: A multi-phase training campaign.

  • Network Separation and Segmentation: Separation of the industrial networks from enterprise networks using a demilitarized zone (DMZ) and the division of the safety network itself into segments and security zones.

  • System Access Control: Controlling network access to the system with firewalls, authentication, authorization, anti-virus and anti-spam software, VPNs, physical port locks, and traditional physical security measures.

  • Device, Workstation and Server Hardening: The process of configuring a device against communication based threats using measures such as password management, access control, disabling unnecessary interfaces, protocols, and services.

  • Network Monitoring and Maintenance: Continual monitoring and system maintenance to meet the challenge of new emerging threats.

NOTE: For more information, read the U.S. Department of Homeland Security document titled: Recommended Practice: Improving Industrial Control System Cybersecurity with Defense-in-Depth Strategies.
