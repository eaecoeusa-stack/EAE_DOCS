Solution Hardening

Within an EcoStruxure Automation Expert solution, services are disabled by default and should only be enabled where they are required within the application.

Thus, the devices are secured by default, the services must be explicitly enabled in the device or declared in the application for them to be opened.

Which means that you should review that only the services required by the application are enabled. To do so, check the properties of each devices in the Properties pad within the Physical Devices editor and the Logical Devices editor.

NOTE: Particular attention should be applied to the archive service, detailed in the Archive - User Guide. In an IEC62443-3-3 compliant system, this archive service must not be enabled.
