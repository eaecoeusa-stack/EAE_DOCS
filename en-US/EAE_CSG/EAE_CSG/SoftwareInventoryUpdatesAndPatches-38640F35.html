Compile Software Inventory

When installing a product from the EcoStruxure Automation Expert, it might be required to install additional software and components to succeed the product installation.

Please find for the following EcoStruxure Automation Expert products where to find their list of additional software and components installed:

  • For EcoStruxure Automation Expert - Buildtime:

    • In the Installation Instructions topic from the EcoStruxure Automation Expert - Release Notes: Find the list of every software installed with EcoStruxure Automation Expert - Buildtime.

    • With the Component Information command: Find all the components installed during the installation of the additional software mentioned in the EcoStruxure Automation Expert - Release Notes.

      NOTE: This command is available in the File menu, in the About dialog, as detailed in the About topic from the EcoStruxure Automation Expert - User Manual.
    NOTE: The EcoStruxure Automation Expert - Release Notes document is available without installation EcoStruxure Automation Expert - Buildtime while the Component Information command is not.

  • For EcoStruxure Automation Expert - HMI and EcoStruxure Automation Expert - Archive (Linux and Windows):

    Find in the Installation Instructions topic from the EcoStruxure Automation Expert - Release Notes.

    NOTE: EcoStruxure Automation Expert - HMI is not supported in a IEC62443-3-3 system as detailed in the Unsupported Configurations topic.

  • For EcoStruxure Automation Expert - Soft dPAC on Linux:

    Refer to the Installing Soft dPAC on Linux chapter from the Soft dPAC - User Guide to know the software and components to install to succeed Soft dPAC on Linux installation.

  • For EcoStruxure Automation Expert - Soft dPAC on Windows:

    Find in the Installation Instructions topic from the EcoStruxure Automation Expert - Release Notes.

    NOTE: EcoStruxure Automation Expert - Soft dPAC on Windows is not supported in a IEC62443-3-3 system as detailed in the Unsupported Configurations topic.

  • For EcoStruxure Automation Expert - High Availability Soft dPAC:

    Refer to the Installation chapter from the High Availability Soft dPAC - User Guide to know the software and components to install to succeed High Availability Soft dPAC installation.

Software Licenses

To deploy a system according to this guidance several licences in addition to the EcoStruxure Automation Expert, AVEVA System Platform, and Windows operating system licences are required, including:

  • Nozomi Guardian.

  • Acronis Cyber Protect Endpoint Protection

  • Acronis Cyber Protect Backup and Restore

  • Cyber Application Platform/CAP

Contact Schneider Electric support team for more information on how to obtain these required licences.
