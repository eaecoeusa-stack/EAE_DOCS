Workstation and Server Hardening

The Schneider Electric Cybersecurity Application Platform installation hardens the Server installation.

To provide defense in depth of the software integrity, it is advised to:

  • Enable UEFI

  • Enable Secure Boot

In addition, the following guidance for removable storage and remote sessions must be followed.

Removable storage

It is recommended that USB removable media access is prohibited. This can be enforced with either Acronis CyberProtect or via the domain group policy. Set disable access to all removable media within the domain group policy. This settings can be found in System > Removable Storage Access > All Removable Storage classesEnable Deny all access.

Remote Sessions

Remote sessions should be configured to timeout. The following settings are recommended, but can be adapted based on the application needs:

  • Active session limit: 8 hours.

  • Idle session limit: 1 hour.

  • Disconnected session limit: 1 hour.

These settings can be found in Windows Components > Remote Desktop Services > Remote Desktop Session Host > Session Time Limits.
