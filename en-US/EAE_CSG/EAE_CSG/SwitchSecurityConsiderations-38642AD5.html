Switch Security Considerations

To enhance the cybersecurity of your architecture, Schneider Electric recommends the use of switch Modicon Networking MCSESM083F23F0.

If you decide not to use this switch, several other switches, considered as alternative, have been certified as ISASecure® compliant components:

  • At SL1: Belden GRS106 L3

  • At SL2: Catalyst IE3100 Rugged Series / Stratix 5200, Catalyst IE3x00 / Stratix 5800 and Catalyst IE9300 Rugged Series

NOTE: Find the entire list of ISASecure® certified components here.

Security considerations on these ISASecure® SL2 compliant switches:

  • Only LDAP and RSTP should be enabled. All other services should be disabled.

  • These switches allow authentication through the LDAP service. Thus, authentication of user access to network switches for configuration is recommended to be via this LDAP service so that a common password policy can be enforced.

    If LDAP is enabled to allow authentication against Active Directory only users in the network administrators group should be mapped to users in the active directory configuration.

  • Consider the number of network connections concentrated on the switch. A large number of connections results in the switch becoming more critical, which may result in the switch having to be physically protected.
