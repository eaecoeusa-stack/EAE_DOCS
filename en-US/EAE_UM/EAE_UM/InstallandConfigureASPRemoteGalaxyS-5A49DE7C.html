0. Install and Configure ASP Remote Galaxy Setup

This chapter describes the steps to install and configure AVEVA System Platform (ASP) remote Galaxy repository.

This installation and configuration are needed when EcoStruxure Automation Expert and the Galaxy repository are installed in different machines:

  1. AVEVA System Platform Galaxy Repository in Server PC

  2. EcoStruxure Automation Expert in Client PC

Complete these few actions to install and configure ASP Remote Galaxy Setup:

  1. Install the ASP Galaxy Repository in the Server PC

  2. Install the Archestra remote development in client PC

  3. Configure DCOM

  4. Test Connection from the client PC with the remove Galaxy repository

  5. AVEVA AutoBuild

Once all these installation and configuration steps completed, it is then possible to follow the procedure described in the topic Asset Link for AVEVA OMI.
