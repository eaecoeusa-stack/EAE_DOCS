6. Adding the SE.EAEPortal Library and an AVEVA Server Device

Adding the SE.EAEPortal Library to the Solution

Before an AVEVA server device can be added to the solution, add the SE.EAEPortal to the Solution.

In the Solution Explorer pad, highlight the solution node, then right-click and select Project Library References to open the Project library references dialog. Then in the Installed Libraries tab, select SE.EAEPortal click Add then click Apply.

The SE.EAEPortal will be added to and available for use in the project library.

Adding an AVEVA Server Device

NOTE: Each solution can include only one (1) AVEVA server device.

To include an AVEVA server device:

Step

Action

1

In the Solution Explorer pad, open the solution, then right click on the System node and select Open to open the System editor.

2

Open the Solution Explorer navigate to and right click on [Solution Name] > System > Devices, then click New Device. The New Device dialog opens.

3

In the New Device dialog, in the Type list, select SE.EAEPortal.AVEVA_SERVER: Accept the default name, or change it. Then click OK.

NOTE: In the Count field, you can select only one (1). If you attempt to select more than one, a message displays indicating only one device will be added. 

4

Select the new AVEVA_SERVER device in the Logical Devices editor to display its properties. Configure the following properties:

  • Configuration > OPC UA Gateway > Server Node

  • Configuration > OPC UA Gateway > Gateway
