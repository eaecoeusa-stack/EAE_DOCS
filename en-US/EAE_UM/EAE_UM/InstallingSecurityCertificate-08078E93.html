Installing Security Certificate

There are two approaches for installing the security certificate:

  • Re-install EcoStruxure Automation Expert - Buildtime

  • Manually install the security certificate

Re-install EcoStruxure Automation Expert - Buildtime

If the security certificate was not properly installed or was subsequently removed, uninstalling EcoStruxure Automation Expert - Buildtime then re-installing it may install the security certificate.

Manually install the Security Certificate

Manually installing the security certificate consists of three tasks:

Task 1: Stop running services, as follows:

  1. In the Windows™ search field, located in the lower left par of the task bar, type Services.

  2. Right-click SE ASPSupervisoryFacetServices EcoStruxure Automation Expert 24.0 and select Stop.

  3. Repeat step 2 for SE AVEVAConnectorService EcoStruxure Automation Expert 24.0.

  4. Repeat step 2 for SE EngineeringHubService EcoStruxure Automation Expert 24.0.

Task 2: Install the certificates:

  1. In the Windows search field, type Command Prompt.

  2. Click on the Command Prompt app icon to open it.

  3. Execute the following six commands in the command prompt:

    certutil.exe -addstore -enterprise root "%programdata%\Schneider Electric\Portal Services\Certificates\CACertificate-2.cer

    certutil.exe -importpfx -f -p "" "%programdata%\Schneider Electric\Portal Services\Certificates\Certificate-v3-2.p12"

    netsh http delete sslcert ipport=0.0.0.0:44850

    netsh http add sslcert ipport=0.0.0.0:44850 certhash=bc6eb6350eeba8be078c288bbaf88dde48dd7bff appid={41a5e7a8-98cb-4949-aab7-d2be41f61cec}

    netsh http delete sslcert ipport=0.0.0.0:44840

    netsh http add sslcert ipport=0.0.0.0:44840 certhash=bc6eb6350eeba8be078c288bbaf88dde48dd7bff appid={ac2f4011-c6a6-4e01-a977-81d151df81a8}

Task 3: Start services that were stopped in Task 1:

  1. In the Windows™ search field, type Services.

  2. Right-click SE ASPSupervisoryFacetServices EcoStruxure Automation Expert 24.0 and select Start.

  3. Repeat step 2 for SE AVEVAConnectorService EcoStruxure Automation Expert 24.0.

  4. Repeat step 2 for SE EngineeringHubService EcoStruxure Automation Expert 24.0.

Follow-up

After implementing one of the above-described procedures, re-try the Asset Link functionality that previously did not succeed. If the condition persists, contact the Schneider Electric support team.
