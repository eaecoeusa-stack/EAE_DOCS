4. Modify Existing Connections and Adjust Attributes

Modify Existing Connections

  • Existing connections can also be modified via context menu after right-clicking the connection line

Context menu commands include:

Command

Description

Update Route

Redraws the connection route, e.g. to mend confusing lines after moving some FBs

Insert Point

Inserts a new point into the connection line, to make manually changes of the route, e.g. to realize a knee or a special route.

Remove Segment

Removes the selected segment and the corresponding turning points from the connection line

Cross Reference

Changes the existing connection into a cross reference, instead of the connection line a symbol on both interfaces of the FBs will mark the connection. A tooltip, by moving the mouse cursor over the symbol at the interface, shows which FB is connected to that interface. Useful for long, confusing connection lines.

Comment...

With this entry, a comment can be added, which will be shown within the tooltip of this connection line

Attributes...

Opens the Attribute Editor to modify attributes of the selected connection

  • If the right-click is executed on a special selection of connections some items in the context menu are not available.

  • Modify connection routing: Select a line and move it via the green spots or when the mouse pointer turns into a black double arrow with pressed mouse key. Further on the connection routing can be adjusted by the use of the arrow-keys when the relevant segment is selected. Alt+arrow-key leads to a selection of the respective next deviation point of the connection line. Ctrl+Arrow-key leads to pixelwise movement of the selected segment. Add new deflection points for the connection: see above.

  • Delete Connections:

    Step

    Description

    1

    Click on a connection to select it

    2

    Press Del-key

Adjust Attributes

Right-click on background or function block, and select Attributes... in the context menu to open the  Attributes dialog.
