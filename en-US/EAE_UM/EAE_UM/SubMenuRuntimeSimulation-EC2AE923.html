Runtime Simulation

NOTE: Runtime Simulation menu commands are enabled only when Active Network Profile is set to (Local Test).

The Action Menu Runtime Simulation item links to the following commands:

Command

Description

Start

Starts the simulation.

Stop

Stops the simulation.

When the simulation is started local addresses (127.0.0.1) are automatically used for dPAC simulated runtimes. But the configured addresses for other devices (for Modbus TCP, OPC UA...) are used as is and running the application may affect these physical devices also in Local Test (simulation) mode.

 WARNING
UNINTENDED EQUIPMENT OPERATION
When using the Local Test mode (simulation), make sure that the potential activation of communications with physical devices, addressed through the Hardware Configuration or anywhere in the application logic (i.e.: Modbus TCP, NETIO, OPC UA, etc...), will not cause unwanted effects to the operations.
Failure to follow these instructions can result in death, serious injury, or equipment damage.
