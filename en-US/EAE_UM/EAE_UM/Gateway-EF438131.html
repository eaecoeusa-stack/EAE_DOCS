PROFIBUS DP Master Gateway

Use the PROFIBUS gateway to connect sensors and actuators to the BusMaster function block within the PROFIBUS DP network.

Follow the first procedure below to configure the PROFIBUS gateway in EcoStruxure Automation Expert software.

Some elements will need to be configured with ProSoft PLX50 © software as detailed in the second procedure. Switch to this second procedure when indicated in the first procedure.

PROFIBUS Gateway Configuration in EcoStruxure Automation Expert Software

Step

Action

1

In EcoStruxure Automation Expert software, open the project in which you want to add a PROFIBUS gateway.

Add the following libraries to your solution:

  • Runtime.Base

  • SE.DPAC

  • SE.Standard

  • Standard.IoModbus

2

Open the System editor and select the Physical Devices editor. In the Automation Devices tree, select the Fieldbus Gateway > PLX51–PBM device and add it to your solution.

3

In the Physical Devices editor, connect the PLX51–PBM device to the relevant elements of your solution and configure its IP address.

4

Follow the procedure to configure the PROFIBUS gateway in ProSoft PLX50 © software.

5

Open the System editor and select the Hardware Configuration editor.

Click Add new master HW-CAT and select Standard.IoModbus.MODBUS.

6

Right-click on MODBUS device and select Add.

The Add To MODBUS window opens.

Double-click on Standard.IoModbus.MODBUSSLAVEND and Standard.IoModbus.MODBUSGENTCPS, then click Close.

7

Right-click on MODBUS device and select Properties.

In the Properties MODBUS window, set the Busid parameter and click OK.

8

Right-click on MODBUSGENTCPS and select Properties.

Set the Stationid parameter to 1 as defined for the Local Node Number in ProSoft PLX50 © configuration utility.

Set the IPaddress parameter identical to the one defined for the PLX51–PBM IP Address in ProSoft PLX50 © configuration utility.

Click OK.

9

Right-click on MODBUSGENTCPS and select Import Modbus Gateway.

The Import Options window opens. Select Suffix variable name with device name as import option and click OK.

Use the browser to locate the CSV file exported from ProSoft PLX50 © configuration utility and click Open.

A Confirmation window will ask you if you want to generate the associated symbolic links. Click Yes.

Result: the function blocks needed for the gateway operation have been imported to your solution.

10

Open the System editor and select the Function Blocks Network editor to display the function blocks.

Connect the function blocks by drawing a line between the free INITO node of your solution and the free INIT node of the gateway.

11

Open the Deploy and Diagnostic pad, expand the menu below Action and select Deploy > Deploy.

Expand again the menu below Action and select Device actions > Run.

Result: the PROFIBUS gateway is deployed.

PROFIBUS Gateway Configuration in ProSoft PLX50 © Software

Step

Action

1

Start ProSoft PLX50 © configuration utility.

2

Click on Create New Project to open the Project Explorer window.

Right-click on New Project and select Add.

3

In the Add New Device window, select the PLX51–PBM device and click OK. The Configuration window opens.

4

In the General tab:

  • Fill in the PLX51–PBM IP Address in the Identity part.

  • Select the StandaloneMaster communication mode in the Operation part.

  • Select the Modbus TCP Slave primary interface in the Operation part.

5

In the Modbus tab, set the Local Node Number to 1 in the Basic Settings part.

6

In the Modbus Addressing tab:

  • Set the Master Control Register address.

  • Set the Device Control Register address.

  • Check the Master Control and the Slave Device Enables boxes.

7

In the Profibus tab, set the BAUD Rate to 500 kbps. Validate your changes by clicking OK.

8

In the Project Explorer window, right-click on Profibus Devices and select Add Profibus Device.

The GSD File Selector window opens.

9

Click Add File and browse to your EDS configuration file.

Click Open to add it to the GSD File Selector.

NOTE: You can import an EDS file from any trustable source to configure Profibus devices in ProSoft PLX50 © software.

Select your imported configuration and click OK. The Configuration window opens.

10

In the Profibus Configuration tab, verify that the Node Address is set to 2.

11

In the Slot Configuration tab, click Add Module.

Choose your module types and configure their properties:

  • Description

  • Data Point

  • Data Type

  • Register Type

Right-click on a module line and select Assign Modbus from Here to configure the Modbus Offset parameter.

12

In the Project Explorer window, right-click on the PLX51–PBM device and select Go Online to connect the gateway.

The Configuration Action window opens. Click Download to download the configuration.

Result: a green light appears in front of the PLX51–PBM device in the Project Explorer window. It indicates that the gateway is in online mode.

13

In the Project Explorer window, right-click on the PLX51–PBM device and select Verify Configuration.

NOTE: Make sure that no detected error message appears in the Verification window. EcoStruxure Automation Expert will not import a configuration which contains detected errors.

14

Save your configuration.

In the Project Explorer window, right-click on the PLX51–PBM device and select Export Modbus Expanded CSV.

Select the folder for exportation and click Save.

Result: your configuration map has been exported as CSV file and is ready to use in EcoStruxure Automation Expert software.
