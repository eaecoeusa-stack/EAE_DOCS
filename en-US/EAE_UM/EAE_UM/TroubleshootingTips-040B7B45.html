Unable to launch Asset Link for Bulk Engineering : SE EngineeringHubService EcoStruxure Automation Expert v25.0 service is not reachable. Check if the service is running, then try again.

Launch Asset Link for Bulk Engineering

Bulk Engineering steps, such as the importing of engineering data, are supported by the following service:

  • SE EngineeringHubService EcoStruxure Automation Expert v25.0

This service was installed as part of the Asset Link for Bulk Engineering Application feature of EcoStruxure Automation Expert - Buildtime installer. However, if this service is not running, you will see this message in the Output pad.

Refer to Starting Asset Link for Bulk Engineering application services.

Unable to load projects : Service 'SE EngineeringHubService EcoStruxure Automation Expert 24.1' is not running.

Selecting the AVEVA Project list in the Configuring AVEVA Engineering project

AVEVA Project configuration step connects with the AIS Service to get the project details. Following services are required to be running:

  • SE EngineeringHubService EcoStruxure Automation Expert v25.0

This service was installed as part of the Asset Link for Bulk Engineering application of EcoStruxure Automation Expert - Buildtime installer. However, if this service is not running, you will see this message in the Output pad.

Refer to Starting Asset Link for Bulk Engineering application services.

Unable to validate user : The SE EngineeringHubService EcoStruxure Automation Expert v25.0 is not running.

Clicking on Save after entering Project and Project user details in the Asset Link for Bulk Engineering Service Server Configuration

AVEVA Project configuration step connects with the AIS Service to validate the user and access to the project. Following services are required to be running:

  • SE EngineeringHubService EcoStruxure Automation Expert v25.0

This service was installed as part of the Asset Link for Bulk Engineering application of EcoStruxure Automation Expert - Buildtime installer. However, if this service is not running, you will see this message in the Output pad.

Refer to Starting Asset Link for Bulk Engineering application services.

Starting Asset Link for Bulk Engineering Application Services

Many event messages can be resolved by starting Asset Link for Bulk Engineering application services, as follows:

Step

Action

1

In the Windows™ search field, located in the lower left part of the task bar, type “services”. Windows presents the list of services.

2

Right-click SE EngineeringHubService EcoStruxure Automation Expert 24.1 and select Restart.
