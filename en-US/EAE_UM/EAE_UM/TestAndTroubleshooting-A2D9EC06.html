Test and Troubleshooting

There are many ways to test and troubleshoot projects in EcoStruxure Automation Expert - Buildtime:

  1. Simulation Runtime

  2. Start Local HMI

  3. Watch and Force, Block Events, Trigger Events

  4. Test Container for Basic Function Blocks

  5. Test Container for CAT_HMI

  6. Watch Trend

  7. Log Files

Simulation Runtime

When using Local test network profile, EcoStruxure Automation Expert is providing a simulation runtime environment for the user in order to test the application logic without the need to deploy it on real hardware equipment. To know more about how to start the simulation runtime, refer to the HMI - User Guide topic Simulation Runtime.

Start Local HMI

Start Local HMI can be used to test whether created canvases operate and display correctly. The functionality is the same as it is with a real HMI. It has to be started using context menu entry Start Local HMI after a right-click on the relevant entry of node Canvases, containing the Canvases to be tested (e.g. 1280x980). For more information, refer to the HMI - User Guide topic Operating the EcoStruxure Automation Expert - HMI Client.

Watch, Force, Block Event, Trigger Event

The watching of data points (variables or event interfaces) serves to obtain detailed information about the operation of the application. That may be very informative by helping you to localize possible errors detected in the project.

When you force a value the actual value of a variable is intentionally overridden by a fixed value to watch the effects within the project.

NOTE: A warning messages appears when you unforce a value by clicking on the check mark next to Force Value.

The blocking of an event is used to suppress this event being triggered, what on his side, depending on the kind of function block, will also suppress data to be circulated or even suppress the execution of a complete project part.

Triggering an event can be used to test a single function block or some functionality in a subsequently part of the project.

The prerequisite for the use of watching, forcing, etc. is that the project is deployed on a device or Simulation Runtime (refer to to the HMI - User Guide topic Operating the EcoStruxure Automation Expert - HMI Client. Load the project to the device by means of the Deploy command in the Deploy and Diagnostic editor Action menu.

Further on it is necessary to be online with the device for having a connection established to the watch server. In order to do so you can use the Login command  in the Deploy and Diagnostic editor Action menu. If the attempt to go online has been successful an information such as: DEV[127.0.0.1:61497]: connection to watch server established will be displayed in the Output pad and column Login State within the Logical Devices editor will show Online.

Now you can watch data points via right-click on one or several selected function blocks within the Resource Editor of the relevant resource or within the Application Editor and a click on the command Add watch from the context menu.

Right-click on a function block interface (i.e., the label of the interface within the function block) to open a context menu to set only this single interface to Watch mode or to Force Value. Using an event input you can switch between Watch, Trigger Event or Block Event. Event outputs cannot be triggered, they can only be blocked.

Any currently watched or forced data points are listed in the Watch pad. Forced values are shown highlighted yellow.

Detailed information about how to work with watch points can be found at Watch.

Watch Trend

Watch Trend means the pad which can be opened via main menu bar View > Debug area > Watch Trend. It serves to show graphically the changes in value of individual variables selected from a list of watched in- and outputs (as described at point 3. on this page above) during runtime over a defined time range. For this purpose, it is necessary that the project is deployed to simulated or real devices and to be online, means logged into the device.

Information about this function can be found at Watch Trend pad.
