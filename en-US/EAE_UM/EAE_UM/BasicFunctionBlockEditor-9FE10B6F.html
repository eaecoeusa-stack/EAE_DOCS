Basic Function Block Editor

The Basic Function Block editor displays a schematic representation of program logic.

Functions and tasks include:

  • Interface editor

  • Create a new state

  • State context menu commands

  • State action (gray box) context menu commands

  • State action (light red box) context menu commands

  • Add actions

  • Algorithms

  • Test function

Interface Editor

Click the Interface icon from the vertical toolbar on the left to open the Interface editor. Use the Interface editor to create the interfaces of the Basic function block.

Create a New State

To add a new state to the ECC logic:

  1. Select the Execution Control Chart icon from the vertical menu on the left.

  2. Right-click on the editor background and select New State from the context menu:

Algorithms

The Algorithms editor  contains one tab for each action used in the function block. This editor contains the algorithm for the action written in structured text in accordance with IEC 61131-3.

IntelliSense (Code Completion) is available by pressing the shortcut Ctrl+Spacebar to make editing easier.

Right-clicking the tab header of any existing algorithm opens a context menu with the following commands:

Command

Description

Add Algorithm

Creates a new Algorithm.

Rename Algorithm

Opens a dialog to rename the selected Algorithm.

Delete Variables

Removes the selected Algorithm. This item is available only when the selected Algorithm currently is not in use in the ECC

NOTE: The context menu is not available if no Algorithm exists.

When no Algorithm exists the context menu is not available, because also no Algorithm tab header exists. To create a new Algorithm in this case, right-click the grey box of an action of a state in the Execution Control Chart editor as described above.

Temporary Variables

This command edits temporary variables using a grid located at the top of the selected Algorithm.

The grid editor can be accessed using the Algorithms command, which is located in the grey vertical menu bar.

The grid allows to enter a new variable and to change its name, to edit the data type of the variable, enter the number of array elements or the minimum and maximum index value valid for the array, to enter an initial value for the variable to start with when entering the algorithm.

Test Function

The Test Container provides the ability to simulate the execution of basic function blocks singularly, even before it is instanced in a project.

A description of how to use the test container can be found at Test Container for Basic Function Blocks.
