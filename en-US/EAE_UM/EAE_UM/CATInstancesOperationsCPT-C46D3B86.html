CAT Instances Operations

The CAT instances operations are a way given by EcoStruxure Automation Expert to bulk create CAT instances, update their offline parameters and their connections with other CAT instances.

These operations are accessible in the CAT Instances node of the Solution Explorer pad tree, as detailed in the topic CAT Instances. They are used to create and update large amount of CAT instances through bulk edit using Excel.

This CAT instances bulk edit is done by:

  • Exporting the existing CAT instances of a project into an Excel.

  • Updating the CAT instances in the Excel to modify the existing ones or create new CAT instances.

  • Import the updates done in the Excel into EcoStruxure Automation Expert.

NOTE: If any issue appears during these steps, check the topic Troubleshoot Tips.
