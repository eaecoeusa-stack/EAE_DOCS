Persistence Data

The Action Menu Persistence Data item links to the following commands:

Command

Description

Back up Persistent Data

Creates a back up file for the values of parameters that were configured as persistent in the Offline Parameterization editor.

Restore Persistent Data
(Single Restore Persistent Data - HA Partner)

Restores the persistent parameter data from a back-up file that was previously created using the Back up persistent data command, so that the persistent parameter values are applied to the device when it is re-initialized.

The Restore Persistent Data command applies to:

  • High availability Soft dPACs at the HA Pair level.

  • Standalone controllers at the individual controller level.

The Single Restore Persistent Data command applies to individual HA Soft dPACs at the HA Partner level.

Delete Persistent Data

Deletes the persistent data for parameters configured as persistent in the Offline Parameterization editor. The original default parameter settings are applied to the device when it is re-initialized.
