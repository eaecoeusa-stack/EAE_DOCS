Asset Link for Bulk Engineering

Use Asset Link for Bulk Engineering to bulk generate CAT instances in an EcoStruxure Automation Expert application based on data sourced from one of these engineering tools:

  • AVEVA Engineering via the AVEVA Integration Service (AIS)

  • An Excel file (or compatible) based on a model and manually filled

  • Uploaded DEXPI Process and Instrumentation Diagram (P&ID) files:

    DEXPI files with version 1.2 schema can be uploaded to import engineering data.

Asset Link for Bulk Engineering extracts the engineering data (equipment, instruments, and their relationships and attributes), then bulk generates CAT instances.

In addition, AVEVA Engineering and Excel sources handle other assets such as plant, area and unit, so you can build a Process view.

Preliminary Tasks

AVEVA Engineering: Configure connection to AIS

Excel File:

  • the sheet name must be SEEngineeringView.

  • the name of the column with the plant and area names must be PlantName and AreaName respectively

  • the name of the column with instrument names must be Name; as it relates to a lowest-level instance, each name must be unique in the column

  • the additional columns are optional; a column of attributes can be created and its content will be extracted

  • the Excel file must be filled to create a bread crumb trail across the columns. A common practice is to start the asset names with an ISA 3–digit code.
