State of a Reliable Cross Communication Connection

The following function blocks are located n the SE.HWCommon library and can be used to identify the state of the Reliable Cross Communication connection between the HA Soft dPAC controller and an HA device:

  • ReliableCrossCommConnectionStateChange: Basic function block.

  • ReliableCrossCommConnectionStateSource and ReliableCrossCommConnectionStateDestination: Composite function blocks.

ReliableCrossCommConnectionStateChange
