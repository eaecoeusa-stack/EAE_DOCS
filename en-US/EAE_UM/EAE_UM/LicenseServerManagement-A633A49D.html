License Server Management

Two types of License Servers are supported with EcoStruxure Automation Expert:

  • Cloud License Server

  • Enterprise License Server

    NOTE: To know their differences, please check the topic EcoStruxure Automation Expert Licenses

Here are the different steps to manage these two License Servers:

  • Cloud License Server

  • Enterprise License Server

  • License Server configuration

  • License Server availability

  • Borrowing floating license from a License Server

  • Reserving borrowed floating licenses for offline use

  • License Server notifications

Cloud License Server Creation and License Assignment

The creation of your Cloud License Server and the license operations on your server can be done on the Software Management portal.
NOTE: In China, contact your local Schneider Electric customer support to request the creation of your Cloud License Server for EcoStruxure Automation Expert.

Complete the following steps to create a Cloud License Server and to add the required licenses as floating licenses on your License Server:

  1. In the License Manager, click the License Portal link.

    Result: The Software Management portal opens.

  2. Login with your MySchneider account.

    NOTE:

    • If you do not have a MySchneider account, you can create one by clicking Register and following the instructions.

    • If there is no Internet connection on your Engineering Workstation, enter the URL displayed in the tooltip of the License Portal link in a web browser, on any device with Internet connection.

  3. Verify if the licenses that you want to use on your Cloud License Server appear in the Software Management portal. If not, then register the licenses Activation IDs to your account.

  4. Click the Devices tab. Then add a new device, select EcoStruxure Automation Expert and enter a device name for your Cloud License Server.

  5. Select the created Cloud License Server and add the licenses that you want to use on this server.

  6. Safely store the Device ID (that is the License Server ID).

    NOTE: The Device ID is required in the License Manager to configure the License Server on any Engineering Workstation, where licenses are borrowed or reserved from this Cloud License Server.

Enterprise License Server Installation and License Management

The Enterprise License Server must be hosted and maintained by your enterprise.

The Enterprise License Server is offered as separate package and can be downloaded for free from the same location where you downloaded the EcoStruxure Automation Expert package.

This package also includes an English guideline, describing the Enterprise License Server requirements and necessary steps to install, configure and manage the licenses on it.

License Server Configuration

To configure your Engineering Workstations to borrow or reserve licenses from your License Server, complete the following steps:

  1. Start EcoStruxure Automation Expert and click Help menu, then click License Manager.

    Result: The License Manager opens and displays the License Information page.

  2. Click the License Server button.

  3. Click New in the last column of the License Server table.

    Result: The Configure License Server page opens.

  4. On the Configure License Server page:

    • For Cloud License Server:

      1. Enter an Alias for the License Server.

      2. Select the type Cloud License Server.

      3. Enter your License Server ID previously retrieved.

      4. Check Connection to verify if the License Server is reachable.

      5. Save the License Server connection once its status is Online. This status indicates that the License Server is running as it is reachable over the network.

        NOTE: With the button Remove, a saved configuration can be removed when needed.

        Result: The Configure License Server page is closed.

    • For Enterprise License Server:

      1. Enter an Alias for the License Server.

      2. Select the type Enterprise License Server.

      3. Select the Hyper Text Transfer Protocol of your License Server. It is strongly recommended to use the secured one (https).

      4. Enter the host name or the IP address and the Port of your Enterprise License Server machine.

      5. Check Connection to verify if the License Server is reachable.

      6. Save the License Server connection once its status is Online. This status indicates that the License Server is running as it is reachable over the network.

        NOTE: With the button Remove, a saved configuration can be removed when needed.

        Result: The Configure License Server page is closed.

  5. If required, in the License Server page, modify the License Server configuration by using the Edit button.
    NOTE: The Edit button is unavailable when licenses are still reserved from the License Server. Releasing the reserved licenses makes the Edit button available again.

  6. Click Close to return to the License Information page.

    Result: One copy of each floating license, which is not yet available as node-locked license, is automatically borrowed from the configured License Server. For the borrowed licenses, the alias name of the License Server is displayed in the Source column of the License Information page. The remaining available licenses on your License Server are displayed under Available Served Licenses, on the License Information page.
    NOTE: After any License Server configuration change, it is recommended to restart other possible running EcoStruxure Automation Expert instances on your PC.

Available Licenses on a License Server

The License Information page shows the EcoStruxure Automation Expert licenses. The licenses, available on the configured License Server, are also displayed.

To see them, click Available Served Licenses to expand a table when a License Server is properly configured and reachable.

The Available Served Licenses table shows the following additional information:

  • Alias: It is the alias name given to the License Server during its configuration.

  • Remaining Copies: The number of available license copies that can be borrowed at the moment by another Engineering Workstation.

    NOTE: If you want to verify if a copy has been released from another Engineering Workstation, collapse, then expand again the Available Served Licenses table to refresh the number. Licenses with zero remaining copies are not displayed, except if they are borrowed or reserved by the current Engineering Workstation.

Borrowing Floating License from a License Server

EcoStruxure Automation Expert borrows automatically, from the configured License Server, one copy of the floating licenses that are not yet available as node-locked licenses:

  • After each successful configuration of your License Server.

  • At each opening of EcoStruxure Automation Expert.

  • After the release of reserved licenses from a License Server.

For the two Asset Link product features, automatic borrowing of the corresponding Addon licenses is only done if these features are installed.

NOTE: The feature Asset Link for Bulk Engineering needs no more Addon license in EcoStruxure Automation Expert V24.0 because it is only for preview.

The conditions to borrow licenses on your Engineering Workstation are:

  • A permanent Internet connection on the Engineering Workstation when borrowing a license from a Cloud License Server.

    NOTE: If this Internet connection is not permanently available, then License Server notifications might appear. A re-connection is required at least once within every four hours.

  • A permanent Intranet connection on the Engineering Workstation when borrowing a license from an Enterprise License Server.

    NOTE: If this Intranet connection is not permanently available, then License Server notifications might appear.

  • A configured License Server within the License Manager of EcoStruxure Automation Expert with Online connection status.

  • An available copy of a Professional Engineering, Standard Engineering, or Addon license, must be available on the License Server for borrowing.

After having successfully borrowed an available license copy from the License Server, information about this license becomes available on the License Information page of the License Manager. The new license appears there with the License Server alias name in the Source column.

NOTE: It is recommended to synchronize the clock of your Engineering Workstation with a time server to prevent problems with license borrowing.

Reserving Borrowed Floating Licenses for Offline Use

Your borrowed floating licenses can be reserved for use on your Engineering Workstation without connection to the License Server, for a maximum period of 14 days.

To reserve borrowed floating licenses, your Engineering Workstation must be connected to the License Server. After the license reservation is confirmed, the Engineering Workstation can be disconnected from the License Server.

The license reservation is available only if a License Server is configured in your EcoStruxure Automation Expert solution.
NOTE: During the reservation period, the License Server configuration on the License Server page cannot be modified.

Step

Action

1

Start EcoStruxure Automation Expert and click Help menu, then click License Manager.

Result: The License Manager opens and your available licenses are displayed.

2

Click the Reserve License button.

Result: The Reserve Licenses page opens.

3

Modify the duration of the reservation period by clicking - or +.
NOTE: The maximum duration is 14 days. The period ends at the end of the reservation date displayed on the same page or in the License Information page.

4

Click Next, then click OK on the confirmation dialog.

Result: The previously borrowed floating licenses are now reserved for the selected period of time. Your Engineering Workstation can be disconnected from the License Server. In the License Manager, the Reserve License button is replaced by the Release Reservation button.

The reserved licenses can be released manually before the end of the reservation period if your Engineering Workstation is connected to the License Server. Otherwise, they are released automatically at the end of the reservation period, even if the Engineering Workstation is not connected to the License Server.

To release a reserved license when connected to the License Server, open the License Manager and click the Release Reservation button, then click OK on the confirmation dialog. The license is released and becomes borrowed again.

NOTE: When the reservation period of one or more licenses expires in less than three days, a message is displayed at the start of EcoStruxure Automation Expert. To extend the reservation period, a connection to the License Server is required to release the current reservation. Then you can reserve the licenses again for a new reservation period.

License Server Notifications

EcoStruxure Automation Expert gives message notifications when specific license use cases happened while borrowing or reserving a floating license from a License Server. Some of these use cases are:

  • The connection to the License Server ends while EcoStruxure Automation Expert runs. Short connection breaks do not lead to messages.

  • Re-connection to the License Server could not be established within a defined time.

  • There is no available copy of an Engineering or Addon license, available on the License Server.

  • There are no borrowed licenses available to reserve for offline use.
