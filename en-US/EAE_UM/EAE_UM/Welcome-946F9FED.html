About the Document

Document Scope

This document describes the EcoStruxure Automation Expert, a software that lets you efficiently engineer distributed systems. The EcoStruxure Automation Expert industrial automation system seamlessly integrates automation tasks in one single tool, offers prefabricated software libraries, enables hardware-independent engineering, and uses IEC 61499 as well as IEC 61131 as their control paradigm.

The integration of all necessary components in one package allows you to experience consistent, scalable and distributable project planning in a new efficiency dimension in accordance with the IEC 61499 norm.


EcoStruxure Automation Expert is a combination of the following components:

  • EcoStruxure Automation Expert - Buildtime - Engineering environment

  • EcoStruxure Automation Expert - Runtime (EcoRT) - Runtime system for distributed control technology

  • EcoStruxure Automation Expert - HMI - Runtime system for visualization

  • Schneider Electric Libraries - hardware-independent libraries

  • Services
