Text Edit Menu

The Text Edit menu is displayed only when focus is on a text editor. It contains a Code area and a Search area.

Code area commands

Command

Description

Toggle fold

Opens/closes the current outline segment.

Toggle all folds

Opens/closes all outline segments of a document.

Show definitions only

Displays definitions only.

Comment / Uncomment selection

Places the selected text into a comment (<!- - SelectedText - ->) and removes selected test from the comment.

Search area commands

Command

Description

Find Next

Opens a dialog for searching through text files.

Replace

Opens a dialog to search and replace text in text files.

Go to line number

Opens a dialog where a line number or an expression can be input to jump directly to the specified location.

Go to matching brace

Jumps to the respective parenthesis at the oppose position of the selected parenthesis expression.
