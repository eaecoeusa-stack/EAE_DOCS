New Algorithm

Use this dialog to create a new algorithm within a Basic function block, and to edit an existing algorithm.

You can access the dialog in two ways:

  • To create a new algorithm, in the ECC logic tab for a function block, right click and select New Algorithm ... from the context menu.

  • To edit an existing algorithm, right-click on the tab of an existing algorithm within Algorithm. tab in a basic function block, then select Add algorithm ... from context menu.

New Algorithm Dialog Commands

The New Algorithm dialog includes the following commands:

Item

Description

Name

The name of the new algorithm has to be defined here.

Go to editor

If check marked, the editor for this algorithm will be opened automatically after the dialog has been confirmed with OK. Otherwise the editor can be opened by opening tab page algorithm and clicking on the relevant tab. The algorithm has to be written in Structured Text. (Enabled only for new algorithms.)

Associate with current action

If check marked, that action within the ECC logic, where the right-click has been done, will be replaced by the new algorithm as soon as you confirm the dialog with OK. Otherwise the existing action can also be replaced after the new one has been created, by executing a right-click again on the action within the ECC Editor. Now the context menu will show the additional action and it can be selected directly from here. (Enabled only for new algorithms.)

Create a new action

If check marked, the new action will be added to the existing action where the right-click has been executed, as soon as you confirm the dialog with OK. (Enabled only for new algorithms.)

Help

Opens context help.

OK

Closes the dialog and creates the new algorithm.

Cancel

Closes the dialog without creating an algorithm.
