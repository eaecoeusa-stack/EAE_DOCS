Transition Editor

Condition to pass a transition

A boolean expression that must return either TRUE or FALSE is attached to each transition.

If this expression has the value TRUE, the state or step (according to the context) following the transition gets activated.

Procedure

  1. Double-click a transition (the graphic, not the label) in the chart editor. This opens a window with a text editor. Default expression: TRUE.

  2. Enter the desired expression. Due to an autofill function, you can enter only the beginning of a variable name and select the full name in the drop-down dictionary or just click Enter.
    NOTE:

    • The dictionary is directly available by pressing the space bar; then select and double-click.

    • The entered symbols can be moved using the mouse.

  3. Click OK.
