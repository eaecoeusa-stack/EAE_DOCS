Get Diagnostics

After the MQTT broker is installed and configured, the following actions have to be completed to get the device diagnostics on HMI:

  1. Configure a Soft dPAC device.

  2. Define the communication with the MQTT broker.

  3. Push the MQTT configuration to the Soft dPAC Manager.

  4. Draw Diagnostics Grid on canvas.

  5. Run the application online.

Configure a Soft dPAC Device

To configure a Soft dPAC device, follow the actions described in Working with Soft dPAC in the Physical Devices Editor in the EcoStruxure Automation Expert - Soft dPAC User Guide.

Define Communication with the MQTT Broker

To define the communication with the MQTT broker, complete the following steps:

  1. Open the Physical devices editor, then the Properties pad.

  2. Add a workstation to define the communication with the MQTT broker.

  3. Select the bottom left part of the workstation (named NIC in the Properties pad). Then in the Properties pad, add the IPv4 address of the engineering workstation, where the MQTT broker is installed.

  4. Select the entire workstation (named Workstation in the Properties pad). Then add the MQTT broker service by:

    1. Right clicking the workstation.

    2. Clicking Manage Services > MQTT Broker > Activate service.

  5. In the Properties pad, define the listener port to secure communication by:

    1. Finding the Logical Port parameter: Properties > Services > MQTT Broker > Logical Port.

    2. Modifying the parameter Logical Port value to 8883.

    This listener port number is the same as the one used during the configuration of the MQTT broker.

Push MQTT Configuration to the Soft dPAC Manager

To push MQTT configuration onto the Soft dPAC Manager, follow these actions for each station running a Soft dPAC:

  1. Open the Physical devices editor.

  2. Configure a Soft dPAC device (see section above).

  3. Define communication with the MQTT broker (see section above).

  4. Select the entire workstation, containing the Soft dPAC device, then push the MQTT configuration by:

    1. Right-clicking the workstation.

    2. Clicking Manage Services > Manage Soft dPAC > Configuration.

    3. Clicking Apply at the bottom right of the window opening.

NOTE: Repeat these pushing steps on any station running a Soft dPAC, each time the broker address is changed.

Result: The MQTT configuration is pushed onto the Soft dPAC Manager as it is shown in the tab Services and in the line Diagnostic service of the window. The Soft dPAC Manager is now making the communication between the MQTT broker and the Soft dPAC device, allowing the diagnostics to be delivered.

Draw Diagnostics Grid on Canvas

To see the diagnostics transferred from the Soft dPAC device, add a Diagnostics Grid on a canvas:

  1. In the Solution Explorer pad, expand the Canvas sub-node.

  2. Open or create a canvas.

  3. Add a Diagnostics Grid into the canvas.

  4. Save and close the canvas.

Result: The Diagnostics Grid is added and the canvas is ready to be started online.

NOTE: To know how the Diagnostics Grid graphic tool works, refer to Graphics Editor Toolbar.

Run the Application Online

To get the diagnostics, make sure to have completed all the previously described actions. Then run the Soft dPAC device online and start the HMI online. The canvas with the Diagnostics Grid displays the diagnostics of the device.
