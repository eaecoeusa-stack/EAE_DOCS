CAT Instances do not Appear Inside the Function Block Network Editor

If the CAT instances do not appear inside the Function Block Network editor of the wanted applications, it means that their Application Name cell in the Excel file have been kept empty or wrongly filled. To find these missing CAT instances, follow the next steps:

  1. Open the Display Profiles list located in the top right of the Solution Explorer pad:

  2. In this list select CAT Instances Sort > Application. With this option, the CAT instances in the Solution Explorer pad will be sorted by application.

  3. Expand the CAT Instances node and discover below it all the applications available in the project plus the sub-node Not Assigned to System Application. All the CAT instances not found inside the Function Block Network editor of their application will be gathered into this one.

To bring them back to the right application, open the Function Block Network editor of the wanted application. Then drag and drop the CAT instances from the Not Assigned to System Application node into the editor. It will automatically assign the CAT instance to this application.

After this action the CAT instances will then be correctly imported into EcoStruxure Automation Expert.
