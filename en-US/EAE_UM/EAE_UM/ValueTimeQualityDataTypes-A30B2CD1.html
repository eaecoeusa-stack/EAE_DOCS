Value, Time, Quality Data Types

This topic describes the use and supporting architecture for value, time, quality (VTQ) data types in the EcoStruxure Automation Expert Buildtime software. 

A set of new (system, Class 1) data types are created for VTQ. Any function block parameters that are specified as being of a VTQ type carry quality and timestamp data, along with the value in all function block connections. 

Using VTQ data types lets you reduce the size and performance impact when compared to existing function blocks. The VTQ data type provides design flexibility regarding which parameters carry the cost of value, time and quality transfers. It also supports the use of VTQ by process control function blocks. 

This topic describes the use of VTQ for system integration by implementing VTQ values at the edge.

NOTE: VTQ does not need to be extended to function blocks that are internal to your solution, if VTQ is otherwise not required.

This topic discusses the following subjects:

  2. Process Automation Function Blocks

  3. VTQ Data Structure

  4. Supported Elementary Data Types

  7. Implicit Data Type Conversions for Function Block Connections
