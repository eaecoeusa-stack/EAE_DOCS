8. Time Checking Functions

All functions described in this topic are conforming to IEC 61131-3 third edition.

IS_VALID

Synopsis 

IS_VALID(IN: DT): BOOL
IS_VALID(IN: DATE): BOOL
 
IS_VALID(IN: LDT): BOOL
IS_VALID(IN: LDATE): BOOL

Description

Checks input parameter IN of type DT, LDT, DATE or LDATE for validity and returns TRUE if the value is within the range defined for the data type in question or FALSE otherwise.

NOTE: Currently this function can only be used to check a date or date and time value only after some functions where noticed.

Errors

None

Example 

VAR
  d1: DATE;
END_VAR
d1 := CONCAT_DATE(2021, 12, 5);
IF IS_VALID(d1) AND d1 = DATE#2021-12-5 THEN
  (* correct *) ;
END_IF;
