Offline Node-Locked License Operations

The following operations are available to manage node-locked licenses on an Engineering Workstation not connected to the Internet (offline):

  • Activate node-locked licenses

  • Return activated node-locked licenses to the Schneider Electric License Server

  • Renew the license information

Activate Node-Locked Licenses

If your Engineering Workstation with EcoStruxure Automation Expert is not connected to the Internet, complete the following steps to activate your node-locked licenses on your workstation:

Prerequisites: A PC with Internet connection is required for use during offline activation operations, for the exchange of files with the Offline License Processing web portal.

  1. Start EcoStruxure Automation Expert on the Engineering Workstation not connected to the Internet.

  2. Open the License Activation page of the License Manager in one of the following ways:

    • Click Help > License Manager, then click Activate.

    • Click File > About, then click either Activate Now or License Manager and click Activate.

    • Click Activate in the license popup message, displayed at the start of EcoStruxure Automation Expert, when no Engineering license is yet activated.

      Result: As the workstation does not have access to the Internet, a message appears to inform you that the Schneider Electric License Server is not reachable, and asks if you want to continue with the offline license activation.

  3. Click Yes in the displayed message, to continue the offline license activation.

    Result: The Offline License Activation page opens, and the license activation steps are displayed.

  4. Enter the Activation IDs of one or more licenses and click Next.

  5. Select the path where the Request File should be stored, and click Next to generate the file.

    Result: The License Manager displays the Web Portal Instructions step, where the steps to complete the offline activation operation on the PC with Internet connection are described. Copy these instructions, including the URL of the Offline License Processing web portal, on the PC with Internet connection to continue the license activation operation.

  6. Transfer the Request File to the PC with Internet connection, for example via a portable media or Bluetooth.

  7. Open the web portal on the PC with Internet connection, using the URL provided in the instructions of the Web Portal Instructions step, and follow the steps to:

    • Upload the Request File to the web portal.

    • Download the Response File from the web portal.

  8. After you complete the specified steps on the web portal, transfer the Response File back to your Engineering Workstation, where you initiated the activation, for example via a portable media or Bluetooth.

  9. On your Engineering Workstation without Internet connection:

    • If the Offline License Activation page is still open, click Next to complete the Web Portal Instructions step, then in the Select Response File step enter the path to the location where the Response File is stored, and click Next.

    • If the Offline License Activation page is closed, click Process Response in the License Manager, then enter the path to the location where the Response File is stored, and click Next.

    Result: When the license activation operation is completed, the Confirmation step informs you if your licenses are activated.
    NOTE: If the activation is not successful, read carefully the displayed error message text with solution proposal. In case you need support, you can click the Copy to Clipboard icon in the Confirmation step and send the error message to your local Schneider Electric customer support.

  10. Click OK to close the page.

Result: The node-locked licenses of EcoStruxure Automation Expert on your Engineering Workstation are activated. You can use the functions of the successfully activated licenses within the EcoStruxure Automation Expert software. The newly activated licenses are displayed on the License Information page of the License Manager.

Return Activated Node-Locked Licenses to the Schneider Electric License Server

Activated node-locked licenses can be returned from their current Engineering Workstation back to the Schneider Electric License Server. This operation is needed when you want to:

  • Move an activated node-locked license to another Engineering Workstation.

  • Reinstall your Engineering Workstation.

In both cases, follow the steps to return the licenses:

Prerequisites: A PC with Internet connection is required for use during offline return operations, for the exchange of files with the Offline License Processing web portal.

  1. Start EcoStruxure Automation Expert, and open the License Manager dialog from the Help menu.

  2. Select one or more node-locked licenses to be returned.
    NOTE: Multiple licenses can be selected by pressing Ctrl and clicking the licenses with the mouse. A trial license and a borrowed or reserved license from a License Server cannot be returned. For each license, a message is displayed in the tooltip of the license status icon to inform you if the license can be returned.

  3. Click Return.

    Result: As the workstation does not have access to the Internet, a message appears to inform you that the Schneider Electric License Server is not reachable, and asks if you want to continue with the offline license return.

  4. Click Yes in the displayed message, to continue the offline license return.

    Result: The Offline License Return page opens, and the license return steps are displayed.

  5. Select a location to store the Request File and click Next to generate the file.

    Result: The License Manager displays the Web Portal Instructions step, where the steps to complete the offline return process on the PC with Internet connection are described. Copy these instructions, including the URL of the Offline License Processing web portal, on the PC with Internet connection to continue the license return operation.

  6. Transfer the Request File to the PC with Internet, for example via a portable media or Bluetooth.

  7. Open the web portal on the PC with Internet connection, using the URL provided in the instructions of the Web Portal Instructions step, and follow the steps to:

    • Upload the Request File to the web portal.

    • Download the Response File from the web portal.

  8. After you complete the specified steps on the web portal, transfer the Response File back to your Engineering Workstation, where you initiated the license return, for example via a portable media or Bluetooth.

  9. On your Engineering Workstation without Internet connection:

    • If the Offline License Return page is still open, click Next to complete the Web Portal Instructions step, then in the Select Response File step enter the path to the location where the Response File is stored, and click Next.

    • If the Offline License Return page is closed, click Process Response in the License Manager, then enter the path to the location where the Response File is stored, and click Next.

    Result: When the license return operation is completed, the Operation Completed message informs you if your licenses are returned.
    NOTE: If the return is not successful, read carefully the displayed error message text with solution proposal. In case you need support, contact your local Schneider Electric customer support.

  10. Click OK to close the page.

To reactivate the licenses, follow one of the following options:

  • If the Engineering Workstation is connected to the Internet, refer to the online activation instructions to reactivate your node-locked licenses.

  • If the Engineering Workstation is not connected to the Internet, refer to the offline activation instructions to reactivate your node-locked licenses.

NOTE: The reactivation of the licenses follows the same steps as an initial license activation.

Renew License Information

The procedure to renew the licenses information offline is very similar to the steps to return offline a node-locked license.

If your Engineering Workstation with EcoStruxure Automation Expert is not connected to the Internet, follow the same steps as the ones to return the licenses, except the step 2: you do not need to select any license to renew the licenses information. The rest of the steps remain the same.

Result: The updated license information from the different License Servers is displayed in the License Information page of the License Manager.
