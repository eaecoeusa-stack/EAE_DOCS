Composite Function Block

Composites are collections of Basic function blocks or other composites with lower level functionality to achieve higher level functionality. Like basic function blocks, composites cannot run distributed. (See also IEC 61499 documentation for Composites)

For generating or deleting look at help item Elements.

For editing help, refer to the topics Function Block Interface Editor and Composite Function Block Editor.

To learn more about how to create a composite function block, watch How to Create a Composite Function Block in EcoStruxure Automation Expert.

The interfaces and associations between input variables (InputVars) and event inputs (EventInputs) — and the same on the output side — are shown in the graphic view of the Composite Function block. The Composite Function block presents the following element groups

  • EventInputs (interfaces)

  • EventOutputs (interfaces)

  • InputVars (variables)

  • OutputVars (variables)

  • Sockets (adapters)

  • Plugs (adapters)

  • AdapterInputs (adapters)

  • AdapterOutputs (adapters)

The following function blocks can be integrated into a Composite function block:

  • Composite

  • Basic

  • Service

    NOTE: CAT or Adapter cannot be integrated into a Composite Function block.
