Search Results Pad

The Search Results pad shows the results of a search. It is situated by default at the bottom in the middle of EcoStruxure Automation Expert - Buildtime .

Click View > General pads > Search Results in the main menu bar to open the pad.

This pad opens automatically when:

  • A search is started by the search function of the toolbar.

  • A search is made in the Search and Replace dialog box.

  • right-clicking a function block within an application, resource or the like and selecting Find all references from the context menu.

Toolbar

Icon

Function

Description

Show last searches

Shows results of the last searches. Search results are listed in context menu and can be selected there.

Select search list mode

Two types of notation can be selected:

  • Flat - all results of a search are listed in one node

  • Per File - the results of a search are assorted in files

Expand all

Expands all nodes within the search result.

Collapse all

Collapses all nodes within the search result.
