Add Modules and Submodules to Devices

After adding an IO-Controller and IO-Devices to your PROFINET network, you can add modules and submodules. Then you can edit submodule parameters.

Adding Modules and Configuring the Watchdog and Port Speed and Duplex Settings

To add Modules to an IO-Device, follow these steps:

  1. In the Hardware Configuration editor, right click on the PN_IOController node and select Configure.

    The PROFINET IO Configurator tool opens.

    NOTE: The IO-Devices you previously added are listed in the Selected Devices list.

  2. Select a device and in the Modules column, hover your cursor over the ellipsis (...), then click Configure.


    NOTE: The PROFINET IO Configurator displays:

    • Device hierarchical level values for Network Configuration > Device.

    • A collection of Available Modules, sorted by module type (for example, DI, DO, AI, AO and so forth).

    • The list of available Slots for the module.

  3. Expand the list of Available Modules and locate the module you want to add.

  4. To add a module to the device, drag and drop a module to a specific slot.

  5. Repeat steps 3 & 4 for every additional module you want to add.

  6. (Optional) Configure the network settings, watchdog, and port speed and duplex settings.

    1. Click Advanced Options.

    2. Expand the Network Settings node and input values for IP Address, and Subnet Mask. Then either input a Default Gateway or select Same as IO-Controller settings.

    3. Expand the General node and select Watchdog Time. The Watchdog Factor and Watchdog Time (ms) settings are displayed.

    4. Select a Watchdog Factor from the list. The corresponding Watchdog Time is automatically applied.

      NOTE:

      • The Watchdog Factor is the permitted number of update cycles that can elapse with no I/O data transmitted.

      • The Watchdog Time = device Update Time x Watchdog Factor, and cannot exceed 1920 ms.

    5. Expand the Port Options node, and select or de-select Enable Autonegotiation for each port. If Enable Autonegotiation is:

      • Selected, the connected ports automatically determine both speed and duplex settings.

      • De-selected, 100BASE-TXFD MAU is applied to the port.

  7. To save your edits now, click Back to return to the main page. In the main page click Update & Close, then click Yes.

    Or you can proceed to add submodules to a module by hovering your cursor over the ellipsis in the Submodules column, then clicking Configure.

Adding Submodules

To add Submodules to a Module, follow these steps:

  1. If you are not proceeding from the Adding Modules procedure, above, then in the PROFINET IO Configurator editor, navigate to the Network Configuration > Device level of the PROFINET network hierarchy:

    1. In the Hardware Configuration editor, right click on the PN_IOController node and select Configure.

    2. Select a device and in the Modules column click Configure.

    3. Select a Module from the Slots list and in the Submodules column hover your cursor over the ellipsis (...), then click Configure.


    NOTE: The PROFINET IO Configurator displays:

    • The module hierarchical level displays values for Network Configuration > Device > Module.

    • A collection of Available Submodules.

    • The list of available Subslots for the submodule.

  2. Expand the list of Available Submodules and locate the submodule you want to add.

  3. To add a submodule to the module, drag and drop a submodule to a specific subslot.

  4. Repeat steps 2 & 3 for every additional submodule you want to add.

  5. To save your edits now, click Back until you return to the main page. In the main page click Update & Close, then click Yes.

    Or you can proceed to edit parameters module by clicking the ellipsis (...) in the Parameters column for a selected submodule. Here you can select Identification and Maintenance to enter descriptive parameter tag information.

Edit Parameters

To edit submodule parameters, follow these steps:

NOTE: If you are not proceeding from the Adding Submodules procedure, above, then in the PROFINET IO Configurator navigate to the Network Configuration > Device > Module > Submodule level of the PROFINET network hierarchy as described below.

  1. In the Hardware Configuration editor, right click on the PN_IOController node and select Configure.

  2. Select a device and in the Modules column, hover your cursor over the ellipsis (...), then click Configure.

  3. Select a Module from the Slots list and in the Submodules column hover your cursor over the ellipsis (...), then click Configure.

  4. Select a Submodule from the Subslots list and in the Parameters column, hover your cursor over the ellipsis (...), then click Configure.

    NOTE: The PROFINET IO Configurator displays:

    • The hierarchical level values for Network Configuration > Device > Module > Submodule.

    • A list of Parameters for the selected submodule.

  5. You can edit the Configured Value for each parameter.

  6. Repeat steps 2 to 5 for every submodule whose parameters you want to edit.

  7. To save your edits now, click Back until you return to the main page. In the main page click Update & Close, then click Yes.
