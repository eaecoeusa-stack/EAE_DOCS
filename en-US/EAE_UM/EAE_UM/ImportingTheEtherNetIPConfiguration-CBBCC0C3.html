Importing the EtherNet/IP Configuration to a Solution

  1. In the Hardware Configuration editor, right click the EIPSCANNER node to open the context menu, then select Import EtherNet/IP scanner....
    Result: The Open dialog opens.
  2. In the Open dialog, navigate to and select the EtherNet/IP configuration zip file, then click Open.
    Result: The configuration from the EtherNet/IP Configurator is added to the Hardware Configuration editor.
  3. Optionally, the following additional actions can be performed:
    1. If you created Symbolic Links, open the Symbolic Links pad to create SymLinks, then assign them to LinkName attributes of input and output items.
    2. Test the solution by opening the Physical Devices editor, and click the Start Monitoring toolbar command to check the status of each device on the EtherNet/IP network.
    3. Open the Deploy and Diagnostic editor where you can Login, Clean, Deploy, and Run the solution.
