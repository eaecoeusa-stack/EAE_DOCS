Archive Solution

The archive mechanism is available in EcoStruxure Automation Expert to any opened solution. This mechanism generates a file containing all the elements in the solution. It allows to share or duplicate the current solution.

Once created, any engineering workstation with EcoStruxure Automation Expert can open it by unarchiving the file of the archived solution.

To archive the current open solution, follow the next steps:

  1. To start an archive of the solution either open:

    • The File menu.

    • The Solution Explorer pad. Then right click on the solution name.

  2. Once there, select Archive.

  3. The dialog box Save referenced libraries as part of the archive opens. This dialog box lists all the referenced libraries of the solution.

    Select the referenced libraries that will be included inside the archive file by:

    • Ticking the check box next to Export to select all the libraries.

    • Ticking the individual check box of libraries to select specific libraries.

    All the ticked libraries will be included inside the archive file at their current referenced version. This ensures that the user unarchiving the solution will not experience compatibility issues.

  4. Click Okay when all the wanted referenced libraries are selected.

  5. In the dialog box appearing, select where to save the archive and how to name it. The default name is <Solutionname>_<ddmmyy>-<hhmm>.sln.zip.

  6. Press Save to create the archive file.

Once created, the Archive Solution dialog box opens giving the following options:

  • Open Explorer: Open the directory where the archive file is stored.

  • Send E-Mail: Start the default E-mail program and compose a new mail with the archive file attached.

  • Close: Close the dialog box.
