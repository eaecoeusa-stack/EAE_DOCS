2. Conversion Functions

Data type conversion functions change the original data type of a value (the source data type) to an explicitly selected different data type (the target data type). For example the "STRING_TO_INT" data type conversion function would change the text string "006" to the integer 6.

EcoStruxure Automation Expert treats data type conversion functions that are expressed in the format <source data type>_TO_<target data type> as keywords.
