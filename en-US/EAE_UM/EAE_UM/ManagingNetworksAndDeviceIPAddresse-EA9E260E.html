Managing Networks and Device IP Addresses

Each Ethernet device exposes one or more interfaces, with each interface having one or more endpoints. Each endpoint can be associated with a network and an IP address. 

Use the Properties pad to manage networks and assign device IP addresses.

NOTE: Some devices, for example workstations and servers, present two (or more) groups of properties in the Properties pad: one property group for the device itself, and one property group for each network interface card (NIC) included in the device.

Adding a Predefined Network

To add a predefined network:

  1. Select a device interface in the workspace area to display its Ethernet settings in the Properties pad.

  2. Navigate to Ethernet Interface > IP Address > Logical Network.

  3. Click the downward pointing arrow to open a list. By default, the list includes:

    • Manage logical networks.

    • NOCONF (default). No network configuration is used.

    • Default Network. Default values are used for the subnet mask, subnet address, and gateway address.

  4. Select Manage logical networks. The Logical Networks editor opens.

  5. In the Logical Networks Editor, click Add to create a new network.

  6. In the network properties area, edit the Identifier (network name), Subnet Mask,Subnet Address,  and Gateway Address.

  7. Click OK to save the network configuration. The new network name is added to the bottom of the logical network list and can be used.

Deleting a Predefined Network

To remove a predefined network:

  1. Select a device interface in the workspace area to display its Ethernet settings in the Properties pad.

  2. Navigate to Ethernet Interface > IP Address > Logical Network.

  3. Click to display the downward pointing arrow.

  4. Click the arrow to open a list.

  5. Select Manage logical networks. The Logical Networks editor opens.

  6. In the Logical Networks Editor, select a network.

  7. Click Remove to remove the selected network.

  8. Click OK. The network name is deleted.

Configuring an IP Address

To configure an IP address:

  1. Select a device interface in the workspace area to display its Ethernet settings in the Properties pad.

  2. Navigate to Ethernet Interface > IP Address > Logical Network.

  3. Click to display the downward pointing arrow.

  4. Click the arrow to open a list.

  5. Select a network.

  6. Click OK to close the Logical Networks editor.

    Result: The lowest available network IP address is assigned to the interface.

Configuring Device Properties

To configure the generic properties of a device, for example, its name:

  1. Select a device in the workspace area to display its properties in the Properties pad.

  2. The properties that appear in bold text are editable.
