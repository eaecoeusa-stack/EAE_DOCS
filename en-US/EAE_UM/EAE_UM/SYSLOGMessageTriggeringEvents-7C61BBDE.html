SYSLOG Message Triggering Events

Syslog Event Categories

Syslog triggering events may be categorized as follows:

  • Set security of a device from the Physical Devices editor in EcoStruxure Automation Expert

  • Successful/unsuccessful device deployment

  • Modification of system time

  • Successful/unsuccessful IP address change (device properties)

  • Local cybersecurity SYSLOG storage capacity has been modified

  • Successful connection to SYSLOG server

  • Loss of connection to SYSLOG server

  • Creation/deletion of a root public key certificate by a certificate authority (CA)

  • Creation/deletion of a child public key certificate by a root certificate

  • Creation of a new user ID/password

  • Modification or deletion of an existing user ID/password

  • Application enters RUN operation state

  • Application enters STOP operation state
