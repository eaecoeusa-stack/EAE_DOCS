Sequence Management

In the State Graph editor

You can:

  • Create a sequence

  • Associate a sequence with a state node

  • Dissociate a sequence from its state node

  • Delete a dissociated sequence.

A right click on a blue node displays the following menu:

Menu Element

Unassociated state (light blue node)

Associated state (deep blue node)

 

None

The box is checked and greyed. All the operations are possible.

The box is unchecked. By checking it, you disassociate the state node from its sequence.

Delete sequence

Hover this menu command to display the sequences available for deletion on the right side. A sequence can only be deleted if it has been dissociated from its state node by checking None.

Click the sequence you want to delete. A Delete Sequence confirmation window appears. Click Yes to confirm or click No to cancel.

Sequences list

The list of existing sequences is displayed.

You can associate one of the sequences with the state node by clicking on a sequence. Therefore several states may be associated to one sequence. In this case, it is a good practice to rename the sequence by right-clicking the tab header in the Sequence Diagram pad.

The list of existing sequences is displayed.

You can associate one of the sequences with the state node by clicking on the chosen sequence.

A tick symbol indicates that this sequence is currently associated with this state node.

New Sequence

Clicking on this menu command gives the same result as a double-clicking a light blue node (see below).

The menu command is greyed. No interaction is possible.

A double click on an unassociated state:

  • Creates a new sequence named after the state Key attribute (if it does not exist already)

  • Associates automatically the new sequence to the state node

  • Displays the new associated sequence in the Sequence Diagram pad.

A double click on an associated state:

  • Displays its associated sequence in the Sequence Diagram pad.

In the Sequence Diagram editor

By right-clicking a header, you can:

  • Rename a sequence

  • Delete a dissociated sequence.
