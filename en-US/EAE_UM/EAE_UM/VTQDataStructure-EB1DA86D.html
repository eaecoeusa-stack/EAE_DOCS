3. VTQ Data Structure

Name

Step

Action

picosecs

UINT

Specifies the number of 10 picoseconds (1,0 e-11 seconds) intervals which shall be added to the timestamp below. 

quality

DWORD

Quality information for the value. 

timestamp

LDT (IEC 61131 third revision type)

The source timestamp for the value. Represented as a 64-bit signed integer which represents the number of nanoseconds since January 1, 1970. 

A value of 0 indicates “no timestamp”.

value

Varies

IEC 61131 Elementary data type. 

The data value. Varies in length.
