14. Starting and Restarting AVEVA Server Device Services

Starting AVEVA Server Device Services

Many event messages can be resolved by starting AVEVA server device services, which can be done as follows:

Step

Action

1

In the Windows ™ search field, located in the lower left part of the task bar, type services.

2

Right-click  SE ASPSupervisoryFacetServices EcoStruxure Automation Expert 24.0 and select Start.

The service Status will change to Running.

Restarting AVEVA Server Device Services

To restart AVEVA server device services:

Step

Action

1

In the Windows™ search field, located in the lower left part of the task bar, type services.

2

Right-click  SE ASPSupervisoryFacetServices EcoStruxure Automation Expert 24.0 and select Restart.

The service will restart and the Status will change to Running.
