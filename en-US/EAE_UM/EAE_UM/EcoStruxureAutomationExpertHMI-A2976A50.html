Graphic Editors for EcoStruxure Automation Expert HMI

 Graphic editors create graphic elements which at runtime can be used to control operations, parameterize, display alarms or information and so forth via HMI.

The basic structure of graphic editors is the same, although some graphic editors may present additional features and options.

Basic Structure of Graphic Editors

Graphic editors in EcoStruxure Automation Expert work the same way as popular editors for graphical programming, e.g. Visual Studio (Microsoft) or Sharp Develop. The following editors are available on the vertical toolbar on the left:

Command

Description

Designer

The graphic editor, comprising a graphic toolbar and a drawing area.

Source Code

A text editor to implement dedicated functions with C# code.

Additionally, the features from the EcoStruxure Automation Expert Buildtime toolbar and the Format main menu can be used.

The Properties pad is used to define the properties of the background as well as of each element created in the editor (color, width, size, visibility, etc.)
