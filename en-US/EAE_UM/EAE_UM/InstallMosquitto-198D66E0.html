Install Mosquitto

To install Mosquitto, follow these steps:

  1. Navigate to the Mosquitto download page.

  2. Download the latest 64-bits Windows version of Mosquitto.

  3. Execute the installation file.

    Result: The Eclipse Mosquitto Setup window opens.

  4. Click Next in the first page.

  5. In the list of components to install, select Service and click Next.

  6. Select a Windows location where to install Mosquitto, and click Install to proceed with the installation.

    NOTE: If Microsoft Visual C++ is not installed on the engineering workstation, it will be installed as well during the Mosquitto installation. Click Accept to accept the license terms.

  7. When the installation is completed, click Finish.

Result: Mosquitto broker is installed on the engineering workstation. It now needs to be configured:

  1. To create self-signed certificates, if not already available.

  2. To configure the Mosquitto broker.
