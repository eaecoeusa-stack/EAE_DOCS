7. Graphic Designer

These settings relate to the properties of graphic elements (e.g. Canvases, Symbols or Faceplates in CATs, elements in Graphic folder etc.)

Topic

Description

Grid Options

Align mode

Snap to Grid : Grid Properties will be available and the settings at Show Grid and Snap to Grid will be applied when a new graphic element is created

Snap Lines : Pre-settings for Show Grid and Snap to Grid will be set to False (in Properties), independently from what is marked in Grid Properties. The adjustments for Width and Height will be applied.

NOTE: A change of these settings will not concern existing elements. Only elements that are created after the change will apply the new settings.

Grid Properties

Available, when Align mode Snap Lines is marked. Affects the pre-settings as described above.

Default Event

Ask if create Event Handler on DoubleClick : If this check box is marked, a double-click on the default graphic element opens a dialog to define whether to create an Event Handler in source file.

Create Event Handler on DoubleClick : If this check box is marked, an Event Handler will be created with a double-click on the default graphic element automatically. (The recently mentioned item has to be unmarked, otherwise the dialog will be opened.)
