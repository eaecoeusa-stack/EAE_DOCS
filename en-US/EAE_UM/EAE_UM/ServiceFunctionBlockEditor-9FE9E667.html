Service Function Block Editor

Service function blocks represent the interface to the hardware and therefore they are already supplied with the libraries in a ready-to-use condition. Use the editor primarily for viewing, but also for editing, the function blocks. The editing options for the service function block are the same as for other function blocks and can be found at Function block Interface Editor, whereas the functionality of a service function block is implemented solely in the C programming language.

The service function block editor includes a Service tab, which can be used for diagnostics. Service function blocks are implemented directly in the runtime environment. This tab can be used to explain the functionality of the function block. That means it serves only a descriptive function, and performs no functional operation on the function block.

NOTE: In EcoStruxure Automation Expert version 24.1 and later, the FM_DLL and the BM_DLL function blocks are no longer included in the Runtime.Base library. If you need assistance replacing one of these function blocks in your application, and in other special cases if you need to create extra service function blocks, please contact. contact the Schneider Electric support team.
