4. Creating AVEVA Industrial Graphics in EcoStruxure Automation Expert

You can create AVEVA Industrial Graphics in EcoStruxure Automation Expert. This process allows you to create both CAT objects and their associated graphics using one tool.

AVEVA Industrial Graphics can be created at two levels:

  • at the CAT level: an AVEVA Industrial Graphic becomes part of the CAT and is included when the CAT is instantiated

  • at the solution level: an AVEVA Industrial Graphic serves as a container for the graphics of the instantiated CAT graphics added to it.

PREREQUISITES

  • When installing EcoStruxure Automation Expert - Buildtime, you need to select the AVEVA Industrial Graphics Editor option.

  • You need to reference the AVEVAIndustrialGraphic

Creating AVEVA Industrial Graphics at CAT level

  1. In the Solution Explorer, right-click a CAT HMI node right under a CAT and select Add AVEVA Industrial Graphic. In the Define New Name dialog enter a name, then click OK.

  2. The graphic editor opens, with tab name <CAT name>.<graphic name>. Create the graphic using the graphic editor tools and Save.

  3. In the CAT Interface editor, make sure that the CAT HMI includes input and output variables with associated events.

  4. If necessary, in the OPCUA Server editor, expose the I/O variables (IO) for the CAT HMI.

Creating AVEVA Industrial Graphics at the Solution level

  1. In the Solution Explorer, right-click either the OMI > MyContent node or the OMI > MyContent > Graphics node, and select New > AVEVA Industrial Graphic.

  2. In the Define New Name dialog enter a name, then click OK. The graphic editor opens.

  3. Drag each CAT instance you want to add to the solution from the <solution> > CAT Instances > Application node and drop it onto the canvas.

    NOTE: If the selected instance has multiple associated graphics, the Select generic symbol dialog opens, where you can specify a graphic then click OK.

  4. Save your edits.
