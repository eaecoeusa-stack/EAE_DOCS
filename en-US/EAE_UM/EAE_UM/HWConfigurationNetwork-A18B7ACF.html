Hardware Configuration Network

The Hardware Configuration Network view is a graphical view depicting the interconnection of network hardware elements. It is not intended for use as an editor, but instead for displaying the hardware interconnections that can make troubleshooting easier. It is available only to users assigned the Advanced user level in the Settings dialog.

 CAUTION
RISK OF UNINTENDED OPERATION
To avoid the loss of connections between hardware elements, make changes to hardware configurations in the Hardware Configuration editor. In the Hardware Configuration editor, the dialogs are arranged library-dependent.
Failure to follow these instructions can result in injury or equipment damage.

Accessing the HW Configuration Network

There are several options to open Hardware Configuration Network:

  • Right-click on a device within the Logical Devices editor and choose Open HW Configuration Network.

    (A detailed description of the remaining items of the context menu can be found in the HMI -User Guide topic Simulation Runtime.

  • Right-click on a blank space within the Resource editor and choose Layers > Show HW configuration network.

    (A detailed description of the remaining items of the context menu can be found at Resource Editor.)

  • Open the drop-down menu for the resource in the breadcrumbs trail at the top of the window and choose Hardware Configuration.

    To switch back to the Resource Editor, right-click and select Layers > Show HW configuration network in the above mentioned context menu (the check mark in front of the entry will be removed), or select Local  in the breadcrumbs trail for the resource.
