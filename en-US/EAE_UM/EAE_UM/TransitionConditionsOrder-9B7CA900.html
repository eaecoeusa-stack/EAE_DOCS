Transition Conditions Order

Use the Transition Conditions Order dialog to determine the order in which state transition conditions are checked.

Each row in the dialog represents a state transition. When you select a row, the arrows for switching the sequence - located at the right-hand window border - are enabled. Use the arrows to move the selected transition to the desired position in your order. Simultaneously, the current order is displayed in the logic view in the ECC and the selected number is highlighted in red.

In the Basic Function Block Editor, right-click on the background in the ECC editor and in the context menu select Edit condition order... to open the Transition Conditions Order dialog.
