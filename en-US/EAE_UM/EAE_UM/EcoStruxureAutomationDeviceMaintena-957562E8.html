EcoStruxure Automation Device Maintenance

EcoStruxure Automation Device Maintenance (EADM) is the standalone commissioning and maintenance tool for Schneider Electric industrial automation devices. EADM can be used to commission and maintain EcoStruxure Automation Expert devices, including 'out-of-the-box' pre-configuration settings (for example, setting the device IP address, name, and so forth) and also for secure firmware management of the device.

Use the EADM tool in the following scenarios for an EcoStruxure Automation Expert device :

  • Initial configuration

  • Firmware update

Initial Configuration of a Device

New 'out-of-the-box' EcoStruxure Automation Expert devices need to be configured with valid network settings before they can be used with the EcoStruxure Automation Expert software. Applying these initial settings is referred to as pre-configuration. Using the EADM tool you can perform the following tasks relating to connected devices:

  • Discover the device in network (using IPv6 and DPWS capability of EcoStruxure Automation Expert devices)

  • Setting the network parameters of the target device (the available network parameters depends on the capabilities of the device and number of Ethernet ports available, in which case the network parameters will be available for each Ethernet port/Network interface), the parameters that can be configured are:

NOTE:

Assignment Mode: This parameter defines how the Device gets his IP, Possible options are:

  • Stored: Device uses stored address (manually configured).

  • DHCP using Device Name: Device gets IP Address from an DHCP server based on the device name.

  • BootP using MAC: Device gets its IP address from an BootP server based on the MAC Address of the Device.

  • DHCP using MAC: Device gets its IP address from an DHCP server based on the MAC Address of the Device.

Assignment Mode Apply: This parameter defines how the Device applies the configuration

  • Save: Device saves the configuration when set from EADM and the new configuration is applied once the device is restarted manually.

  • Save and Apply:  Device saves the configuration when set from EADM and the device is rebooted to apply the new configuration.

To pre-configure the EcoStruxure Automation Expert devices:

Step

Action

1

Confirm that you have connected the device that needs to be configured on the same network as the EADM tool (some anti–virus and firewall software may block the discovery. If you are unable to discover the device, disable the firewall.) Select the Discovery icon in the header area of the EADM tool (for detailed information on how to use the discovery function, refer to the EADM user manual).

Upon completion of discovery the device is listed in the Device List.

2

Click the User icon in the listed device, and in the Device login dialog, enter the credentials (Device User Name: installer; Device password: Inst@ller1). Then click Save.

The device status should be now online (indicated by the small icon ). The device is now ready for configuration.

3

Select the Configuration option by clicking the browse button in the upper right corner. This opens a context menu for Preconfiguration, with the following items:

  • Optical:  Allows you to locate the device physically in the network, the Device LED will blink to indicate the selected device.

  • Optical and Acoustical:  Allows you to locate the device physically in the network, the Device LED will blink to indicate the selected device along with an audible sound.

  • Properties: Brings up the dialog for configuring the Pre-Configuration parameters.

4

Click Properties to open the pre-configuration dialog, which can be used to configure the device Network and Interface parameters.

5

Click OK to finish the Pre Configuration, the configuration will now be send to the device. Restart the target device for the new configuration to take effect.

Updating Device Firmware

The EcoStruxure Automation Expert devices support secure firmware packages and secure firmware transfer capabilities. The EADM tool allows you to manage the different firmware packages and easily flash the target devices with the correctly specified firmware packages. EADM supports new firmware packages, called as SEDP packages. To update the target device, you need to get a valid SEDP package supported by the target Device. SEDP packages supported by EADM include:

  • M580 distributed controller

  • M251 distributed controller

  • M262 distributed controller

  • ATV dPac

For detailed information on how to use the above features, please refer to the EcoStruxure Automation Device Maintenance Firmware Upgrade Tool Online Help (EIO0000004033), which is available after installing the tool.

To update device firmware:

Step

Action

1

Confirm that you have connected the device that needs to be configured on the same network as the EADM tool. (Some of the anti–virus and firewall software may block the discovery. If you are unable to discover the device, disable firewall.) Select the Discovery icon in the header area of the EADM tool (for detailed information on how to use the discovery function, refer to the EADM user manual).

Upon completion of discovery the device is listed in the Device List.

2

Click the User icon in the listed device, and in the Device login icon in the listed device, and in the Device login dialog, enter the credentials (Device User Name: installer; Device password: Inst@ller1). Then click Save.

The device status should be now online (indicated by the small icon ). The device is now ready for configuration.

3

In the Data Package list for the device, select the required Firmware version. (Refer to the EADM user manual on how to add new Device Packages).

4

Click Update Firmware at the bottom of the Main view. The EADM tool will update the target device with the new Firmware and notify once done.

NOTE: Restart the device to apply the new firmware
