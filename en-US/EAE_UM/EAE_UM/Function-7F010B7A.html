Function

Use the Function node to create your own custom functions. EcoStruxure Automation Expert automatically includes your custom functions in the lists of functions and function blocks available throughout your solution.

Custom functions can be created only as Structured Text functions.

Context Menu Commands

Use the Function node’s context menu to create a new custom function (New Item...), create a New Folder for functions, Rename or Remove functions, Paste copied functions, and Export and Import functions.

Use a functions’s context menu to Open, Rename, Export, Copy, and Remove functions, and to Find all references of the function in your solution.

Creating a New Function

To create a custom function:

  1. Right click on the Solution > Function node and select New Item...

  2. In the File Wizard, input (or accept the default) function Name.

    NOTE: The Type (Function) and Language (Structured Text) are pre-defined.

    A new tab is created for the new function.

  3. In the new function tab, enter the elements that define the new function:

    • Create the input and output variables for the new function.

    • Add any required structured text code for the new function

  4. Save your edits.
