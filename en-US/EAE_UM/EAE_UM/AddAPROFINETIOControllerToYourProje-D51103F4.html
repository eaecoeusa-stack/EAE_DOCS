Add a PROFINET IO-Controller to your Project

After adding the required library references to your project, follow these steps to add a PROFINET IO Controller to your project:

  1. Click on the System File icon in the quick access toolbar to open the System Editor.

  2. Open the Hardware Configuration editor.

  3. Navigate to the target resource, for example, Devices > DEV > RES0.

  4. Right click and select Add.

  5. In the Add Bus dialog:

    • Select Element Type: Bus.

    • Select Type: Standard.IoProfinet.PN_IOController.

    • Type a Name for the controller, for example, PN_IOController01.

    • Click Add.

The PN_IOController object is added to the Hardware Configuration editor.
