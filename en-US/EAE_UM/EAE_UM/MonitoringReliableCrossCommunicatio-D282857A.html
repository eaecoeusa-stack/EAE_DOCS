Resource Statistics

Path Value Description

Network.CrossCommunication.ResourceUtilization.

Index

Reliable Cross Communication resource Index.

Network.CrossCommunication.ResourceUtilization.

Name

The resource/freelist name.

Network.CrossCommunication.ResourceUtilization.

Size

The capacity of the resource (bytes, freelist entries, …)

Network.CrossCommunication.ResourceUtilization.

Used

The maximum during the last period.

Network.CrossCommunication.ResourceUtilization.

MaxUsed

The maximum utilization since the start of the project, i.e. since the controller started.

Network.CrossCommunication.ResourceUtilization.

UpdateCounter

Measure of change to the resource utilization during the last period.


NOTE:

  • The last period property is defined in the Logical Devices editor at Communication > ReliableCrossComm > Monitoring > EvaluationIntervalSec.
