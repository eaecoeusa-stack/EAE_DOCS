5. Sync AutoMapped CATs to AVEVA System Platform

Prerequisites

  • The Sync AutoMapped AVEVA SP Templates command is available only if you select both of the following when installing EcoStruxure Automation Expert:

    • AVEVA Industrial Graphics

    • Asset Link for AVEVA OMI.

  • The latest patch of the AVEVA System Platform version 2023 R2 SP1 is installed.

    NOTE: To know the what is the latest patch of the AVEVA System Platform 2023 R2 SP1 supported with EcoStruxure Automation Expert, refer to the Software Compatibilities topic of the Compatibilities and Dependencies - User Guide.

Procedure

To sync Automapped CATs and EcoStruxure Automation Expert graphics to the AVEVA System Platform, follow these steps:

  1. Select CAT > Application > Sync AutoMapped AVEVA SP Templates to create Galaxy ASP templates that include the mapped attributes, the enabled State alarm for the selected attributes, plus associated AVEVA Industrial graphics. This may take a while, and progress is shown in the Output window.

  2. (Optional) Check the AVEVA System Platform to confirm that the attributes appear in the ASP Template Toolbox, and the graphics appear in the Graphic Toolbox tab.

NOTE: All these steps are similar for CAT Composite instances and SubApp instances.
