Adding EtherNet/IP Devices to the Physical View

  1. In the Physical devices editor, drag and drop an EtherNetIP compatible device to the View.
    Example: ATV6 or EtherNetIP device.
    Result: The device properties are displayed and the DTM property is available. If a specific DTM is required for the device, the Vendor and the DTM Name details are filled.
  2. Configure the properties for the devices added to the Physical devices, including:

    • Name

    • IP Address

    • Logical Network

  3. If the device requires a communication card (for example, ATS480, or ATV6) to use EtherNet/IP, configure the Communication Card property with an ATV Communication card.
  4. Right-click the device and select Open configuration editor.
    Result: The DTM catalog is updated, if not up to date, then the EtherNet/IP Configurator opens.
