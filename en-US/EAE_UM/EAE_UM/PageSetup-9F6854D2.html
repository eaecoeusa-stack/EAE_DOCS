Page Setup

Use the Page Setup dialog to select settings for printing pages from editors in EcoStruxure Automation Expert.

NOTE: Before selecting settings in the Page Setup dialog, first specify Network Editor Page Layout settings.

Network Editor Page Layout

Configure page layout template printing settings before printing editor pages. The template settings you select will provide options for, and apply to, the editor pages you subsequently print.

To access the configurable editor page template, click the File menu, then in the Backstage select Settings and navigate to IEC Network Editor > Network editor page layout > Page layout. Template settings include:

Page Layout Page Layout

Select the desired Page layout from the list:

  • A pre-defined page layout, e.g. A3h, A3q, A4h, A4q (where 'q' stands for horizontal orientation and 'h' for vertical). The Page size settings are automatically set.

  • None

    . The editors will not present a print preview as no page size is defined.

  • Custom. The Page size settings are configurable. Expand Page Layout > Page size and choose the desired units, width, and height of the page.

After the desired format has been selected, the page layout preview in the editor can be switched on and off using the Show/Hide Print Layout icon  on the toolbar.

This is a printer software-independent page template. That is, the pages are printed as shown on the screen. The aspect ratio is preserved.

NOTE: The Navigator icon at the upper right corner of the editor reacts according to the page layout settings when the page layout preview in the editor is activated by the Show/Hide Print Layout icon .

Refer toFile menu for how to begin printing.

Page Setup

Open the Page Setup dialog from the Home menu in the Print group. It serves to set the page layout for the print.

The settings for orientation and margins (accessed via the Home menu in the Print area) are effective in any event, even in page layout 'None'.

If a page layout has been defined, such as ' A4h ', then a dialog which allows the settings for print scale opens first.

Description of the settings in dialog Page Setupin window Scaled Print:

Command

Description

Scale

The field Scale allows a direct indication of the scale of the print. It is available only when the check mark was removed at Scale to Fit Media.

Scale to Fit Media

Selects the appropriate scale automatically, according to the selected page size.

Auto Rotate Page

Rotate the page automatically to match the selected orientation.

Split Page

Specifies whether the page is divided to several pages with complete contents printed, or to print only the part of the relevant page which fits with the current setting ( scaling , margin definition , etc. ) to each leaf. In the latter case those parts are cut that do not fit on the sheet.
