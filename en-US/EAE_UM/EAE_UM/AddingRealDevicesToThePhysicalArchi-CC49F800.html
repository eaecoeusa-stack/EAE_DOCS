Adding Discovered Devices to the Physical Architecture

One way to commission the physical architecture and reconcile it to the discovered architecture, is to use EcoStruxure Automation Expert to add discovered devices directly to the physical architecture of the project.

In this way, you can automatically create devices in the Physical view devices from the discovered devices from the Discovery view. The created devices are set with some parameters of the discovered devices.

To add a discovered device::

  1. Discover the devices.

  2. In the Discovery view of the Physical devices editor, select the discovered device to be added.

  3. Select the command Add device in the project.

This command adds to the top left of the Physical view a device with the same type as the selected discovered device. That device has its Name, Part Number and IP address set to the same values as the parameters of the selected discovered device.

Adding Command Rules:

  • This command is only available for the following devices: M251d, M262d, M580d, ATV6xx, ATV9xx, ATV340x, ATS480, TeSysT and the BMECRD0100.

  • The name given to the added device is the same as the discovered device. If that name already exists, the name given will be an incremented version of it.

  • All the IP addresses received from the discovered device will be added to the configured device.

  • The command is disabled if a device of the same type with the same IP address already exists in the Physical view.

After these steps are completed, and these rules followed, the real device will be added to the physical architecture. The final step for communication with the real device is to set security of the devices.
