EtherNet/IP Fieldbus Scanner

With the EtherNetIP fieldbus scanner feature, you can integrate an EtherNetIP fieldbus scanner into an EcoStruxure Automation Expert application. Two types of EtherNetIP scanners are available:

  • EIPSCANNER: used to create, configure and map an EtherNetIP fieldbus scanner from the Hardware Configuration.

  • EIPSCANNER2: used to map the devices configured in the physical view, to an EtherNetIP fieldbus scanner.

The Standard.IoEtherNetIP.EIPSCANNER is the scanner used also in earlier EcoStruxure Automation Expert versions.

The Standard.IoEtherNetIP.EIPSCANNER2, offers facilitated configuration and change management. However, this scanner is not supported by some configurations (for example, with BMECRD0100).

Prerequisites

Before configuring an EtherNet/IP fieldbus scanner for your solution, the following actions are required:

  1. Add the Standard.IoEtherNetIP library as a Project Reference for your solution. Only libraries that have been added to the list of installed libraries are available to be added as project references:

    NOTE:
    If you change the version of the Standard.IoEtherNetIP library from 21.1.x to 21.2.x, you need to force a rebuild by:

    1. Making any change in the application.

    2. Executing the Build command.

    If you do not force a rebuild, up-versioning the Standard.IoEtherNetIP library has no effect on the deployed solution.

    Refer to the programming manuals of the EtherNet/IP devices for information on the capabilities of the devices and the properties that can be accessed.

  2. Install and reference the SE.FieldDevice library in your solution.

  3. Install on the PC the DTMs of the Schneider Electric devices added to the physical view, and any required specific device DTMs, or import EDS files.
