Edit Custom Layout

In this topic is described the different element from Edit custom layouts dialog box. This dialog box is used to edit the custom layouts available in the View menu.

NOTE: To know more how to edit custom layouts, check the topic Handle Custom Layout.

  1. In this section are listed all the custom layouts already created, each of them having a unique name. One line per custom layout.

  2. The new custom layouts are created by clicking in this line.

  3. This button deletes the selected custom layout.

  4. These buttons moves in the list the order of the custom layouts.

  5. This button applies the modifications made in this dialog box and closes it.

  6. This button cancels the modifications made in this dialog box and closes it.
