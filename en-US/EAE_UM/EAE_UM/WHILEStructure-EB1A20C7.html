4. WHILE Structure

Structured text defines a WHILE-loop as a control structure, with which a group of statements (ST statements) can be repeated until a control condition becomes TRUE. When <expression> already produces FALSE on the first call, the statements are not executed.

Syntax of the WHILE control structure: 


WHILE <expression> DO
  <ST statements>;
END_WHILE;
Example 1: 
WHILE index < value DO
  out0 := out0 + 1;
END_WHILE;
Example 2: 
(* Statements will never execute *)
WHILE index <> index DO
  out0 := out0 + 1;
  index := index + 2;
END_WHILE;
