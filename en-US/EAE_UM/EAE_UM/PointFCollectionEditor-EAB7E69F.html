3. PointF Collection Editor

Use the PointF Collection Editor to edit the points of an element such as e.g. Poly Line, Pipe, Polygon. Access this dialog by selecting the element in the graphic editor, then clicking the Misc > Points ellipsis () in the Properties pad. The PointF Collection Editor includes the following items:

Element/Command

Description

Members:

The existing points are listed here and the order of the points defines how to connect them. Use the arrow buttons on the right edge of the field Members to change the order of the entries.

The coordinates of the points are merely listed here. If you want to edit them use the x,y properties area.

Add

Adds a point to the end of the list, with the coordinates 0,0.

Remove

Deletes the selected point from the collection Members.

Use the arrow buttons to change the order of the entries. The currently selected entry will be moved one position into the direction of the arrow on the actuated button.

x,y properties:

Shows the coordinates of the selected point. To adjust the coordinates directly just edit the desired values here in the field on the right side of X resp. Y.

View of properties in category group.

View of properties in alphabetical order.

OK

Closes the window and saves the modifications.

Cancel

Closes the window without any modifications.
