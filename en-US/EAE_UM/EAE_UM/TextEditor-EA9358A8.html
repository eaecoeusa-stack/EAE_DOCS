3. Text Editor

These settings relate to character-based source code files (e.g. *.cnv.cs).

Topic

Item

Description

General

General Options

Word wrap : enables/disables text line wrapping

Folding : enables/disables the fold (i.e. collapse) code regions

Enable Change Marker Margin : enables/disables margin for marking text changes

Show Quick ClassBrowser Panel : enables/disables Browser panel at the upper edge of the editor

Show line numbers : enables/disables line numbering

Show hidden definitions : displays hidden definitions

Font

Contains font setting and enable/disable the possibility of zooming within editor by using Ctrl and mouse wheel

Markers and Rulers

Markers and Rulers

Show spaces : shows points for spaces inserted before the beginning of a code line

Show tabs : shows a double greater-than sign (>>) for each tab stop before the beginning of a code line

Show EOL markers : enables/disables end-of-line markers

Underline errors : shows a red wave line beneath detected error if marked

Highlight matching bracket : highlights corresponding brackets, whereby you can choose whether to have the cursor located before or after the highlighted bracket (the caret will be located similarly)

Highlight current line: none or yellow highlighting the full line

Highlight symbols: inserts a thin vertical line at the desired column

Enable animations : none or yellow highlighting the full line

Show column ruler at column # : hides or shows column ruler at selected column #

Behavior

Tabs

Adjustment for tabulator positions

Behavior

General text editor behavior, including:

Mouse wheel zooming: mouse wheel behavior

Hide mouse cursor while typing : mouse cursor can be hidden, only caret shows the position where the next character will be inserted

Cut or Copy entire line when nothing is selected : uses the entire line where cursor is positioned if no selection has been made

Enable virtual space :

Use Ctrl + Click for GoTo definition : key combination to jump to definition of selected screen object

Auto insert curly braces : inserts the corresponding close bracket for every opened bracket automatically

Code Completion

Enable/disable code completion, the automatically displaying of input help for the code. This concerns the C# files of symbols and faceplates, Structure Text files of Basic function blocks and XML files.

NOTE: Ctrl + Space opens the code completion window manually while typing the code in C# and Structured Text.

Code completion detail settings

detailed settings for the use of code completion

Tooltip settings

Selections for tooltip behavior for code.
