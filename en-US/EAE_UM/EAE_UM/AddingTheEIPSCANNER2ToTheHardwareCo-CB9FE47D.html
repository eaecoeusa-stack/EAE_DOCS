Adding the EIPSCANNER2 to the Hardware Configuration

  1. Open the Hardware Configuration editor.
  2. Right click the editor background and select Add from the context menu to open the Add Bus dialog.
  3. Select the Standard.IoEtherNetIP.EIPSCANNER2 as the Type and click OK.
    NOTE: The EIPSCANNER2 selection is available only if you referenced the Standard.IoEtherNetIP library.
Result: An EIPSCANNER2 is added in the Hardware Configuration editor, which presents the following structure:

  • autoStart

  • enableSymlinkOC

  • phase

  • busCycleTime

  • busCycleTolerance

  • busCycleActionWhenMissed
