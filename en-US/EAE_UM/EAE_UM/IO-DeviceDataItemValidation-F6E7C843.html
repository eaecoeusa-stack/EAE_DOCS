IO-Device Data Item Validation

The PROFINET Configurator validates each data item and the entire collection of data items. Validation displays an explanatory message for each of the following detected configuration errors:

  • Invalid length of a single data item (values outside the supported range of 1 to 1024 bytes).

  • Invalid length of the collection of all data items (value greater than 1440 bytes).

  • Invalid Symlink name (the assigned name includes one or more non-supported characters).

  • Maximum number of data items exceeded (greater than 32 data items).

If the PROFINET Configuration detects an error in a device configuration, the detected error icon ( ) appears next to the Advanced... command in the Network Configuration main page.
