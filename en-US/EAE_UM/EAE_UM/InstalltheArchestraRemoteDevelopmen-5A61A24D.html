Install the Archestra Remote Development in Client PC

To install the Archestra remote development in the client PC:

  1. Run the latest patch of the AVEVA System Platform 2023 R2 SP1 installation wizard in the same PC where EcoStruxure Automation Expert is installed.
    NOTE: To know the what is the latest patch of the AVEVA System Platform 2023 R2 SP1 supported with EcoStruxure Automation Expert, refer to the Software Compatibilities topic of the Compatibilities and Dependencies - User Guide.

  2. Choose the additional option AVEVA Application Server Development to install.

  3. Click on Next and check the components that will be installed.

  4. Click on Next and process through all the remaining steps to complete the installation, including the AVEVA configure wizard.

NOTE: For further details about the installation process, check the Installation Guide of the latest patch of AVEVA System Platform 2023 R2 SP1.

Once done, configure the Archestra Remote Development in the client PC by:

  1. Open the Configuration executable.

  2. Configure the following two sections:

    • System Management Server by connecting to an existing System Management Server (first option). Here enter the Galaxy Repository server name.

    • AVEVA Entreprise License Server by setting all the required information.
