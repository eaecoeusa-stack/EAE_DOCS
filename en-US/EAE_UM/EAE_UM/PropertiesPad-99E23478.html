Properties Pad

The Properties pad makes it possible to view and edit properties and events for a selected object. In the main menu, select  View > General pads > Properties to open this pad.

The available options for editing and viewing vary, depending on the context, e.g. IEC Editor or HMI Editor.

To clear or reset the value of an overwritten option, right-click the option name and select Reset in the context menu.
