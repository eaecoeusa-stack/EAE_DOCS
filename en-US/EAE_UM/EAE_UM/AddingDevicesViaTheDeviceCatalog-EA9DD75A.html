Using the Device Catalog

To design your physical network, the Physical devices editor gives access to the device catalog, containing a list of supported products.

The device catalog is organizing the production, depending on their type and their range. All the devices with the same purpose are then gathered at the same place — in the same Product Type and then in the same Product Range. For example, all the EcoRT devices are located together, inside the group type Distributed Automation Controllers and inside the group range Modicon.

  1. Expands the device catalog to display all product types and product ranges.

  2. Collapses the device catalog. Only the product types are displayed.

  3. Type in a value to display only devices with that value in the identifier, version, or description field.

  4. Toggles on and off the advanced sorting and filtering tools, described as items 5, 6, and 7.

  5. Toggles the device catalog display by product type:

    • Arrow pointing up sorts in ascending alphanumeric order.

    • Arrow pointing down sorts in descending alphanumeric order.

  6. Toggles the device catalog display by product range

    • Arrow pointing up sorts in ascending alphanumeric order

    • Arrow pointing down sorts in descending alphanumeric order

  7. Type in values used to filter the display on the combined bases of identifier, version and description.

  8. Opens a list of value delimiters for filtering the list of devices, including:

    • Is equal to

    • Is not equal to

    • Contains (default)

    • Does not contain

    • Starts with

    • Ends with

    • Is null

    • Is not null

    • Is empty

    • Is not empty

    Select a delimiter to apply the filter. 

  9. Clears the filter.

  10. Product type level group.

  11. Product range level group

The catalog presents the following information for each device:

  • Identifier

  • Version

  • Description

Adding a Device

To add a device:

  1. Select the device in the Device Catalog.

  2. Drag it to and drop it into the workspace area.

Copying an Existing Device

To copy a device in the workspace area:

  1. Select a device.

  2. Right-click and select Copy.

  3. Right-click anywhere in the workspace area and select Paste.

NOTE: When you copy and paste a device, you also copy all the device properties, including the IP address. Only the device Name, which is generated automatically, is changed.
