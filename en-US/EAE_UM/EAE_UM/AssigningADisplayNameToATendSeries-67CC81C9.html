Assigning a Display Name to a Trend Series

You can assign a display name to a Trend series, which will be displayed during runtime in the canvas that contains the Trend control.

  1. In Solution Explorer, navigate to the symbol object that contains the Trend series. For example:

    <solution name> > CAT > Application > CAT1 > iThis : CAT1_HMI > sDefault

  2. Select the Trend control to show its Properties.

  3. Select Misc > Series and click the ellipsis (...).

  4. In the TrendSeries Collection Editor:

    • In the Members area, select a member.

    • In the right pane, enter a Legend Name.

      NOTE: The Legend Name can be created in a localized language, and displayed in the HMI in that language.

  5. Repeat step 4. for each Trend series member you want to assign a display name.

  6. Click OK.
