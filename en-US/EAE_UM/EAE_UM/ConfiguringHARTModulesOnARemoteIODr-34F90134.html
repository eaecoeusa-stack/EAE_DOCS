Configuring HART Modules on a Remote I/O Drop

The BMEAHI0812 and BMEAHO0412 Mx80 modules can be added only to the local remote I/O backplane of a Soft dPAC or an HA Soft dPAC system with a BMECRD0100 adapter module.

To configure BMEAHI0812 and BMEAHO0412, perform the following tasks.

Build the Remote I/O Drop

  1. In the Hardware Configuration editor, build one or more RIO drops by adding at minimum:

    • Ethernet Backplane (for example SE.IoX80:BMEXBP0800)

    • Power supply (for example SE.IoX80:BMECPS3500)

    • Adapter module (SE.IoX80:BMECRD0100)

    • HART input module (SE.IoX80:BMEAHI0812)

    • HART output module (SE.IoX80:BMEAHO0412)

  2. Enter the rack address (rackAdr) of the backplane.

  3. Enter the slot number (SlotNr) of each I/O module.

  4. Enter a sufficiently large Bus Master Cycle Time (busCycleTime for SE.HwCommon:BM_RIO), for example, 100ms.

  5. Enter Promptness values for each I/O module.

    NOTE: Each I/O module Promptness value needs to be larger than the Bus Master Cycle Time.

Configure IP Addressing for each BMECRD0100 Adapter Module

  1. Right click on the SE.HwCommon.BC_RIO node and select Properties...

  2. Input the IP Address in two places:

    • DeployURI

      NOTE: Overwrite only the IP Address portion of this value. For example, if the default settings are 'ws://127.0.0.1:51499' replace only the '127.0.0.1' with the module’s actual IP Address..

    • connectinfo > ipaddress.

  3. Click OK.

Configure the EtherNet/IP Scanner DTM

As a pre-requisite, install the Schneider Electric HART Gateway DTM, version 1.1.20 or later, on your PC.

  1. Add an EtherNet/IP Scanner (Standard.IoEtherNetIP:BC_RIO_EIP).

  2. Right click on this node and click Configure EtherNet/IP Scanner...

    The EtherNet/IP Configurator opens.

  3. In Network View, select EtherNet/IP scanner.

  4. Select Select Channel Properties > TCP/IP.

  5. Configure the IP address settings of the BMECRD0100:

    • Module IP Address

    • Sub-Network Mask

    • Gateway IP Address

  6. Click Apply.

Add the BMEAHO0412 and BMEAHI0812 DTMs

  1. In the Network View, right click EtherNet/IP Scanner and select Add.

  2. Select BMEAHI0812 and BMEAHO0412 and click OK.

  3. Select Device List > BMEAHO0412.

  4. In the Address Setting tab, enter:

    • IP Address

    • Identifier: In the format AAA0_NN_AHO0412, where:

      • AAA is the alpha-numeric user-defined Rackid

        NOTE: The Rackid value for each BMECRD0100 remote I/O drop must be unique, otherwise the remote I/O drop becomes inoperable.

      • 0 is a required character

      • NN is the slot number (decimal)

      • AHO0412 is the part number

      NOTE: The Identifier value for each BMEAHO0412 module must be unique. If modules with duplicate Identifier values are added, the remote I/O drop becomes inoperable.

  5. Select Device List > BMEAHI0812.

  6. In the Address Setting tab, enter:

    • IP Address

    • Identifier: In the format AAA0_NN_AHI0812, where:

      • AAA is the alpha-numeric user-defined Rackid

        NOTE: The Rackid value for each BMECRD0100 remote I/O drop must be unique, otherwise the remote I/O drop becomes inoperable.

      • 0 is a required character

      • NN is the slot number (decimal)

      • AHI0812 is the part number

      NOTE: The Identifier value for each BMEAHI0812 module must be unique. If modules with duplicate Identifier values are added, the remote I/O drop becomes inoperable.

  7. Click Apply.

Add HART Device DTMs

  1. In the Network View right click a HART module, either the BMEAHI0812 or the BMEAHO0412, then select Add.

  2. Navigate the list of available HART devices, then click OK.

  3. Right-click the HART device, and in the Channel Selection dialog select the channel to use, then click OK.

  4. Repeat steps 1 - 3 for all added HART devices.

Configure the BMEAHO0412 DTM

  1. In Network View, select EtherNet/IP scanner > BMEAHO0412.

  2. Select Process Data, then enable (select) the used channels and disable (de-select) the unused channels.

  3. Click Apply.

Configure the BMEAHI0812 DTM

Repeat the steps described in the previous section for the HART input module.

Configure HART Device DTMs

  1. In the Network View double-click a HART device previously added.

  2. Configure the device-specific settings for the HART device.

  3. Repeat steps 1 - 3 for all added HART devices.

Export and Import Configuration

  1. Close the EtherNet/IP Configurator and during the Save, Export the configuration to a zip file.

  2. In the Hardware Configuration editor, select the Standard.IoEtherNetIP:BC_RIO_EIP node.

  3. Right click on this node and click Import EtherNet/IP Scanner....

  4. Navigate to and select the exported configuration zip file, then click Open.
