Sequential Control

Introduction

Sequential control enables step-by-step execution of a process with control passing from one state to the next state depending on conditions. Sequential control can be used, for example, to describe the manufacturing of products as event-controlled processes (recipes).

In EcoStruxure Automation Expert, sequential control applications are created by combining two CATs:

  • The State Manager which allows defining the states and their relationships.

  • The Procedural CAT which allows associating sequences with the different states of the State Manager.

The overall procedure is as follows:

  1. Create the State Manager CAT and paste the states diagram xml code therein.

  2. Create the Procedural CAT and edit the sequence charts therein.

  3. In the Procedural CAT, define the actions attached to steps and define the conditions for transition (see Transition Editor).
