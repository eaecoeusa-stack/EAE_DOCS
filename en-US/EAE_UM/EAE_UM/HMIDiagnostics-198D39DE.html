Diagnostics

The diagnostic mechanism offers a standard way to diagnose the EcoStruxure Automation Expert system. In this version, diagnostic information is supported only for Soft dPAC Linux.

This diagnostic is given through a Diagnostics Grid instance displayed on a canvas. Once this canvas running, the Diagnostic grid on the HMI will automatically display the current diagnostics emitted by the configured devices.

MQTT is a lightweight publish-subscribe protocol where a central broker is responsible for collecting messages from all publishers and propagating messages to all subscribers.

Thus, to perform the Soft dPAC devices diagnostics on EcoStruxure Automation Expert, follow these actions:

  • Verify that the MQTT broker used is meeting the requirements.

  • If no MQTT broker is available, set one.

  • Set up in EcoStruxure Automation Expert to get the diagnostics.

NOTE: To know more about the MQTT protocol, please check the official MQTT website here.
