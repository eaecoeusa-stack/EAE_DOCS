Resource Editor

Use the Resource Editor to view, create, delete or modify elements that are mapped to this resource (from an application) or created directly in this resource. Connections which have been created in an application are also shown in this view if the function blocks are mapped to this resource. Elements that are added directly to this resource are also assigned to this resource.

A description of the general functions of the toolbar can be found in the Function Blocks Network editor.

This editor opens if you:

  • Double-click a resource within the System node of the solution in the Solution Explorer (open the node of the appropriate device first).

  • Double-click an item in the Logical Devices editor. 

  • Click on a device in the Device editor.

Change to the respectively next level by double-clicking on an object or on the background.

Click the Mapping button on the upper border to display the sub view mapping.

Resource Editor Context Menu

A right-click on the background within the editor opens a context menu:

Command

Description

FB

Insert function blocks from available libraries via submenus.

CAT

Insert CATs from available libraries via submenus.

Frame

Insert frames or comments. Frames are useful to hold the project clearly arranged and function blocks located in the same frame can also be moved together (see Editors Overview).

Layers

  • Show communication channels. Opens the Communication Network editor).

  • Show HW configuration network: Opens the view for the hardware configuration network (see HW Configuration Network).

Auto Layout

Arranges the function blocks and automatically creates frames among FBs belonging together (for information about frames see Editors).

Go To ***

Switch to current device editor, that means one level up (this can also be effected by a double-click on the background of the Resource editor).

Go To Definition

Switch to respective resource type.

Attributes

Opens the  Attributes dialog.

Redraw connections

After moving the position of one or several connections between function blocks, click Redraw connections to return the connections to their default position, i.e., the position EcoStruxure Automation Expert automatically drew when you first created connections between function blocks.

Resource Editor Function Block Context Menu

A right-click on one or several selected function blocks within the Resource Editor opens a context menu. Some entries may vary depending on the project and status of the project (deployed and logged in or not, etc.).

This context menu is described in Function Blocks Network editor.
