Boot Project Impact on Action Commands

In a device, the stored boot project can be the same as – or different from – the currently loaded project. EcoStruxure Automation Expert - Buildtime notifies you of the comparative state of the boot project and the loaded application by means of icons in the Deploy and Diagnostic editor.

You need to consider this because, after a firmware restart or a device power cycle, the device will execute the boot project program which is potentially different from the program previously executed, thereby directly impacting operations. In this case, EcoStruxure Automation Expert displays a dialog for actions that would result in a logout. This presents you an opportunity to resolve a boot project discrepancy before logout.

 WARNING
UNINTENDED EQUIPMENT OPERATION
Verify that the Boot projects set in the devices will not cause unwanted effects on equipment operation if they become active (after a firmware restart or power cycle).
Failure to follow these instructions can result in death, serious injury, or equipment damage.
