EcoStruxure Automation Expert Licenses

The use of EcoStruxure Automation Expert requires the activation of licenses to unlock and access all features.

The following license types are available:

  • Trial

  • Engineering (Lite, Standard and Professional)

  • Addon

  • Application

NOTE: In the absence of a valid trial or Engineering license, the product can be launched but solutions cannot be opened nor created.

Trial License

Each EcoStruxure Automation Expert version includes a new trial license that has the same version number as the software.

The trial license:

  • Is automatically activated with the first launch of the installed EcoStruxure Automation Expert version.

  • Expires after 42 days (remaining time is displayed in the About page and in the License Manager).

  • Unlocks the use of basic product features to create control applications deployable in simulation mode (Local Test profile in Deploy and Diagnostic editor). The deployment on real devices (Default profile) requires an Engineering license.

Engineering License

Three types of Engineering licenses are available with EcoStruxure Automation Expert, Lite, Standard and Professional. The Engineering licenses unlock the use of basic product features to create control applications deployable in simulation mode (Local Test profile in Deploy and Diagnostic editor), as well as to physical devices (Default profile).

The Lite Engineering license is designed for simple solutions and has following limitations on the basic product features:

  • Soft dPAC High Availability can not be used, there are no limitations for the other device types.

  • Custom libraries cannot be created.

The Standard Engineering license has no limitations on the device types to be used for a control application. In addition to the basic product features, optional Addon licenses can be purchased and activated to unlock these special product features for the Standard Engineering license.

The Professional Engineering license offers the same features as the Standard Engineering license, and it unlocks also all the special product features of the Addon licenses.

NOTE: The Lite Engineering license is not backwards compatible with EcoStruxure Automation Expert versions earlier than V24.1.
NOTE: The Professional Engineering license is not backwards compatible with EcoStruxure Automation Expert versions earlier than 23.0.

Addon License

Addon licenses unlock specific product features in addition to the basic product features that the Standard Engineering license unlocks. The Professional Engineering license includes and unlocks also all the specific product features of the Addon licenses.

The following product features of EcoStruxure Automation Expert need an Addon license to be able to unlock and use them:

  • Asset Link for AVEVA OMI.

  • High Availability

  • Procedural Automation: Using procedural automation functionality in a control application requires a new Addon license starting with EcoStruxure Automation Expert V24.1.

NOTE: The product feature Asset Link for Bulk Engineering needs no more an Addon license starting with EcoStruxure Automation Expert V24.0 because it is only a preview feature.

Application License

Application licenses are required for each solution that is created with EcoStruxure Automation Expert. Contact your local Schneider Electric sales team if you need help to understand which Application licenses are needed.

Starting with EcoStruxure Automation Expert V25.0, activate also your Application licenses, otherwise there is shown a license non-compliance notice if these licenses are not available. You can activate the Application licenses in the same way as the Engineering and Addon licenses.

NOTE: If you want to work from multiple Engineering Workstations on the same EcoStruxure Automation Expert solution then activate the Application licenses on a License Server.
NOTE: If you activate the Application licenses as node-locked licenses, the License Manager is showing one line per Activation ID for the license. Please ensure that the corresponding number of Application or Device licenses are purchased for the number of IO Devices contained in your solutions. This can be checked in your Entitlement emails or in the Software Management portal (https://licensing.se.com/) which both show the number of copies of your Application or Device licenses.

Managing Engineering, Addon and Application Licenses

Engineering, Addon and Applications licenses can be purchased at Schneider Electric. They are delivered to your e-mail address after purchase, and come with a license certificate that includes your Activation IDs.

Depending on your needs, each copy of your Engineering, Addon and Application license can be used as:

Node-Locked License

To use your license as a node-locked license, you need to activate it on your Engineering Workstation.

The activation process can be done either with:

Online Activation Method

This is the recommended and the fastest license activation method and can be used if the Engineering Workstation with EcoStruxure Automation Expert can access the Internet.

Offline Activation Method

This method can be used only if your Engineering Workstation with EcoStruxure Automation Expert cannot access the Internet. Offline activation requires the use of a second PC with access to the Internet and the ability to reach the Offline License Processing web portal.

After the license is activated on your Engineering Workstation, EcoStruxure Automation Expert can be used with or without Internet connection on the workstation.

If the license needs to be moved to another Engineering Workstation, it has to be returned to the Schneider Electric License Server and then reactivated on the other Engineering Workstation.

Floating License

To use your license as a floating license, you need a License Server to be able to share it with multiple Engineering Workstations. Multiple Engineering, Addon or Applications licenses can be assigned on a configured License Server.

Floating licenses can be borrowed or reserved for your Engineering Workstation from the License Server.

There are two available types of License Servers:

Cloud License Server

This License Server is hosted in a cloud and handled by Schneider Electric. The benefit of using a Cloud License Server is that you do not need to install and maintain it. A permanent Internet connection on the Engineering Workstation is required to borrow floating licenses from the Cloud License Server. Only a temporary connection to the Internet is required during the reservation and the release of floating licenses for offline use.

Enterprise License Server

This License Server must be hosted and maintained on a Windows 64-bit machine by your enterprise. The License Server must be connected to the same enterprise Intranet where your Engineering Workstations are connected. A permanent connection to the Intranet on the Engineering Workstation is required to borrow floating licenses from the Enterprise License Server. Only a temporary connection to the Intranet is required during the reservation and the release of floating licenses for offline use. The license operations on the Enterprise License Server can be done either online or offline.

For both License Servers, when EcoStruxure Automation Expert is started on an Engineering Workstation, it first verifies if valid node-locked Engineering, Addon and Application licenses are available, and then uses them. If no node-locked licenses are found, it tries to borrow one available copy of your Engineering, Addon, or Application licenses from the configured License Server. When EcoStruxure Automation Expert is closed, the successfully borrowed licenses are automatically returned to the License Server. The licenses are then available to be borrowed by any other Engineering Workstation.

NOTE: If an Addon license is needed on a specific Engineering Workstation, then activate it there as node-locked license. Otherwise, if it is activated on a License Server as floating license, it is borrowed by any Engineering Workstation that connects to the License Server when EcoStruxure Automation Expert is started.
Example of Engineering or Addon license use: With the purchase of a license with ten copies for a single Activation ID:

  • Three copies could be activated as node-locked licenses on three different Engineering Workstations.

  • Seven copies could be assigned to your License Server and be used as floating licenses on seven other Engineering Workstations at the same time.

NOTE: In this example, the number of copies and how they are used is purely arbitrary.

NOTE: After activation, the license is compatible, with any existing EcoStruxure Automation Expert version and with later versions released until August 2025.

To remain compatible and to continue using this license for future versions of EcoStruxure Automation Expert beyond the first 12 months, read the Advantage Service Plan chapter.

A permanent Engineering and Addon license is compatible with the currently used EcoStruxure Automation Expert version if the displayed compatibility is greater or equal the release version displayed at the bottom right in the License Manager. These licenses are also backwards compatible with previous EcoStruxure Automation Expert versions. The License Manager of these previous versions only displays a dash in the version column for these licenses because they have no version number.

NOTE:

  • Permanent Engineering and Addon licenses in V23 are still compatible with EcoStruxure Automation Expert 23.x and 24.x. These licenses are displayed in the compatibility column of the License Manager with 2024.12.

  • Professional Engineering licenses are not backwards compatible with EcoStruxure Automation Expert versions earlier than 23.0.

  • Lite Engineering licenses are not backwards compatible with EcoStruxure Automation Expert versions earlier than V24.1.

  • Licenses displayed with compatibility “Universal” are compatible with any EcoStruxure Automation Expert version.

  • Trial licenses still have a version number displayed in the compatibility column indicating for which version of EcoStruxure Automation Expert they are valid.

Advantage Service Plan

Advantage Service Plan is a software maintenance program, that provides access to the latest versions of EcoStruxure Automation Expert, without the need to purchase new licenses or activate update licenses for new releases.

Newly purchased permanent Engineering and Addon licenses include a 12‑month Software Assurance, that starts with the first activation date of the license. Your activated licenses remain compatible with EcoStruxure Automation Expert versions released within these 12 months.

You can purchase an Advantage Service Plan that includes Software Assurance to extend the compatibility of your licenses every 12 months and to continue using the same licenses for all future EcoStruxure Automation Expert versions.

After purchasing an Advantage Service Plan, it is necessary to update the license information to apply the new compatibility to the activated licenses.

The license information is updated for activated licenses in the following manners:

  • On a Cloud License Server, the new compatibility is applied automatically after the purchase of the Advantage Service Plan.

  • On an Enterprise License Server perform a renew operation which is described in the Enterprise License Server Guide.

  • In the License Manager, perform a renew operation by clicking the Renew button.

NOTE: In the License Manager, when the License Server is configured and connected, click the Renew button to update the license information displayed in the Compatibility column.
