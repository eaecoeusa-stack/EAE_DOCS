File Menu

Click on the File menu to access the EcoStruxure Automation Expert - Buildtime Backstage.  In the Backstage, you can access commands relating to:

  • Project and File

  • Development Environment

Backstage Commands

Command

Description

<Back Arrow>

Returns you to the EcoStruxure Automation Expert - Buildtime interface.

New

Opens the New Solution dialog, where you can create, name and save a new solution.

NOTE: Path and file name length is limited by Microsoft Windows restrictions. The fully qualified file name cannot exceed 259 characters, the path name cannot exceed 247 characters. If an error of this kind is detected, create the project with a shorter name, and/or use fewer subfolders.
Open

Presents the following tools for opening an existing solution:

  • Recent: a list of last opened solution files, displaying solution name and path.

  • This PC: a list of local PC drives for navigating to local solution files.  Only files with the extension *.sln are selectable.

  • Browse: opens the Open dialog for locating and opening an existing solution. Only files with the extension *.sln are selectable.

  • Unarchive: Opens the dialog to unarchive a project folder.
    NOTE: Path and file name length is limited by Microsoft Windows restrictions. The fully qualified file name cannot exceed 259 characters, the path name cannot exceed 247 characters. If an error of this kind is detected, unarchive the project in another folder, somewhere higher in the file hierarchy.
  • Checkout Opens a dialog to checkout a project which is already managed in an existing archive. Select the path of the repository, select the respective project from within the repository, define the Checkout directory, where the working copy of the project will be stored.
    NOTE:  Use only an empty folder, because the entire content of the folder will be overwritten.
    .
Archive

Starts the archiving of a project folder.

Save a Copy

Saves a copy of the currently open solution, without altering the original version.

Close

Closes the currently open solution.

Settings

Opens a dialog with settings for the development environment EcoStruxure Automation Expert - Buildtime (see Settings)

Logs

Opens by domain a console that lists the information, warnings and errors. The domains are selected in Settings > System Logs.

Installed libraries

Displays a list of the installed libraries and serves to manage the libraries on the machine, that means for installing and uninstalling libraries.

Installed Plug-ins

Displays a list of the installed device plug-ins.

About

Opens the About dialog, which displays version information for EcoStruxure Automation Expert - Buildtime.

Exit

Closes the program.
