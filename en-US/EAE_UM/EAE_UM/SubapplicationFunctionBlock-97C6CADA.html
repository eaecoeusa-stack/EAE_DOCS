Subapplication Function Block

Subapplication function blocks are collections of basic, composite, or other subapplication function blocks. But they can run distributed, unlike basic or composite function blocks. Function blocks within a subapplication function block also can run distributed to different resources. (See also IEC 61499 documentation for Subapplications)

For generating or deleting a Subapplication function block, refer to the topic Elements.

For editing help, refer to the topics Function Block Interface Editor and Sub Application Function Block Editor.
