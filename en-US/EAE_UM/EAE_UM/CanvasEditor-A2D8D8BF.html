Canvas Editor

Use the Canvas Editor to create and edit the canvas, i.e., the graphical view on the HMI. Create a canvas in the desired resolution at first, as described in the topic  Canvas.

The Canvas Editor structure is described in the topic Graphic Editors.

Adding Symbols

In addition to adding elements from the Canvas Editor toolbar, CAT_HMI Symbols can be added to the canvas by dragging an instance of the CAT into the canvas:

NOTE: There is a limitation with the Multi-OS HMI used for the Harmony ST6xxx devices. Beside the library SE.AppBase, all the HMI parts of the other application libraries are not supported on HMI ST6xxx. This limitation comes from external DLL files. Once added to a library, it will prohibit this one to be used with Multi-OS HMI.

You can also insert an image of the real plant, facility, heating system, etc. into the canvas. Either you can insert one image covering the entire canvas (select the background and choose the desired image at Misc > ImageName in the properties) or you can insert several smaller images by creating rectangles or some other container elements and filling them with images.

Inserted hardware instances need to be aligned exactly, easily and quickly. To achieve this, select the elements to be aligned, right-click one of them and choose Align > Tops (Middles or Bottoms) and Horizontal Spacing > Remove from context menu. (The context menu is described in the topic Graphic Editors for .Net HMI.)

To pre-test the newly created canvas, right-click the respective resolution in the Solution Explorer and select the  Start Local HMI item, as described in the HMI - User Guide topic Operating EcoStruxure Automation Expert - HMI Client.

In the Solution Explorer, right-click on the resolution and in the context menu use the Redefine Start Canvas with wizard command to specify the Start Canvas.
