Device Configuration Commands

A list of commands is displayed when you right-click a device in the physical view.

Depending on the type of device, some commands are unavailable.

Some commands are accessible only when system diagnostic is started.

Command

Description

Delete

Used to delete the device from the project.

Copy

Used to copy a device. The command is unavailable if Paste command is available.

Paste

Used to paste a copied device. The command is unavailable if Copy command is available.

NOTE: If the device is configured, the configuration is also copied to the pasted device.

Open device Webpage

Used to display the webpage of a device.

Open configuration editor

Opens the EtherNet/IP Configurator, where you can configure a device DTM for the use of EtherNet/IP protocol with EIPSCANNER2. This command is available only for devices, scannable with EtherNet/IP.

Manage Services

Offers access to the following commands:

  • Manage Soft dPAC

  • MQTT broker

  • SYLSOG server

These commands offer actions to select for the activation, configuration, or deactivation of each service.

Locate

Used to physically identify the devices in the physical view or the discovery view.

Get identity

Used to identify a discovered device.

Configure device settings

Used to modify the parameters of a connected device.

Configure security

Used to secure the device. The command is accessible when the device is configured, identified, and the system diagnostic is started.

Reset security

Used to reset the configured security of a device. The command is accessible when the device is configured, identified, secured, and the system diagnostic is started.

Set current date and time

Used to set manually the date and time in a device. The command is accessible when the device is configured, identified, secured, and the system diagnostic is started.

Fetch logs

Used to generate a log file when the device is configured, identified, secured and enters STOP state. The command is accessible when the system diagnostic is started. The command is available only for BMECRD0100 and Modicon M580 Distributed PAC.
