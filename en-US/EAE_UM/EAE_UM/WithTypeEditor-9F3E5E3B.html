With Type Editor

Use this dialog to define the association between event and data ports of a function block without having to first change the Interface Editor.

Accessing the Editor

This dialog can be accessed in three ways:

  1. The dialog opens automatically when and then using the newly offered port to connect the line. By setting the respective check mark, the association (the with-construct) of the data port to the specified event port is created. Multiple choice is possible, because each data port can be associated to several events at the same time.

    1. a new port is added to an HMI function block in the Composite Editor view of a CAT, by dragging a new connection line from another function block in the Composite Editor  to the HMI function block, then

    2. using the newly offered port to connect the line.

    By setting the check mark for the event in the With Type Editor, the association (the with-construct) of the data port to the specified event port is created. Multiple choice is possible, because each data port can be associated to several events at the same time.

  2. This dialog can also be opened when:

    1. Anew event output is created from within the ECC editor, by a right-click on an event output of a state in the ECC editor and a click on New Event Output ... in the context menu:

    2. Next, the Define new name dialog opens where the Name of the new event output can be defined.

    3. When you click OK to close the dialog, the With Type Editor dialog opens and presents a list of the available data outputs to associate the new event output with.

    4. Select with a checkmark the relevant data outputs and, after confirming the dialog with OK, the new event output will be created and inserted on that position where the right-click has been made. If an event output had been defined there it will now be replaced by the new one.

  3. This editor can also be opened using the ellipsis button in the With column for events within the Interface Editor.

    NOTE: To create a 'with construct' (an association) between an event output and a data output, you can drag the data output and drop it on the event output using your mouse. The same holds true for inputs. Executing the same action again would delete the 'with construct'.
