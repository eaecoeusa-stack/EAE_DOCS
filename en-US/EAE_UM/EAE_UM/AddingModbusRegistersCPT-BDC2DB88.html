Adding Modbus Registers

The first step of the manual development is to add the Modbus registers into the Modbus device previously added into the Hardware Configuration editor.

To add manually a Modbus register into these elements:

  1. Select Modbus device line and click on Add new HW-CAT button:

  2. The Add to [Modbus device Name] dialog window opens with the Element Type Modbus register selected.

  3. The list is containing all the Modbus registers available to add into the Modbus device. As described in the chapter Modbus, there are:

    • The Modbus Gateway registers at the top of the list.

    • The Standard.IoModbus registers at the bottom of the list.

    As specified in the Modbus chapter, the development of Modbus communication shown in this documentation will only be using Modbus registers from the library SE.ModbusGateway.

  4. In this Modbus Gateway list of Modbus registers, select the one fitting to the variable to transfer. To know which one will fit, check the following aspects:

    • Usage: If the variable to transfer through Modbus is sent, select a writing Modbus register starting by SE.ModbusGateway.MB_W, the W standing for write. If the variable is received, select a reading Modbus

      register starting by SE.ModbusGateway.MB_R, the R standing for read.

    • Type: Choose a Modbus register transmitting the same type than the one from the variable to transfer. Select a Modbus register finishing with the right variable type (BOOL, WORD, INT and so on).

      NOTE: The read Modbus registers automatically convert variable type chosen to its VTQ version (BOOL to VTQBOOL and so on). For more information about VTQ, refer to Transmitting and Interpreting Online Values topic.
      NOTE: For other information, refer to the SE.Modbus Gateway - Library Guide topic List of Type Handled.

  5. Select the Modbus register applying to these two rules, enter its name and click Add.

Even though the dialog window won’t close after clicking on Add, the Modbus register will still be added into the Modbus device. As the dialog window remains open, multiple Modbus registers can be added at the same time. Click on Close when all the Modbus registers have be added.

When all Modbus registers have been added into the Modbus device, it is necessary to configure them so they are transmitting correctly the expected variable. To do so, check the topic Setting Modbus Registers Parameters.
