Creating an AVEVA Model Plant View

To create a new AVEVA model plant view and get the Galaxy hierarchy, follow the next steps:

  1. Ensure to have Asset Link for AVEVA OMI installed and configured as defined in the Asset Link for AVEVA OMI chapter.

  2. Configure Galaxy as detailed in AVEVA SP configuration.

  3. Open Hierarchy View pad; this opens the AVEVA Hierarchy view with the Galaxy name as its view name.

  4. Click the icon; this loads the data from the Galaxy.

  5. Check out that:

    • Synchronization information shows in the Output pad.

    • Hierarchy View pad indicates Sync In Progress.

The AVEVA Hierarchy view contains:

  • all the folders with same order of Area object instances as in the configured Galaxy

  • all the application instances from the Galaxy that have the same name as the ones in the EcoStruxure Automation Expert project.

Then add other application instances and organize them manually into the AVEVA Hierarchy view as described in the Handling Application Instances topic.
