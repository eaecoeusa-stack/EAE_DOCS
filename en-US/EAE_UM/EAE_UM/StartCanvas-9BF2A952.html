Start Canvas

Use the Start Canvas wizard to create a new resolution. A resolution is a node that can contain multiple canvases in the specified resolution for a topology.

How to create canvases is discussed at Canvas.

The Start Canvas wizard includes the following steps:

Step 1: Resolution

Step 2: Canvas Logo

This wizard opens by a right-click on node Canvases > New Resolution.

Resolution

In step 1, specify the size of the canvas in pixels, the behavior of the canvas when displayed at different resolutions, and the style and layout of the navigation bars that appear in the canvas.

Command

Description

Size

Select a specific canvas size in pixels, or select Other.

Width

Available if Size Other is selected.

Height

Available if Size Other is selected.

Resize Behavior

Behavior for re-size selectable:

  • Standard: Canvas elements remain at the same size in the canvas. Thus, if the canvas size is bigger than the HMI window size then canvas elements might be not fitting inside the HMI window. To see them, use scroll bar, maximize button or resize the HMI window.

  • Fit: The canvas always fit to the HMI window. Thus, if the canvas is bigger than the HMI window, then all canvas elements are automatically zoomed out to reduce their size and fit into the HMI window. With Fit, no scrolling bar, maximize button nor resize HMI window option are available.

Show Top Navigation Bar

Creates a navigation bar at the top of the start canvas, for both Windows HMI and Multi OS HMI. Selected by default.

Show Side Navigation Bar

Create a navigation bar at the left side of the start canvas, for only Multi OS HMI. Selected by default.

Navigation control

Select the style of navigation bar to display at the top of the start canvas:

  • (None)

  • Breadcrumb

  • Rose (not available on Multi-OS solutions)

Login Button

Creates a Login Button in the start canvas.

Current User Label

Creates a field in the start canvas to show the current user label.

Language Button

Creates a button in the start canvas to select the language.

Runtime Connection

Displays a list of controllers connected to the HMI application.

New Version Deployed

Informs the HMI operator when a new HMI version is available. The operator can implement the new HMI version by exiting then re-starting the HMI runtime application at an appropriate time.

Logger

Creates a button in the start canvas that, when clicked, shows the log outputs of the HMI runtime.

Display Alias Name Toggle Button

Creates a button () in the start canvas to toggle among object name options:

  • Asset tag: the text assigned as the Asset Tag Name (default)

  • HMI Alias: an alias name you can define

  • Instance name: the object path name (Runtime.Resource.HMI)

Exit Button

Check box active creates an Exit button in the start canvas.

Show Warning by click on Exit button

Creates a warning text dialog for exit via exit button. (This dialog does not appear when the canvas is shown in a window, instead of full screen mode, and the window is closed using the x-button on the upper right corner of the window frame.)

Warning Text

Input the text to be displayed if Show Warning by click on Exit button is selected. If left blank and Show Warning by click on Exit button is selected, the following text is used: "Do you really want to exit?"

Help

Opens the context help.

Back

Not available, because the current dialog window is the first in this sequence.

Next

Steps forward to dialog window Canvas Logo.

Finish

Shuts the dialog and creates a new resolution using the current settings.

Cancel

Shuts the dialog without creating a new resolution.

Canvas Logo

In step 2, the Canvas Logo can be defined, using the following fields:

Command

Description

Logo

Shows the path of the current used logo image. In default setting: (Default).

...

Opens standard dialog 'Open' to select the file.

Reset

Resets the logo image to default logo again.

Help

Opens context help.

Back

Steps back to dialog window Canvas Type.

Next

Not available, because the current dialog window is the last in this sequence.

Finish

Shuts the dialog and creates a new resolution using the current settings.

Cancel

Shuts the dialog without creating a new resolution.

NOTE: The color scheme of the canvases can be defined with the Theme setting in the context menu for the Distributed PAC project.

A right-click on an existing resolution and a click on the context menu entry: Redefine Start Canvas with wizard also opens the Start Canvas Logo dialog, where you can redefine the logo which is used in the Start Canvas.
