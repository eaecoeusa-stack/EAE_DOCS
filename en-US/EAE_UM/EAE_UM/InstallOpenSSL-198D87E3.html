Install OpenSSL

To install OpenSSL, follow these steps:

  1. Navigate to the OpenSSL download page.

  2. Find the latest 64-bits Windows normal version of OpenSSL and click theEXE file to download it.

  3. Execute the installation file.

    NOTE: If Microsoft Visual C++ is not installed on the engineering workstation, it is installed during the OpenSSL installation. After this installation is completed, run the OpenSSL installer again and follow the wizard steps.

    Result: OpenSSL is installed.

Few Windows environment variables have to be defined before to be able to generate the self-signed certificates. Then either:

  • Set Windows environment variables temporarily, for the current session only:

    1. Open the Windows command prompt.

    2. Enter the following commands:

      • set OPENSSL_CONF=C:\Program Files\OpenSSL-Win64\bin\openssl.cfg

      • set Path=%Path%;C:\Program Files\OpenSSL-Win64\bin

  • Or, set Windows environment variables permanently:

    1. In the Windows search bar, write environment.

    2. Select Edit the system environment variable.

      Result: The System Properties window opens.

    3. Click the Advanced tab, and then click the Environment Variables button.

      Result: The Environment Variable window opens.

    4. In the System variables section, click the New button.

      Result: The New System Variable window opens.

    5. Add the variable names and the variable values provided in the following table, and click OK:

      Environment Variable Number

      Variable Name

      Variable Value

      1

      OPENSSL_CONF

      C:\Program Files\OpenSSL-Win64\bin\openssl.cfg

      2

      Path

      C:\Program Files\OpenSSL-Win64\bin

    6. Click OK on each previously opened window to validate and close it.

Result: OpenSSL is now installed and configured. The added environment variables are taken into account with the next Windows command prompt opening. You can create self-signed certificates.
