Install the ASP Galaxy Repository in the Server PC

To install the ASP Galaxy repository in the server PC:

  1. Run the installation wizard in the server PC of the latest patch of AVEVA System Platform 2023 R2 SP1.

    NOTE: To know the what is the latest patch of the AVEVA System Platform 2023 R2 SP1 supported with EcoStruxure Automation Expert, refer to the Software Compatibilities topic of the Compatibilities and Dependencies - User Guide.

  2. In the AVEVA System Platform 2023 R2 SP1 Installation dialog box opening, select AVEVA System Platform Computer Roles and click on Next. This will lunch a role-based installation instead of a Product Based Selection.

  3. Select the All-In-One Node option to install and click on Next.

  4. The AVEVA System Platform 2023 R2 SP1 Installation dialog now shows the components that will be installed.

    In the list there is the Historian Server component. If it shall be placed in another machine, customize the installation and deselect the Historian Server component here.

  5. Click on Next until reaching to the account creation page.

    This page is used to create the user for the Archestra Network services.

  6. Add a User Name, Password and select the check box Create Local Account.

  7. Once done, click on Next and proceed through all the remaining steps to complete the installation, including the AVEVA configure wizard.

NOTE: For further details about the installation process, check the Installation Guide of the latest patch of AVEVA System Platform 2023 R2 SP1 Installation Guide.

Once done, configure the ASP Galaxy Repository in the server PC by:

  1. Opening the Configuration executable.

  2. Configuring the following two sections:

    • System Management Server by setting this machine as the System Management Server (second option).

    • AVEVA Entreprise License Server by setting all the required information.

  3. Finally, active the ASP license.
