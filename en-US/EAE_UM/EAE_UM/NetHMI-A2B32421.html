Graphic Editors for .Net HMI

The graphic editors for HMI are available when the project has been created with HMI support.

Graphic editors, which serve to create .Net graphic elements to be used in HMI, can have different features (depending on their purpose). Use the:

  • Symbol Editor to create HMI symbols within a CAT, to be used in Canvas directly by dragging the instance of the CAT into the Canvas Editor and choosing the respective symbol from the dialog.

  •  Faceplate Editor to create HMI faceplates within a CAT, to be linked with a graphic element in a symbol of a CAT, to use the faceplate quasi as a subordinated part of one or more symbols.

  • Graphic Editor to create entire objects which can be inserted in CAT symbols or faceplates of one or several CATs (e.g. a switch, lamp or sunblind symbol which is used often) or in Canvases directly.

  • Canvas Editor to create the canvases, which contain the elements used for controlling to be displayed on the HMI.

The drawing elements of the toolbar and the graphic editor formatting ribbon commands are described in the topic Graphic Editor .

NOTE: To keep the view of opened graphic editors with a dependency to other graphic editors up-to-date (e.g. to get the modification on a CAT_Symbol also displayed in the appropriate canvas), recompile the project. This reloads the editors and modifications are displayed other open editors.

Graphic Item Menu Commands

Right-click on a graphic element to display context menu commands:

Command

Description

Order

Opens a sub menu to set the selected element to front, back or even only one level up or down. Bring to Front or Send to Back is also available via Iconbar, as described below.

Align

Opens a sub menu to align the selected elements horizontally or vertically or to the grid. Horizontally and vertically alignment can also be realized via Iconbar, as described below.

Make Same Size

Opens a sub menu to set the selected elements to identical height or width, or both together

Horizontal Spacing

Opens a sub menu to adjust the horizontal spacing between the selected elements

Vertical Spacing

Opens a sub menu to adjust the vertical spacing between the selected elements

Select

Provides the selecting of a single element of a group via sub menu. The sub menu lists those elements extending into the area of the mouse cursor at the time of the mouse click, therefore the list will not necessarily show all elements of the group.

TIP: If you want to select a special element, but it is not shown in the list, make another right-click with slightly shifted mouse position, as central as possible over the element to be selected.

Alt+click is a further option to select a specific element of a group.

Further on the submenu offers, besides the individual elements, also those groups being a part of the group. That means, if several groups of several levels are pooled to one big group, also the smaller groups are offered at choice as far as they are in the range of the mouse pointer.

Group

Joins the selected elements to a group. An additional element 'Group' will be created. The name of the element can be adjusted in the properties. The elements belonging to the group, for example, can be moved together at design time, and it can be programmed in that way, that they will execute the same action, e.g. becoming visible by mouse click, or something like this, at runtime.

It is possible to create several levels of groups, by creating group in group etc.

NOTE: By creating a new group a new element is created as well, called element 'groupX'. Because newly created elements are located on top, it is possible that an element which does not belong to the group, but had been on top of an element of the newly created group, is now hidden by that element of the new group. Adjust the order in this special case again (see topmost entry in this context menu → Order). The order of the elements within the group stays unchanged.
Ungroup

Dissolves a group to its individual elements. This way, the element 'group' will be deleted.

This procedure is not necessary to modify or delete one specific element of the group. In this case select the specific element as described at Select. Now it can be deleted or modified. The group can still be used.

NOTE: By dissolving a group it might be, that elements, which had belonged to the group are hiding neighboring elements now, if the neighboring elements had been lying on top of an element of the group, without belonging to the group. By dissolving a group the element 'groupX' will be deleted and the single elements of the former group are raised to the top of the order. Adjust the order in this special case again (see topmost entry in this context menu → Order). The order of the elements to one another of the dissolved group stays unchanged.
Set Layer

Opens a sub menu to set the specific level of the selected element. Newly created elements belong to the currently active layer (see the description of the Layer Active icon of the graphic item menu bar below).

The layer of groups cannot be set. It depends on the layer of its single elements. (That means, if a group is composed of single elements which are located in different layers and one of the elements is located in an non-visible or unselectable layer, the group is accessible just by elements located in visible and selectable layers. Though the elements which are located in invisible or unselectable layers belong to the group, they still remain in the chosen state.)

Cut

Cuts the selected elements and stores them to the clipboard.

Copy

Copies the selected elements to the clipboard.

Paste

Pastes the elements stored in the clipboard.

Delete

Delete

Deletes the selected elements.

Properties

Opens the Properties pad, or sets the focus on it, when it is already open.
