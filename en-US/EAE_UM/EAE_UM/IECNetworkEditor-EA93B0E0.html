5. IEC Network Editor

The relevant information for each entry is displayed in the lower area of the dialog window.

Topic

Item

Description

Network editor

 -

These settings concern all network editors (the editor for applications, the resource editor, the composite editor in Composites and CATs, etc.)

Network editor page layout

 -

Here you can adjust the settings for printing the editor view. See also Page Setup

Library Element Appearance

 

Parameters

These settings concern the appearance of constants, which can be set at data inputs of function blocks.

Element Colors

These settings affect the appearance of function blocks. Either the same color scheme can be used for all function blocks, or some types can be set with special settings. The check box Enabled must be selected for the respective element, to have the default settings modifiable and then the element will be displayed with the new settings as well.
