Import CAT Instances

Once the Excel file containing the updated CAT instances is done, the updates of the CAT instances can be imported into EcoStruxure Automation Expert.

Follow the next steps to import the Excel file contents:

  1. Select the Import CAT instances command of the node CAT Instances from the Solution Explorer pad.

  2. With the Windows folder opening, select the Excel file and click Open.

  3. In the Output pad, all the issues found in the Excel file are listed (if there are some) and the Import CAT instances pop up appears to ask for proceeding import and for giving recommendations.

  4. After having read the recommendations and if the results of the issues are acceptable, click on Continue to start the import of the CAT instances. If the results are not acceptable, then go back to edit the Excel file to solve the issue and retry this process.

The Excel file will be processed and the CAT instances from the Excel file will be updated.

The updated CAT instances, along with the updated connections, will be located under the application and layer in Function Block Network editor as defined in the Excel file.

If a new element mentioned in the Excel file does not exist in the application, like a layer or a connection betweent two CAT instances, the element will be created and the associated CAT instance will be assigned or linked accordingly.

If the layer mentioned in the Excel file does not exist in the application, it will be created and the CAT instance will be assigned accordingly.

NOTE: If any issue appears during the process, check the topic Troubleshoot Tips.
