Multilanguage Dictionaries

Two Collections of Multilanguage Dictionaries

The Multilanguage Dictionaries node comprises two collections of multilanguage dictionaries:

  • HMI Translation: For HMI symbols.

  • Device Translation: For device properties accessed from the Logical Devices editor.

Dictionary Editor Features

Each dictionary editor operates identically and presents the following commands:

  • Add Language: Opens a list box of available languages. The chosen language is shown as a new column in the editor.

  • Delete Language: Opens a list box of already added languages. A click on an entry deletes the language, thus the entire column is removed.

  • Export: Select the languages to be exported, then select a name and location for the exported .zip file.

  • Import: Opens the Import Language dialog. Click Yes to import an existing translation file, then navigate to and select the translation .zip file to import.

Each dictionary editor offers both a Full Text Search function and a Filter function. Click on the filter icon () to open the filtering dialog and make filtering choices on values in that column.

Creating Dictionary Items

Enter an ID in the Key column that uniquely describes that item. Special characters are not allowed.

NOTE: The Key value is required for each item.

In each added language column, enter the translated text for the respective item.

The text in column Neutral is displayed in the canvas, symbol or faceplate while programming. It also is used as a fallback language.

If the Neutral value is not populated, the Key value is used as a secondary fallback.

For each added language, you need to define only those items that are different from the Neutral value.

To delete an entire row of the dictionary, select the row and press the Delete key.

Using a Dictionary Item

To use the multilanguage text in the HMI, you need to adjust the properties of the inserted text by using the Key.

In the Properties pad for a Free Text or Label element added to the HMI, the Key text can be specified instead of inputting only a single language text string. The easiest way to do this is to use the drop down menu for the attribute TextKey. The available Keys are presented. The key value selected in the TextKey attribute is inserted automatically with a prefixed $key in the field Text.

Switching to Another Language in the HMI

In the HMI, the button to switch between different languages - indicated by an icon of the language’s nation flag - will be activated automatically when one or more languages are added. This button is already included in the Start Canvas. (Refer to this topic for information on where to find the button, if you want to use it on other places or if it has been removed erroneously). Use the button during runtime to switch between languages.

Another example for using multilingual text can be found at: Define Ranges at item Text.

NOTE: The language displayed at startup depends on various language settings and how the HMI is run, as described in the following section HMI Translation Language Display.

If it is necessary to use a different language for startup, it can be defined within the C#-code in the StartCanvas-Constructor: 

ILanguageService languageService = (ILanguageService)ServiceProvider.GetService(typeof(ILanguageService))
      if (languageService != null)
      {
        languageService.CurrentLanguage = "en"
        (this as ILanguageSensitive).OnLanguageChanged()
      }

In the example above English is used as startup-language.

HMI Translation Language Display

The language displayed in the HMI depends on both:

  • Where the HMI is run from:

    • Canvases node of the Solution Explorer

    • Run command in the Deploy and Diagnostic editor (Active Network Profile = Default)

    • Start (Runtime Simulation) command in the Deploy and Diagnostic editor (Active Network Profile = Local Test )

    And

  • The language setting:

    • EcoStruxure Automation Expert Language (EAE Language): Set in Settings > General > User Interface Language

    • Device Language: Set in Logical Devices Editor > Language

    • HMI Translation Language: Set in Solution > Multilanguage dictionaries > HMI Translation
