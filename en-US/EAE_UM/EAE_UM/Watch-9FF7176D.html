Watch

A right-click on a function block within an application or resource editor will show the context menu entries Add watch and Remove watch  (refer to Function Blocks Network Editor Actions). These commands serve to add or remove all watch points for the entire function block at once (or for several selected function blocks). The entries are only available in online mode. That means, the project first has to be deployed in simulation mode (refer to the HMI -User Guide topic Simulation Runtime) or on real devices, and the device(s) have to be online (see the Login command of the Deploy and Diagnostic editor). You can add watch points when the project is in the ErrorHalt state, Stopped state, Running state or is ready to enter the Running state. Previously added watch points can be also switched to force mode by simple overwriting the shown value within the editor. The value will be shown highlighted yellow now to point out that it has been forced.

To set a watch point on just a single variable or to remove one, or to switch between watching and forcing, execute a right-click on an Input or Output of a function block and select an appropriate action.
