Fieldbus Components Editor

The Fieldbus Components Editor lists, in the left part of the editor, the available CAT types. Select the CAT types that you want to include in your project as fieldbus components by placing a checkmark next to each item.

In the right part of the window, in the column Device Master, add a label to each entry marked as fieldbus components using the drop down list.

The Fieldbus Components Editor is available only for libraries permitting a source-code access. This functionality is not available for previously completed libraries, which are installed on the machine merely for end use.

To open the Fieldbus Components Editor, in the Solution Explorer, beneath the project node, navigate to CAT > Hardware, then right click and select Fieldbus components... in the context menu.
