Configuring Event Driven PROFINET Communication

PROFINET Function Blocks

EcoStruxure Automation Expert provides function blocks for configuring event driven PROFINET communication. Event driven communications adds flexibility to the management of your PROFINET network. After you have Configuring your PROFINET network, use the following function blocks to respond to triggering events:

  • PN_READ_REC to read PROFINET record data.

  • PN_WRITE_REC to write PROFINET data.

  • PN_RECEIVE_ALARM to receive alarms from PROFINET IO-Devices for handling by application logic.
