Solution Explorer Pad

The Solution Explorer shows the currently loaded project folder in the EcoStruxure Automation Expert - Buildtime Integrated Development Environment (IDE). To open it, in the View menu, click General pads > Solution Explorer.

The individual objects within the projects and libraries are shown in groups (e.g. Composite or Basic Function Blocks), with each group identified by its distinctive icon.

You can filter content by:

  • Selecting a Profile. The profile can be one you previously created using the toolbar Save as... command, or one of the default selections: All or Canvases.

  • Entering Search characters including the wildcard characters of * and ? –

Toolbar

The Solution Explorer toolbar presents the following icons and functions:

Icon

Function

Description

 

Search

Searches for the entered search text in the tree of the Project Folder Overview. Useful feature to find a particular object within the project and its (referenced) libraries very quickly.

NOTE: The wildcard characters * and ? are allowed in a search.

Delete current search

Deletes the current search and again displays the entire tree, depending on the selected profile.

Profile list

Opens list of profile filters you can apply to the Solution Explorer. The active profile is updated when a filter is changed.

Display profiles

Opens the Current Profile sub menu, described below.

Collapse parent

Collapses the selected node in the Solution Explorer. 

Collapse more

Collapses the content of the selected node one level.

Expand more

Expands the content of the selected node one level.

Collapse all

Fully collapses the selected node and its child node contents.

Expand all

Fully expands the selected node and its child node contents.

Current Profile Menu

Click on the Display Profiles icon ( ) to display the current profile commands:

Command

Description

Save as ...

Saves the current profile under a new name

Delete ...

Deletes the current profile

CAT instances Sort

Selects the type according to which the CAT instances are sorted. Multiple selections can be made from among the following:

  • None

  • Origin

  • Application

  • Device

  • Group

  • Type

Display Filter

Opens the sub menu to filter according to various technology groups (Composite, Basic, ...)

  • All: Sets all visible.

  • None: Sets all invisible.

  • Show <Element>: Set the visibility for the selected element.

Referenced Library Menu

Right-click on a library node to display the following menu:

Command

Description

Show Info

Opens the info box containing the meta data of the selected library (the same box can be opened by clicking the Info button in dialog Installed Libraries).

References

Displays all referenced libraries of this library. The command is grayed out when the library has no further references.

A click on the subcontext entry of a referenced library opens the info box containing the meta data of the referenced library (same box can be opened by clicking the Info button in dialog Installed Libraries).

Help

Opens the context help

NOTE: Referenced libraries are displayed in the tree control if they are not referenced by the main project itself, but only by a subproject or a referenced library.

System Menu

Right-click on the System node to display the following menu:

Command

Description

Rename

Use this command to rename the System node.

Open

Opens the System Editor. The same can be achieved by double-clicking the node or clicking on the button  Open System File from the toolbar.

Help

Opens the context help.
