Exporting Data Type

DT_TO_DATE, DATE_AND_TIME_TO_DATE

Synopsis 

DT_TO_DATE(IN1: DT): DATE
DATE_AND_TIME_TO_DATE(IN1: DATE_AND_TIME): DATE

Description

Splits the input parameter IN1 of type DT or DATE_AND_TIME to the type DATE by keeping only the date information of the input value. The result of this conversion is then returned with the DATE type.

Errors

Error ERR_RT_CONVERSION detected when the split value is inconsistent. Because of time zone difference, the split value can be out of the range.

Warning WRN_NONSTD_TYPED_FCT detected when DATE_AND_TIME_TO_DATE function is used as this function format may lead to compatibility issue.

Example 

VAR_TEMP
  someDT : DT := DT#2021-12-22-21:12:02;
  convDATE : DATE;
END_VAR
convDATE := DT_TO_DATE(someDT);
