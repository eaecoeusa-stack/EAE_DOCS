Manually Set the Date and Time in a Device

Set Current Date and Time

Before setting the date and time inside the target device, ensure that:

  • You are logged in, for this the configured device needs to be secured. For more information, refer to Secure the Devices.

  • The device is not in RUN state to perform this operation, otherwise the execution is rejected.

Follow the next steps to set the current date and time:

  1. Using EcoStruxure Automation Expert, open the Physical Devices editor.

  2. Select the device you want to configure, then right-click and select Set current date and time to open the dialog box.

  3. Set the device date and time.

    Field

    Description

    Current Device Date Time

    Coordinated Universal Time (UTC) Date and Time read from device when dialog opens.

    Current PC Date Time

    UTC Date and Time from PC clock and keep it refreshed with the latest value.

    Set Device Date Time

    Same as Current PC time

    UTC Date and Time values same as PC time (Dynamic).

    Custom

    You can specify a custom date and time.

    NOTE: On Modicon M251 Distributed PAC and Modicon M262 Distributed PAC if the date and time value set on the custom device is too early, the device will automatically set the date on which the product was first launched. If the value is too far in the future, the device will generate a new certificate that must be trusted again by EcoStruxure Automation Expert.

    *The firmware version, available in the EcoStruxure™ Automation Device Maintenance (EADM) tool, is used to deduce this value as follow: V23.0.<YY><DDD>.00 with <YY> being the digits of the year and <DDD> being the digits of the current day of the year.

    Example: V23.1.23314.00: <YY> = 23 (means the year 2023) and <DDD> = 314 ( means the day 314 of the year = November 10th).[[{

NOTE: There is no need to restart the device to apply these settings.

Applicability

  • Modicon M262 Distributed PAC

  • Modicon M251 Distributed PAC

  • Modicon M580 Distributed PAC

  • Altivar dPAC Module VW3A3530D
