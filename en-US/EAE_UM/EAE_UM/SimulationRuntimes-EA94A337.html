8. Simulation Runtimes

Topic

Description

Simulation runtimes

Use the Add, Remove, and Configure commands in this area to manage simulation locations.

Simulation HMI \Security Active

Toggle the Security Active setting to turn on and off cybersecurity features, including login, when operating in simulation mode with Active Network Profile set to (Local Test).

NOTE: This setting does not apply to normal operations when Active Network Profile is set to Default.
