Substitute Library Element

Use this dialog to replace an instance of a function block with another function block of a different type.

NOTE: The function block type definition is displayed in the center of the function block.

If the connections of the new type differs from the connections of the old type, this dialog opens automatically. 

If the connections of the new type and the original type completely match, the type name in the center of the function block is replaced without opening the dialog.

Reconnecting Inputs, Outputs, and Connections

Use the dialog to adapt existing connection lines to the new function block type and the number and data type of the connections for generic blocks can be defined.

In the dialog:

  • The input and output arrows represent the connections of the original instantiated block.

  • The block depicted in the dialog represents the substitute block type.

By setting the connection lines between the arrows and the connections of the block in the dialog, the connection lines for the new block type are inserted accordingly.

A basic description of generic blocks is located in Generic Interface Editor.

  • OK accepts the current connections and block settings and closes the dialog. The block type is replaced.

  • Cancel closes the dialog, cancels the procedure of substituting of the type and retains the original block type in the network. Any changes made in the dialog are discarded.

See also the remarks on substituting function block types in the topic Editors.
