Settings

Configuring the Settings

Use the Settings dialog to view and configure settings that govern the operation of the EcoStruxure Automation Expert software.

Select a topic in the hierarchical structure of the left pane to display in the right pane the settings details related to that topic.

Reopen the solution after making modifications of settings to update your view.

Topic areas (in the left pane) include:

  1. General

  2. Coding

  3. Text Editor

  4. Subversion (SVN)

  5. IEC Network Editor

  6. Libraries

  7. Graphic Designer

  8. Simulation Runtimes

  9. Hardware Configuration

  10. System Logs.

To open this dialog, click on the File menu  to open the Backstage, then click Settings.

Resetting the Settings

In some cases, it may not be possible to start EcoStruxure Automation Expert. This may be caused by, for example:

  • Overwriting an installation of version of EcoStruxure Automation Expert with a different version. This may occur even though the previous settings are automatically saved and migrated to the new version.

  • Deleting the EcoStruxure Automation Expert settings that had been saved to the local disk.

If EcoStruxure Automation Expert cannot be restarted, you need to make a full reset of the settings.

This action will set back EcoStruxure Automation Expert to its factory default settings.

You can reset the settings in two ways:

  • Using the Reset tool settings button: Accessible in the top right of the Settings dialog, this button will open the Reset tool settings dialog. Review and accept the default settings to reset of the settings. This action will restart EcoStruxure Automation Expert with all the settings set to their factory default values.

  • At the end of the EcoStruxure Automation Expert installation process, the EcoStruxure Automation Expert - Buildtime wizard gives you the option to Reset the existing tool settings of current version of EcoStruxure Automation Expert. Select this and finish the installation to reset the settings.

At the end of these processes, the settings - except the language setting - are reset. The language setting remains unchanged.

You can then restart EcoStruxure Automation Expert.
