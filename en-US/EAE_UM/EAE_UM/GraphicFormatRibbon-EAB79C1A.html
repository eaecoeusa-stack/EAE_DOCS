1. Graphic Format Ribbon

The Graphic Format ribbon is available when a graphic object, for example a canvas, is open for editing. Click Graphic at the top of the main window to display the ribbon.

The Graphic Format ribbon contains the following icons and commands:

Command

Description

Align commands:

Align left

Justifies the selected elements left-aligned.

Align horizontally center

Justifies the selected elements by aligning them at their vertical middle line.

Align right

Justifies the selected elements right-aligned.

Align top

Justifies the selected elements by aligning them at their top line.

Align vertically center

Justifies the selected elements by aligning them at their horizontal middle line.

Align bottom

Justifies the selected elements by aligning them at their bottom line.

To Grid

Aligns the selected elements to the grid.

Make Same Size commands:

Width

Make the selected elements the same width.

Height

Make the selected elements the same height.

Both

Make the selected elements both the same height and the same width.

Spacing commands:

Make Equal (horizontal)

Make the selected elements equally spaced horizontally.

Increase (horizontal)

Increase the horizontal space between the selected elements.

Decrease (horizontal)

Decrease the horizontal space between the selected elements.

Make Equal (vertical)

Make the selected elements equally spaced vertically.

Remove (horizontal)

Remove the horizontal space between the selected elements.

Increase (vertical)

Increase the vertical space between the selected elements.

Decrease (vertical)

Decrease the vertical space between the selected elements.

Remove (vertical)

Remove the vertical space between the selected elements.

Orientation commands:

Mirror Horizontal

Mirrors the selected element about its vertical central axis. If there are several elements selected, each element will be flipped about its own vertical central axis. If you want to mirror a composition of several elements about their common, contrived vertical central axis, create a group out of them and select the group. Elements as e.g. text, button, trend or pie cannot be mirrored.

Mirror Vertical

Mirrors the selected element about its horizontal central axis. If there are several elements selected, each element will be flipped about its own horizontal central axis. If you want to mirror a composition of several elements about their common, contrived horizontal central axis, create a group out of them and select the group. Elements as e.g. text, button, trend or pie cannot be mirrored.

Rotate 90° left

Rotates the selected element 90° counterclockwise about its center. If there are several elements selected, each element will be rotated about its own center. If you want to rotate a composition of several elements about their common, contrived center, create a group out of them and select the group. Elements as e.g. text, button, trend or pie cannot be rotated.

Rotate 90° right

Rotates the selected element 90° clockwise about its center. If there are several elements selected, each element will be rotated about its own center. If you want to rotate a composition of several elements about their common, contrived center, create a group out of them and select the group. Elements as e.g. text, button, trend or pie cannot be rotated.

Order commands:

Bring to Front

Brings the selected elements to the top level.

Send to Back

Sends the selected elements to the lowest level.

Bring Forward One Shape

Brings the selected element forward one shape layer.

Send Backward One Shape

Sends the selected element backward one shape layer.

Layer commands:

Layer Visible

Opens a selection list which allows determining which layers are currently visible. By default all layers are visible.

Layer Selectable

Opens a selection list which allows determining which layers are currently selectable. If there are elements located on a layer which is set to be not selectable, these elements cannot be selected in the editor. By default all layers are selected.

Layer Active

Opens a selection list which allows determining which layer is currently active. Newly added elements are assigned to this layer. Layer 0 is active by default.

The context menu, which can be opened by right-clicking on the drawing area of the Graphic Editor or on selected elements, is described on page Graphic Editors for .Net HMI.
