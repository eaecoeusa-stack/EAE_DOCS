Add PROFINET IO-Devices

After adding an IO-Controller, configuring the IO-Controller, and importing GSDML files, you can proceed to add IO-Devices to your PROFINET network.

The device parameters for each device you added to the Selected Devices list include

  • Station Name: Input the unique device. The controller calls this name to check the presence of the device.

    NOTE: PROFINET IO Configurator cannot be used to assign a Station Name to IO-Devices.

  • IP Address: Input the correct IP address, if different from the presently configured value. The controller will assign this value to the device.

  • Update Time (ms): Select a value that is greater than the controller Send Clock time.

    NOTE: Update Time is the frequency at which the device should send data to the controller. A larger value can be used for devices sending non-critical data that does need to be updated frequently.

To add IO-Devices to your PROFINET network, follow these steps.

  1. In the Hardware Configuration editor, right click on the PN_IOController node and select Configure.

    The PROFINET IO Configurator tool opens.

    NOTE: The GSDML files you previously imported are listed in the Available Devices list.

  2. To add a device from the Available Devices list, drag an available device from the Available Devices to the Selected Devices list.

  3. To save your edits now, click Update & Close, then click Yes.

    Or you can proceed to add modules to devices by hovering your cursor over the ellipsis (...) then clicking Configure for a selected device.
