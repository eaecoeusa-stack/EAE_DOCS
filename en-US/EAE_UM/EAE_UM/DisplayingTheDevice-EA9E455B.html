Displaying the Device

To display the network settings of a device:

  1. Place your cursor over an Ethernet endpoint device in the Physical Devices editor, for example a controller or a network interface card (NIC) in a workstation.

  2. The properties that appear in bold text are editable.

NOTE:

  • If the Ethernet endpoint device includes multiple Ethernet interfaces, the tooltip displays multiple entries.

  • If a network is configured, the name of the network is displayed.

  • If no network is configured, NOCONF is displayed.

  • If an IP address is configured, it is displayed following the network name (or NOCONF).

  • If no IP address is configured, the IP address part is left blank.

  • If you hover your cursor directly over the port, and not the device or NIC, the name of the tooltip displays the port name.
