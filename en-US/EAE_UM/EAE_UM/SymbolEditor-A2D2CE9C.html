Symbol Editor

Use the Symbol Editor to create and edit the graphical view of CAT function blocks. This graphical view can be instanced within a Canvas and can thus be made available on the HMI at runtime. The Symbol Editor presents the same basic functions as described in the topic Graphic Editors.

This topic discusses the following features of the Symbol editor:

  1. Interface Editor View

  2. HMI Symbols Library

  3. HMI Base Symbols Library

  4. Customizing Generic Symbol Icons

  5. Symbol Size when used in Canvas
