Code Templates

Allows creating of new code template groups as well as removing them. By selecting a specific template you can view and edit it within the editor.

The group, which is shown in drop down menu Extensions, defines where the respective template comes into operation. (e.g. Templates that are in operation for Structure Text - tab Algorithms of a Basic.fbt function block - can be found at .fbt; C# templates for HMI.cnv.cs files of CAT function blocks can be found at .cs, etc.)

Usage of templates: By typing the template abbreviation (e.g. ife for the IF ELSE template) followed by a blank, it will be inserted automatically as defined in templates.

Code Generation

Contains settings for generating the code and whether comments will be generated as well
