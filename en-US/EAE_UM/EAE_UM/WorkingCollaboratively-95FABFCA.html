Manage Multiuser

EcoStruxure Automation Expert allows several users to connect to and interact with the same devices, to work collaboratively with the same solution. This multi-user capability requires caution to use, especially to ensure that users' actions are coherent towards a common and shared goal for the solution. (Otherwise unexpected results might occur).

NOTE: For more information about the multi-user collaboration, refer to the Multi-User Collaboration topic.
 WARNING
CONCURRENT ACCESS TO DEVICES
When allowing several users to perform simultaneous or sequential modifications to the application under execution in the devices, ensure that these modifications do not have adverse effects on the safety of the local and system level application execution.
Failure to follow these instructions can result in death, serious injury, or equipment damage.

EcoStruxure Automation Expert offers two different ways to work in a multi-user mode in EcoStruxure Automation Expert - Buildtime:

  • Using the Multi-User environment in a collaborative way coming along with EcoStruxure Automation Expert Platform.

    NOTE: For more information, refer to the Multi-User Environment - User Guide.

  • Using Apache® Subversion® system (SVN):

    • SVN Integration

    • Merge Tool

    • SVN Limitations

    NOTE: This functionality might become obsolete in a few EcoStruxure Automation Expert version.
