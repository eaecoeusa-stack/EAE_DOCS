Graphic Format Menu

The Graphic Format menu ribbon is displayed only when focus is on a graphics editor.

The contents of the Graphic Format menu display can vary, depending on the selected screen object. At minimum, it contains an Align area and an Order area.

NOTE: Refer to Graphic Editors for EcoStruxure Automation Expert HMI for additional information about the ribbon contents.

Align Area Commands

Command

Description

Lefts

Left aligns the selected elements.

Centers

Center aligns the selected elements by their vertical center.

Rights

Right aligns the selected elements.

Tops

Top aligns the selected elements.

Middles

Center aligns the selected elements by their horizontal center.

Bottoms

Bottom aligns the selected elements.

Order Area Commands

Command

Description

Bring to Front

Brings the selected elements to the top level of the graphic.

Send to Back

Sends the selected elements to the lowest level of the graphic.
