2. Frame and Comment Objects

Frame Objects

  • Right click in the editor, then select Frame > Add Frame to insert a rectangular frame. Use frames to contain groups of function blocks. A frame can be moved and resized using the mouse to grab the frame border handles. Move function blocks onto a frame and they will belong to and move with this frame. Move function blocks outside of a frame to free them from the frame.

  • Right click on the frame, then select Frame > Properties to open the frame's property dialog. The MoveStyle property defines the objects that will move with the frame, including:

    • Only objects that are completely contained  (AnyContained), or

    • Only objects that are partially contained (AnyIntersectsBounds), or

    • No objects at all (None) will move with the frame. (If None is selected, the frame acts as a graphical subdivision for the network, not a group of elements).

  • New frames are graphically placed on top of pre-existing ones. Frames can be grouped within other frames. Double-click on a frame, or use the select Frame > Propertiescontext menu command to open a property dialog where you can set the frame color and caption.

  • Only the first line of the caption is displayed in the frame. Other lines (after the first carriage return) appear in the tooltip.

  • Double-click on the caption of a frame to open it in a dialog for direct editing.

Comment Objects

Right click in the editor, then select Frame > Add Comment to add a comment object. Use comment objects to display textual information. Comment objects can be grouped within Frames. The first line of text is displayed as a title on top of the comment object; subsequent lines appear only within its body.
