Changing the I-Device Role

Changing the I-Device Role for a configured I-Device changes that device configuration.

Changing Role from IO Controller to IO Device

If you change the configured device Role from:

  • IO Controller to IO Device

    or

  • IO Controller and IO Device to IO Device

The following changes are made to the configuration:

  • The Role changes to IO Device.

  • The Advanced… command, which lets you navigate to the IO Device configuration is enabled.

  • The Available Devices list, the Manage Catalog command, and the configured devices grid are disabled.

When you click Update and Close in the Network Configuration main page, any previously configured IO-Devices are removed from the configuration.

Changing Role from IO Device to IO Controller

If you change the configured device Role from:

  • IO Device to IO Controller

    or

  • IO Controller and IO Device to IO Controller

The following changes are made to the configuration:

  • The Role changes to IO Controller.

  • The Available Devices list, the Manage Catalog command, and the configured devices grid are enabled.

  • The Advanced… command, which lets you navigate to the IO Device configuration is disabled.

When you click Update and Close in the Network Configuration main page, any previously configured IO data items are removed from the configuration.
