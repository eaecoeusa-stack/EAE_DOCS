State Graph Pane

Overview

When creating a Procedural CAT, the State Graph pane of a Procedural CAT displays the states defined by the related State Manager and their relationships. The State Graph is built automatically from the State Manager CAT.

You cannot modify the State Graph in this pane, but only in the State Management diagram editor.

However the State Graph pane allows you to associate states with sequences created in the Sequence Diagram Editor (see Associating States with Sequences).

State Graph description

Here is an example of State Graph:

Icon

Description

Black nodes are static states. They cannot be associated with a sequence (no PhaseCmd attribute).

Light blue nodes are sequence—less states. They can be associated with a sequence, but this association is not made (empty nodes).

Deep blue nodes are associated states. Light blue nodes become deep blue nodes when a sequence is associated to them.

Dashed rectangles materialize groups. Groups allow to simplify the graphic by replacing several transitions with a common destination by a single transition.

Arrows materialize transitions. Transitions can exist from one state to another, or from a group to a state.

Grey rectangles are optional tags used to label transitions (Cmd attribute).
