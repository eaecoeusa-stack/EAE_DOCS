6. EXIT Structure

Structured text defines an EXIT statement as a statement for stopping a looped control structure. Syntax of the EXIT statement: 


(* FOR *)
FOR <VARIABLE := <ANY_INT TO <ANY_INT BY <ANY_INT_CONSTANT> DO
  <ST Statements>
  EXIT;
  <ST Statements>
END_FOR;

(* WHILE *)
WHILE <expression> DO
  <ST Statements>
  EXIT;
  <ST Statements>
END_WHILE;

(* REPEAT *)
REPEAT
  <ST Statements>
  EXIT;
  <ST Statements>
UNTIL <expression END_REPEAT;
Example 1: 
out0 := 0;
FOR index := 0 TO 10 DO
  out0 := 1;
  EXIT;
  out0 := 2;
END_FOR;
Example 2: 
out0 := 0;
index := value - 1;
WHILE index < value DO
  out0 := 1;
  EXIT;
  out0 := 2;
  index := index +1;
END_WHILE;
Example 3: 
out0 := 0;
index := value - 1;
REPEAT
  out0 := 1;
  EXIT;
  out0 := 2;
  index := index + 1;
UNTIL index < value END_REPEAT;
