Log file

The Action Menu Log file item links to the following commands:

Command

Description

Fetch Log Files

Display the device log file in the Log Files Viewer.

NOTE: only available for logged in devices.

The Log Files Viewer toolbar:

Command

Description

File

Load Log: Opens the Open Log File dialog, where you can open:

  • a Cyclic Log File, or

  • a Device Log File

NOTE: Select a log file type͢—cyclic.log or device.log— to display the available log files of that type.

Log Level Filters

Trace/Debug/Info/Warn/Error/Unrecoverable - possible filters, incremental, means: Fatal - only messages of category Fatal are shown; Error - messages of category Fatal and Error are shown; Warn - messages of category Fatal, Error, and Warn are shown; etc., Trace shows all messages.

Enable / Disable All Logs

All log files enabled or disabled

Go to First Log Message

Scrolls up to the first log message

Toggle Auto Log to Last Log Message

The automatic jump message, when opening the Log File window, can be switched on or off. The icon is displayed grey, when switched off.

Go to Last Log Message

Scrolls down to the last log message

Zoom Out Log List Font

Decreases font size

Zoom In Log List Font

Increases font size

Clear Log Messages

Hide all messages from the display (messages are not removed from the log file)

Search Text in Log Messages

Display only messages that contain the search string

Details

Show or hide Message Details window

Settings

To configure the display aspect and the viewer behavior

Pin On Top

Pins log file window on top of all displayed windows

Quit

Close the log file window

Message Details

Shows details of the selected message

Copy

Copies the message details to clipboard

For information on how to send information captured in the Log Files window to Schneider Electric technical support, refer to the Log Files help topic.
