Add/Update Constants Dialog Box

The Add/Update Constants dialog box allows users to rename existing variables or add new ones within the following types of Function Blocks:

  • Basic Function Block

  • Composite Function Block

  • Service Function Block

  • Adapters

NOTE: The selected Function Block must contain at least one string-type input variable for the dialog box to be accessible.

Accessing the Add/Update Constants Dialog Box

The Add/Update Constants dialog box can be open in two ways:

Right-Click Menu Options:

  • Single Function Block: Right-click on a Function Block or use the keyboard shortcut Ctrl + R to open the dialog box.

  • Multiple Constants within One Function Block: Select multiple string input variables from the same Function Block, then right-click to access the dialog box.

  • Function Block + One Constant: Select an Function Block and one of its input variables, then right-click.

Limitations

The following actions do not grant access to the Add/Update Constants dialog box:

  • Selecting multiple Function Blocks at once.

  • Selecting Function Blocks that do not contain string-type input variables.

  • Selecting one variable from one Function Block and another from a different Function Block.

NOTE: Using Ctrl + R on unsupported selections will trigger an error message.

Multi-User Access Restrictions

When a user reserves the solution, other users cannot access the Add/Update Constants dialog box.

If another user attempts to do so, a warning message will appear.

Using the Update Constants Dialog Box

Right-click on a variable to open the Update Constants dialog box, the selected variables are displayed (up to 6 rows).

If more than 6 variables are selected:

  • The first 4 variables are displayed.

  • Row 5 displays an ellipsis (…) to indicate hidden items.

  • Row 6 shows the last selected variable.

Match

Type the word or segment you want to search for within variable names. Once text is entered, the “Update To” field becomes “Replace With”, allowing you to define what the found word should be replaced with. Variables containing the matched word will be displayed in the Preview area.

Update To / Replace With

Use this field to rename or update variables by using a single word or multiple segments separated by slashes (/). After typing, click Update to apply changes to the selected variables.

Current Name Button

Clicking this button inserts the $& symbol into the field, that will make appear the original name of the variable in the Preview. You can modify constants by adding a prefix or suffix, or by combining them with numbers. If the content is long, hover to view it via tooltip.

Start ascending from

Set a starting number when applying sequence numbers to variables using the Ascending or Descending options.

Ascending / Descending Buttons

Automatically append numeric values to variable names. Numbers begin from the value defined in the Start ascending from. If the Descending value is less than the number of selected variables, a warning message will appear.

Using the Preview section

variables added to the list are displayed in the Preview (e.g., six items).When you select three variables, only those three will be editable in the preview. The Update button remains disabled unless you make a change.

Using the Add/Update Constants Dialog Box

Right-click on a Function Block or use the keyboard shortcut Ctrl + R to open the Add/Update Constants dialog box.

The Function Block context menu retains the same behavior as described above, with one additional control:

Overwriter for the existing constant names checkbox.

  • When unchecked, it shows only the variables not yet added.

  • When checked, it overwrites all added variables.

This checkbox allows precise control over whether updates affect the full list or only new entries.
