Automatic Generation

The automatic generation of the Modbus registers is an option given in the Hardware Configuration editor. This option will automatically generate and set up Modbus registers accordingly to the Modbus mapping file containing settings information filled by the user. This Modbus mapping file is an XML file made through Excel containing all the settings information about the Modbus registers.

Doing this automatic generation will avoid the user to set up manually most of the parameters of the Modbus registers imported (few parameters are not handled by the automatic generation). With a large number of Modbus registers to set, this automatic generation can reduce the searching time of parameters as all their settings is done in Excel. It can also reduce mistakes or oversights during the settings of the parameters.

To automatically generate the Modbus registers with Modbus Gateway, follow these next steps:

  1. Create the Modbus mapping file.

  2. Import this Modbus mapping file into EcoStruxure Automation Expert.

  3. Finalize the generation by adding few settings to the Modbus registers not handled by Modbus Gateway.

NOTE: Before starting this procedure, make sure that the latest version of the SE.ModbusGateway library is referenced in the project. To learn more about this library and its function blocks, press F1 on it in the list of project references.
