Installed Libraries

Use the Installed Libraries dialog to install, uninstall, and view libraries on the host PC. Only libraries that have been installed are available to be added as a solution resource.
NOTE: You can install or uninstall a library only when no solution is open.

Features

Click the File menu, then in the Backstage area select Installed Libraries to display this dialog, its elements and commands.

Element

Description

Name

Name of the installed library. The icon shows already whether the library can be used in a correct way.

The Library icon   shows that the library is correctly installed.

Version

Version number of the library.

Click the Filter icon to open the Filter Libraries dialog. Use this feature to filter the library list according to the library version or name.

Storage

Shows the storage area of the library.

Click the Filter icon to open the Filter Storage Location dialog. Use this feature to filter the library list according to the storage location.

Install

Opens a dialog for installing new libraries.

NOTE:  Close current projects before installing new libraries.

This function installs new libraries on the computer. If you want only to create references to already installed libraries in a project, refer to the topic Project Library References

Uninstall

Removes the selected library completely from the computer.

NOTE: Close current projects before uninstalling libraries. If you want to remove the reference of libraries from a project: Project Library References Info Shows information about the library and whether the selected library is correctly installed. Here you can get an exact description of referenced, missing library
Info

Opens a read-only dialog describing the attributes of the selected library.

When installing libraries, in some cases the same library may be already installed on this machine at another storage location. In this case, the Library installed multiple times dialog opens immediately after the installation of the library (this may also occur after installing the EcoStruxure Automation Expert - Buildtime), which lets you remove the obsolete library. You can use this dialog to:

  • Specify where the library will be stored.

  • Exit the installation.

  • Click Help to get assistance on how to proceed. 

Newly installed libraries are stored at C: \ProgramData\CATLibraries\Shared. Usually the predefined storage is the right one, the duplications on all other locations will be removed. Only in some special cases it might be necessary to select another storage location. That might occur if an older library has to be used, or with the use of a library that can also be installed at a user defined location, as discussed above.

EcoStruxure Automation Expert - Buildtime can start only after multiple libraries have been removed from the machine manually or by the use of the dialog Library installed multiple times. The button Exit can be used to close the dialog without uninstalling something to check the machine for the proper storage location. When restarting the EcoStruxure Automation Expert - Buildtime, the dialog shows up again, until the multiple libraries have been removed.

While installing or uninstalling libraries you may be informed you lack administrator privileges. You can use the dialog to proceed and logon with administrator privileges.
