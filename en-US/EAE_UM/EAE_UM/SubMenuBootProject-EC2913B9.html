Boot project

The Action menu Boot project project item links to the following commands:

Command

Description

Set Active Project As Boot Project

Stores the application deployed in a Device as its Boot Project, to make it available when the device restarts.

The Set Active Project as Boot Project command, which is set to False by default, is located in the Properties tab of the Logical Devices editor. When this property set to True,the user does not need to manually execute the Set Active Project as Boot Project command when deploying a device in the Deploy and Diagnostic editor. User can also set the active project as boot project while Deploy To Run Assistant by checking the Set Active Project as Boot Project checkbox" ,provided this property is set as True.

If the Set Active Project as Boot Project property is set to False in

the Properties tab of the Logical Devices editor, the active project will not boot automatically after deployment is complete. To boot a project, navigate to the Deploy and Diagnostic editor, select Boot project from drop down menu and click on Set Active Project As Boot Project.

Load Boot Project
(HA Pair)

Sets the Boot Project of two paired high availability controllers as their active application. . Refer to the topic describing the HA Pair level command Load Boot Project in the EcoStruxure Automation Expert – HA Soft dPAC User Guide.

Load Boot Project
(Single Load Boot Project - HA Partner)

Sets the Boot Project of a device as its active application.

NOTE: The behavior of the application depends on the attribute (set in the Logical Devices editor) Runtime Configuration > Boot > Behavior on reboot, as follows:

  • Stop: the device enters Stopped state.

  • Run: the device enters Running state, if a boot project exists.

  • LastState: the device re-enters the state it was in before the reboot.

  • LastStateIgnoreErr: the device attempts to re-enter the state it was in before the reboot, even if that state was the detected error state or the halt state.

Refer to the topic Action Commands Impact on Operating States for information on how the Load Boot Project command impacts device operating states.

Delete Boot Project

Deletes the current Boot Project stored inside the device

Refer to the topic Other Action Menu Commands for information on how the Delete Boot Project command impacts device operating states.
