Configuring the EtherNet/IP Fieldbus Scanner

When you right-click the device to be scanned in the Physical devices, and you select Open configuration editor, the EtherNet/IP Configurator opens.

Use the configurator to perform the following actions:


If you configure a device with specific DTM, referenced in the hardware catalog:

  1. In the device DTM configuration page of the EtherNet/IP Configurator, set and fill out the required parameters.

    NOTE: You can find DTM information by clicking the Help button on the toolbar.

  2. Click OK, then click Yes to save your configuration and close the configurator.

    Result: Predefined I/O mapping is generated.


If you configure a generic device, or a device with DTM which is not referenced in the hardware catalog:

  1. In the DTM Selector of the EtherNet/IP Configurator, select the corresponding DTM in the catalog and click OK.

    NOTE: For generic devices, any DTM in the catalog can be selected.

    Result: The device DTM configuration page opens.

  2. Set and fill out the required parameters.

    NOTE: You can find DTM information by clicking the Help button on the toolbar.

  3. Click OK, then click Yes to save your configuration and close the configurator.

    Result: I/O mapping is generated, based on the DTM configuration.


NOTE: After the device is configured, you can Copy it, if you want to create multiple devices, containing the same configuration.
