Handle Custom Layouts

As detailed in the topic Working with Pads, the EcoStruxure Automation Expert - Buildtime Integrated Development Environment (IDE) is totally flexible.

The user can then arrange all the pads and editors as wanted.

To help the user managing the IDE, few predefined layout are available as listed in the topic View menu.

In addition, custom layouts can be created to define specific IDE. Each of them being possibly unique, the user can then change anytime from a custom layout to another depending on the needs.

This topic then details all the following actions helping to handle the custom layouts:

  • Open Edit Custom Layout dialog box.

  • Create new custom layout.

  • Set to active a custom layout.

  • Define custom layout.

  • Reset the custom layout to its defined IDE.

  • Organize custom layouts.

  • Delete a custom layout.

Open Edit Custom Layout Dialog Box

To open the dialog box Edit Custom Layout, follow the next steps:

  1. In EcoStruxure Automation Expert, click on the View menu.

  2. Click on the Window Layout.

  3. In the drop down list, select Edit the custom layouts.

The Edit Custom Layout dialog box opens.

Create New Custom Layout

To create a new custom layout, follow the next steps:

  1. Open Edit Custom Layout dialog box.

  2. Click on the line named New custom layout.

  3. The line name disappears, enter instead the new custom layout name.

  4. Press Enter to validate the name of the new custom layout.

  5. Click on OK to validate the creation of the new custom layout.

The new custom layout is now created and appears in the Window Layout drop down list.

NOTE: The name of an existing custom layout can be modified anytime by following the same actions as shown here.

Set to Active a Custom Layout

To set a custom layout to active, follow the next steps:

  1. Ensure to have a custom layout created.

  2. Click on the View menu.

  3. Click on the Window Layout.

  4. In the drop down list, select the wanted custom layout.

The layout then updates to the wanted layout as previously defined.

Define Custom Layout

To define the layout of a custom layout to a wanted IDE, follow the next steps:

  1. Ensure to have a custom layout created.

  2. Set to active that custom layout.

  3. Then modify the IDE as wanted.

  4. One done, click on the View menu.

  5. Click on the Window Layout.

  6. In the drop down list, select Update the layout.

The custom layout is then updated to this new IDE. It then becomes the new default IDE of the custom layout.

Reset the Custom Layout to its Defined IDE

To reset a custom layout to its defined IDE after having modified it, follow the next steps:

  1. Ensure to have the custom layout set as active.

  2. One done, click on the View menu.

  3. Click on the Window Layout.

  4. In the drop down list, select Reset the layout.

The custom layout is then reset to its last defined IDE discarding all the modifications recently made on the IDE.

Organize Custom Layouts

For additional fine tuning, all the custom layouts can be reordered in the Window Layout list as described in the next steps:

  1. Open Edit Custom Layout dialog box.

  2. Select the custom layout to reorder.

  3. Click on the up or down arrow depending on the need.

  4. Repeat this action to all the custom layouts requiring reorder.

  5. Click on OK to save the modifications done.

The order of the custom layouts is then reorganized. Thus, see this new custom layouts order by clicking on the Window Layout.

Delete a Custom Layout

To delete a custom layout, follow the next steps:

  1. Open Edit Custom Layout dialog box.

  2. Select the custom layout to delete.

  3. Click on the following button:

  4. In the Delete layout dialog box appearing, click on Yes.

  5. Click on OK to save the delete.

The custom layout is then not available anymore.

NOTE: If that deleted custom layout was the one active, the selected layout automatically becomes Default.
