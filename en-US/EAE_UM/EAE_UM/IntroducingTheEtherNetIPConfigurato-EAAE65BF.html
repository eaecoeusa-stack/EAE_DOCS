Introducing the EtherNet/IP Configurator

Use the EtherNet/IP Configurator to create and edit configurations for the EtherNet/IP scanner. Only the EtherNet/IP protocol can be configured using this tool. Launch the EtherNet/IP Configurator from the Hardware Configuration editor. When the EtherNet/IP configurator first opens, an EtherNet/IP scanner object is added by default.

The EtherNet/IP Configurator contains the following areas:

Area

Description

Network View pane

(located on the far left)

In this pane you can:

  • Use the context menu commands to add, remove, and select EtherNet/IP devices as part of the EtherNet/IP configuration. The available context menu commands are described below.

  • Edit the names of the EtherNet/IP scanner and the EtherNet/IP devices to match the names used in EcoStruxure Automation Expert.

Configurator toolbar

(header area)

The toolbar contains the following commands (left to right) for the EtherNet/IP configurator:

  • Save: Saves your configuration edits, and asks if you also want to export the configuration.

  • Close: Closes the currently opened configuration, and asks if you want to save any unsaved changes.

  • Cut: Removes the selected device from the Network View pane.

  • Copy: Copies the selected device configuration, which can then be used to create a new device object by pasting it into the Network View pane.

  • Paste: Creates a new device configuration by adding a copied device configuration.

  • View > Catalog: Opens the catalog of available DTMs that can be added to the EtherNet/IP configuration. This command provides the same functionality as the EtherNet/IP scanner context menu Add command.

DTM configuration area

(located center and right)

This area contains DTM configuration tabs for the EtherNet/IP scanner and for EtherNet/IP devices that have been added to the configuration.

NOTE: Use the Help button to access information explaining how to use the DTM configuration tabs.

Network View Pane Context Menu Commands:

The Network View pane offers commands relating only to the configuration of the EtherNet/IP scanner and the EtherNet/IP devices. Commands that relate to the operation of the solution or of individual devices that might appear in the context menus have been disabled. Only the enabled commands are described below:

Command

Description

Add...

Opens the DTM Selector dialog that you can use to add DTMs to the EtherNet/IP configuration.

When you select a device DTM and click Ok, the selected device DTM is added in two places:

  • In the Network View pane beneath the EtherNet/IP scanner.

  • In the DTM configuration area Device List.

NOTE: When adding a new device DTM, replace the default name with a unique, meaningful name at the time it is created. The Network View pane and the DTM configuration area Device List do not always display device DTMs in the same order. As a result, if you decide to Remove a device DTM – especially where multiple, similarly named device DTMs of the same type exist – you might unintentionally remove a device DTM other than the one intended.

This dialog includes a filtering function. You can perform a simple search by typing characters into the Search Catalog... field and the dialog will display only devices with those characters in one of the fields (Device, Version, Date, Vendor). Simply delete your input characters to clear the filter and display all DTMs.

Or you can click on Advanced to open additional search tools, that let you specify field-specific search criteria for one or more of these fields.

To add a search filter:

  1. Select a field in the Device List.
    NOTE: Device, Version, and Vendor are available.

  2. Click Add Filter.

  3. Type in a filtering value (for example, device or version), or select a filtering value from the list (for example, Vendor).

  4. The filter is immediately applied.

  5. Click Clear All Filters to end filtering.

Paste

Creates a new EtherNet/IP device with the same settings copied using the Copy command for an EtherNet/IP device.

Additional Functions

Contains a collection of commands that you can use to:

  • Add and remove DTMs from the library.

  • Export the entire EtherNet/IP scanner configuration as a zip file.

  • Display information about the SoftdPAC DTM.

Import

Used to select and import a previously exported EtherNet/IP device configuration xml file.

Create PDF Report

Used to create a report in PDF format describing the EtherNet/IP configuration settings.

EtherNet/IP device context menu commands include:

Command

Description

Copy

Copies the EtherNet/IP device configuration settings. Once copied, the EtherNet/IP scanner Paste command can be used to create another device with the same settings.

Remove

Deletes the selected EtherNet/IP device from the configuration.

Configuration

Opens the selected device DTM as a tab in the DTM Configuration area.

Additional Functions

Displays information about the selected device DTM.

Upload Parameters from device

Copies the parameter settings from the device to the EtherNet/IP Configurator.

Download Parameters to device

Copies the parameter settings from the EtherNet/IP Configurator to the device.

Export

Exports the configuration settings of the selected EtherNet/IP device DTM in xml format. Once exported, the EtherNet/IP scanner context menu Paste command can be used to create a new device with the same configuration settings.
