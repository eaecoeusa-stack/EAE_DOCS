2. CASE Structure

The CASE statement executes one of several statement groups, depending on the value of the expression. The expression is analysed and compared with the specified values. If there is a match, the statements are executed in accordance with the specified value.

Syntax of the CASE control structure: 


CASE <INT-Expression> OF
<SINGLE_VALUE|LIST_OF_VALUES|VALUE_RANGE>:
<ST Statements>;
[<SINGLE_VALUE|LIST_OF_VALUES|VALUE_RANGE>:
<ST Statements>;...]
[ELSE
<ST Statements>;]
END_CASE;
Example 1: 
CASE in0 OF
1:
  out0 := TRUE;
2:
  out1 := TRUE;
3:
  out2 := TRUE;
END_CASE;
Example 2: 
CASE in0 OF
1:
  out0 := TRUE;
2,3,4,5:
  out1 := TRUE;
6..10:
  out2 := TRUE;
ELSE
  out0 := FALSE;
  out1 := FALSE;
  out2 := FALSE;
END_CASE;
