Personalizing a Process View

Once a process view has been created inside the Hierarchy View pad, it can be personalized by adding folders.

The folders can contain CAT instance, SubApp instances, SubCAT instances and other folders. The folders are used to classify and organize the process view.

To personalize the process view, a few actions are available:

  • Renaming a view

  • Deleting a view

  • Creating a folder

  • Renaming a folder

  • Deleting a folder

Renaming a view

To rename a view, select the wanted Process view in the drop-down, then click the icon and select Rename View; the Rename Process View dialog box opens; enter the new name and click OK.

Deleting a view

To delete a view, select the wanted Process view in the drop-down, then click the icon and select Delete View. Confirm the choice.

As a result, the potential ● marks before the assigned application instance names disappear in the solution tree-view.

Creating a folder

This action allows you to create a folder under the root or under another folder.

To create a new folder, follow these steps:

  1. Open the Hierarchy View pad and select the wanted Process view in the drop-down.

  2. Right-click on the process view root node or on another folder under which the new folder must be added.

  3. Select Create New Folder; this opens the Create New Folder dialog box.

  4. Enter the name of the new folder in the Name section.

  5. Click OK.

NOTE: Folders with the same name are allowed except at the same level of the same branch.

Renaming a folder

To rename a folder, select the wanted Process view in the drop-down, right-click the wanted folder name and select Rename Folder; the Rename Folder dialog box opens; enter the new name and click OK.

Deleting a folder

To delete a folder, select the wanted folder name, right-click and select Delete Folder. Be aware that deleting a folder will also remove from the Process view all the elements under it: selecting Yes deletes its sub-folders and unassign the related CAT, SubApp or SubCAT instances from the view.
