About

The About dialog displays the version and revision number of the EcoStruxure Automation Expert - Buildtime software installed on your PC.

Click the File menu, then in the Backstage area select About to display this dialog.

Software Information

The About dialog presents the following commands:

Command

Description

Copy Details

Copies the component information details to your PC clipboard.

License Agreement

Opens the end user  license agreement.

Component Information

Opens a dialog displaying the name, version and description of the EcoStruxure Automation Expert Buildtime components loaded as part of your PC's installation.

System Information

Opens the System Information dialog, which displays a list of hardware, software and components installed on the host PC. This dialog also presents a search function to more easily locate a list item.

URL

The URL of the Schneider Electric website.
