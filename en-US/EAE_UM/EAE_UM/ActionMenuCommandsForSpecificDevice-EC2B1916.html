Commands for Specific Devices

The AVEVA_SERVER device found in the SE.EAEPortal library presents unique Action menu commands.

NOTE: These commands can succeed only when connected to a Galaxy in the ArchestrA System Management Console.

The Action Menu Create Project Database item links to the following commands:

Command

Description

Compile

Compiles the application for the device, and extracts the CAT instances and their exposed objects.

Sync to AVEVA System Platform

Reads information for all CAT instances and creates/updates application objects in AVEVA System Platform with OPC UA references.

Launch System Platform IDE

Opens the connected AVEVA System Platform integrated development environment (IDE).

Test AVEVA OMI on local computer

When Active Network Profile is set to (Local Test), this command pushes all the created objects to the runtime HMI, binding the application objects created in the AVEVA System Platform to the EcoStruxure Automation Expert CAT references.

For additional information on the use of the AVEVA_SERVER device and its commands refer to the topic Asset Link for EcoStruxure Automation Expert.
