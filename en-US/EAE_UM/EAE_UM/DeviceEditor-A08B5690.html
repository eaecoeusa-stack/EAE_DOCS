Device Editor

The Device Editor shows the resources of a device graphically. Possible modifications of resources depend on the features of the device.

Open the Device Editor by:

  • Double-clicking a device in the System node of the solution in the Solution Explorer pad.

  • Double-clicking an item in the Logical Devices editor.

  • Double-clicking a device at the system level of the Function Blocks Network editor

Double-clicking a resource item the Resource editor to open the selected resource.

Double-click the background of the Device editor to go up one level from the current device to the System level, where the created devices of the project are shown.

If function blocks are added directly to a resource, they do not require mapping; they are directly assigned to this resource.

Editor Context Menu

Right-click on the background of the editor to open a context menu:

Command

Description

Go To System

Switch to the system, that means one level up (this can also be effected by a double-click on the background of the device editor)

Go To Definition

Switch to respective element, that means e.g. device type, function block type, etc. The view of the element opens.

Attributes

Opens dialog Attribute

Redraw connections

After moving the position of one or several connections between function blocks, click Redraw connections to return the connections to their default position, i.e., the position EcoStruxure Automation Expert automatically drew when you first created connections between function blocks.

Device Context Menu

A right-click on a selected resource within the Device editor opens a context menu. The menu content will vary depending on the project and status of the project (deployed and logged in or not, etc.).

This context menu is described in the Function Blocks Network editor.
