Modicon Edge I/O NTS



Modicon Edge I/O NTS is a distributed I/O option within the control system architecture that uses Ethernet IP. You can configure, monitor and parameterize the modules in connected mode.


User Workflow


Topology Editor
This view helps to add Modicon Edge I/O NTS island in Physical view :

  • Drag and Drop the island from device list.

  • Configure settings (IP address).

  • Open Configuration Editor.

  • Configure the island (to configure the island refer to the SE.ModEdgeIONTS - Library Guide).



Communication Mapping Editor
This helps you to map Modicon Edge I/O NTS island to EcoRT device under EtherNet/IP (EIP) protocol.

To open the tool, you can :

  • Open Hardware Configuration Editor and add EtherNet/IP Busmaster.

  • Open Communication Mapping Editor and select EtherNet/IP protocol.

  • Select the Modicon Edge I/O NTS island from the unscanned device list and assign it to the scanned device list.

  • You can configure Requested Packet Interval and Timeout parameters.

  • Apply the configuration .

  • Modicon Edge I/O NTS modules configured are automatically instantiated in the Hardware Configuration Editor.



Connected Mode

This view helps you to connect the Modicon Edge I/O NTS editor in secured way. You can connect online to monitor the health and status of the configured modules. It also helps you to download and upload configuration in connected mode.


To connect to the head :

  • Click on Login button to connect a device using the IP address configured in the project. Each connection to the head ask you the Credential and the Connection IP address.

  • You can change the Connection IP address in the connection windows. A dedicated screen requires you to trust the head Certificate, if not already done. At each connect, the proxy checks that the certificate in the Head matches the one trusted by the end user.

  • When you have trusted the certificate, a login panel opens and prompts you for your credentials. Each connection to the head asks you the Credential and the Connection IP address.

  • You can change the Connection IP address in the connection windows. When you click attempting to be authenticated. OK, the login panel is closed, and you are

  • When the secure connection is established and you are authenticated, the Login button is changed into Logout and remain enabled. The Download and Upload buttons are also enabled.

    NOTE: if the head connection is established, and the certificate has not been trusted, a popup with certificate information opens. You can trust it or cancel.

  • When you click on Disconnect button, Logout button is changed into Login and remain enabled. Download and Upload button are disabled.

NOTE: When the Configuration Data is opened, then you cannot modify its configuration in the physical view.
