Test Container for CAT HMI

CAT_HMI symbols or faceplates can be tested for a correct workflow by means of a separate test container. Use the Run Test Container icon  from the toolbar. This button is only available while the editor for a CAT_HMI symbol or faceplate is currently open and selected. A click on the button automatically re-compiles the project and then opens the test view.

Testing CAT HMI

Set the relevant check marks for variables of type BOOL and enter a value for variables of type REAL, INT, and so forth. Then take over the changes in the function block and start the test process by a click on the relevant event button  (which triggers the event). The effects can be checked in the test window Tester.

Additionally, this simulation includes the ability to create screenshots of the symbol or faceplate including added comments for documentation purposes.

To do so, switch from Runtime mode  to Describe mode , which will enable you to use the icons for creating comments (Bubble)  and numbering (Number) .

  • For further information about documentation see Editors at item 5. Write Documentation for Elements.

  • For general information about element CAT see CAT.
