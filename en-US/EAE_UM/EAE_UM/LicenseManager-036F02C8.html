License Manager

Overview

The License Manager is a dialog window in EcoStruxure Automation Expert, accessible by clicking License Manager in the Help menu .

The License Manager is where all the licensing information and operations are gathered.

The following elements are shown in the License Manager dialog:

  • Toolbar: Contains the buttons required for license management.

  • License Information: Provides information about the available licenses in columns with status icons for each license.

  • License Portal link: Opens the Software Management portal.

  • Device ID: The Device ID of the Engineering Workstation whose licenses are connected through the license activation process.
    NOTE: Read more information of the Device ID in chapter Important License Information.

  • Release: Displays the release date of the current EcoStruxure Automation Expert version in a yyyy.mm format.

Toolbar Buttons

Name

Icon

Description

Activate

This button opens:

  • The Online License Activation page when the PC has Internet connection. This page is used to activate online node-locked licenses.

  • The Offline License Activation page when the PC has no Internet connection. This page is used to activate offline node-locked licenses.

NOTE: For both, online and offline license activation, you can activate multiple node-locked licenses with a single license request.

Return

This button opens:

  • The Online License Return page when the PC has Internet connection. This page is used to return online node-locked licenses only.

  • The Offline License Return page when the PC has no Internet connection. This page is used to return offline node-locked licenses only.

NOTE: For both, online and offline license return, you can return multiple node-locked licenses with a single license request.

Renew

This button opens:

  • The Online License Renewal page when the PC has Internet connection. This page is used to renew the license information for all licenses online.

  • The Offline License Renewal page when the PC has no Internet connection. This page is used to renew the license information for all licenses offline.

Refresh

This button refreshes the licenses information in the License Information page. This page lists the available licenses on the Engineering Workstation. The License Information page displays the following information for each available license:

  • Status icon with a tooltip (see the different status available for the licenses in EcoStruxure Automation Expert Licenses)

  • Name

  • Version

  • Part Number

  • Activation ID

  • Expiration Date

  • Source

NOTE: For reserved licenses, under Expiration Date are displayed, the expiration date of the reserved license on the Engineering Workstation, and its expiration date on the License Server.

Process Response

This button opens the Process Response File page used to complete offline license operations like : offline node-locked activation, offline node-locked renew and offline node-locked return.

License Server

This button opens the License Server page. In this page, one type of License Server can be configured, either a Cloud License Server or an Enterprise License Server.

Reserve License

This button opens the Reserve Licenses page. The button is available only when a License Server is configured and no licenses are currently reserved.

Release Reservation

This button releases the reserved floating licenses. The button is available only when a License Server is configured and licenses are currently reserved.

Settings

This button gives access to setting options. The License View option can be used to hide all incompatible, invalid and expired licenses on the License Information page of the License Manager. A filter symbol in the Status column indicates that a filter is active. The filter settings can be selected individually per Windows user.

License Portal Access

The License Portal link is situated on the far-right of the toolbar in the License Manager.

This link opens the Software Management portal where you login with a MySchneider account. If you do not have a MySchneider account, you can create one by clicking Register and filling the required information.

After logging in to the portal, you can manage all your Schneider Electric licenses, registered to your account. The license registration is either done with your license order, or you can do it in the portal, using the Activation IDs of your licenses.

The following information and actions are available on the Software Management portal:

  • Search and view license information (for example, available and total number of copies, license expiration or duration, part number, Device ID, Software Assurance information, etc.).

  • Register licenses to your account.

  • Group your licenses.

  • View and manage devices.

    NOTE: For EcoStruxure Automation Expert you can create Cloud License Servers from the Devices tab.

  • Offline license operations.

  • Manage user access.

More information on the Software Management portal is available on the portal when you click on Get started and help.

NOTE: The Software Management portal is not supported in China.

Important License Information

  • It is recommended to close your EcoStruxure Automation Expert solution before making any license operation or License Server configuration in the License Manager. Otherwise, if a used license becomes unavailable, the solution closes automatically and proposes to save unsaved changes first.

  • The license activation process connects the licenses to the Device ID of the Engineering Workstation. The Device ID of the machine, used with your node-locked licenses is displayed in the License Manager and in the Software Management portal for your registered licenses.

    In the following specific cases, the Device ID of your Engineering Workstation can change, which sets the status of your licenses to Invalid. Invalid licenses cannot be used anymore as they become orphaned.

    • Renaming the Engineering Workstation:

      Before renaming an Engineering Workstation, first return all activated node-locked licenses, otherwise they become orphaned.

      Resolving this orphaned license issue can be done by reverting the name change, rebooting, then perform a renew operation by clicking the Renew button in the License Manager. Afterwards, return all node-locked licenses if you need to rename the Engineering Workstation again. The returned licenses can then be activated again on the renamed Engineering Workstation. If these actions do not solve the issue, contact your local Schneider Electric support.

    • Changing the network configuration on a physical machine:

      Removing or disabling of network adapters, which were present at node-locked license activation, can change the Device ID and the licenses can become orphaned.

      Resolving this orphaned license issue can be done by reverting the network adapter change to the one at license activation, restarting EcoStruxure Automation Expert, then perform a renew operation by clicking the Renew button in the License Manager. If these actions do not solve the issue, contact your local Schneider Electric support.

    • Cloning a virtual machine:

      Before cloning a virtual machine, first return all activated node-locked licenses, otherwise they become orphaned. The returned licenses can be activated again on the virtual machine after it has been cloned.

      Resolving this orphaned license issue can only be done by your local Schneider Electric support.

  • It is recommended to synchronize the clock of your Engineering Workstation with a time server. Otherwise, if the clock is set in the past, your licenses might be displayed as Invalid.

  • Only one copy of an Engineering or Addon license is needed per Engineering Workstation to enable the corresponding product features. This copy is shared with all users logged in on this workstation.

  • If a valid node-locked Engineering or Addon license is activated on an Engineering Workstation, it does not in addition borrow an Engineering or Addon license from a possibly configured License Server.

  • If you use your licenses from a License Server and different versions of EcoStruxure Automation Expert are installed in parallel on your Engineering Workstation, it is recommended to use only one EcoStruxure Automation Expert version at a time. If you need to use multiple versions at the same time, start the latest version first, otherwise all licensed features might not be unlocked.

  • The license reservation is supported only by EcoStruxure Automation Expert 23.1 and later. If you run EcoStruxure Automation Expert 23.0 or earlier while licenses are reserved from a later version, the reserved licenses can be released automatically from your Engineering Workstation but not from the License Server. In this case, the released licenses become available again on the License Server only after the reservation period ends.
