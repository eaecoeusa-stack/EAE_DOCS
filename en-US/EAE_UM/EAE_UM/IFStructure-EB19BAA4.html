1. IF Structure

An IF statement consists of a condition and a code section, which, in turn, consists of one or more statements. When the programme is run, if the conditional statement is reached, the first condition is then analysed, and if it applies, the code section is then executed. Then, in each case, the programme continues to be executed with the statements following the conditional statement.

Syntax of the IF control structure: 


IF <Boolean Expression> THEN
<ST Statements>;
[ELSIF <Boolean Expression>THEN
  <ST Statements>;...]
[ELSE<ST Statements>;]
END_IF;
Example 1: 
IF in0 = TRUE THEN
out0 := TRUE;
ELSE
out0 := FALSE;
END_IF;
Example 2: 
IF in0 = TRUE THEN
out0 := TRUE;
ELSIF in1 = TRUE THEN
out1 := TRUE;
ELSIF in2 = TRUE THEN
out2 := TRUE;
ELSE
out0 := FALSE;
out1 := FALSE;
out2 := FALSE;
END_IF;
