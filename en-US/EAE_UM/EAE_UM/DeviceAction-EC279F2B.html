Actions

Click Action to display the menu available for this editor.

NOTE: The specific set of commands may vary depending on the device:

Command

Description

Login

Establishes a connection to the selected devices and logs into the devices.

For troubleshooting information refer to Operating States and Commands Notes for this command.

Logout

Breaks the current connection to the devices.

Compile

Compiles the application for the selected device.

Compile for Online Change

Performs an incremental compile of changed parts of the application for the selected device. Available for:

  • Standard (non-HA) controllers

  • HA Paired controllers (but not an individual HA Partner controller)

Create Project Database

For solutions that perform variable and alarm archiving, this command is available for Archive_Database objects and creates a database in the PostgreSQL database that is automatically installed when EcoStruxure Automation Expert Runtime is installed. 

 WARNING
APPLICATION START AFTER DEPLOYMENT
Before sending a Deploy or Start command to devices, ensure that the application can start safely even if the devices start asynchronously or if some of them do not start at all.
Failure to follow these instructions can result in death, serious injury, or equipment damage.
 WARNING
APPLICATION MISMATCH
Before deployment, ensure robust communication links exist with the devices. After deployment verify that all targeted devices successfully run the deployed application. At application level, ensure that in case of unsuccessful deployment of the application on some devices, the system can still run safely.
Failure to follow these instructions can result in death, serious injury, or equipment damage.
