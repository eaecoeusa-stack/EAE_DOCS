Editors Overview

Use editors to create and modify function blocks (including their connections) and other components of a project. Refer to the graphic editors topic for information on editing HMI-related elements.

NOTE: The interface editor is the same for all function blocks, and is discussed at Function Block Interface Editor.

Several editors are described in this document - including Composite Function Block Editor, Sub Application Function Block Editor, Application Editor, and Resource Editor - which share the following features:

  1. Instantiate Function Block Objects

  2. Frame and Comment Objects

  3. Connection Objects

  4. Modify Existing Connections and Adjust Attributes

  5. Write Documentation for Elements
