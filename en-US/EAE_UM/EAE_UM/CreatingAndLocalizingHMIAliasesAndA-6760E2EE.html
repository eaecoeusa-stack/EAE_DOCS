Creating and Localizing different CAT and Sub-CAT Instance Names

In EcoStruxure Automation Expert, different alternative names can be given to a CAT or sub-CAT instance. The alternative names are:

  • The Instance name: Name given to the CAT or sub-CAT instance when creating it.

  • The Asset Tag name: It is a name linked with the Hierarchy View pad. To add an Asset Tag name, check the topic Creating Asset Tag.

  • The HMI Alias name: It is a CAT or sub-CAT instance attribute from the Attributes dialog box. To add an HMI Alias name, check the topic Creating and Displaying HMI Alias Name.

Each of these different names can be displayed dynamically on the HMI symbol of a CAT or sub-CAT instance as described in the topic Display Instance name, Asset Tag Name and HMI Alias Name.

In addition, you can also assign a display name to a Trend series.

The Asset Tag, HMI Alias, and Trend series display name can be created in a localized language, and displayed in the HMI in that language.
