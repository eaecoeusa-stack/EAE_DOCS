4. Launch Asset Link for Bulk Engineering & Configure

Step

Action

1

Right-click the node of your Solution and select Launch Asset Link for Bulk Engineering.

This opens the Asset Link for Bulk Engineering tab. After reading the introduction, click Start workflow to proceed. The milestones of the workflow show.

NOTE: You can switch from this tab to a CAT tab, to adjust a rule, save it and resume the workflow.

2

Click the icon. This opens a threefold window (the first tab is used only in the manual mapping Engineering instance – CAT).
NOTE: To know more about the Library Rule Configuration tab, check the topic Library CATs mapping criteria and exclude library CATs in auto-map.

3

In the Library Rules Configuration tab, you can override the library rules to succeed in matching engineering instances to SubCATs; you can also exclude a whole library from the auto-mapping process. Click Save.

NOTE: For more details, check the topic Override library CATs mapping criteria and exclude library CATs from auto-mapping.

4

In the Hierarchy Mapping tab, pair the levels in the water fall (default generic names) with one of the proposed names. This way the names in the Hierarchy view will be in accordance with the AVEVA model or the Excel headers. Click Save.

NOTE: With DEXPI files, this tab is void.

5

In the Application Layer Mapping tab, pair the high levels in the AVEVA model or in the Excel file (usually PlantName and AreaName headers) to the Application and Layer containers on which EcoStruxure Automation Expert is based . Click Save.

6

Close the threefold window.
