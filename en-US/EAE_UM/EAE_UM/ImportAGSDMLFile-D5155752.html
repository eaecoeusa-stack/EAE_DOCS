Import a GSDML File

After adding a PROFINET IO-Controller to the project, follow these steps to import one or more GSDML (General Station Description Markup Language) IO-Device files:

  1. In the Hardware Configuration editor, right click on the PN_IOController node and select Configure.

    The PROFINET IO Configurator tool opens.

    NOTE: If one or more GSDML files were previously imported, they are listed in the Available Devices list.

  2. Click Manage Catalog to open the GSDML Catalog.

  3. Click Import GSDML in the top-right area of the catalog.

  4. In the Open dialog, navigate to and select one or more GSDML files, then click Open.

    The selected files appear in the GSDML Catalog.

  5. Click Back to close the catalog and return to the Available Devices display

    You can see a list of the IO-Devices whose GSDML files you imported.
