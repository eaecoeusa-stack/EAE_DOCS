Configure a PROFINET IO-Controller - Procedure

Follow these steps to configure the PROFINET IO-Controller you previously added:

  1. In the Hardware Configuration editor, right click on the PN_IOController node and select Configure. The PROFINET IO Configurator tool opens.

  2. In the Controller Configuration area, view and edit values for:

    • IP Address

      NOTE: Enter the existing address of the port to be used for PROFINET communications. The editor cannot assign a new address. PROFINET bus master uses this IP address to locate the Ethernet port used by PROFINET-IO.

    • Subnet Mask

    • Default Gateway

    • Bus ID: (read-only) The name of the controller node entered in the Hardware Configuration editor.

    • Station Name: Enter a PROFINET network name for the controller.

    • Send Clock (ms): The time allocated for the controller to process its inputs and outputs.

      NOTE: Enter a Send Clock time setting that is less than the device update time. The smaller the Send Clock time setting, the greater the controller CPU usage, as more data must be processed quickly.

    • Start Mode: Settings include:

      • Automatic: A PROFINET BusMaster INIT event is triggered when the controller enters RUN state. The PROFINET controller starts using the configuration deployed in EcoStruxure Automation Expert - Runtime.

      • Manual: The PROFINET BusMaster needs to be manually triggered using an EVENTCHAINHEAD function block with the NAME = HW_INIT. The PROFINET IO-Controller will not start until manually triggered.

    • VTQ Types:

      • Enable supports the use of VTQ data types in your PROFINET network.

      • Disable supports ordinary data types, without time and quality data.

    • Role: Select IO-Controller to configure the Soft dPAC as only an IO-Controller.

      NOTE: Refer to Configuring a Soft dPAC Controller as an I-Device for instructions on how to configure the Soft dPAC as an IO-Device or as both an IO-Controller and an IO-Device.

  3. To save your edits, click Update and Close, then click Yes to confirm.
