7. RETURN Statement

Structured text defines a RETURN statement as a STOP statement for the algorithm or block.

Syntax of the RETURN statement: 


<ST Statements>
RETURN;
(* code after a RETURN statement is not reachable *)
<ST Statements>
Example 1: 
out0 := 0;
FOR index := 0 TO 10 DO
  out0 := out0 + 1;
END_FOR
RETURN;
(* not reachable code *)
out0 := 99;
Example 2: 
out0 := 0;
index := value - 1;
WHILE index < value DO
  out0 := out0 + 1;
  index := index +1;
END_WHILE;
RETURN
(* not reachable code *)
out0 := 999;
Example 3: 
out0 := 0;
index := value - 1;
REPEAT
  out0 := out0 + 1;
  index := index + 1;
UNTIL index < value END_REPEAT;
(* not reachable code *)
out0 := 9999;
