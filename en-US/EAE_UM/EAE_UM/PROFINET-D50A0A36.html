PROFINET

PROFINET is an industry standard for real-time, unsynchronized data communication over industrial Ethernet. It collects data from and controls equipment in industrial automation systems operating under demanding time constraints.

Use the PROFINET IO Configurator in EcoStruxure Automation Expert to configure, control, and diagnose data exchanges between controllers (IO-Controllers) and Ethernet field devices (IO-Devices).

EcoStruxure Automation Expert supports both cyclic and acyclic PROFINET communications for Soft dPAC controllers.

Refer to Configuring a PROFINET IO-Controller for information how to configure a PROFINET IO-Controller network for continuous periodic cyclic communications.

Refer to Configuring a Soft dPAC Controller as an I-Device for instructions on how to configure a Soft dPAC controller to act in the roles of only an IO-Device or as both an IO-Device and an IO-Controller.

Refer to Configuring Event Driven PROFINET Communication for information how to configure event driven read, write and error detection acyclic messaging for your PROFINET network.

Required Libraries

Use of the PROFINET IO Configurator requires that the following library be referenced in your solution:

  • Standard.IoProfinet

    NOTE: The Standard.IoProfinet library is not included in the EcoStruxure Automation Expert installation package. To obtain this library, contact Schneider Electric support.

For instructions on how to:

  • Add a library to your solution, refer to the topic Define Project Library References.

  • Add a library to your EcoStruxure Automation Expert application, refer to the topic Installed Libraries.
