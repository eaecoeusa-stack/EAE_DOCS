Edit Physical Devices Connections

Connect devices using their Ethernet ports: 

Legend:

  1. Connected Ethernet ports

  2. Unconnected Ethernet ports

  3. "Wire" indicating a physical connection

To connect two devices:

  1. Select a port on the first device.

  2. Hold down the left mouse button.

  3. Drag the cursor to the destination port on the second device.

  4. Release the mouse button. The connection is created.

NOTE: Creating wiring lines does not automatically create any IP address or network settings.

To change the connection point of an existing connection:

  1. Click on the connection wire. A round handle appears at each end of the wire.

  2. Select and drag a handle to the destination port on the second device.

  3. Release the mouse button. The connection is changed.
