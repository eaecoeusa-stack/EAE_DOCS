Configuring the EtherNet/IP Fieldbus Scanner and Adapters

When you select Configure EtherNet/IP scanner... in the Hardware Configuration editor, the EtherNet/IP Configurator opens. Use the configurator to perform these tasks:

  1. Configure the IP address settings of the EtherNet/IP scanner:
    1. In the Network View pane, double-click the scanner node to display its DTM in the DTM configuration area.
    2. In the DTM configuration area, under Channel Properties, select TCP/IP.
      Result: The IP address settings are displayed.
    3. Edit the Module IP Address, Sub-Network Mask, and Gateway IP Address settings to match the settings in your EcoStruxure Automation Expert solution.
    4. Click Apply to save your changes.
  2. Add EtherNet/IP device DTMs to the configuration:
    1. In the Network View pane, right-click the EtherNet/IP scanner node and select Add....
      Result: The DTM Selector dialog opens.
    2. In the DTM Selector dialog, select the corresponding device DTM, then click Ok.
      Result: The selected device is added to both the Network View pane beneath the EtherNet/IP scanner, and the DTM configuration area Device List.
    NOTE: Repeat these steps for each device DTM that you want to add.
  3. Edit the names of the added EtherNet/IP devices, removing the default name and using the names to be used in the EcoStruxure Automation Expert solution:
    1. In the Network View pane, double-click on a device name.
      Result: The name becomes editable.
    2. Edit the highlighted text name to match the desired device name and hit Enter.
    NOTE: Repeat these steps for each device DTM name that you need to edit.
  4. Configure the I/O connection settings for each EtherNet/IP device:
    NOTE: An I/O connection consists of a block of input and output data to be exchanged between the scanner and each device. The scanner can scan up to 128 EtherNet/IP I/O connections.
    1. In the Network View pane, double-click a device to display its DTM.
    2. In the DTM navigation tree, either select an existing connection, or create a new connection and select it.
    3. For each I/O device, configure RPI, Input size, and Output size.
    4. Click Apply to save your edits.
    NOTE: Repeat these steps for each device I/O connection.
    NOTE: The specific DTM configuration screens can vary depending on the selected device. For information on how to create and configure connections for specific device types, click the Help button.
  5. Configure communication settings by adding IP address settings for each EtherNet/IP device:
    1. In the Network View pane, double-click the scanner node to display its DTM in the DTM configuration area, or select the scanner DTM configuration tab if it is open.
    2. Under the Device List node, select a device DTM, this is the top-most node for that device.
      Result: The Properties and Address Setting tabs are displayed.
    3. Select the Properties tab, then set Import Mode to Manual.
    4. Select the Address Setting tab, then configure IP address settings for the selected device: IP Address, Subnet Mask, and Gateway.
    5. Click Apply.
    NOTE: Repeat these steps for each EtherNet/IP I/O device.
  6. Configure input and output items for each EtherNet/IP I/O device:
    1. In the Network View pane, double-click the scanner node to display its DTM in the DTM configuration area, or select the scanner DTM configuration tab if it is open.
    2. Under the Device List node, select the Items node for the selected device DTM. Expanding the connection node might be required to see this node.
    3. Use the Input, Input (bit), Output, and Output (bit) tabs to create, position, and name I/O items.
    4. Click Apply.
    NOTE: Repeat these steps for the I/O items for each device.
    NOTE:  For information on how to create, position and name I/O items, click the Help button.
  7. When all configuration edits are completed, close the EtherNet/IP Configurator, while saving and exporting the configuration:
    1. Click the Close toolbar command ().
      Result: A dialog is displayed, asking if you want to save your changes.
    2. Click Yes.
      Result: The Saving the Project dialog is displayed, asking if you want to export the EtherNet/IP configuration.
    3. In the Saving the Project dialog:

      • ClickYes if you are finished editing the configuration.

      • Click No to close the EtherNet/IP Configurator.

      Result: If you click Yes, a Save As dialog opens.
    4. In the Save As dialog, enter a File name for the configuration file, then Save it to a zip file.
      Result: The EtherNet/IP Configurator closes and you return to EcoStruxure Automation Expert - Buildtime.
