Add PROFINET Network Cyclic Data as Symbolic Links

You can specify the PROFINET input and output device data that you want to add to your EcoStruxure Automation Expert solution. The added data appears as symbolic links (Symlinks) in the Hardware Configuration editor.

NOTE: EcoStruxure Automation Expert supports the selection of a maximum of 251 I/O data items per device to be used to create Symbolic links.

  1. In the Hardware Configuration editor, right click on the PN_IOController node and select Configure.

    The PROFINET IO Configurator tool opens.

    NOTE: The IO-Devices you previously added are listed in the Selected Devices list.

  2. Select a device and in the Modules column, hover your cursor over the ellipsis (...), then click Configure.


  3. Click Cyclic Data. A window opens displaying the available input and output data. The following information is displayed:

    • Collective information for the selected data items:

      • Symlinks Count: The number of Symlinks HWCAT I/O with the Exclude attribute de-selected.

      • Input Length: The size in bytes of input data configured, including status information.

      • Output Length: The size in bytes of output data configured, including status information.

    • Information for each individual data item:

      • Slot: Module slot location.

      • Subslot: Submodule subslot location.

      • In/Out: In = input data. Out = output data.

      • Index: Slot.Subslot address index (zero-based).

      • Type: Data type.

      • Length: Size of the data item.

      • Exclude: Only de-selected items are included in the cyclic data transferred to the Hardware Configuration editor for the device.

      • Description: Type of data item.

    NOTE: Use the Exclude All, and Include All commands (described below) to help you create the cyclic data collection to be transferred to the Hardware Configuration editor for the device.

  4. Repeat steps 2 & 3 for each device.

  5. After you finish selecting device I/O data items, click Back to return to the Network Configuration page

  6. Click Update & Close, then click Yes to update the hardware configuration.

Cyclic Data Selection Tools

The Cyclic Data selection page of the PROFINET IO Configurator presents the following commands you can use when creating a list of I/O data items to add as Symlinks in the Hardware Configuration editor:

  • Exclude All: Selects each data item in the Exclude column for the selected device. These data items will not be included in the collection of data to be transferred to the hardware configuration.

  • Include All: De-selects each data item in the Exclude column for the selected device. These data items will be included in the collection of data to be transferred to the hardware configuration.
