Creating a Process View

The creation of additional process view is needed when all CAT, SubApp or SubCAT instances can’t be assigned under the same process view, for organizational purpose.

Having several process views available into the Hierarchy View pad allows then the user to have more custom ways of organizing the CAT, the SubApp and SubCAT instances of its plant.

Now, to create a new process view, follow the next steps:

  1. Select the action Create view, as mentioned in the Overview topic.

  2. The Create View dialog box opens.

  3. Enter the name of the new process view.

  4. Press Finish to create the view.

At this point, the new process view is created and automatically selected in the view section, as mentioned in the Overview topic.
