Modicon Edge I/O NTS

Overview

Modicon Edge I/O NTS is a future-ready distributed I/O system designed to bring intelligence and flexibility to industrial automation. EcoStruxure Automation Expert will allow user to integrate communication head module as distributed​ IO node over EtherNet/IP in control system architecture. This release will enhance interoperability and I/O integration to simplify automation system design and deployment.

For more information about Modicon Edge I/O NTS, refer to the Modicon Edge I/O NTS – Professionnels | Schneider Electric France.

New Features

EcoStruxure Automation Expert Modicon Edge I/O NTS editor will offer toolbar to enhance interoperability. Unlike the last release, this release will allow user to connect in online mode to Modicon Edge I/O NTS head.

The toolbar provides a central control point for managing devices during engineering and maintenance. It will allow users to:

  • Log in to establish a secure connection with the target device.

  • Download project configurations to apply updates on the live system.

  • Upload existing configurations from the device for validation or backup.

  • Toggle between Online/Offline mode to Monitor and analyze the device’s real-time health, status, and error conditions.

    By consolidating these actions, the toolbar streamlines configuration management, ensures consistency between the EcoStruxure Automation Expert Editor and Modicon Edge I/O NTS head, and simplifies troubleshooting in online mode.

Compatibility

Modicon Edge I/O NTS is supported by all dPAC controllers :

  • Seamless integration with all devices supporting Ethernet/IP.

  • Manual integration with all devices supporting Modbus.

Full performance tests has been done on SoftdPAC Linux and M262d.

NOTE: Integration of the Modicon Edge I/O NTS over Modbus TCP/IP is not yet seamless. However, users can engineer modules using the Generic Modbus TCP device hardware CAT and still exchange data over Modbus TCP/IP.
