Device Information

In the Deploy and Diagnostic editor, general information for each device is given into their device lines (Status, Next Step...).

In addition to this general information, extra information shows by clicking on the button . One extra window appears below the device line.

NOTE: Click on the button to hide this window.

In this window, there is information concerning:

  • The solution (In the solution): The information in this column is related to the current open solution.

  • The device (Running the device): The information in this column is related to the running application in the selected device. Login and deploy the device to get the information from this column.

  • The boot application (In the boot application): The information in this column is related to the boot project of the selected device. Login and Set Running Project As Boot Project to the device to get the information from this column.

The following information is given specifically for each columns:

  • Solution name: Corresponds to the name of the solution.

  • Device name: Corresponds to the name of the device.

  • Studio version: Corresponds to the EcoStruxure Automation Expert studio version.

  • SCC revision (Source Code Control revision): Corresponds to the revision number of the solution when SVN multiuser management is used into the solution.

  • Last update: Corresponds to the date and time of the last update made.

All this information is only concerning the tab Application of the extra window. This tab is always available.

In addition, other tabs can appear once logged in to a device depending on the libraries referenced to the solution. These tabs give information about the device MIBs and their diagnostic data.
