Create new solution and map your CAT instance

Create a solution on EcoStruxure Automation Expert software that should have an ATV dPAC Module

To Configure an ATV dPAC display, you must create an ATV CAT instance

Refer to SE.IoATV Altivar Distributed PAC Module VW3A3530D Library Guide, chapter ATVDisplay - Hardware CAT, to learn how to:

  • Configure your physical device.

  • Configure your logical device.

  • Configure the hardware.

  • Configure the Function Blocks Network.

  • Connect the Symbolic Links.

  • Deploy your solution.

These steps will create the necessary CAT instances based on your application, which you will then use to configure your ATVDisplay Templates.

To add and select a value in the different templates, simply drag and drop the CAT instance into the configuration view of the template.

All the inputs and outputs present in this CAT instance can be used and configured by the ATV dPAC display template.
