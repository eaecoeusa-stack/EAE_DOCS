2. HMI Symbols Library

Regarding the HMI Symbols library:

  • Sort and number of offered elements are not dependent only on the data type of the variable, but also on the activated libraries. A right-click on the project node in the Solution Explorer opens a context menu. A click on the entry Options opens a sub context menu. Choose HMI. The tab HMI Libraries opens (see also Project Options). The currently available libraries are listed here and you can activate the desired library via check box. Multiselection is possible (see screenshot below).

  • The HMISymbols library is an older library version, which is available to provide backwards compatibility. This permits older projects to be re-engineered. Activate this library only to modify older projects, which have been engineered applying this library. If you want to begin a new project, it is highly recommended to de-activate this library and activate only the HMIBaseSymbols library. It is a remastered library that contains also the elements of the HMIBaseSymbols library, but with extended functions and a new look.
