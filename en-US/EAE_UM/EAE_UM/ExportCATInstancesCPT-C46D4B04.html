Export CAT Instances

The export of CAT instances is one of the operations available to create and modify CAT instances.

This operation is exporting a selection of CAT instances and offline parameters into an Excel file.

This selection is defining:

  • Which CAT instances are exported.

  • What is the export configuration of the information into the Excel file.

  • What offline parameters are exported.

Knowing that, follow the next steps to export the CAT instances:

  1. Select the Export CAT Instances command of the node CAT Instances from the Solution Explorer pad.

  2. The Export CAT instances dialog window opens. The window is divided into three steps: Select CAT instances, Configure worksheet breakdown and Select offline parameters. Each of these steps is covering a different aspect of the CAT instances export selection.

  3. In the first step are selected the CAT instances to export.

    This selection is done by ticking manually the box of the CAT instance or by ticking the top box which will:

    • Select All CAT instances if not all were selected yet.

    • Unselect All CAT instances if all were selected.

      NOTE: This top box has the same behavior in the two next steps.

  4. Click on Next when the selection of CAT instances to export is complete. The Export CAT Instances dialog window moves to the second step.

    NOTE: At least one CAT instance need to be selected to move to the second step.

  5. In the second step are defined how the information of the CAT instances will be organized in the Excel file. The organization depends on what sub CAT of the CAT instance is selected:

    • When a CAT is selected, all its information is gathered in one line of a specific Excel sheet, depending on the CAT type.

    • All the deselected sub CATs of the selected CAT will also have their information given in the same line of the Excel sheet.

    • A sheet is specific to a particular type of CAT. Which means that it will gather the information of all the selected CAT with the same type. Each of them having their information gathered in one line.

    • As soon as a sub CAT is selected, then its information will not be gathered in the sheet of its CAT anymore. Instead its information will be added to the sheet of its type with the information of all its deselected sub-sub CATs.

    • By default, only the box of the CAT instance is selected, which means that all the information of itself and its sub CATs are gathered in one line of the CAT instance type sheet.

    Thus, select the CAT requiring to have their information separated into their own sheet and click on Next. The Export CAT Instances dialog window moves to its third and last step.

  6. In the third step is listed all the offline parameters of the CAT instances and their sub CATs available.

    Select the offline parameters to export. Their name and value will be added to the sheet of the CAT type they are part of.

    The deselected offline parameters will not be mentioned in the Excel file.

  7. Click on Export when the selection of the offline parameters to export is complete.

    NOTE: No offline parameter needs to be selected to be able to export the CAT instances.

  8. A Window folder opens. Choose a file name and a location to save the export file. Once done, click on Save to generate the Excel file.

At this point the Excel file has been generated with inside all the information selected by the user. This file can be exploited to update the CAT instances.
