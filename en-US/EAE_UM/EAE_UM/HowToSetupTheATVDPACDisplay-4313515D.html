ATV dPAC Display

Overview

In EcoStruxure Automation Expert the ATV dPAC module (VW3A3530D) allows you to display application variables on the Altivar Graphic Display Terminal. You can customize the following elements:

  • Monitoring and Setting screens, based on user-selected CAT instance variables.

  • Information screens.

  • Menus and sub-menus.

A Design Engineer can create the ATV HMI tree view in EcoStruxure Automation Expert. Once deployed, this view is shown on the ATV Graphic Display Terminal and can be used by the Operator.

The ATV dPAC Display feature works with these drive products:

  • Altivar Process ATV6•• (used as an example in this document)

  • Altivar Process ATV9••.

  • Altivar Machine ATV3••.

This chapter explains the feature and provides an application example.
