Define Ranges

Use the Define Ranges dialog to define and color-code value ranges for elements that include the Ranges property. These elements include specific HMI Base Symbols (for example Free Text, Label, LED, or Image). Elements that contain this property can be created within a CAT_Symbol or Faceplate following the process described in the topic Select Generic Symbol.

To access this dialog:

  1. Open the Properties pad for an element that includes the Ranges property.

  2. Navigate to Appearance > Ranges

  3. Click on the ellipsis (...).

The dialog Define Ranges may appear in different styles, depending on the type of variable and element.

Define Ranges Dialog

Command

Description

Click this to create new ranges. The Define Enum Value dialog opens, which you can use to specify a value range. 

Depending on the type of variable which is used, the ranges can be composed of only a single value (e.g. with INT) or a value range (e.g. REAL).

Type in an Enum value, then configure its properties (for example: Text, TextColor, Brush, Pen and so forth. Repeat the same procedure for each value you want to display with an distinct appearance at runtime.

INT types are implemented in form of enum. That means, that the use of an INT variable type offers a series of fixed values to be equipped with specific properties, instead of value ranges, which are offered with variable type REAL. Therefore it is possible not to define individual properties for all permissible values of the variable. When using HMI Base Symbol Image this would lead to showing a pink rectangle for undefined values at runtime, instead of an image. Or, on the other hand, when using HMI Base Symbol Led it would show a grey, unlighted led for the undefined values.

NOTE: To show an image at runtime instead of a pink rectangle for values which are not extra defined, consider the following.

  • Undefined values are shown applying the fallback properties. These can be defined at the node Ranges. Open the node by a click on the + in front of Ranges. The same properties, which are offered in the dialog Define Ranges, are also available here. Because for Images the default adjustment (None) is active. Just select a fitting image, which has to be shown for invalid or undefined values.

    NOTE: In the same way, for example, it is also possible to set the default color of an LED.

  • In the case of a value range (e.g. for variable type REAL), the maximum value of the range needs to be defined, when creating a new range. The beginning of the range is defined automatically by the maximum value of the last range. Consequently, the very first range which has to be defined begins with '-Infinity' and ends up with the value which can be defined in the dialog window Split Range.

NOTE: If somehow the need arises to use value ranges with INT data types, the variable can be converted to REAL or LREAL. By this, the definition of value ranges will be offered in the dialog Define Ranges automatically. However, first verify that the INT variable can be converted to REAL or LREAL without any loss of data, in order not to produce unintended results.

Click this to remove the selected range.

Defines the start and end value of the range more precisely. You can select the relational operators >, >=, = respectively between <, >=, =. A modification of the definition at the start of a range will automatically change the end of the range beforehand as well, and vice versa.

[value field]

To modify an existing range, the adjusted value can also be changed subsequently.

Text

Defines an alternative text to be shown at runtime. The default text is ${Value}. This shows the current value at runtime. Set the check box on the left to overwrite the entry by an alternative text or to add some additional text to '${Value}'.

NOTE: You can also use multilingual text in this case. Refer to the topic Multilanguage Dictionaries. Analogously to this description, multilingual text can be used also with dialog 'Define Ranges.' Replace the multilingual text by the text key expression that previously has been defined in the dictionary, for example ${Key:TextKey}. In addition, it is possible to use each further combination with the value of the variable as well as text pieces without multilingualism.

Example: ${Key:VariableValue} ${Value}°C

At design time it is expected that the same text, which is defined at column neutral in the dictionary, be displayed in the faceplate or HMI symbol. If this is not the case and it is shown ${Key:TextKey}instead or even simply the text key itself, please check the entry in the dictionary as well as the entry at Text in the dialog Define Ranges (resp. in the fallback properties of the variable as described above in the description of the command) for typos. The text key has to be identically spelled in both spots and the spelling has to be up to the example as mentioned above.

TextColor

This field serves to define a text color for the active range or value.

Brush

This field serves to define the brush (background). This is relevant for symbol label.

Pen

This field serves to define the pen (frame). This is relevant for symbol label.

NOTE: Switch to FixedSingle at Appearance/BorderStyle to have the modification take effect. Press enter after modifying the Pen to confirm the changes. Click on OK without pressing Enter will lead to discarding the modification of Pen.

[definition field for color]

A different color for each defined range can be adjusted. This is relevant for symbol led (see image directly above the table).

Image Selector Sub Dialog

When using the symbol Image, the Image Selector sub dialog can be opened from the dialog Define Ranges to adjust the relevant image.

Command

Description

From Catalogue:

A click on the ...-button opens the Image List Editor. A description of the editor can be found at Image Storage.

NOTE: When defining an image at From Catalogue: an eventually defined image at From File: will be deleted automatically. The same is true in reverse, because only one source can be active at the same time.
From File:

Click the ellipsis to open Windows Explorer. Navigate to and select the desired image.

Reset

Deletes a previously made image selection and sets to default condition.

OK

Closes sub dialog and saves modifications.

Cancel

Closes sub dialog without saving modifications.
