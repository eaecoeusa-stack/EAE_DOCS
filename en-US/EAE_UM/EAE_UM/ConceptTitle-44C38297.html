Reset the Device Security

Reset the device security by:

  1. Setting the device to STOP state, using the solution that was used to secure the device.

  2. Right-clicking on the device in the physical view and selecting Reset security.

  3. A warning dialog box appears. Enter the user credentials and then select Yes.

    NOTE: If you forgot the User or the Password credentials for the device, refer to one of the following topics:

    • ATVd: Local reset of the ATV Distributed PAC security topic in the Altivar Distributed PAC Module VW3A3530D - User Guide

    • CRD and M580d: Cybersecurity Rotary Switch topic in the Modicon M580 Distributed PAC - User Guide

    • High Availability Soft dPAC: Physical Devices Editor topic in the High Availability Soft dPAC - User Guide

    • M251d: SD Card Commands topic in the Modicon M251 Distributed PAC - User Guide

    • M262d: SD Card Commands topic in the Modicon M262 Distributed PAC - User Guide

    • Soft dPAC Linux: Reset a Soft dPAC device in the section of the Soft dPAC - User Guide

  4. Erase Galaxy database files from AVEVA Operator Stations:

  5. Erase backup files stored in secure backup locations.

  6. Re-image workstations to their original platforms.

  7. Consider using a data destruction program to erase data from hard drives.

  8. Format hard disks.

  9. Physically disintegrate and pulverize decommissioned controllers. If this is not possible, securely store the disposed boards.

The device is rebooted and the security is reset. The device is then decommissioned.
