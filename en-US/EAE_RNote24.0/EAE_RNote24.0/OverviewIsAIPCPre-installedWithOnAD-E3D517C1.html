EcoStruxure Automation Expert — Essential Edge

Overview

EcoStruxure Automation Expert — Essential Edge is a Schneider Electric iPC, pre-installed with EcoStruxure Automation Expert - Soft dPAC on a Debian Linux operating system.

The EcoStruxure Automation Expert — Essential Edge is used for the integration of IT applications and tools in a single hardware, along with application control. The following hardware functionalities are assembled under one single hardware: analytics, archiving and control.

To preserve maximum stability and performance of the EcoStruxure Automation Expert - Soft dPAC, it is recommended that you do not install or run other software that may consume hardware resources necessary for EcoStruxure Automation Expert - Soft dPAC operability. For more information, refer to the EcoStruxure Automation Expert - Essential Edge User Guide.
