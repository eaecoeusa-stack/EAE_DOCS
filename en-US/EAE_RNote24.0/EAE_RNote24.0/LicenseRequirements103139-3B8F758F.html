License Requirements


Overview

The use of EcoStruxure Automation Expert requires the activation of licenses to unlock and access all features.

The following license types are available:

  • Trial license

  • Engineering license (Standard and Professional)

  • Addon license

Trial License

Each EcoStruxure Automation Expert version includes a new trial license that has the same version number as the software.

The trial license:

  • Is automatically activated with the first launch of the installed EcoStruxure Automation Expert version.

  • Expires after 42 days (remaining time is displayed in the About page and in the License Manager).

  • Unlocks the use of basic product features to create control applications, deployable in simulation mode (Local Test profile in the Deploy and Diagnostic editor). The deployment on real devices (Default profile) requires an Engineering license.

Engineering License

Two types of Engineering license are available for EcoStruxure Automation Expert — Standard and Professional. With the Standard Engineering license, you can unlock the use of basic product features to create control applications deployable in simulation mode (Local Test profile in Deploy and Diagnostic editor), as well as to real devices (Default profile). The Professional Engineering license allows the same use as the Standard license with the addition of the special features of the addon licenses. The Professional Engineering license is not compatible with EcoStruxure Automation Expert versions earlier than V23.0.

Addon License

The addon licenses unlock specific product features in addition to the basic product features that the Standard Engineering license unlocks.

The following product features of EcoStruxure Automation Expert need an addon license to be able to unlock and use them:

  • Asset Link for AVEVA OMI

  • High Availability

NOTE: All listed product features can be used only when a Standard Engineering license and the according addon license are activated.
NOTE: The feature Asset Link for Bulk Engineering no longer needs Addon license in EcoStruxure Automation Expert V24.0 because it is only for preview.  

License Usage

Depending on your needs, each copy of your Engineering and addon license can be used as node-locked or floating license.

If EcoStruxure Automation Expert does not find any valid Engineering license at launch, a dialog box is displayed and allows the activation of your licenses as node-locked licenses. The same dialog box allows to open the License Manager to configure the connection to your Cloud License Server or your Enterprise License Server. From there, you can use your licenses as floating licenses and reserve floating licenses for offline use. You can also launch the License Manager from the About dialog box or from the Help ribbon bar.

The Enterprise License Server is a separate package that you can download for free on the same location as the EcoStruxure Automation Expert installation package. The Enterprise License Server package contains the Enterprise License Server Guide (English only), describing the requirements and the steps to install, configure and manage the EcoStruxure Automation Expert licenses with the Enterprise License Server.

You can activate your licenses at any time, even when the trial period is active.

You can activate your licenses with online or offline methods.
