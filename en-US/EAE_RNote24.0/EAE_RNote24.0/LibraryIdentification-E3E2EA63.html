Known Limitations

WEBSOCKET_SERVER

When you configure the WEBSOCKET_SERVER function block, select an unused port. The ports 51443 and 51499 are already used by other services.

Known Anomalies

EVENTCHAIN, EVENTCHAINHEAD

EVENTCHAIN and EVENTCHAINHEAD function blocks can be linked altogether only inside the same resource. 

Insert ButtonTagname to Display the Instance Name using Multi-OS EcoStruxure Automation Expert - HMI

Follow this procedure to manually add ButtonTagname to a canvas: 

  • Open a canvas 

  • Go to Solution Explorer, type ButtonTagname, grab and drop in the top left corner of the canvas 

  • On EcoStruxure Automation Expert - HMI, you can use this button to display or not the instance name 

Refer to EcoStruxure Automation Expert - HMI Graphic chapter in the online help for more details. 

SE.FieldDevice Library

AtvSpeedControl MB Serial

The new AtvSpeedControl MB Serial allows controlling Altivar drives using the MB Serial port of M262 dPAC, M251 dPAC and Soft dPAC.

Standard.IoModbus Library

MODBUSGENTCPS and MODBUSGENBC

"connStatus" input provided with Modbus Buscouplers devices,  MODBUSGENTCPS and MODBUSGENBC, can be used to halt and resume bus communication from  user application. The input is typically used in the event any configured server is undergoing maintenance. 

Modbus client (TCP & RTU) VTQ

When reading a Modbus coil or register using a VTQ, we used to receive a value, a timestamp, and a quality with only 3 possible quality values: Good, Bad, and Uncertain. However, we have now enhanced our support for quality coming from the Modbus server device to reflect Modbus errors thus helping during commissioning.
