System Requirements

Screen Resolution

An incorrect screen resolution impacts the use of :

  • EcoStruxure Automation Expert - Buildtime

  • EcoStruxure Automation Expert - HMI (Microsoft Windows)

  • EcoStruxure Automation Expert - Soft dPAC (Microsoft Windows)

  • EcoStruxure Automation Expert - Archive

This table indicates performances and limitations of screen resolutions on the main screen:

Resolution on main screen

Impact

< or = 640x480

Not supported

From 640x480 to 1280x1024

Supported with limitations (incorrect display of several dialog windows, functions unavailable)

From 1280x1024 to 1920x1080

Supported with time to time layout limitations.

> or = 1920x1080

Recommended.

EcoStruxure Automation Device Maintenance

EcoStruxure Automation Device Maintenance is a standalone commissioning and maintenance tool for Schneider Electric industrial automation devices. EcoStruxure Automation Device Maintenance V3.0 or greater is required to configure network settings (for example, setting the device IP address, name, and so forth) and to update the firmware of the devices. EcoStruxure Automation Device Maintenance is included with EcoStruxure Automation Expert package.
