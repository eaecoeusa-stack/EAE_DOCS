Positionner library

The AtvPosControl Hardware CAT and AtvPosInterface FB must be used to interface with the Positionner library. 

ATV dPAC and Graphic Display Terminal

The ATV dPAC can display EcoStruxure Automation Expert application variables on the Graphic Display Terminal of its host Altivar drive, by customizing the following UI elements:

  • Monitoring and Setting screens, based on user-selected CAT instance variables. 

  • Information screens. 

  • Menus and sub-menus. 

The Design Engineer can create the ATV HMI tree view in EcoStruxure Automation Expert. After deployment, it is displayed on the ATV Graphic Display Terminal, and can be used by the Operator. 

Local reset to factory settings

Is it now possible to reset the security of the ATV dPAC, using the drive's Graphic Display Terminal (for example if the device password has been forgotten).

Known Limitations

Drive Systems

If you want to update the firmware of the ATV drive systems, or to integrate an ATV dPAC, please contact Customer Care Center. 

EtherNet/IP Device Management

The Altivar dPAC Module VW3A3530D does not support EtherNet/IP devices. 

The Altivar drives EtherNet/IP hardware CATs can be used with other dPAC controllers. 

Firmware Update

Before upgrading or downgrading the firmware, read carefully the Altivar dPAC Module VW3A3530D User Guide. 

The document describes: 

  • If you can upgrade or downgrade the firmware from one version to another. 

  • The steps to perform an upgrade or a downgrade of the firmware. 

The firmware delivered along with this EcoStruxure Automation Expert offer is signed. Therefore, if the ATV dPAC is already loaded with a signed firmware, it is not possible to downgrade it, for example, to V20.1 compatible firmware. Please contact Customer Care Center. 

The firmware of the ATV dPAC module for EcoStruxure Automation Expert 21.2 and later can be updated using EcoStruxure Automation Device Maintenance V3. x software with: 

  • Modbus TCP IPV4 connection and .fwp firmware packages (the connection point will appear as mbap). 

  • DPWS IPV6 connection and .sedp firmware packages (the connection point will appear as https). 

A firmware update operation may trigger an SPFC Error: restart the drive to clear this error. 

HMI Faceplates Scaling

The faceplates and the symbols of the ATV dPAC HW CATs does not scale correctly when used with Embedded IO. 

Modbus TCP Device Management

The Altivar dPAC Module VW3A3530D supports up to 8 HW CAT Modbus TCP devices. 

When using MotorVS application CAT, the OPC-UA server may need to be disabled to free-up memory. 

Operating Modes

The LED lights indicating the operating states of the device are not supported by EcoRT. 

PositionerVSDBase Application CAT with Altivar dPAC Module VW3A3530D

There are certain limitations associated with using the PositionerVSDBase application CAT in conjunction with ATV dPAC. These limitations can impact the functionality and performance of the system. Be aware of these limitations when configuring and using the PositionerVSDBase application CAT with an Altivar dPAC Module VW3A3530D.

The Altivar dPAC Module VW3A3530D: 

  • The value of busCycleTime from ATVdPACxVW3A3530D hardware CAT should be set to a minimum of 10 ms or greater. 

  • The value of busCycleTime in the ATVdPACxVW3A3530D hardware CAT should be lower than or equal to the value of Cycle Time in the PositionerVSDBase application CAT. 

OPC UA Server

The OPC UA server of the Altivar dPAC Module VW3A3530D is turned off by default. You can turn it on from EcoStruxure Automation Expert. It is activated after the configuration deployment. 

Time Management

In Altivar drives, the battery used to save the time is located inside the Graphic Display Terminal. In order to keep the clock correct in conditions of loss of power, the Graphic Display Terminal must be connected before the drive is powered off. 

In Altivar Machine ATV340 range, the Graphic Display Terminal must be purchased separately. The product reference is VW3A1111
