Microsoft .NET Framework 3.5 Installation

The Microsoft .NET Framework 3.5 installation package is no longer available as offline installer for Microsoft Windows 10 and later, and for Microsoft Windows 2019 and later. Therefore, .NET Framework 3.5 is no longer installed with the EcoStruxure Automation Expert - Buildtime installer. To install .NET Framework 3.5 manually:

On a Windows computer with Internet connection:

  1. Open the Windows Features dialog from the Search bar.

  2. Select the .NET Framework 3.5 (includes .NET 2.0 and 3.0) checkbox and click OK.

On a Windows computer without Internet connection:

  1. Download the disk image file (ISO) from the Windows version installed on the computer.

  2. Mount the ISO image file on the Windows computer.

  3. Open command prompt as Administrator and run the following command:


    Dism.exe /online /enable-feature /featurename:NetFX3 /All /source: <pathto-sources-sxs-on-mounted-iso> /LimitAccess
    NOTE: The path-to-sources-sxs-on-mounted-iso is the path to the D:\sources\sxs folder of the mounted ISO file.

Automatic Root Certificates Group Policy

The group policy for automatic root certificates update must be set to Not Configured (default value) or Disabled on the computer, otherwise one of the following might occur:

  • The EcoStruxure Automation Expert installation might be unsuccessful and an Invalid digital signature error message might appear.

  • EcoStruxure Automation Expert might not function properly and error messages might appear.

To configure the automatic root certificates update:

  1. From the Windows Start menu, open the Local Group Policy Editor.

  2. Click the Local Computer Policy setting, then double-click Computer Configuration.

  3. Double-click Administrative Templates > > System > > Internet Communication Management > > Internet Communication settings > > Turn off Automatic Root Certificates Update.

  4. Click either Not Configured or Disabled, if the setting is enabled.

  5. Click OK.
