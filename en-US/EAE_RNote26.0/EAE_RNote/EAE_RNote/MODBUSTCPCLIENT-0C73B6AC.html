New Features

Modbus TCP Client: Online Change

This feature allows you to perform full hardware configuration updates dynamically (Deploy online change), without requiring a system restart (Full deploy). Online hardware catalog (HWCAT) changes for the MODBUS TCP Client will be available exclusively on the SoftdPAC platform.

Add or remove the following components in a single cycle:

  • Bus Masters

  • Bus Couplers

  • Registers / Coils

Import the entire hardware configuration (HWConfig) in one operation.

Known Limitations

Individual component-level changes are not supported with Online Change

Individual component-level changes (e.g., adding or deleting a single bus component) are not supported. Only full configuration updates are allowed.
