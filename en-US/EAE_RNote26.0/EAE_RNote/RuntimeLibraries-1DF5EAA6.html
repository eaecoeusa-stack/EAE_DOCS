New Features

NET IO Secure

NET IO service function block is now compatible with IOT and Web server standard "Server Name Indication" (SNI) for improved cyber security and compatibility. An additional input parameter named SNI was introduced to the NET IO function block, allowing users to specify the target domain name. This would enable the server to present the appropriate SSL/TLS certificate during the handshake, thereby establishing a secure connection. This will enable seamless communication with widely used servers, supporting IT/ OT convergence and unlocking new integration possibilities. It will reinforce cybersecurity by adopting industry-standard encryption protocols (SSL/TLS), enable secure data exchange. User Experience will be simplified by enabling the use of a single IP address for multiple secure services, reducing configuration complexity and improving manageability.
