New Features

Online Change

This feature allows to perform a full hardware configuration updates dynamically (Deploy online change), without requiring a system restart (Full deploy). For more details refer to Modbus TCP Online Change section.

New Protocols Supported: EtherCAT and Profinet IO Device

Two protocols are now available with Soft dPAC:

  • EtherCAT

  • Profinet IO Device

New Edge Controllers Compatible: AP310 and M590 dPAC

Two edge controllers are now compatible and pre-installed with Soft dPAC:

  • AP310

  • M590 dPAC

HTTPS support for IOT Connectivity

 HTTPS is now supported on Softdpac for IOT Connectivity.

NET IO service function block is now compatible with IOT and Web server standard "Server Name Indication" (SNI) for improved cyber security and compatibility.

NOTE: For more information about Net IO, refer to the Net IO Secure Topic.

40K OPC-UA Tags

In your EcoStruxure Automation Expert application, the OPC UA Server supports a default maximum of 20,000 OPC UA variables. If your application requires more variables, this limit can be extended to 40,000 variables by using a specific version of the SE.DPAC library, which is available upon request from Schneider Electric Support.

Vulnerability Fixed

A vulnerability has been fixed by restricting access to the launcher web server. It is now only accessible by the EcoStruxure Automation Expert - Soft dPAC Manager on the local network.

Known Limitations

NOTE: To learn more about EcoStruxure Automation Expert - Soft dPAC limitations introduced in a previous release version and still active today , please refer to the following link :Soft dPAC limitations. For new limitations, please see the two following sections.

Docker Compatibility

Soft dPAC supports only Docker versions prior to 28. Installation will fail if an incompatible version is detected. Version 27.x is required, with 27.5 recommended.

Syslog Server Configuration

To have access to the Syslog server functionality, you must configure Syslog server with the following port:

  • Unsecure port: 601

  • Secure port: 6514

This limitation will be fixed in the next EcoStruxure Automation Expert release.
