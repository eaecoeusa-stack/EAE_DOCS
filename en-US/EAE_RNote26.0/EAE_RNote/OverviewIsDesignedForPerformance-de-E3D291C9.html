New Features

Bus Diagnostic for TM3

The TM3 diagnostic feature leverages the BusStatusSymlik to deliver real-time visibility into communications between controllers and TM3 modules. This powerful tool detects module presence, bus errors, mismatches and communication faults with precision. By converting diagnostic values into binary format, user can quickly identify affected modules and error types, ensuring faster troubleshooting and improved system reliability.

HTTPS support for IOT Connectivity

 HTTPS is now supported on Modicon M262 Distributed PAC for IOT Connectivity.

NET IO service function block is now compatible with IOT and Web server standard "Server Name Indication" (SNI) for improved cyber security and compatibility.

NOTE: For more information about Net IO, refer to the Net IO Secure Topic.

Known Limitations

Modbus Serial RTU Polarization

The polarization on the Modbus Serial RTU is not effective.
