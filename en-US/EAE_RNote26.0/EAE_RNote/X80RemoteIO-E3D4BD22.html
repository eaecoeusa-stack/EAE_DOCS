X80 Ethernet Remote I/O Adapter for dPAC

Identification

X80 Ethernet Remote I/O Adapter for dPAC module can be used as an I/O adapter in each pair of high availability remote I/Os. The X80 Ethernet Remote I/O Adapter for dPAC components (that is, the associated controller and I/Os) are configurable in the Physical Devices editor.

The X80 Ethernet Remote I/O Adapter for dPAC module includes three external Ethernet ports with Ethernet backplane. Two external ports serve as integrated EtherNet/IP connections to the device network and the remaining Ethernet port is the service port. Input data is collected and published to the remote controller and output modules are updated with the data received from the controller. The two network ports support RSTP and can be combined with Modbus TCP/IP and EtherNet/IP in a network topology.

The SE.IoX80 library provides an HMI CAT for the X80 Ethernet Remote I/O Adapter for dPAC module.

For more information about X80 Ethernet Remote I/O Adapter for dPAC, refer to the BMECRD0100 - EAE - dPAC - Module de communication Modicon X80 pour rack déporté ERIO - Stand - Professionnels | Schneider Electric France.

Known Limitations

The X80 Ethernet Remote I/O Adapter for dPAC deployment and I/O communication do not provide cryptographically protected data integrity or confidentially. 

BM●A●●●●●● modules cannot adjust input alignment, filter, range threshold, and output fallback while running. This can be done during buildtime in the Hardware Configuration editor.
