Documentation

All the EcoStruxure Automation Expert documents are available on the web here. There are also available in EcoStruxure Automation Expert - Buildtime through the Help ribbon or with the contextual help (F1).

NOTE: Some of these documents are also available in PDF in the installation package of EcoStruxure Automation Expert to help you install your devices.

New High Availailability Disruption Avoidance User Guide

High Availability Disruption Avoidance Soft dPAC - User Guide describes instructions about how to install, configure and operate HA Softdpac when using HADA orchestrator provided by EcoStruxure Automation Expert Platform.

A second document will supplement this one for the installation and configuration of the orchestrator and its policies within EcoStruxure Automation Expert Platform.

New EtherCAT Configuration Tool User Guide

The EtherCAT Configuration Tool - User Guide describes all the user interface of the EtherCAT Configuration Tool. This user guide also describes how to use this tool to configure your EtherCAT network.

New AP310 User Guide

The new AP310 - User Guides describes all the instructions about how to install, configure and operate the AP310.

New M590 dPAC User Guide

The two new M590 dPAC user guides describe all the instructions about how to install, configure and operate the hardware and Software of the M590 dPAC.

Cybersecurity Guide updates

This release includes some enhancements and important revisions to define a cyber secured Architecture. Please refer to the Cybersecurity - Guide for more details.

Release Documentation Updates

There is no more Read Me document from this version. All product updates and informations of each new EcoStruxure Automation Expert release are now only described in the Release Notes. For users requiring a non-English version of this document in the installer, it might not be up to date. It is recommend to always refer to the documents available on the Online Help Center for the latest version.
