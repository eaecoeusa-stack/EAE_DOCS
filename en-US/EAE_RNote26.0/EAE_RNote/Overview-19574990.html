Overview

EcoStruxure Automation Expert (EAE) is Schneider Electric’s software-centric industrial automation system designed to bring flexibility, openness, and interoperability to process and hybrid industries. It’s part of the EcoStruxure architecture, which focuses on digital transformation and IIoT (Industrial Internet of Things).

EcoStruxure Automation Expert is based on a set of components:

  • Software, to design the Automation solution and monitor the system.

  • Libraries (software extension), to structure collection of reusable components usable to build automation applications more efficiently.

  • Devices, to manage and coordinates automated processes (control).

  • Protocols, to ensure data exchange between the various components of the automation system (controllers, I/O, third-party systems, SCADA, MES, etc.).

Software

  • EcoStruxure Automation Expert - BuildTime encompasses the design, engineering and monitoring of the application.

  • EcoStruxure Automation Expert - HMI, a fully integrated, object-oriented industrial visualization solution.

  • EcoStruxure Automation Expert - Archive, a centralized solution for the historization of process data, alarms, and trends.

  • AVEVA OMI, through AVEVA System Platform, to create application objects (AppObjects) in an automated workflow.

Devices

Distributed Programmable Automation Controller (dPAC) platforms with the Universalautomation.org Shared Source Runtime engine:

  • ATV dPAC for Altivar

  • Modicon M251 dPAC

  • Modicon M262 dPAC

  • Modicon M580 dPAC

Plus, innovative new Software-based Controllers hosted on Edge Controllers(Essential Edge, M590d, AP310):

  • Soft dPAC for Linux™ and Windows™, for standalone configurations.

  • Soft dPAC for Linux™, for High Availability configuration in order to promote continuous operation and minimize downtime in critical applications.

Both, controlling a set of Distributed I/O devices:

  • Modicon Edge I/O NTS (compatible all Automation/Edge Controller)

  • Modicon X80 range (compatible M580d and Edge Controllers)

  • Modicon TM3 range (compatible M2xx)

NOTE: UniversalAutomation.org is a non-profit organization dedicated to overseeing the implementation of an industrial shared-source runtime execution engine. For more information, please visit universalautomation.org website.

Protocols

Communication protocols like OPC-UA, Ethernet IP, MQTT, Modbus (RTU/TCP), PROFINET, EtherCAT, HART, Open TCP/IP.
