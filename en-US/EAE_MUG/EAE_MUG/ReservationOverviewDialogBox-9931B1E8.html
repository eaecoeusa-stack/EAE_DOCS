Reservation Overview Dialog Box

The Reservation Overview dialog box shows the status of:

  • Your zones in the My Zones tab.

  • Every user zones in the All Zones tab.

This Reservation Overview dialog box is opened when clicking on the Manage reservations commands in the Session menu.

My Zones Tab

This is the Reservation Overview dialog box when the My Zones tab is selected:

  1. Displays all the zones that contain in their zone name or status the characters typed in this search bar.

  2. Selects all your reserved zones.

  3. Selects one specific reserved zones.

  4. Gives the full breadcrumb of all your reserved zones.

  5. Gives the status of all your reserved zones.

  6. Releases the selected zones, if these zones are in status Delivered or NothingToDeliver.

    NOTE: If one of the selected zone is in ReadyToDeliver status, then this button opens the Deliver my Modifications dialog box.
    NOTE: This button is disabled when no zone is selected.

  7. Opens the Deliver my Modifications dialog box. The opened Deliver my Modifications dialog box then includes the selected zones in it.

    NOTE: This button is disabled when no zone is selected.

  8. Opens filter dialog box. Use this to filter the zones according to the zone breadcrumb or zone status.

All Zones Tab

This is the Reservation Overview dialog box when the All Zones tab is selected:

  1. Displays all the zones that contain in their zone name, status or person the characters typed in this search bar.

  2. Selects all the reserved zones.

    NOTE: This checkbox is visible only if you are the owner of the Automation Solution in EcoStruxure Automation Expert Platform.

  3. Selects one specific reserved zones.

    NOTE: These checkboxes are visible only if you are the owner of the Automation Solution in EcoStruxure Automation Expert Platform.

  4. Gives the full breadcrumb of all reserved zones.

  5. Gives the status of all reserved zones.

  6. Gives for each zone the name of the user who reserved it.

  7. Releases the selected zones, even if they are not yours, if these zones are in status Delivered or NothingToDeliver.

    NOTE: This button is disabled when no zone is selected.
    NOTE: This button is visible only if you are the owner of the Automation Solution in EcoStruxure Automation Expert Platform.

  8. Opens filter dialog box. Use this to filter the zones according to the zone breadcrumb, zone status or user names.
