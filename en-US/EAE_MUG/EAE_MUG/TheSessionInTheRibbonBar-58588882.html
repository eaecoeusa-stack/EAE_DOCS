Session Menu

The Session menu appears in the Ribbon bar when EcoStruxure Automation Expert is opened through EcoStruxure Automation Expert Platform.

The Session menu contains all the commands required to Multi-User purposes as shown below:

  1. Reserves the zone that is currently selected, i.e. the tab currently opened and selected.

    NOTE: This command is disabled when the current-zone is already reserved by you or another user.

  2. Opens the Reservation Overview dialog box.

  3. Ends the reservation of the zone that is currently selected only if that zone is in Delivered or NothingToDeliver status.
    NOTE: This command is disabled when the current-zone is not reserved.

  4. Updates your session to add all the latest modifications performed by the other users.

    NOTE: This command is disabled when there is no update from other users available.
    NOTE: If your session is not saved when clicking on the Update command, the Save Changes in System pop up appears to ask if you want to save the current changes. It is recommended to save the changes before updating the session.

  5. Opens the Deliver my Modifications dialog box.

    NOTE: This command is enabled if at least one zone is reserved and none is in Unsaved status.

  6. Opens this Multi-User Environment- User Guide.

  7. Ends the reservation of all the zones that are in Delivered and NothingToDeliver status, the zones with other status remain reserved. Messages then appear to inform you about the actions performed for the different zones.

    NOTE: This command is disable when you have no zone reserved.
    NOTE: If at least one zone is in Unsaved status, then no action is performed when you click on this command.
