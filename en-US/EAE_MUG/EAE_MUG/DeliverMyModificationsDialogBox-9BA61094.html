Deliver my Modifications Dialog Box

With the Deliver my Modifications dialog box, you can select the zones to deliver and comment this delivery.

This dialog box is opened when clicking either on:

  • Deliver command in the Session menu.

  • Deliver button in the Reservation Overview dialog box.

  1. Gives the full breadcrumb of all the zones listed in this dialog box.

  2. Selects all the zones listed in this dialog box.

  3. Selects one specific zone from the ones listed in this dialog box.

  4. Writes here a commit message describing the delivery modifications.

    NOTE: One commit message is always needed to deliver modifications.

  5. Gives the status of all the zones listed in this dialog box.

  6. Opens filter dialog box. Use this to filter the zones according to the zone breadcrumb or zone status.

  7. Click on it to deliver the modifications done on the selected zones.

    NOTE: This button is enabled when at least one zone is selected and when you wrote a commit message.

  8. Cancels the delivery and closes this dialog box.

  9. Cancels the delivery, closes this dialog box and opens back the Reservation Overview dialog box.

    NOTE: This button is available only when the Deliver my Modifications dialog box was opened through the Reservation Overview dialog box.
