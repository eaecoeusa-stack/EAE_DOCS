Reserve Commands

The reserve commands are used to reserve a zone within EcoStruxure Automation Expert.

The reserve commands are available in the:

  • Solution Explorer pad.

  • Deploy and Diagnostic editor.

Reserve command in Solution Explorer Pad

To access it in the Solution Explorer pad:

  1. Right click on an element.

    NOTE: To know all the elements that can be reserved, refer to the Multi-User Zone Feature topic.

  2. In the command list appearing select Reserve.

NOTE: When right clicking on one of these elements, the Reserve command is always enabled, however, it doesn’t mean that the reservation always occur.

For instance, if that zone or a sub-zone of it is already reserved by another user, the reservation is not performed and a message appears to indicate by whom the zone is currently reserved.

Reserve command in Deploy and Diagnostic Editor

To access it in the Deploy and Diagnostic editor:

  1. Perform one of these actions:

    • Right click on one device.

    • Click on the drop down list in the Action column of one device.

    • Select the checkbox of multiple devices and click in the general Action button.

  2. In the command list appearing select:

    • Acquire Reservation; To reserve one (or more) physical device.

    • Release Reservation; To release the reservation of one (or more) a physical device.

      NOTE: You can also end the reservation of a physical device with the Reservation Overview dialog box, or the End all reservations command in the Session menu.
