On-screen Keyboard Does Not Pop-up Automatically

In Ubuntu OS, the on-screen keyboard may not popup with EcoStruxure Automation Expert - HMI when clicking on an edit control such as a text field.

To make a change in a text field using on-screen keyboard, follow this procedure:

  1. Ensure Screen Keyboard is set to enabled in the Ubuntu accessibility settings.

  2. Swipe up from the bottom of the screen to display the on-screen keyboard.

  3. After the keyboard appears, click in the text field to select it.

  4. Use the on-screen keyboard to enter text.

NOTE: Click on the frame of the HMI window or any other window, to hide the on-screen keyboard.
