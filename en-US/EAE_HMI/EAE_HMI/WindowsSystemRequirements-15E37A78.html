Microsoft Windows Configuration

Microsoft Windows operating system functions, including but not limited to, automatic update, power options, or system notifications may interrupt operation of EcoStruxure Automation Expert - HMI. To avoid such interruptions, Microsoft Windows operating system may need additional configuration. Contact your local administrator prior to installing EcoStruxure Automation Expert - HMI.

Automatic Root Certificates Root Policy

The group policy for automatic root certificates update must be set to Not Configured (default value) or Disabled on the computer, otherwise one of the following might occur:

  • The EcoStruxure Automation Expert installation might be unsuccessful and an Invalid digital signature error message might appear.

  • EcoStruxure Automation Expert might not function properly and error messages might appear.

To configure the automatic root certificates update:

  1. From the Windows Start menu, open the Local Group Policy Editor.

  2. Click the Local Computer Policy setting, then double-click Computer Configuration.

  3. Double-click Administrative Templates > System > Internet Communication Management > Internet Communication settings > Turn off Automatic Root Certificates Update

  4. Click either Not Configured or Disabled, if the setting is enabled.

  5. Click OK.
