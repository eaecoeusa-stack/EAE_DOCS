Upgrade EcoStruxure Automation Expert - HMI Runtime

You can upgrade an existing installation of EcoStruxure Automation Expert - HMI runtime without uninstalling the original installation.

Windows

To upgrade EcoStruxure Automation Expert - HMI for Windows, reinstall EcoStruxure Automation Expert - Runtime

For instructions on how to install EcoStruxure Automation Expert - Runtime, refer to the EcoStruxure Automation Expert User Manual topic Installing EcoStruxure Automation Expert - Runtime.

Ubuntu Linux

To upgrade EcoStruxure Automation Expert - HMI for Ubuntu Linux, reinstall it.
