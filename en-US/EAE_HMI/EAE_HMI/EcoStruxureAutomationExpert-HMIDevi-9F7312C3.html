EcoStruxure Automation Expert - HMI Manager

The HMI Manager receives compiled runtime data for the HMI client. After starting up, the HMI Manager icon appears in the Windows System Tray.

HMI Manager Functions

Right-click on the icon to access the following menu:

Command

Description

Start

Starts the HMI Manager with the most recently deployed HMI client.

Stop

Stops the HMI client.

Start Log

Opens the log file of the HMI Manager.

Stop Log

Closes the log file of the HMI Manager.

Settings

Displays the HMI Manager - Settings window. See HMI Manager Settings below.

Exit

Shuts down the HMI Manager.

HMI Manager Settings

The following settings are available in the HMI Manager - Settings window:

Command

Description

IP Address

The IP address of the HMI client.

Port Number

For HMI-NET type devices, available in the library SE. Standard (Version 24.0.0.4 and later), the communication between EcoStruxure Automation Expert - Buildtime and EcoStruxure Automation Expert - HMI is secured by HTTPS port 51443 by default.

If the HMI client uses a different Logical port than 51443, select a secured Port Number in the HMI Manager — Settings.
NOTE: Version 24.0 and later of HMI Manager is always secured. When using an earlier version of the SE.Standard Library and the port is set to 61999 then deploying the HMI fails as the connection between Buildtime and HMI Runtime is not established.
Launch at Windows Start

Select this checkbox to launch the HMI client automatically following a Windows restart.

When Windows starts up, the HMI Manager is launched and if a previously deployed HMI client was running it is automatically launched.

If this option is not selected, only the HMI Manager is launched.

Click Update to apply any modified settings and close the window.

Starting the HMI Manager

The HMI Manager can be started:

  • Automatically at Windows startup, using the Launch at Windows Start option described previously.

  • From the Windows Start menu:

    Start > EcoStruxure Automation Expert > HMI Manager

  • By running the .exe file located in the installation folder of EcoStruxure Automation Expert - Runtime in the SoftHMI\Windows folder.

Information about the startup configuration of the HMI client can be found at HMI Client.
