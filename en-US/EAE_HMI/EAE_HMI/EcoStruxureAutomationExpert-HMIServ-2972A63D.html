Operating the HMI Service through Linux Commands

You can use the following commands to operate the EcoStruxure Automation Expert - HMI service:

  • Status: 

    $ sudo systemctl --user status eae-hmi-deploy
  • Start the service: 
    $ sudo systemctl --user start eae-hmi-deploy
  • Stop the service: 
    $ sudo systemctl --user stop eae-hmi-deploy
  • Re-start the service: 
    $ sudo systemctl --user restart eae-hmi-deploy
  • Display the major version: 
    $ sudo apt-cache show eae-hmi
  • Find HMI-related configuration and monitoring settings: 
    $ sudo eae-hmi-config --help
