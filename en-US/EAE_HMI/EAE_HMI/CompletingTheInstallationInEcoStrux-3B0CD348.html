Completing the Installation in EcoStruxure Automation Expert

Make the following configuration settings in the EcoStruxure Automation Expert Physical Devices editor to complete the installation:

  1. Select a host workstation, server, or Essential Edge controller (HMIBX1A0NDA).

  2. In the Properties pad, select the Runtimes row and click the ellipsis (...) to open the Runtimes Editor.

  3. Click Add then select Soft HMI.

  4. Click OK.
