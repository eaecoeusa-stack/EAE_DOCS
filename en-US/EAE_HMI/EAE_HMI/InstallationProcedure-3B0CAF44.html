EcoStruxure Automation Expert - HMI Installation Procedure

Depending on your architecture, you have the choice between installing one of the following packages:

  • hmi_24.0.24192.04_arm64.deb

  • hmi_24.0.24192.04_amd64.deb

To install EcoStruxure Automation Expert - HMI, follow these steps:

  1. Download the EcoStruxure Automation Expert - HMI installation package and copy it to the host iPC, for example, to the Home folder.

    NOTE: Depending on your architecture, the installation package name will vary, as noted above. For example, it may be:
    eae-hmi_<version>_amd64.deb
    where <version> may look like 24.0.24093.02.
    Alternatively _arm64 may be replaced with _amd64.

  2. Navigate within the folder where the .deb file is copied, right click in an empty space, then select Open in Terminal.

  3. Execute the command to install the EcoStruxure Automation Expert - HMI, for example:
    $ sudo apt install ./eae-hmi_<version>_amd64.deb
    where <version> may look like 24.0.24093.02.

NOTE: If you receive a detected error message indicating the user is not included in the sudoers file, follow these steps to add the user:

  1. Switch to the root user: $ su root

  2. Enter the password for the root user.

  3. Use the following command to add a new user:
    $ sudo adduser <UserName> sudo
    where <UserName> is the name you assign to the new user.
