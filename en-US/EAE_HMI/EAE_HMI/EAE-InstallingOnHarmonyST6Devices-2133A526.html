Install EcoStruxure Automation Expert - HMI Runtime on Harmony ST6xxx Devices

EcoStruxure Automation Expert HMI Runtime can be installed on the following Harmony ST6xxx touch panel HMI devices:

  • HMIST6200

  • HMIST6400

  • HMIST6500

  • HMIST6600

  • HMIST6700

These devices are delivered with Linux-based EcoStruxure Operator Terminal Expert software pre-installed. This software can be replaced by the EcoStruxure Automation Expert HMI Runtime software, enabling EcoStruxure Automation Expert HMI applications to be installed on the devices.

It is possible to subsequently restart the EcoStruxure Operator Terminal Expert software if necessary.

Installation Checklist

The installation of the EcoStruxure Automation Expert - HMI Runtime on Harmony ST6xxx Devices includes performance of the following tasks:

  • Verify system requirements (Linux or Windows) before installation.

  • Download the correct EcoStruxure Automation Expert - HMI package for your architecture (arm64 or amd64).

  • Perform signature verification using GPG:

    • Import Schneider Electric public key (SE_GPG.asc).

    • Verify .deb package signature with the command gpg --verify

  • Install the HMI package using sudo apt install ./<package>.deb (Linux)

  • Configure EcoStruxure Automation Expert settings:

    • Add Soft HMI runtime in the Physical Devices editor.

  • For Harmony ST6 devices:

    • Configure the IP address and network settings,

    • Enable Ethernet downloads.

    • Install EcoStruxure Automation Expert HMI Runtime using installer command (EcoStruxureAutomationExpertHMIST6Installer.exe).

    • Set the local time zone using the command --tzset

  • Deploy HMI application from EcoStruxure Automation Expert Buildtime.

  • Confirm successful installation and reboot device.

  • Do not disconnect power during installation or update process.

Installation Steps

The steps to install EcoStruxure Automation Expert HMI Runtime on Harmony ST6xxx devices are:

  1. Configuring the IP address.

  2. Enabling Ethernet downloads.

  3. Installing EcoStruxure Automation Expert HMI Runtime.

  4. Setting the local time zone.

  5. Creating and deploying an HMI application.

If an EcoStruxure Automation Expert HMI Runtime is already available on Harmony ST6xxx devices and update has to be made, the steps to follow are:

  1. Starting the Factory Installed Software on the Harmony Device.

  2. Configuring the IP address (without its step 1 since the HW Configuration would already be open).

  3. Enabling Ethernet downloads.

  4. Updating EcoStruxure Automation Expert HMI Runtime.

  5. Creating and deploying an HMI application.

Configuring the IP Address

Step

Action

1

On the Welcome screen of the Harmony device, press HW Configuration to open the configuration screen:

2

Press the Ethernet1 field and enter the device IP address, the Gateway address and the Network mask.

Alternatively, you can press the field to the right of the Ethernet1 field, then press the IP Address field and enter the IP address. Additional configuration options are available using this method.

NOTICE
risk of loss of network connection and “no route to host” detected error
When configuring network settings on the ST6 device and only one Ethernet interface is being used (e.g., Ethernet1), you must also assign the same Gateway address and Network mask to the other Ethernet interfaces (e.g., Ethernet2), even if the other interfaces are not actively used.
Failure to follow these instructions can result in equipment damage.

3

Press Save and Reboot.

Result: The IP address is configured and the device reboots.

NOTE: It may be necessary to repeat the process and reboot again if the IP address is not correctly configured immediately.

Enabling Ethernet Downloads

The Harmony device must now be configured to allow remote downloading of projects over an Ethernet connection:

Step

Action

1

Connect an Ethernet cable to the ETH1 port on the Harmony device and the Ethernet port on the PC.

2

On the Welcome screen of the Harmony device, press HW Configuration to open the configuration screen.

3

Scroll down to the Ethernet Download field.

4

Press the Enable field.

Result: A popup window appears confirming that the Ethernet port is open and projects can now be downloaded using the Ethernet connection.

Installing EcoStruxure Automation Expert HMI Runtime on a Harmony ST6xxx Device

Prerequisites: EcoStruxure Automation Expert Buildtime and Runtime, version 21.2 or greater, must be installed on the PC.

NOTE: The following steps are similar for installing and updating EcoStruxure Automation Expert HMI Runtime on a Harmony ST6xxx Device. In the update case, the executable used in this section is an updated version of the one already installed in the device.

Step

Action

1

Navigate to the following folder:

C:\Program Files\Schneider Electric\EcoStruxure Automation Expert - Buildtime xx.x\Studio\bin

where xx.x is the software version, for example, 24.0.

Once there, open the command prompt window by typing cmd in the folder’s breadcrumb.

2

In the command prompt window, enter the following command to ping the HMI:

EcoStruxureAutomationExpertHMIST6Installer.exe --ping 192.168.xxx.xx

where 192.168.xxx.xx is the previously configured IP address of the Harmony device.

NOTE: If the ping is not responding as shown in the image above, please verify the Configuring the IP address and Enabling Ethernet downloads steps.

3

Finally, run the following command to install EcoStruxure Automation Expert HMI Runtime on the HMI:

EcoStruxureAutomationExpertHMIST6Installer.exe --install 192.168.xxx.xx

where 192.168.xxx.xx is the previously configured IP address of the Harmony device.

Type y to begin installation.

Result:

  1. A popup window appears on the Harmony device with the message Now downloading...

  2. Confirmation of successful installation is displayed in the command window of the PC.

  3. The Harmony device reboots.

  4. The messages Application is starting... then No project has been deployed to the device appear on the screen of the Harmony device.

NOTE: Do not disconnect the power cable from the device during this period.

The EcoStruxure Automation Expert HMI Runtime software is now installed on the Harmony device.

Setting the Local Time Zone on the Harmony Device

HMI applications can include alarm controls that display the local time. To set the local time zone on the Harmony device:

Step

Action

1

Ensure the Ethernet service is configured in the HW Configuration screen on the Harmony device.

2

Navigate to the following folder:

C:\Program Files\Schneider Electric\EcoStruxure Automation Expert - Buildtime xx.x\Studio\bin

where xx.x is the software version, for example, 24.0.

Once there, open the command prompt window by typing cmd in the folder’s breadcrumb.

3

In the command prompt window, run the following command:

EcoStruxureAutomationExpertHMIST6Installer.exe --tzset timezone ipaddress

where:

  • timezone is the standard tz database name of the time zone. Enter the name exactly as specified in the tz database.

  • ipaddress is the previously configured IP address of the Harmony device.

For example:

EcoStruxureAutomationExpertHMIST6Installer.exe --tzset Asia/Shanghai 192.168.0.1

4

Cycle the power on the Harmony device.

Creating and Deploying an HMI Application

Step

Action

1

Using EcoStruxure Automation Expert, create a new solution (refer to the EcoStruxure Automation Expert User Manual topic Create New Solution). On the first screen, select the option EcoStruxure Automation Expert - Starter Kit (Multi-OS HMI) to provide support for Linux-based devices.

2

In the Solution pad, create a new resolution (Canvases > New Resolution).

3

On the Resolution window, select the screen Size matching that of the target Harmony device, then click Finish.

NOTE: For details on the other options available, refer to the EcoStruxure Automation Expert User Manual topic Start Canvas.

4

Right-click on the new resolution under the Canvases node and choose New item. Proceed to design the canvases and graphical elements of the HMI runtime application, as described in the Graphic Editors chapter of the EcoStruxure Automation Expert User Manual.

NOTE: Multi-OS HMI applications can contain a maximum of 140 numeric values for each canvas.

5

In the quick access toolbar, click the System File icon to open the System editor.

6

Click the Physical devices icon from the vertical toolbar on the left.

7

In the Device catalog on the left:

  1. Select Harmony > Harmony operating panel > Harmony ST6.

  2. Drag and drop the device onto the workspace area.

8

In the workspace area, select the Harmony ST6 device to display its properties in the Properties pad.

NOTE: If the Properties pad is not visible, select View > General pads > Properties in the main menu to display it.

9

In the Ethernet interface section, set the IP Address to that previously configured in the Harmony device.

10

In the Solution Explorer:

  1. Navigate to and select <Solution Name> > System > Devices.

  2. Right-click and select New Device.

  3. In the New Device dialog:

    • Enter a Name

    • Select a Count

    • Select the TypeSE.Standard:HMI_NET

    ... then click OK.

11

In the Logical Devices editor, select the new standard HMI object, then double-click in the Physical Device column and select the Harmony ST6 device to link it to the logical device.

12

On the Deploy and Diagnostic pad, compile the device and deploy it.

13

On the Harmony device, the following message is displayed:

A new version is deployed. Do you want to restart the HMI?

Press YES.

Result: The following message is displayed:

New application is starting…

The new project is then displayed.

If a project was already running, press New version deployed, then press YES.

Starting the Factory Installed Software on the Harmony Device

Step

Action

1

Power on the Harmony device. Startup of the currently installed HMI runtime application begins. The device is responsive to commands for the first 5 seconds of the startup phase.

When the progress bar is in the middle, the message Initializing EcoStruxure Automation Expert HMI appears.

2

Touch the upper left third of the screen, drag down to the lower left third of the screen, and release:

Result: The message Starting the original runtime... appears.

The Welcome screen appears if no runtime application is loaded on the device. Otherwise, the EcoStruxure Operator Terminal Expert application starts.

3

Because a project has already been installed on the device using EcoStruxure Operator Terminal Expert, the Welcome screen, shown in the Configuring the IP address section, is no longer displayed.

To access the configuration screen, press once in the upper left corner of the screen then once in the lower right corner of the screen as soon as the loading icon appears in the middle of the screen.
