What is EcoStruxure Automation Expert?

EcoStruxure Automation Expert is an industrial automation software solution developed by Schneider Electric. This integrated development environment (IDE) serves as a suite of tools for automation developers.

While EcoStruxure Automation Expert is not a standalone framework or programming language, it adheres to the IEC61499 standard.

NOTE: For more information about the IEC61499 standard, refer to IEC61499 Standard Overview topic.

EcoStruxure Automation Expert is designed to support the entire automation development lifecycle, providing essential tools for coding, debugging, testing, and deploying applications across various hardware platforms with a software-centric approach.

Key features of EcoStruxure Automation Expert are:

  • The capability for adaptation to process changes.

  • Integration between Information Technology (IT) and Operational Technology (OT) systems.

  • Vendor-agnostic deployment options.

Additionally, EcoStruxure Automation Expert integrates with source control tools such as Git, facilitating streamlined version control and enhancing team collaboration. Beyond basic code editing, EcoStruxure Automation Expert offers advanced debugging tools, refactoring support, profiling and performance analysis, real-time collaboration features, and a variety of testing frameworks that align with contemporary development requirements. It also encompasses templates, automation tools, and plugins to optimize development processes for both small-scale projects and large-scale automation applications.
