Startup and Normal Operation

Before initiating the Startup Procedure, first complete the Preparation Procedure

Then, verify that the initial conditions are completed:

  • Unit 600 is set to atmospheric pressure and ambient temperature.

  • The actuated valves are securely closed.

  • All electrical equipment is powered off.

Additionally, when transitioning the PID loops to automatic mode, it is mandatory to use the last known operating setpoint as the initial value.

Next only can start the normal operation procedure:

  1. Set valve 6XV-001 to automatic mode.

  2. Set PIC-C1-3 to automatic mode.

  3. Initiate the operation of the vacuum pump PV-1. After the startup phase, it transitions to full functionality and operates at the designated setpoint conditions.

  4. Allow the system pressure to decrease to 100 mbar.

  5. Activate U-601 to start heating in the TFE. The operator provides the temperature setpoint.

    NOTE: Adjust the temperature according to the feed composition. If potential challenges with solid removal in SS-1 are anticipated, beginning with a lower temperature setpoint is advisable.

  6. Once the TFE reaches the specified temperature setpoint, initiate A-SS1.

  7. Set LIC-SS1-1 to automatic mode.

  8. Set TIC-SS1-1 to automatic mode.

  9. Start the TFE feed by opening 6XV-002 and manually configuring either the flow rate setpoint or the valve opening percentage on FIT-C1-1 or FCV-D1-1.

  10. Gradually increase the feed flow rate to the nominal value of 60 kg/h. A careful incremental rise in the flow rate setpoint is necessary.

  11. Set FIC-D1-1 to automatic mode.

  12. When the minimum level in C-1 is attained (LAL-C1-2A is inactive), manually activate E-601, noting that a temperature setpoint will be required.

  13. Configure FCV-C1-1 to 100% open.

  14. When the minimum level in D-3 is reached (LAL-D-3 is inactive), set LIC-D-3 to automatic mode.

  15. Adjust the temperature setpoint in C-1 until it reaches a steady-state condition, so that the level setpoint in C-1 remains constant.

  16. Once the level in C-1 stabilizes, set LIC-C1-2 to automatic mode.

  17. Gradually decrease the opening percentage of FCV-C1-1 to regulate the column top temperature (TE-C1-2) effectively.

    NOTE: Validation through pilot trials is necessary to determine whether a specific value can be established as the setpoint without introducing any transients.

  18. Open TCV-C1-3 to the designated setpoint percentage.

  19. When the minimum level in D-4 is achieved, set LIC-D-4 to automatic mode.

  20. Once TE-C1-2 is sufficiently close to the expected value, transition the flow rate control of FIC-C1-1 to automatic mode.

  21. When the top column reflux reaches stability, fully transition FIC-C1-1 to automatic mode.

  22. Allow time for the column to reach a steady operational state.

  23. Gradually modify the temperature setpoint of TE-C1-3 until it aligns with the expected temperature. It is important that the column operats steadily before implementing any changes.

  24. Upon achieving stable operation of the column, set TIC-C1-3 to automatic mode.

  25. Activate PD1-C1 at the minimum operational speed.

  26. The speed of PD1-C1 can be adjusted gradually, keeping both the level constant and the temperature TE-C1-4 from decreasing significantly.

  27. Finally, set TIC-C1-4 to automatic mode. The startup procedure can be considered complete once the whole system is operating steadily.

The system is functioning consistently, and no further actions are necessary during normal operations.

A final confirmation follows the completion of the pilot plant operations. In the event of a brief interruption in the feed to Unit 600, the equipment maintains the specified operating pressure while refraining from processing any material.
