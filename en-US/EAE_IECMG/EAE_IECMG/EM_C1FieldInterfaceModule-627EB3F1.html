Upon identifying all components for the field interface, this step also involves incorporating objects that accurately represent their behavior.

In EcoStruxure Automation Expert are available advanced library elements (e.g. Analog Input (AI), Analog Output (AO), Digital Input (DI), Digital Output (DO)...). These objects facilitate read and write operations for sensor values and provide additional features like:

  • Visualization

  • Value scaling

  • Filtering

  • OPC UA data accessibility

  • Etc...

NOTE: For scenarios requiring only read or write operations, employ native SYMLINKMULTISRC and SYMLINKMULTIDST function blocks.

To effectively use the sensor values within the broader system logic, let’s expose these values through a single adapter, allowing for seamless integration with the relevant components, like the control logic module. Additionally, if a sensor value is needed directly by another component, like a PID controller, a dedicated adapter can be established to help efficient data flow to the PID function block.
