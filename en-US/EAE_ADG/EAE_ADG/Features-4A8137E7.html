Features

The following features can be used to observe the system behavior at function block level and temporarily interfere with the execution of the application to experiment with optimization options. The investigation principles are to search for connected function blocks that create significant amounts of events under given circumstances.

Watches

Watches can be used to see not only the event values but also the event counters and any unexpectedly high number of events.

Blocking Event

Events can be blocked to interrupt chains of execution and identify (or isolate) execution paths generating high events counts.
