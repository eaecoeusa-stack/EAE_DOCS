The performance of an EcoStruxure Automation Expert system depends on several factors, including the type of hardware devices used as controllers (dPAC platforms) and the type and dimensioning of the services used in the solution.

The following table presents performance data that was generated, subject to the following limitations:

  • Software Limitation (SWL):

    The software limitation arises from the EcoStruxure Automation Expert - Buildtime or the supporting software stack. By design, EcoStruxure Automation Expert - Buildtime supports maximum values for services usage, which cannot be exceeded for a given platform. If necessary, you can instead use a platform offering greater capabilities.

  • Recommended Limitation (REC):

    This limitation represents a recommended configuration designed to produce optimal system performance. If greater sizing is necessary, a platform offering more capabilities can be used. You can also attempt to use a larger sizing with adequate testing. In this case, contact the Schneider Electric Customer Care Center to confirm adequate sizing and provide the necessary support.
