Optimizing On-Demand Utilization Performance of the Resources

When a canvas is opened in the HMI, data is used on-demand, so that only the objects appearing inside this canvas are updated. Then use faceplates for optimizing performances as all objects in the faceplates are not updated when the faceplates are closed.

Then to improve HMI clarity and resource performance, modulate where the HMI object are located by balancing between flat design and on-demand utilization.
