HMI ST6 Performance

HMI ST6 Hardware:

  • HMI ST6 has different resolutions available but they run with the same processor

  • ST6200, ST6400, ST6500, ST600, ST6700

  • From 480x272px to 1366x768px

  • Processor: ARM Cortex-A8, 800 MHz

Maximum number of PLC objects per canvas: 256

NOTE: A PLC object represents any symbol that shows 1 variable coming from PLC. It can be an integer, a float, a boolean, a string and so on.

Recommended HMI refresh frequencies:

  • The HMI refresh cycle depends on the frequency of events arriving to the HMI CAT function block.

  • Faster cycles require to reduce the number of PLC objects per canvas.
