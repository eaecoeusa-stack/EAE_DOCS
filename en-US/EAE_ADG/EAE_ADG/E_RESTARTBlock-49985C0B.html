E_RESTART Function Block

In each resource, the E_RESTART function block is instantiated and triggers the initial events to initiate the logic execution:

  • COLD: Raise one event after a cold start (power-on boot) of the resource.

  • WARM: Not supported yet. This output event is reserved for future enhancements.

  • ONLINECHANGE: Raise when an online change is performed.

E_RESTART function block:
NOTE: It is recommended that the STOP event not be used, as current implementation of the software does not manage the STOP event.
NOTE: Refer to the Runtime.Base library help for more information.
