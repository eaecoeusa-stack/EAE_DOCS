HMI Configuration

In the Physical devices editor, a HMI device can be configured as:

  • Harmony iPC: Available as HMIPSOS552D1801 in the list.

    NOTE: HMI Runtime is automatically added when dropping it in the editor, only configure the IP address.

  • Harmony ST6: Available as Harmony ST6 in the list.

    NOTE: HMI Runtime is automatically added when dropping it in the editor, only configure the IP address.

  • Workstation: Available as Workstation in the list.

    NOTE: HMI Runtime need to be manually added in the Properties pad after dropping it in the editor, then also configure the IP address.

Once added into the Physical devices editor, link the HMI device to a logical device in the Logical Devices editor.

Finally, manage the HMI device on the Deploy and Diagnostics editor of EcoStruxure Automation Expert.

From there, only two actions are available for HMI devices:

  • Compile

  • Deploy

To deploy the HMI device, follow the next steps:

  1. Start the HMI Manager service on the target machine.

  2. The HMI Manager - Settings dialog box opens.

  3. Set the IP Address with the same value as the IPv4 of the machine.

  4. Set the Port Number.

HMI Manager service then starts the HMI runtime on that targeted machine.

NOTE: To successfully establish the communication between HMI and the EcoStruxure Automation Expert - Runtime, set a parameter in the Logical Device editor. Click on the logical device and in the Properties pad set the parameter Security settings > Use Encryption to:

  • False: For the Soft dPAC using Windows operating system.

  • True: For the others distributed PAC.
