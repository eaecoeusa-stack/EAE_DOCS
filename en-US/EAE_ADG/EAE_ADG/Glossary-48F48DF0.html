Glossary

  • Adapters: interface description for complex interactions. This type defines a set of event inputs, event outputs, data inputs, and data outputs. It allows to model complex relationships between assets and to create a single connection between them at application level.

  • Application: logic to run to manage a process or to pilot a machine.

  • CAT: Composite Automation Type: Extensible object type with aspects including logic (behavior), graphical representation, documentation, I/O connectivity, etc..

  • Deployment: action to send all or part of the application to one or more devices.

  • EcoRT: runtime implementing the IEC61499 standard in EcoRT-Platforms.

  • EcoRT-Platforms: several hardware or digital devices hosting the EcoRT and capable of executing EcoStruxure Automation Expert applications.

  • EcoStruxure Automation Expert: buildtime software allowing to develop, deploy and monitor the application logic and to build and deploy the HMI.

  • Event: Instantaneous occurrence that is significant to raise the execution of an algorithm. An event may be associated to a list of variables.

  • HMI: graphical engine which allows the operator to interact with the process/machine.

  • IEC 61499: IEC standard for event driven automation systems.

  • Mapping: In this document may refer to different actions:

    • Mapping of a logical block instance to a resource (IEC61499 term).

    • Mapping as a symbolic link (symlink) assignment to physical I/O.

    • Mapping of logical device to the physical device.

  • Resource: part of the EcoRT which executes events.

  • Symlink: an abbreviation for 'symbolic link'. This abbreviation is used in the Service function block naming convention. It is advisable to use the wording 'symbolic link' whilst referring to the documentation.

  • System: encompasses the whole ecosystem with new EcoRT controllers, EcoStruxure Automation Expert buildtime etc.

  • Watch: a facility to view the real-time parameter value, timestamp and quality information, as well as event count.
