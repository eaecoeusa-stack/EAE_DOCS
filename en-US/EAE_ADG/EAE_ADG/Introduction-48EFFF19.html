Introduction

The goal of this document is to provide recommendations and examples for the design of applications with the new IEC61499 based EcoStruxure Automation Expert offer.

This offer comprehends:

  • EcoStruxure Automation Expert software: For the design and deployment of the application and the HMI.

  • EcoStruxure Automation Expert - Runtime firmware (EcoRT): For the execution of the application and the communication management within EcoRT platforms.

  • EcoRT Platforms: Hardware or digital devices hosting EcoRT and capable of executing EcoStruxure Automation Expert applications.

  • EcoStruxure Automation Expert - Archive: SQL database and software for managing data archives.

  • EcoStruxure Automation Expert - HMI: The HMI connection to the distributed runtime system EcoRT.

This document does not cover the full IEC61499 standard definition nor does it intend to describe all EcoStruxure Automation Expert functions and features. It presupposes a basic knowledge of IEC61499 concepts and doesn’t replace a proper training on that standard nor on the offer.
