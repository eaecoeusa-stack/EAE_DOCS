SPLIT

Use the SPLIT function block to extract individual members of a struct data input and assign each member to a data output. A maximum of 254 outputs can be created.

If any struct used to create a SPLIT function block is renamed, the name of that struct in the SPLIT function block is renamed.

If any member of the struct is changed, for example renamed or its data type changed, all SPLIT function blocks using that struct are recreated to include the changes.

If any struct used to create a SPLIT function block is deleted, the SPLIT function block using the deleted struct is reset to its default configuration.

Automatically Creating an SPLIT Function Block

  1. In the Function Block Network, right click on an existing function block with a struct output.

  2. Select Create SPLIT function block.

    A new SPLIT function block is created and added to the network that includes:

    • A data output for each struct member of the existing function block data output.

    • An Output pin containing the original struct data output.

Manually Creating a SPLIT Function Block

  1. Right click in the Function Block Network.

  2. Select FB > Project > Runtime.Base > Runtime.Standard > SPLIT.

  3. Right click on the SPLIT function block and select Interface Editor....

  4. Select an existing Input variable type (i.e. a struct), for the Input pin.

  5. Choose the struct members to include as data outputs.

  6. Optional: Select Create "Output" variable from "Input" variable to add an output struct variable with the same members as the selected Input struct.

  7. Click OK.

NOTE: If you selected Create "Output" variable from "Input" variable when creating the SPLIT function block, you can right-click on the Output pin and select Create SPLIT function block. A new SPLIT function block is created and the Output pin of the original function block is connected to the Input pin of the new function block.
