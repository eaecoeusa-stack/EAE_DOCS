Local HMI Test

In the Solution Explorer pad, right clicking on the canvas resolution node to access the command Start Local HMI.

This command starts the local HMI. It is used to test if the created canvases operate and display correctly.

The local HMI has the same functionalities as the real HMI.

NOTE: Starting the local test HMI does not require to have the HMI device running.

By default the HMI canvas positioned on top in the Solution Explorer pad is displayed first. Change which canvas opens first by right clicking on the resolution node and selecting the Startup Canvas.
