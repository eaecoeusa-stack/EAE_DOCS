AGGREGATE

Use the AGGREGATE function block to assemble multiple input data objects into a single output data item of the struct data type. The output data item is passed from the AGGREGATE function block to a struct data type input of another function block.

You can create an AGGREGATE function block either manually or automatically.

Automatically Creating an AGGREGATE Function Block

  1. In the Function Block Network, right click on an existing function block with a struct input.

  2. Select Create AGGREGATE FB.

    A new AGGREGATE function block is created and added to the network that includes:

    • A data input for each struct member of the existing function block data input.

    • A single data output that is connected to the selected struct Input on the existing function block.

NOTE: If you connect an AGGREGATE data output to another function block that does not include a struct input, you will be asked if you want to add an input of the struct type to that function block.

Manually Creating an AGGREGATE Function Block

  1. Right click in theFunction Block Network.

  2. Select FB > Project > Runtime.Base > Runtime.Standard > AGGREGATE.

  3. Right click on the AGGREGATE function block and select Interface Editor....

  4. Select an existing struct.

  5. Choose the struct members to include as data inputs.

  6. Click OK.

Updating an Existing AGGREGATE Function Block

If the struct data variable on which an AGGREGATE function block is based is changed, you can use the AGGREGATE Block Interface Editor to edit the data inputs and struct output of the block.
