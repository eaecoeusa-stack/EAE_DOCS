Strategies for Initialization in EcoStruxure Automation Expert

A controller owns a set of resources (at least one, by default), that act as containers for the application logic and the different hardware buses used with the hardware configuration.

Once created, each of these resources have by default an E_RESTART function block integrated inside of it as detailed in the topic E_RESTART Function Block.

All these application logics and the hardware buses of the resources must be initialized. There are two options to initialize them:

  • DPAC_FULLINIT and plcStart Function Blocks. This option for initializing is recommended.

  • EVENTCHAINHEAD and EVENTCHAIN Function Blocks.
